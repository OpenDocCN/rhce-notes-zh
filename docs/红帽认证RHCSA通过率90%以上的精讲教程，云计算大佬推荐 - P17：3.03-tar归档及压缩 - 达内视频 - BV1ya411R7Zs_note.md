## 归档与压缩的概念 🔍

上一节我们介绍了`tar`命令的重要性，本节中我们来看看归档与压缩这两个核心概念。

在Linux系统中，归档和压缩是两个不同的操作。
*   **归档** 指的是将多个文件或目录合并成一个单一的文件，便于管理和传输。归档操作本身**不会减少**文件占用的总空间。例如，将10个各100MB的文件归档后，得到的归档文件大小约为1000MB。
*   **压缩** 指的是通过特定算法减少文件占用的磁盘空间。在Linux中，压缩通常由如`gzip`、`bzip2`、`xz`等独立工具完成。

`tar`命令最初只负责归档。后来，通过增加选项，它可以自动调用压缩工具，从而**一步完成归档和压缩**，生成我们常见的 `.tar.gz`、`.tar.bz2` 等格式的压缩包。

---

## 基础tar命令操作 🛠️

了解了基本概念后，我们开始学习`tar`命令的基础操作，包括创建、查看和恢复归档文件。

`tar`命令的基本格式如下：
```bash
tar [选项] 归档文件名 要归档的文件或目录...
```

以下是`tar`命令最常用的几个操作选项：

*   **`-c`**：创建新的归档文件。
*   **`-t`**：列出归档文件的内容。
*   **`-x`**：从归档文件中提取（解压）文件。
*   **`-f`**：指定归档文件的文件名。**此选项必须紧跟文件名**。
*   **`-v`**：显示命令执行的详细过程。
*   **`-P`**：保留文件的绝对路径（默认会移除路径开头的 `/`）。
*   **`-p`**：保留文件的原始权限属性（如所有者、时间戳等）。

![](img/692b1437a2512e934936eaee1dd94082_3.png)

![](img/692b1437a2512e934936eaee1dd94082_5.png)

### 创建归档
例如，将 `file1.txt` 和 `dir2` 归档到 `backup.tar`：
```bash
tar -cvf backup.tar file1.txt dir2
```

![](img/692b1437a2512e934936eaee1dd94082_7.png)

![](img/692b1437a2512e934936eaee1dd94082_8.png)

### 查看归档内容
查看 `backup.tar` 中包含哪些文件：
```bash
tar -tvf backup.tar
```

### 解压归档
将 `backup.tar` 中的文件解压到当前目录：
```bash
tar -xvf backup.tar
```
如果想解压到指定目录，例如 `/opt`，可以使用 `-C` 选项：
```bash
tar -xvf backup.tar -C /opt
```

## 结合压缩工具 📦➡️🗜️

上一节我们学会了基本的归档操作，本节中我们来看看如何将归档与压缩结合起来，以节省存储空间。

![](img/692b1437a2512e934936eaee1dd94082_10.png)

`tar`命令通过特定选项可以调用不同的压缩工具，实现归档后自动压缩。

以下是三种常见压缩格式对应的`tar`命令选项：

![](img/692b1437a2512e934936eaee1dd94082_12.png)

![](img/692b1437a2512e934936eaee1dd94082_14.png)

*   **`-z`**：调用 `gzip` 工具，生成 `.tar.gz` 或 `.tgz` 文件。
*   **`-j`**：调用 `bzip2` 工具，生成 `.tar.bz2` 文件。
*   **`-J`**：调用 `xz` 工具，生成 `.tar.xz` 文件。

![](img/692b1437a2512e934936eaee1dd94082_16.png)

![](img/692b1437a2512e934936eaee1dd94082_18.png)

**注意**：使用 `-j` 或 `-J` 选项前，请确保系统已安装 `bzip2` 或 `xz` 软件包。

![](img/692b1437a2512e934936eaee1dd94082_20.png)

### 创建压缩归档包
创建一个包含 `/usr/local` 目录的压缩归档包，并保留绝对路径：
```bash
# 使用gzip压缩
tar -czvPf backup.tar.gz /usr/local
# 使用bzip2压缩
tar -cjvPf backup.tar.bz2 /usr/local
# 使用xz压缩
tar -cJvPf backup.tar.xz /usr/local
```

### 解压压缩归档包
解压相应的压缩包，`tar`命令会自动识别压缩格式：
```bash
# 解压 .tar.gz 文件
tar -xzvf backup.tar.gz
# 解压 .tar.bz2 文件
tar -xjvf backup.tar.bz2
# 解压 .tar.xz 文件
tar -xJvf backup.tar.xz
```

## 关键选项详解：-P 与 -p 🔑

在备份和恢复系统文件时，`-P`和`-p`选项尤为重要，它们决定了文件路径和属性的处理方式。

![](img/692b1437a2512e934936eaee1dd94082_26.png)

*   **`-P` (大写)**：保留被归档文件的**绝对路径**。在恢复时，如果也使用 `-P` 选项，文件会被精确还原到原始路径。这对于系统备份至关重要。
*   **`-p` (小写)**：保留文件的原始**权限、所有者和时间戳**等属性。这是许多操作的默认行为，但显式指定可以确保属性不被改变。

![](img/692b1437a2512e934936eaee1dd94082_27.png)

![](img/692b1437a2512e934936eaee1dd94082_29.png)

**最佳实践建议**：在进行重要备份时，组合使用 `-cP` 选项。这样在恢复时使用 `-xP`，可以确保文件“从哪来回哪去”，避免路径错乱。

## 实战例题演练 📝

![](img/692b1437a2512e934936eaee1dd94082_38.png)

![](img/692b1437a2512e934936eaee1dd94082_40.png)

现在，让我们应用所学知识来解决一个典型的RHCSA考题。

**题目要求**：在 `/root` 目录下创建一个名为 `backup.tar.bz2` 的归档压缩文件，其中应包含 `/usr/local` 目录下的所有内容。

![](img/692b1437a2512e934936eaee1dd94082_42.png)

**解答步骤**：
1.  确保 `tar` 和 `bzip2` 软件包已安装。
2.  使用 `tar` 命令，结合创建(`-c`)、使用bzip2压缩(`-j`)、显示详情(`-v`)、保留绝对路径(`-P`)、指定文件名(`-f`)选项。
3.  执行命令：
    ```bash
    tar -cjvPf /root/backup.tar.bz2 /usr/local
    ```
4.  验证结果，可以使用 `file` 命令检查文件类型，或用 `tar -tjf` 命令查看压缩包内容。

![](img/692b1437a2512e934936eaee1dd94082_44.png)

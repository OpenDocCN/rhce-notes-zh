## 课程与考试介绍

红帽认证体系从7.0版本升级到了8.0版本。在7.0时代，RHCE认证需要学习三门课程：RH124、RH134和RH254。其中，RH124和RH134构成了初级认证RHCSA，而RH254则是中高级的RHCE课程。

在8.0版本中，前两门基础课程（RH124和RH134）保持不变，但第三门课程变为了**RH294**。RH294课程的核心内容是**Ansible自动化运维**，这与7.0版本中侧重于服务搭建（如Samba、DNS）的RH254课程有较大区别。

![](img/c0233af71b53e156c8de6e9c8a0ca864_8.png)

![](img/c0233af71b53e156c8de6e9c8a0ca864_10.png)

认证考试需要逐级进行，不能跳级。首先需要通过RHCSA（考试代码EX200），之后才能参加RHCE（考试代码EX300）的考试。考试通常在同一天进行，上午考RHCSA（2.5小时），下午考RHCE（3.5小时）。满分300分，210分即可通过。成绩通常在三个工作日内通过邮件发送，证书需在红帽官网申请。

![](img/c0233af71b53e156c8de6e9c8a0ca864_44.png)

**终端**（Terminal）提供了一个命令行界面，它运行着一个称为**Shell**的命令解释器。与Windows的CMD相比，Linux的Shell功能更强大，能够执行复杂的脚本任务。

## 配置网络与远程登录

![](img/c0233af71b53e156c8de6e9c8a0ca864_50.png)

![](img/c0233af71b53e156c8de6e9c8a0ca864_52.png)

![](img/c0233af71b53e156c8de6e9c8a0ca864_54.png)

为了方便操作，我们通常通过远程工具连接Linux服务器。首先，需要确保Linux系统已获取IP地址。

![](img/c0233af71b53e156c8de6e9c8a0ca864_56.png)

![](img/c0233af71b53e156c8de6e9c8a0ca864_58.png)

以下是查看IP地址的命令：
```bash
ifconfig
```
执行该命令后，找到 `inet` 后面的地址，例如 `192.168.127.128`，这就是系统的IP地址。

![](img/c0233af71b53e156c8de6e9c8a0ca864_60.png)

![](img/c0233af71b53e156c8de6e9c8a0ca864_62.png)

获取IP地址后，可以使用如MobaXterm、Xshell、SecureCRT等SSH客户端进行远程连接。以MobaXterm为例，新建一个SSH会话，输入目标主机的IP地址和用户名（如`root`），即可登录。

![](img/c0233af71b53e156c8de6e9c8a0ca864_64.png)

## Linux命令基础语法

![](img/c0233af71b53e156c8de6e9c8a0ca864_68.png)

![](img/c0233af71b53e156c8de6e9c8a0ca864_70.png)

Linux命令通常遵循几种基本结构，理解这些结构是高效使用命令行的关键。

### 1. 单独命令
这种形式只有命令本身，用于执行一个简单的动作。
```bash
ifconfig  # 查看网络配置
pwd       # 打印当前工作目录
whoami    # 显示当前登录用户名
uname -a  # 显示系统详细信息
```

### 2. 命令 + 执行对象
在命令后跟上要操作的具体目标（文件或目录）。
```bash
ls /root          # 查看 /root 目录下的内容
cd /root/Desktop  # 切换到 /root/Desktop 目录
```
**注意**：命令与参数、对象之间必须用**空格**分隔。按 `Tab` 键可以自动补全命令或路径。

### 3. 命令 + 选项
选项（通常以 `-` 或 `--` 开头）用于修改命令的行为，提供更详细或特定格式的输出。
```bash
ls -l        # 以长列表格式显示详细信息
ls -lh       # 以长列表格式显示，并以人类可读的形式显示文件大小（如K，M）
ls -a        # 显示所有文件，包括隐藏文件（以 . 开头的文件）
```
多个单字符选项可以合并：
```bash
ls -lh  # 等价于 ls -l -h
```

### 4. 命令 + 选项 + 执行对象
这是最完整的命令形式，结合了上述所有部分。
```bash
ls -lh /root  # 以详细且易读的格式查看 /root 目录的内容
```

## 获取命令帮助

当不清楚某个命令的具体用法时，可以使用帮助系统。最常用的方法是使用 `--help` 选项。
```bash
ls --help
```
`--help` 选项会输出命令的使用方法（Usage）和所有可用选项的说明。例如，`-l` 表示使用长列表格式，`-h` 表示以人类可读的格式显示文件大小（通常与 `-l` 一起使用），`-a` 表示显示所有文件（包括隐藏文件），`-d` 表示仅显示目录本身而非其内容。

## Linux目录结构初探

Linux的文件系统采用一种**倒挂树形结构**。最顶层的目录称为**根目录**，用单个斜杠 **`/`** 表示。

![](img/c0233af71b53e156c8de6e9c8a0ca864_78.png)

这与Windows不同，Windows有多个顶级目录（如C:， D:）。在Linux中，一切皆从 `/` 开始。

在根目录下，存在许多重要的子目录，例如：
*   `/bin`：存放基本命令程序。
*   `/dev`：存放设备文件。
*   `/home`：普通用户的家目录。
*   `/root`：系统管理员root的家目录。
*   `/etc`：存放系统配置文件。

![](img/c0233af71b53e156c8de6e9c8a0ca864_80.png)

使用 `ls /` 命令可以查看根目录下的内容。

![](img/c0233af71b53e156c8de6e9c8a0ca864_82.png)

在文件列表中，有两个特殊的符号：
*   **`.`** （一个点）：代表当前目录本身。
*   **`..`** （两个点）：代表上一级目录（父目录）。
例如，`cd ..` 命令可以切换到上级目录。

## 概述

CentOS是一款免费且在企业服务器领域广泛使用的Linux发行版。它与Red Hat Enterprise Linux（RHEL）高度兼容，学习CentOS等同于学习RHEL。本节课我们将使用VMware Workstation虚拟机软件，通过系统镜像文件来安装CentOS 7系统。

## 准备工作：下载所需文件

在开始安装前，需要从提供的“VIP大礼包”中下载必要的软件和系统镜像。

以下是需要下载到本地电脑的文件列表：
*   VMware Workstation 虚拟机软件及其激活密钥。
*   Xshell 远程连接工具（如果已有其他工具可忽略）。
*   CentOS 7 系统镜像文件（例如 CentOS-7-x86_64-DVD-XXXX.iso）。

## 创建新的虚拟机

上一节我们准备好了安装所需的文件，本节中我们来看看如何在VMware中创建一台新的虚拟机。

1.  打开VMware Workstation软件。
2.  点击“创建新的虚拟机”。
3.  在新建虚拟机向导中，选择“典型”配置，然后点击“下一步”。
4.  在安装客户机操作系统界面，选择“稍后安装操作系统”，然后点击“下一步”。
5.  选择客户机操作系统为“Linux”，版本选择“CentOS 7 64位”，然后点击“下一步”。
6.  为虚拟机命名（例如 `CentOS7-学习机-90`），并选择一个磁盘空间充足的安装位置，然后点击“下一步”。
7.  指定磁盘容量，默认20GB即可，选择“将虚拟磁盘存储为单个文件”，然后点击“下一步”。
8.  在最后一步，先不要点击“完成”，而是点击“自定义硬件”。

## 配置虚拟机硬件

虚拟机软件的核心功能是虚拟出一套完整的硬件平台（如CPU、内存、硬盘、网卡），以便在其上安装操作系统。现在我们来配置这些虚拟硬件。

![](img/4ebffa09db7c594b3b42a8fadd759c4e_29.png)

1.  **内存**：建议分配 **2048 MB**（即2GB）。这能保证系统运行流畅。请注意，虚拟机只有在开机时才会占用真机的物理内存。
2.  **处理器**：根据真机CPU的核心数，为虚拟机分配适当的处理器数量。例如，可以分配 **1个处理器，2个核心**。
3.  **新CD/DVD**：这是安装系统的关键。选择“使用ISO映像文件”，然后点击“浏览”，找到并选中之前下载的CentOS 7镜像文件。
4.  **网络适配器**：选择网络连接模式。这里我们选择 **NAT模式**。这种模式下，虚拟机可以通过真机访问外网，且虚拟机和真机的IP地址无需在同一网段也能相互通信，非常适合笔记本电脑等网络环境会变化的场景。
5.  **移除无用硬件**：选中“USB控制器”、“声卡”、“打印机”等本次用不到的硬件，点击“移除”以节省资源。
6.  硬件配置完成后，点击“关闭”，然后点击向导页面的“完成”。至此，一台空白虚拟机就创建好了。

## 安装CentOS 7操作系统

虚拟机创建完毕后，我们启动它并开始安装CentOS系统。

![](img/4ebffa09db7c594b3b42a8fadd759c4e_31.png)

![](img/4ebffa09db7c594b3b42a8fadd759c4e_33.png)

![](img/4ebffa09db7c594b3b42a8fadd759c4e_35.png)

![](img/4ebffa09db7c594b3b42a8fadd759c4e_37.png)

1.  选中新创建的虚拟机，点击“开启此虚拟机”。
2.  虚拟机启动后会加载镜像，出现安装菜单。使用键盘上下键选择第一项 **“Install CentOS 7”**，然后按回车键。
3.  系统开始加载，进入图形化安装界面。首先选择安装过程中的显示语言，可以选择“中文 - 简体中文”，然后点击“继续”。
4.  进入安装信息摘要界面，需要进行以下关键配置：
    *   **软件选择**：点击进入。对于服务器和学习环境，建议选择 **“最小安装”** 并勾选“兼容性程序库”和“开发工具”。这符合企业生产环境标准，且后续需要的软件可以手动安装。点击“完成”。
    *   **安装位置**：点击进入。确认选中了之前创建的20GB虚拟磁盘，分区方案选择默认的自动配置即可，点击“完成”。
    *   **网络和主机名**：点击进入。首先，将主机名修改为一个易记的名称（如 `centos7`）。然后，点击“配置”来设置网络。
        *   在配置窗口中，选择“IPv4 设置”选项卡。
        *   方法选择“手动”。
        *   点击“添加”按钮，输入以下信息：
            *   **地址**：`192.168.0.90` （可根据规划修改最后一位）
            *   **子网掩码**：`255.255.255.0` 或 `24`
            *   **网关**：`192.168.0.254` （根据实际网络环境填写）
            *   **DNS服务器**：`223.5.5.5` （阿里云DNS）
        *   点击“保存”。回到上一界面，将右上角的网络连接开关打开，显示“已连接”。点击“完成”。
5.  所有配置检查无误后，点击右下角的 **“开始安装”**。
6.  在安装过程中，需要设置 **root管理员密码**。点击“ROOT 密码”进行设置，输入并确认密码（密码需有一定复杂度）。设置完成后点击“完成”。
7.  也可以选择“创建用户”来建立一个普通用户账号（可选）。
8.  等待安装进度条完成。安装结束后，点击“重启”按钮。

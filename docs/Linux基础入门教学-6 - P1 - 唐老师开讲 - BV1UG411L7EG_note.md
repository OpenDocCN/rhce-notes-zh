## 权限基础：理解文件与文件夹的访问控制

![](img/61b279fa07d2176a52ca7724d0a534a7_14.png)

当我们使用 `ll` 命令查看文件或目录的详细信息时，会看到类似 `-rw-r--r--` 的字符串。这串字符就是权限标识。

![](img/61b279fa07d2176a52ca7724d0a534a7_16.png)

*   **权限位结构**：这串字符共有9位，通常分为三组，每组三位。
    *   前三位代表**文件所有者（Owner）**的权限。
    *   中间三位代表**所属组（Group）**的权限。
    *   后三位代表**其他用户（Other）**的权限。

![](img/61b279fa07d2176a52ca7724d0a534a7_18.png)

![](img/61b279fa07d2176a52ca7724d0a534a7_20.png)

*   **权限字符含义**：
    *   `r` 代表**读**权限。
    *   `w` 代表**写**权限。
    *   `x` 代表**执行**权限。
    *   `-` 代表没有对应权限。

例如，权限 `-rw-r--r--` 表示：所有者可读可写，所属组和其他用户仅可读。

## 数字表示法：快速计算与设置权限

除了字符表示，权限也可以用数字表示，这在设置权限时非常方便。

*   **数字对应关系**：
    *   `r` (读) = **4**
    *   `w` (写) = **2**
    *   `x` (执行) = **1**
    *   `-` (无) = **0**

*   **计算方法**：将一组（三位）权限对应的数字相加，即得到该组的权限值。
    *   例如 `rwx` = 4+2+1 = **7**
    *   例如 `r-x` = 4+0+1 = **5**
    *   例如 `r--` = 4+0+0 = **4**

![](img/61b279fa07d2176a52ca7724d0a534a7_22.png)

![](img/61b279fa07d2176a52ca7724d0a534a7_24.png)

![](img/61b279fa07d2176a52ca7724d0a534a7_26.png)

因此，权限 `-rw-r--r--` 用数字表示就是 **644**。

**注意**：对于**目录**而言，`x` 权限代表**进入目录**的权限。没有 `x` 权限，即使有 `r` 权限也无法列出目录内容。默认情况下，新建文件的权限是 **644**，新建目录的权限是 **755**。

## 修改权限：`chmod` 命令的使用

![](img/61b279fa07d2176a52ca7724d0a534a7_28.png)

![](img/61b279fa07d2176a52ca7724d0a534a7_30.png)

`chmod` 命令用于修改文件或目录的权限。权限是针对**文件或目录本身**设置的，而不是针对用户。

![](img/61b279fa07d2176a52ca7724d0a534a7_32.png)

以下是修改权限的两种主要方法：

![](img/61b279fa07d2176a52ca7724d0a534a7_34.png)

![](img/61b279fa07d2176a52ca7724d0a534a7_36.png)

**1. 数字模式（推荐）**
直接使用计算好的三位数字来设置权限。
```bash
chmod 755 filename  # 将文件权限设置为 rwxr-xr-x
chmod 644 filename  # 将文件权限恢复为默认的 rw-r--r--
```

![](img/61b279fa07d2176a52ca7724d0a534a7_38.png)

![](img/61b279fa07d2176a52ca7724d0a534a7_40.png)

**2. 符号模式**
使用 `u` (所有者), `g` (所属组), `o` (其他用户), `a` (所有用户) 配合 `+` (添加), `-` (移除), `=` (设定) 来操作。
```bash
chmod u+x filename      # 给所有者添加执行权限
chmod g-w filename      # 移除所属组的写权限
chmod o=rx filename     # 设置其他用户的权限为读和执行
chmod a+r filename      # 给所有用户添加读权限
```

## 命令行效率工具：快捷键与历史命令

熟练使用命令行快捷键可以大幅提升操作速度。

以下是几个常用快捷键：
*   `Ctrl + a`：光标跳到**行首**。
*   `Ctrl + e`：光标跳到**行尾**。
*   `Ctrl + u`：删除从光标处到**行首**的所有内容。
*   `Ctrl + k`：删除从光标处到**行尾**的所有内容。
*   `Ctrl + r`：进入**反向搜索历史命令**模式，输入关键词可查找并执行过往命令。
*   `Tab` 键：**命令或路径补全**，是Linux下最重要的快捷键。

**历史命令操作**
*   `history`：查看之前执行过的所有命令列表。
*   `!n`：执行历史记录中编号为 `n` 的命令（`n` 是 `history` 命令列出的序号）。
*   `!!`：执行**上一条**命令。
*   `!string`：执行历史记录中**最近一条**以 `string` 开头的命令。

## 特殊符号与设备：`$?` 与 `/dev/null`

**`$?`：上一条命令的退出状态**
这个特殊变量保存了**上一条命令执行后的状态码**。
*   **0**：表示命令**成功执行**。
*   **非0**：表示命令**执行出错**（不同数字代表不同错误）。
```bash
echo $?  # 查看上一条命令是否成功
```

**`/dev/null`：空设备**
这是一个特殊的“黑洞”设备。任何写入它的数据都会被丢弃，常用来屏蔽命令的输出。
```bash
ping -c 2 localhost > /dev/null  # 执行ping命令，但将输出信息丢弃，只看结果（通过 $? 判断）
echo $?  # 如果返回0，说明ping通了
```

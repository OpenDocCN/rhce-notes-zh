在本节课中，我们将要学习Linux系统中一个极其重要的工具——vim文本编辑器。文本编辑器是创建和修改文件内容的必备工具，就像Windows系统中的“记事本”一样。掌握vim是进行系统配置、编写脚本和日常文件管理的基础。

![](img/fc03deca9f832c0bebd215f0bfcce042_11.png)

## 什么是文本编辑器？ 🤔

![](img/fc03deca9f832c0bebd215f0bfcce042_13.png)

在Windows系统中，当你右击桌面选择“新建文本文档”时，系统会创建一个`.txt`文件。双击打开它进行编辑的工具，就是“记事本”。这个“记事本”就是一个文本编辑器，它具备创建文件、以及对文件内容进行增、删、改、查的功能。

![](img/fc03deca9f832c0bebd215f0bfcce042_15.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_17.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_19.png)

在Linux系统中，最常用的文本编辑器是`vi`和`vim`。`vim`是`vi`的增强版本，两者在核心操作上基本一致。主要的区别在于，`vim`支持语法高亮显示，这对于编写脚本或代码非常有帮助，而`vi`则没有这个功能。对于日常修改配置文件，使用哪一个都可以。

![](img/fc03deca9f832c0bebd215f0bfcce042_21.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_23.png)

## vim的基本使用 🚀

vim的命令格式非常简单：
```bash
vim 文件名
```
这个命令有两个特点：
*   若文件**不存在**，则创建新文件并进入编辑状态。
*   若文件**已存在**，则直接打开该文件进行编辑。

![](img/fc03deca9f832c0bebd215f0bfcce042_25.png)

例如，要修改系统主机名，可以编辑`/etc/hostname`文件：
```bash
vim /etc/hostname
```
如果文件存在，vim会直接打开它。如果编辑一个不存在的文件，vim会先创建它。

## vim的三种模式 🔄

![](img/fc03deca9f832c0bebd215f0bfcce042_33.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_35.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_37.png)

vim的操作逻辑与记事本不同，它分为三种模式，这是初学者需要理解的核心概念。

**1. 命令模式**
当我们使用`vim 文件名`启动编辑器后，首先进入的就是命令模式。在此模式下，键盘的按键会被识别为各种编辑命令，而不是直接输入字符。

**2. 输入模式**
在命令模式下，按下特定键（如 `i`, `a`, `o`）会进入输入模式。此时，你可以像使用普通记事本一样，自由地输入和编辑文本。按 `Esc` 键可以退出输入模式，返回命令模式。

**3. 底线命令模式**
在命令模式下，按下英文冒号 `:` 会进入底线命令模式。此时光标会移动到屏幕底部，可以输入保存、退出、搜索等高级命令。

三种模式的转换关系如下：
**命令模式** -> (按 `i`, `a`, `o`) -> **输入模式**
**输入模式** -> (按 `Esc`) -> **命令模式**
**命令模式** -> (按 `:`) -> **底线命令模式**
**底线命令模式** -> (执行命令后或按 `Esc`) -> **命令模式**

## 模式切换与基础编辑 ✍️

上一节我们介绍了vim的三种模式，本节中我们来看看如何进行最基础的编辑操作——进入输入模式并输入文字。

![](img/fc03deca9f832c0bebd215f0bfcce042_39.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_41.png)

在命令模式下，有多个键可以切换到输入模式，它们的主要区别在于光标插入的位置：
*   `i`: 在**当前光标所在字符前**插入。
*   `a`: 在**当前光标所在字符后**插入。
*   `o`: 在**当前光标所在行的下方**新建一行并插入。

对于初学者，记住 `i` (insert) 即可。进入输入模式后，屏幕下方通常会显示 `-- INSERT --` 字样，此时可以自由输入文本。使用方向键移动光标，按 `Enter` 键换行。

编辑完成后，按 `Esc` 键退出输入模式，回到命令模式。

## 保存与退出 💾

![](img/fc03deca9f832c0bebd215f0bfcce042_43.png)

编辑完文件后，我们需要保存更改并退出vim。这个操作需要在底线命令模式下完成。

![](img/fc03deca9f832c0bebd215f0bfcce042_45.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_47.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_49.png)

从命令模式按 `:` 进入底线命令模式后，可以执行以下常用命令：
*   `:w` - 保存文件。
*   `:q` - 退出vim（如果文件有修改且未保存，会提示错误）。
*   `:wq` - 保存文件并退出。
*   `:q!` - **强制退出，不保存**任何修改。
*   `:wq!` - **强制保存并退出**（用于修改只读文件，需要root权限）。

![](img/fc03deca9f832c0bebd215f0bfcce042_51.png)

此外，在命令模式下，按 `Shift + zz` (即连续按两次大写的 `Z`) 也可以快速保存并退出。

## 命令模式下的常用操作 ⚙️

掌握了保存退出后，我们回到命令模式，学习更多高效的编辑命令。以下是命令模式下的一些核心操作：

**光标移动**
*   `0` (数字零): 将光标移动到**行首**。
*   `$`: 将光标移动到**行尾**。
*   `gg`: 将光标移动到**文件第一行**。
*   `G`: 将光标移动到**文件最后一行**。
*   `:行号`: 在底线命令模式下，快速跳转到指定行（如 `:100` 跳转到第100行）。

**删除与剪切**
*   `x`: 删除**当前光标所在**的一个字符。
*   `dd`: **剪切**当前光标所在的一整行（如果后续不粘贴，效果等同于删除）。
*   `数字dd`: 剪切从当前行开始的指定数字行（如 `3dd` 剪切3行）。
*   `C`: 删除从光标位置到**行尾**的所有字符，并进入输入模式。

**复制与粘贴**
*   `yy`: **复制**当前光标所在的一整行。
*   `数字yy`: 复制从当前行开始的指定数字行（如 `2yy` 复制2行）。
*   `p`: 将剪切或复制的内容**粘贴**到当前光标所在行的**下方**。

**撤销与重做**
*   `u`: **撤销**上一次的操作。
*   `Ctrl + r`: **重做**被撤销的操作（即撤销的“撤销”）。

![](img/fc03deca9f832c0bebd215f0bfcce042_53.png)

**搜索**
*   `/关键词`: 在底线命令模式下，从当前光标位置**向下**搜索指定关键词，按 `n` 跳转到下一个匹配项。

## 底线命令模式进阶操作 🛠️

![](img/fc03deca9f832c0bebd215f0bfcce042_55.png)

![](img/fc03deca9f832c0bebd215f0bfcce042_57.png)

除了保存退出，底线命令模式还提供了一些提升效率的功能。

**显示与取消行号**
*   `:set nu`: 显示文件的行号。
*   `:set nonu`: 取消行号显示。

**替换文本**
替换命令的格式为 `:范围 s/旧内容/新内容/选项`。
*   `:%s/old/new/g`: 将**全文**中所有的 `old` 替换为 `new`。`%` 代表全文，`g` 代表一行内所有匹配项。
*   `:10,20s/old/new/g`: 将第10行到第20行之间所有的 `old` 替换为 `new`。

**取消搜索高亮**
执行搜索后，匹配项会高亮显示。使用 `:noh` 可以暂时取消高亮，直到下一次搜索。

本节课中我们一起学习了Linux下强大的vim文本编辑器。我们首先理解了文本编辑器的概念，然后重点掌握了vim的三种工作模式：命令模式、输入模式和底线命令模式。我们学习了如何在这三种模式间切换，并实践了创建文件、输入文本、保存退出等基本操作。

![](img/fc03deca9f832c0bebd215f0bfcce042_61.png)

此外，我们还掌握了许多高效编辑命令，如光标快速跳转(`gg`, `G`, `$`)、行删除与剪切(`dd`)、复制粘贴(`yy`, `p`)、文本搜索(`/`)和全局替换(`:%s/old/new/g`)等。虽然vim初学时有门槛，但一旦熟悉这些核心操作，它将成为你在Linux系统中处理文本文件的得力助手。请务必多加练习，形成肌肉记忆。

# Linux基础入门：03.2：通过名称指定文件 📂

在本节课中，我们将学习如何在Linux系统中通过名称来指定和定位文件。核心内容包括理解文件系统的两种视图、掌握绝对路径与相对路径的概念，以及学习用于文件导航的三个基本命令：`pwd`、`cd`和`ls`。

## 文件系统的两种视图

在开始之前，我们先了解描述Linux文件系统的两种常见视图。

第一种视图通常出现在桌面版Linux操作系统的文件浏览器中。它类似于Windows的资源管理器，左侧是导航栏，可以逐级展开目录树，右侧显示当前目录的内容。这种视图在早期的Red Hat 6/7系统中尤为明显。

第二种视图是自上而下的树状结构图，常用于系统设计或描述目录拓扑。我们在第一章介绍Linux文件系统目录结构时使用的就是这种视图。

![](img/798779571d255fcae1052b4e6b878e9e_1.png)

无论哪种视图，都能清晰地展示整个目录树的结构。

## 绝对路径与相对路径

![](img/798779571d255fcae1052b4e6b878e9e_3.png)

上一节我们了解了文件系统的结构，本节中我们来看看如何精确地描述一个文件的位置，这就需要用到路径。路径主要分为两种：绝对路径和相对路径。

### 绝对路径

**绝对路径**是一个完全限定的名称，它在整个文件系统中是唯一且绝对的。其最显著的特点是**一定从根目录（`/`）开始**。

**公式**：`/path/to/location`

**示例**：`/home/student` 就是一个绝对路径。
*   第一个 `/` 代表根目录。
*   后面的 `/` 是路径分隔符。

![](img/798779571d255fcae1052b4e6b878e9e_5.png)

### 相对路径

![](img/798779571d255fcae1052b4e6b878e9e_7.png)

![](img/798779571d255fcae1052b4e6b878e9e_8.png)

**相对路径**是相对于用户当前所在的工作目录而言的路径。它描述了从“当前位置”到“目标位置”的路线。

![](img/798779571d255fcae1052b4e6b878e9e_10.png)

在Bash Shell中，使用特定符号来表示位置：
*   **`.`** 代表当前目录。
*   **`..`** 代表上一级目录（父目录）。
*   **`~`** 代表当前用户的**家目录**。

**示例**：假设当前目录是 `/home/student`，想切换到 `/tmp`。
使用相对路径的方法是：`cd ../../tmp`
这个命令的意思是：先向上两级（到根目录 `/`），再进入 `tmp` 目录。

![](img/798779571d255fcae1052b4e6b878e9e_12.png)

绝对路径和相对路径各有用途。例如，软件开发中，编译源代码通常在源代码所在目录使用相对路径；而安装软件时，则需要将文件复制到如 `/usr`、`/etc` 等绝对路径下，以确保所有用户都能正常使用。

![](img/798779571d255fcae1052b4e6b878e9e_14.png)

## 文件导航命令

![](img/798779571d255fcae1052b4e6b878e9e_16.png)

理解了路径的概念后，我们就可以使用命令在文件系统中自由导航了。以下是三个核心命令。

![](img/798779571d255fcae1052b4e6b878e9e_18.png)

### 1. pwd 命令

![](img/798779571d255fcae1052b4e6b878e9e_20.png)

![](img/798779571d255fcae1052b4e6b878e9e_22.png)

`pwd` 命令用于**查看当前的工作目录**。

![](img/798779571d255fcae1052b4e6b878e9e_24.png)

![](img/798779571d255fcae1052b4e6b878e9e_25.png)

**代码示例**：
```bash
pwd
```
执行后会输出类似 `/home/student` 的绝对路径。

![](img/798779571d255fcae1052b4e6b878e9e_27.png)

![](img/798779571d255fcae1052b4e6b878e9e_29.png)

![](img/798779571d255fcae1052b4e6b878e9e_30.png)

另一个有用的命令是 `readlink -f`，它可以打印出指定文件或目录的绝对路径。

![](img/798779571d255fcae1052b4e6b878e9e_32.png)

![](img/798779571d255fcae1052b4e6b878e9e_34.png)

**代码示例**：
```bash
readlink -f /etc/passwd
```

![](img/798779571d255fcae1052b4e6b878e9e_36.png)

### 2. cd 命令

![](img/798779571d255fcae1052b4e6b878e9e_40.png)

![](img/798779571d255fcae1052b4e6b878e9e_42.png)

![](img/798779571d255fcae1052b4e6b878e9e_44.png)

`cd` 是 “change directory” 的缩写，用于**更改当前工作目录**。

![](img/798779571d255fcae1052b4e6b878e9e_46.png)

**基本用法**：`cd` 后接目标路径（绝对或相对均可）。

**使用技巧**：
*   `cd ~` 或直接 `cd`：切换到当前用户的家目录。
*   `cd -`：切换到上一次访问的目录。

![](img/798779571d255fcae1052b4e6b878e9e_48.png)

### 3. ls 命令

`ls` 命令用于**列出目录下的文件和子目录**。

![](img/798779571d255fcae1052b4e6b878e9e_50.png)

直接执行 `ls` 会列出当前目录下的非隐藏项目。它有许多常用选项来定制输出内容：

![](img/798779571d255fcae1052b4e6b878e9e_52.png)

![](img/798779571d255fcae1052b4e6b878e9e_54.png)

以下是 `ls` 命令的一些常用选项介绍：

![](img/798779571d255fcae1052b4e6b878e9e_56.png)

![](img/798779571d255fcae1052b4e6b878e9e_58.png)

*   **`-a`**：列出所有文件，包括以 `.` 开头的隐藏文件。
*   **`-A`**：与 `-a` 类似，但不列出 `.`（当前目录）和 `..`（父目录）这两个特殊条目。
*   **`-d`**：仅列出目录本身，而不是目录下的内容。常与其他选项联用，如 `ls -ld /etc`。
*   **`-l`**：以长格式列出详细信息，这是最常用的选项之一。
*   **`-h`**：与 `-l` 联用，以人类可读的格式（如K、M、G）显示文件大小。
*   **`-R`**：递归列出子目录的内容。

![](img/798779571d255fcae1052b4e6b878e9e_60.png)

#### 详解 `ls -l` 输出

`ls -l` 命令的输出包含多列信息，每一列都有特定含义（以 `drwxr-xr-x. 2 student student 6 Dec 1 10:00 Videos` 为例）：

1.  **文件类型与权限**：第一列（如 `drwxr-xr-x.`）。
    *   第一个字符表示**文件类型**：
        *   `-`：普通文件
        *   `d`：目录
        *   `l`：符号链接（软链接）
        *   `b`：块设备文件（如磁盘）
        *   `c`：字符设备文件
        *   `p`：管道文件
        *   `s`：套接字文件
    *   后续9个字符表示**权限位**。
    *   最后的 `.` 与SELinux上下文有关。
2.  **硬链接数**：第二列（如 `2`）。
3.  **文件所有者**：第三列（如 `student`）。
4.  **文件所属组**：第四列（如 `student`）。
5.  **文件大小**：第五列（默认单位为字节，结合 `-h` 选项可显示为K、M等）。
6.  **最后修改时间**：第六列。
7.  **文件或目录名**：第七列。

## 总结

![](img/798779571d255fcae1052b4e6b878e9e_64.png)

本节课中我们一起学习了在Linux中通过名称指定文件的核心知识。我们首先了解了文件系统的两种视图，然后重点区分了**绝对路径**和**相对路径**的概念与用法。最后，我们掌握了三个基础且强大的导航命令：用 `pwd` 查看当前位置，用 `cd` 切换目录，以及用 `ls` 及其各种选项（特别是 `-l`）来查看目录内容的详细信息。这些是操作Linux文件系统的基石，请务必熟练掌握。

## 查看时间和日历

首先，我们可以使用`date`命令来显示当前的系统时间和日期。

![](img/b7df1aa3192a19944e8de53c45161486_7.png)

![](img/b7df1aa3192a19944e8de53c45161486_9.png)

```bash
date
```

![](img/b7df1aa3192a19944e8de53c45161486_11.png)

![](img/b7df1aa3192a19944e8de53c45161486_13.png)

执行该命令会输出类似“2020年 4月 11日 星期六 16:19:30 CST”的信息，具体取决于你的系统时区。

接下来，我们可以使用`cal`命令来查看日历。

```bash
cal
```

这个命令会显示当前月份的日历。

## 识别当前用户

在Linux终端中，命令提示符本身就包含了用户信息。提示符的格式通常是 `[用户名@主机名 当前目录]`。例如，`root@localhost ~` 表示当前用户是`root`，主机名是`localhost`，`~`符号代表当前位于用户的家目录。

提示符的最后一个字符也能指示用户身份：
*   **`#`**：表示当前用户是超级管理员`root`。
*   **`$`**：表示当前用户是普通用户。

除了观察提示符，我们还可以使用`id`命令来明确显示当前用户的身份信息。

```bash
id
```

`id`命令不仅可以查看自己，还可以检查系统中是否存在某个特定用户。

```bash
id 用户名
```

![](img/b7df1aa3192a19944e8de53c45161486_15.png)

![](img/b7df1aa3192a19944e8de53c45161486_17.png)

例如，`id zhangsan`。如果用户存在，命令会显示该用户的ID信息；如果用户不存在，则会提示 `no such user`。

## 切换用户

Linux是一个多用户操作系统，允许在线切换用户，无需注销。切换用户的命令是`su`。

以下是切换用户的基本步骤：
1.  从普通用户切换到`root`用户需要输入`root`的密码。
    ```bash
    su - root
    ```
2.  从`root`用户切换到任何其他用户则不需要输入密码。
    ```bash
    su - 用户名
    ```
3.  使用`exit`命令可以退出当前用户会话，返回到上一个用户。

**重要提示**：`root`用户拥有最高权限，切换至其他用户时无需密码。为避免会话嵌套混乱，建议切换用户后使用`exit`退出，或打开新的终端窗口进行操作，而不是连续使用`su`命令。

## 创建用户

## 修改用户密码

![](img/b7df1aa3192a19944e8de53c45161486_31.png)

![](img/b7df1aa3192a19944e8de53c45161486_32.png)

![](img/b7df1aa3192a19944e8de53c45161486_34.png)

为用户设置或修改密码的命令是`passwd`。

![](img/b7df1aa3192a19944e8de53c45161486_36.png)

以下是不同场景下的密码修改方法：
*   **`root`用户修改自己的密码**：直接运行`passwd`，根据提示输入两次新密码即可。系统对密码强度有要求，例如长度通常需不少于8个字符。
    ```bash
    passwd
    ```
*   **`root`用户修改其他用户的密码**：在`passwd`后加上用户名，无需知道该用户的旧密码。
    ```bash
    passwd 用户名
    ```
*   **普通用户修改自己的密码**：直接运行`passwd`，但必须首先输入自己当前的密码进行验证。
    ```bash
    passwd
    ```
*   **普通用户无法为其他用户修改密码**：如果尝试，系统会报错提示权限不足。

**关键点总结**：只有`root`用户可以为任意用户修改密码；普通用户只能修改自己的密码，且需要验证旧密码。

## 编辑文件

在Linux中，我们经常需要创建或修改文本文件。下面介绍两个常用的文本编辑器。

### 使用 `nano` 编辑器

`nano`是一个简单易用的文本编辑器。直接输入`nano`可以新建文件，输入`nano 文件名`可以编辑现有文件。

```bash
nano test.txt
```

![](img/b7df1aa3192a19944e8de53c45161486_38.png)

![](img/b7df1aa3192a19944e8de53c45161486_40.png)

在`nano`中编辑完成后，按 `Ctrl + X` 退出。系统会询问是否保存，按 `Y` 确认保存，然后输入文件名，或直接回车使用原文件名保存。

### 使用 `vim` 编辑器

![](img/b7df1aa3192a19944e8de53c45161486_42.png)

`vim`是一个功能强大但稍复杂的编辑器，是后续学习的重点。其基本操作模式如下：

1.  **打开文件**：`vim 文件名`（如果文件不存在，则会新建）。
    ```bash
    vim test.txt
    ```
2.  **进入插入模式**：打开文件后，默认处于“普通模式”，无法直接输入文字。按下 `i` 键，进入“插入模式”，此时屏幕左下角会显示 `-- INSERT --`，这时可以正常输入文本。
3.  **保存与退出**：
    *   首先按 `Esc` 键，从插入模式返回普通模式。
    *   然后输入冒号 `:`，进入“命令模式”。
    *   输入 `wq` 并回车，表示保存并退出（`w`=write，`q`=quit）。
    *   如果不想保存修改，输入 `q!` 并回车，表示强制退出不保存。

![](img/b7df1aa3192a19944e8de53c45161486_44.png)

![](img/b7df1aa3192a19944e8de53c45161486_46.png)

**`vim` 基本操作备忘**：
*   `i` -> 进入插入模式（开始编辑）。
*   `Esc` -> 退出插入模式，返回普通模式。
*   `:wq` -> 保存文件并退出。
*   `:q!` -> 不保存文件，强制退出。

## 修改其他用户密码 🔑

上一节我们介绍了用户切换，本节中我们来看看如何以管理员身份修改其他用户的密码。

普通用户只能修改自己的密码。若想修改其他用户的密码，需要使用 `sudo` 命令来获取管理员权限。直接使用 `passwd` 命令修改其他用户密码会失败。

**示例：**
```bash
$ passwd user2
```
执行上述命令会提示权限不足，因为 `user1` 不在 `/etc/sudoers` 授权文件中。

管理员 `root` 用户可以为普通用户授权。授权后，普通用户即可使用 `sudo` 命令来执行管理操作。

**授权并修改密码的步骤：**
1.  `root` 用户编辑 `/etc/sudoers` 文件，为 `user1` 添加相应权限。
2.  切换回 `user1` 用户。
3.  使用 `sudo passwd user2` 命令，输入 `user1` 的密码进行身份验证。
4.  验证通过后，系统会提示为 `user2` 设置新密码。此时可以设置任意密码（如 `123456`），操作成功。

这说明 `user1` 现在是以 `root` 的权限执行了密码修改操作。

## 查看用户信息 👤

`id` 命令用于显示用户的基本信息。

*   直接输入 `id` 会显示**当前用户**的信息。
*   输入 `id username` 可以查看指定用户的信息。如果用户存在，会显示一串包含用户ID和组ID的数字信息；如果用户不存在，则会提示 `no such user`。

普通用户也可以执行 `id` 命令查看自己的信息。

## 命令行效率工具 ⚡

### Tab键自动补全

Tab键 (`↹`) 是Linux命令行中最重要的效率工具之一，用于自动补全命令或目录/文件名。

**其作用包括：**
1.  **补全命令**：输入命令的前几个字母，按Tab键可以自动补全。如果以这几个字母开头的命令只有一个，则会直接补全；如果有多个，按两次Tab会列出所有可能的选择。
2.  **补全路径**：输入目录或文件路径时，按Tab键可以自动补全。
3.  **检验输入**：如果按Tab键没有反应，通常意味着当前输入有误，或者存在多个可能性需要进一步输入或按两次Tab查看。

**示例：**
输入 `pas` 后按两次Tab，会列出所有以 `pas` 开头的命令。
输入 `cd /etc/sysconfig/net` 时，可以在输入部分路径后按Tab键补全。

熟练掌握Tab键可以极大提升命令输入的速度和准确性。

以下是几种常用的历史命令调用技巧：

![](img/d08c46cfedbd820dc2cba2133f42d2e3_7.png)

![](img/d08c46cfedbd820dc2cba2133f42d2e3_9.png)

![](img/d08c46cfedbd820dc2cba2133f42d2e3_11.png)

![](img/d08c46cfedbd820dc2cba2133f42d2e3_12.png)

1.  **`history` 命令**：直接输入 `history` 查看完整列表。使用 `history N` 可以查看最近N条记录。
2.  **`!序号`**：根据 `history` 列出的序号，使用 `!71` 可以直接执行序号为71的那条命令。
3.  **`!开头字母`**：使用 `!i` 会执行历史记录中**最近一条**以 `i` 开头的命令。
4.  **`Ctrl + R`**：进入反向搜索模式，输入关键词（如 `id`）可以搜索并调用包含该关键词的最近一条命令。
5.  **上下方向键**：最常用的方法，按上箭头 `↑` 可以逐条查看并调用之前执行过的命令。
6.  **`Esc + .`**：这是一个非常实用的快捷键，可以调用**上一条命令的最后一个参数**。例如，上一条命令是 `ls /etc/sysconfig/network-scripts/`，那么按 `Esc` 后再按 `.`，就会直接输入 `/etc/sysconfig/network-scripts/`。

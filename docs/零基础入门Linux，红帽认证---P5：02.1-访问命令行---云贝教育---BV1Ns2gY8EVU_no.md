# Linux入门：02.1：访问命令行 💻

在本节课中，我们将要学习如何访问Linux系统的命令行界面。命令行是Linux系统管理的核心工具，掌握它是学习Linux的第一步。我们将从基本概念讲起，了解什么是Shell、终端和控制台，并学习如何登录系统、执行命令以及安全地退出会话。

## 什么是命令行？

上一节我们介绍了课程的整体目标，本节中我们来看看什么是命令行。

命令行是一个基于文本的界面。用户在这个界面中输入字符指令。这些指令通过一个称为Shell的介质，传输到系统的内核。内核解释并执行这些指令，最终输出结果。

这里的Shell是一个解释器。它的作用是将用户输入的命令，翻译成计算机内核能够识别的语言。内核根据指令调动CPU、内存等资源进行运算，并输出最终结果。

在红帽企业版Linux中，默认的Shell是GNU Bourne-Again Shell，简称为**bash**或**Bash Shell**。

## Shell的种类

![](img/4ea0e0478bc70ec3422047fb421de57c_1.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_3.png)

Shell是一个软件，是用户与内核交互的媒介。Shell的种类很多，以下是几种常见的Shell：

![](img/4ea0e0478bc70ec3422047fb421de57c_5.png)

*   **Bash Shell (bash)**：现代主流Linux发行版（如Ubuntu、Fedora、CentOS、红帽RHEL）默认使用的Shell，应用非常广泛。
*   **SH Shell (sh)**：早期Unix系统使用的Shell。在现代Linux中，`sh`通常是`bash`的一个快捷方式或兼容模式。
*   **Z Shell (zsh)**：功能强大的Shell，拥有丰富的插件和主题。新版macOS和RHEL 9也开始支持。
*   **Fish Shell (fish)**：一个智能且用户友好的Shell，拥有自动建议和语法高亮等功能。
*   **Power Shell (pwsh)**：最初为Windows设计，现在也可安装在Linux上，特别擅长处理对象和结构化数据。

## 终端、控制台与TTY

![](img/4ea0e0478bc70ec3422047fb421de57c_16.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_18.png)

在学习Shell时，需要了解三个相关术语：终端 (Terminal)、控制台 (Console) 和 TTY。

![](img/4ea0e0478bc70ec3422047fb421de57c_20.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_22.png)

1.  **终端 (Terminal)**：早期指连接大型主机的输入输出设备。在现代计算机中，我们通过软件模拟终端，例如在图形界面中运行的 **GNOME Terminal** 程序。任何可以执行命令的界面，广义上都可以称为终端。
2.  **控制台 (Console)**：早期指直接控制计算机运行的硬件设备。在现代，直接连接在计算机主机上的显示器被视为**本地控制台**。例如，虚拟机窗口就是该虚拟机的本地控制台。
3.  **TTY (Teletypewriter，电传打字机)**：早期通过电话线连接的输入输出设备。在Linux字符界面中，承载Shell的介质就是TTY。可以通过 `tty` 命令查看当前所在的TTY设备编号，例如 `/dev/tty1`。

无论是哪种设备，现在都可以统称为**终端**。只要能够执行命令，我们都可以称之为终端。

## 命令提示符与命令结构

![](img/4ea0e0478bc70ec3422047fb421de57c_26.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_28.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_30.png)

当我们打开终端时，会看到闪烁的光标和一行提示信息，这就是**命令提示符**。它等待用户输入命令。

![](img/4ea0e0478bc70ec3422047fb421de57c_32.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_34.png)

在Linux中，命令提示符有两种常见形式：
*   **`$`**：代表当前是**普通用户**身份。
*   **`#`**：代表当前是**超级管理员 (root)** 身份。看到此符号需格外谨慎，因为执行的操作可能对系统造成严重影响。

命令提示符的格式通常为：`[用户名@主机名 当前目录]提示符`
例如：`[student@workstation ~]$`
*   `student`：当前登录的用户。
*   `workstation`：当前主机的主机名。
*   `~`：代表当前位于用户的家目录（主目录）。
*   `$`：普通用户提示符。

在提示符后，我们可以输入命令。一个完整的Shell命令通常由三部分组成：
`command [options] [arguments]`
*   **command (命令)**：要运行的程序或工具，本质上是可执行的二进制文件。
*   **options (选项)**：用于调整命令行为的参数，通常以 `-` 或 `--` 开头。
*   **arguments (参数)**：命令作用的目标对象。

可以将命令比作**汽车**，选项比作**导航设备**，参数比作**目的地**。开车（命令）是必须的，而是否使用导航（选项）以及去往哪个目的地（参数），则根据需要决定。

选项主要有三种风格：
1.  **短格式 (BSD/Unix风格)**：通常是一个字母的缩写，前面加一个短横线 `-`。例如 `ls -l -h`。多个短选项可以合并，如 `ls -lh`。
2.  **长格式 (GNU风格)**：是选项的全称，前面加两个短横线 `--`。例如 `ls --help`。长格式选项不能合并。
3.  **无横线格式**：较少见，选项前没有短横线。例如 `ps aux` 中的 `a`，`u`，`x`。

有些命令的选项自身也可以带有参数。

## 登录系统

了解了基本概念后，我们来看看如何登录到Linux系统。登录方式主要分为本地登录和远程登录。

### 本地登录

![](img/4ea0e0478bc70ec3422047fb421de57c_36.png)

本地登录分为文本界面和图形界面。

![](img/4ea0e0478bc70ec3422047fb421de57c_38.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_40.png)

*   **文本界面登录**：在系统启动后的黑色字符界面，直接输入用户名和密码即可登录。在RHEL 9中，默认有6个虚拟控制台（对应TTY1-TTY6），可以通过快捷键 `Ctrl+Alt+F1` 到 `Ctrl+Alt+F6` 进行切换。如果安装了图形界面，它通常会占用第一个虚拟控制台（TTY1或TTY2）。
*   **图形界面登录**：在系统提供的图形化登录界面，选择用户并输入密码登录。登录后，可以在桌面环境中打开终端程序（如GNOME Terminal）来使用命令行。

![](img/4ea0e0478bc70ec3422047fb421de57c_44.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_48.png)

对于没有连接键盘、显示器的服务器，可以通过串口控制台或管理口进行带外管理。在极端情况下，也可以通过配置VNC来远程查看图形或字符安装界面。

### 远程登录

远程登录是管理服务器最常用的方式，主要通过SSH协议实现。

SSH (Secure Shell) 是一种安全的网络协议，用于加密远程登录和管理系统。其基本语法是：
`ssh [user@]hostname`
登录时主要使用两种验证方式：
*   **密码验证**：首次连接陌生主机时，会提示确认主机密钥，输入 `yes` 后，再输入用户密码即可登录。
*   **密钥对验证**：使用公钥和私钥进行验证，比密码方式更安全。例如：`ssh -i ~/.ssh/id_rsa user@hostname`

## 注销登录

![](img/4ea0e0478bc70ec3422047fb421de57c_56.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_58.png)

![](img/4ea0e0478bc70ec3422047fb421de57c_59.png)

完成操作后，务必注销登录以保证安全。

![](img/4ea0e0478bc70ec3422047fb421de57c_63.png)

以下是退出登录的方法：
*   **图形界面**：点击桌面右上角，选择“注销”或“Log Out”。
*   **终端/字符界面**：可以执行以下任一命令：
    *   `exit`
    *   `logout`
    *   或使用快捷键 `Ctrl+D`

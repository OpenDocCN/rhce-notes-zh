## 归档与压缩的概念 🔍

在Linux系统中，归档和压缩是两个不同的概念，通常由不同的工具完成。

*   **归档**：指将多个文件或目录合并成一个单一的文件，便于管理和传输。归档操作本身**不会减少**文件占用的总空间。例如，将10个各10MB的文件归档后，得到的归档文件大小约为100MB。
*   **压缩**：指通过特定算法减少文件占用的磁盘空间。在Linux中，有专门的工具如`gzip`、`bzip2`、`xz`等来完成压缩。

`tar`命令最初只负责归档。后来，通过增加选项，它可以自动调用压缩工具，从而**一步完成归档和压缩**，生成我们常见的`.tar.gz`、`.tar.bz2`等格式的压缩包。

---

## 使用tar命令进行基本归档操作 🛠️

上一节我们介绍了归档与压缩的基本概念，本节中我们来看看如何使用`tar`命令进行最基本的归档操作。

`tar`命令用于创建归档的核心选项是 `-c` (create)，同时必须使用 `-f` (file) 选项来指定生成的归档文件名。

基本命令格式如下：
```bash
tar -cf <归档文件名> <要归档的文件或目录1> <要归档的文件或目录2> ...
```

**示例**：将当前目录下的 `under` 和 `out` 文件归档为 `backup.tar`。
```bash
tar -cf backup.tar under out
```

## 查看与解压归档文件 👀

创建归档文件后，我们可能需要查看其内容或将其中的文件还原出来。

以下是相关的操作选项：
*   `-t`：列出归档文件的内容。
*   `-x`：从归档文件中提取（解压）文件。
*   `-v`：在处理过程中显示详细信息（verbose）。
*   `-C`：指定解压到的目标目录。

**示例操作**：
1.  **查看归档内容**：
    ```bash
    tar -tf backup.tar
    ```
2.  **解压到当前目录**：
    ```bash
    tar -xf backup.tar
    ```
3.  **解压到指定目录（如 `/opt`）**：
    ```bash
    tar -xf backup.tar -C /opt
    ```
4.  **显示详细信息的解压过程**：
    ```bash
    tar -xvf backup.tar
    ```

## 保留绝对路径与文件属性 📂

![](img/56de8c703a2680b7f5212f8d5e944aa3_12.png)

在备份系统文件时，保留文件的原始路径和属性非常重要。`tar`命令提供了相应的选项。

![](img/56de8c703a2680b7f5212f8d5e944aa3_14.png)

![](img/56de8c703a2680b7f5212f8d5e944aa3_16.png)

以下是相关的选项说明：
*   `-P` (大写)：归档时保留文件的绝对路径。这样在解压时，配合 `-P` 选项，文件可以精确还原到原始位置。
*   `-p` (小写)：保留文件的原始权限、所有权和时间戳等属性。在备份和恢复时，这是推荐的做法。

**最佳实践建议**：在进行重要备份时，组合使用 `-cP` 选项。这能确保归档文件包含完整的路径信息，便于日后精准恢复。
```bash
tar -cPf system_backup.tar /usr/local
```

## 结合压缩工具：一步完成归档与压缩 ⚡

![](img/56de8c703a2680b7f5212f8d5e944aa3_20.png)

单独使用`gzip`等工具只能压缩单个文件。要想将多个文件打包并压缩，`tar`命令的 `-z`, `-j`, `-J` 选项就派上了用场。

这些选项可以自动调用对应的压缩工具，在归档的同时进行压缩：
*   `-z`：调用 `gzip` 程序，生成 `.tar.gz` 或 `.tgz` 文件。
*   `-j`：调用 `bzip2` 程序，生成 `.tar.bz2` 文件。
*   `-J`：调用 `xz` 程序，生成 `.tar.xz` 文件。

![](img/56de8c703a2680b7f5212f8d5e944aa3_22.png)

![](img/56de8c703a2680b7f5212f8d5e944aa3_23.png)

![](img/56de8c703a2680b7f5212f8d5e944aa3_25.png)

**示例**：将 `/usr/local` 目录归档并用 `gzip` 压缩。
```bash
tar -czPf backup.tar.gz /usr/local
```
**注意**：使用 `-j` 或 `-J` 选项前，请确保系统已安装 `bzip2` 或 `xz` 软件包。

## 命令选项顺序与考试实战演练 🎯

![](img/56de8c703a2680b7f5212f8d5e944aa3_34.png)

![](img/56de8c703a2680b7f5212f8d5e944aa3_36.png)

`tar`命令的选项顺序有一定灵活性，但有一个关键点需要注意：`-f` 选项必须紧跟在它所指定的文件名之前，它们是一个整体。

以下是正确的选项顺序示例：
```bash
tar -czvPf backup.tar.gz /path/to/dir
```
以下则是错误的，因为 `-f` 没有直接关联到文件名：
```bash
tar -cfzv backup.tar.gz /path/to/dir # 错误示例
```

![](img/56de8c703a2680b7f5212f8d5e944aa3_38.png)

**考试题目模拟**：在 `/root` 目录下创建一个名为 `backup.tar.bz2` 的归档压缩文件，其中应包含 `/usr/local` 目录的内容。

![](img/56de8c703a2680b7f5212f8d5e944aa3_40.png)

解答命令如下：
```bash
tar -cjPf /root/backup.tar.bz2 /usr/local
```
完成后，可以使用 `file` 命令验证文件格式：
```bash
file /root/backup.tar.bz2
```

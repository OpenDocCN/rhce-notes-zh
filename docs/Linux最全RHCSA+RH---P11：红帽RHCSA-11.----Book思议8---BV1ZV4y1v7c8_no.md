## 查看内核信息 🧠

上一节我们介绍了系统的基本操作，本节中我们来看看如何查看系统的内核信息。内核是Linux操作系统的核心，了解其版本对于系统兼容性和问题诊断非常重要。

![](img/84ad0082164c303e4339835ef9569ae9_13.png)

![](img/84ad0082164c303e4339835ef9569ae9_15.png)

![](img/84ad0082164c303e4339835ef9569ae9_17.png)

使用 `uname` 命令可以查看内核的相关信息，包括内核名称和版本。

![](img/84ad0082164c303e4339835ef9569ae9_19.png)

![](img/84ad0082164c303e4339835ef9569ae9_21.png)

![](img/84ad0082164c303e4339835ef9569ae9_23.png)

![](img/84ad0082164c303e4339835ef9569ae9_25.png)

![](img/84ad0082164c303e4339835ef9569ae9_27.png)

![](img/84ad0082164c303e4339835ef9569ae9_29.png)

以下是查看内核信息的常用命令选项：

![](img/84ad0082164c303e4339835ef9569ae9_31.png)

![](img/84ad0082164c303e4339835ef9569ae9_33.png)

*   `uname`：默认只显示内核名称。
*   `uname -r`：显示内核的发行版本号。
*   `uname -s`：显示内核名称（默认行为）。
*   `uname -rs`：结合显示内核名称和版本号，这是最常用的方式。

![](img/84ad0082164c303e4339835ef9569ae9_35.png)

![](img/84ad0082164c303e4339835ef9569ae9_37.png)

![](img/84ad0082164c303e4339835ef9569ae9_39.png)

![](img/84ad0082164c303e4339835ef9569ae9_41.png)

例如，执行 `uname -rs` 可能会输出类似 `Linux 3.10.0-1160.el7.x86_64` 的结果，表示这是Linux内核，版本是3.10.0，适用于红帽企业版Linux 7，CPU架构是x86_64位。

## 查看CPU信息 💻

![](img/84ad0082164c303e4339835ef9569ae9_51.png)

![](img/84ad0082164c303e4339835ef9569ae9_53.png)

![](img/84ad0082164c303e4339835ef9569ae9_55.png)

了解了内核信息后，我们来看看如何查看CPU的详细信息。CPU是计算机的大脑，其型号、核心数等参数直接影响系统性能。

![](img/84ad0082164c303e4339835ef9569ae9_57.png)

![](img/84ad0082164c303e4339835ef9569ae9_59.png)

![](img/84ad0082164c303e4339835ef9569ae9_61.png)

查看CPU信息主要有两种方法：查看特定文件和使用专用命令。

![](img/84ad0082164c303e4339835ef9569ae9_63.png)

![](img/84ad0082164c303e4339835ef9569ae9_65.png)

以下是两种查看CPU信息的方法：

1.  **查看 `/proc/cpuinfo` 文件**：这个文件存储了CPU的详细信息。可以使用 `cat /proc/cpuinfo` 命令查看。文件内容非常详细，包括处理器型号、制造商、核心数量、缓存大小等。对于初学者，最需要关注的是 `processor`（逻辑处理器编号，可反映CPU核心/线程数）和 `model name`（CPU型号）字段。
2.  **使用 `lscpu` 命令**：这条命令能以更结构化的方式显示CPU信息，更易于阅读。它会汇总显示CPU架构、操作模式（32位/64位）、核心数（`CPU(s)`）、每颗物理CPU的核心数、线程数等关键信息。

> **注意**：在虚拟机中查看到的CPU信息通常与物理主机的CPU型号一致，因为虚拟机是通过虚拟化技术模拟出来的。

## 查看内存信息 🧠

![](img/84ad0082164c303e4339835ef9569ae9_67.png)

![](img/84ad0082164c303e4339835ef9569ae9_69.png)

接下来，我们学习如何查看系统的内存使用情况。内存是系统运行程序和处理数据的临时工作区，监控其使用情况至关重要。

![](img/84ad0082164c303e4339835ef9569ae9_71.png)

![](img/84ad0082164c303e4339835ef9569ae9_73.png)

查看内存信息同样有两种主要方式：查看特定文件和使用命令。

![](img/84ad0082164c303e4339835ef9569ae9_75.png)

![](img/84ad0082164c303e4339835ef9569ae9_77.png)

![](img/84ad0082164c303e4339835ef9569ae9_79.png)

以下是查看内存信息的方法：

1.  **查看 `/proc/meminfo` 文件**：这个文件包含了系统内存的详细统计数据。使用 `cat /proc/meminfo` 命令查看。对于日常管理，我们通常只关注前两行：`MemTotal`（总内存大小）和 `MemFree`（空闲内存大小）。
2.  **使用 `free` 命令**：这是最常用的查看内存使用情况的命令。直接输入 `free` 会以字节为单位显示，可读性较差。建议使用 `free -h` 命令，它会以人类易读的单位（K, M, G）显示内存大小。

执行 `free -h` 后，输出主要分为两部分：`Mem`（物理内存）和 `Swap`（交换空间）。
对于 `Mem` 行，我们需要关注：
*   `total`：物理内存总量。
*   `used`：已使用的内存量。
*   `free`：空闲的内存量。

**关于Swap交换空间**：当物理内存不足时，系统会使用硬盘的一部分空间来模拟内存，这就是Swap。但这会严重降低系统性能，因为硬盘速度远慢于内存。因此，在现代服务器中，通常建议关闭Swap或确保有充足的物理内存。可以使用 `swapoff -a` 命令**临时关闭**Swap功能。

## 查看网卡信息 🌐

![](img/84ad0082164c303e4339835ef9569ae9_85.png)

现在，我们来看看如何查看和了解网络接口卡（网卡）的配置信息。网卡是计算机连接网络的硬件设备。

![](img/84ad0082164c303e4339835ef9569ae9_87.png)

![](img/84ad0082164c303e4339835ef9569ae9_89.png)

查看网卡信息可以通过查看其配置文件或使用网络命令。

![](img/84ad0082164c303e4339835ef9569ae9_91.png)

![](img/84ad0082164c303e4339835ef9569ae9_93.png)

![](img/84ad0082164c303e4339835ef9569ae9_95.png)

![](img/84ad0082164c303e4339835ef9569ae9_97.png)

以下是查看网卡信息的方法：

![](img/84ad0082164c303e4339835ef9569ae9_99.png)

1.  **查看网卡配置文件**：网卡配置通常保存在 `/etc/sysconfig/network-scripts/` 目录下，文件名类似 `ifcfg-ens32` 或 `ifcfg-eth0`。使用 `cat` 命令查看该文件，关键参数包括：
    *   `BOOTPROTO`：获取IP地址的方式（`static`/`none` 为静态，`dhcp` 为动态）。
    *   `ONBOOT`：是否在系统启动时激活此网卡（应为 `yes`）。
    *   `IPADDR`：IP地址。
    *   `NETMASK`：子网掩码。
    *   `GATEWAY`：默认网关。
    *   `DNS1`：首选DNS服务器地址。

![](img/84ad0082164c303e4339835ef9569ae9_101.png)

![](img/84ad0082164c303e4339835ef9569ae9_103.png)

![](img/84ad0082164c303e4339835ef9569ae9_104.png)

2.  **使用 `ifconfig` 或 `ip` 命令**：
    *   `ifconfig`：传统命令，可以显示所有网卡的详细信息，包括IP地址、MAC地址、接收/发送的数据包统计等。如果系统未安装，可使用 `yum -y install net-tools` 安装。
    *   `ip addr show` 或简写 `ip a`：更现代的命令，用于显示IP地址信息，输出比 `ifconfig` 更简洁。

在 `ifconfig` 的输出中，`ens32` 或 `eth0` 通常代表物理网卡，而 `lo` 是本地回环网卡（IP为127.0.0.1），用于本机内部通信和测试。

## 查看与修改主机名 🏷️

![](img/84ad0082164c303e4339835ef9569ae9_106.png)

![](img/84ad0082164c303e4339835ef9569ae9_108.png)

最后，我们来学习如何查看和修改系统的主机名。主机名是网络中对设备的标识。

![](img/84ad0082164c303e4339835ef9569ae9_110.png)

查看和修改主机名非常简单，可以通过文件或命令完成。

![](img/84ad0082164c303e4339835ef9569ae9_112.png)

以下是查看与修改主机名的方法：

![](img/84ad0082164c303e4339835ef9569ae9_114.png)

![](img/84ad0082164c303e4339835ef9569ae9_116.png)

1.  **查看主机名**：
    *   **查看文件**：主机名保存在 `/etc/hostname` 文件中，使用 `cat /etc/hostname` 查看。
    *   **使用命令**：直接运行 `hostname` 命令即可显示当前主机名。

2.  **修改主机名**：
    *   **临时修改**：使用 `hostname <新主机名>`，例如 `hostname test`。修改立即生效，但重启系统后会恢复。
    *   **永久修改**：使用 `hostnamectl set-hostname <新主机名>`，例如 `hostnamectl set-hostname myserver`。此命令会同时修改 `/etc/hostname` 文件，并立即生效，重启后也不会改变。修改后需要重新登录Shell才能看到提示符中的变化。

![](img/84ad0082164c303e4339835ef9569ae9_118.png)

![](img/84ad0082164c303e4339835ef9569ae9_120.png)

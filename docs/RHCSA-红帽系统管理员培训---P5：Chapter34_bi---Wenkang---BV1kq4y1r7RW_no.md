## 远程连接SSH 🔗

![](img/b440ef5d9e50b6e377a24c2a72d555c6_19.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_21.png)

上一节我们介绍了在本地终端操作，本节中我们来看看如何通过SSH远程连接到服务器。这对于管理不在本地的服务器至关重要。

![](img/b440ef5d9e50b6e377a24c2a72d555c6_23.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_24.png)

首先，登录到您的Linux系统。您可以使用图形界面登录，也可以使用控制台。系统中有两个用户：`root`（管理员）和`klk`（普通用户）。我们使用`root`登录，密码是`Asimov`。

![](img/b440ef5d9e50b6e377a24c2a72d555c6_26.png)

登录后，需要获取服务器的IP地址。使用命令：
```bash
ip addr
```
或者
```bash
ifconfig
```
在输出信息中找到以`en`开头（如`ens160`）的网卡，其`inet`后面的地址就是服务器的IP。

![](img/b440ef5d9e50b6e377a24c2a72d555c6_28.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_30.png)

确保您的网络是通的，可以使用`ping`命令测试：
```bash
ping <您的IP地址>
```
如果能通，就可以进行SSH连接。

![](img/b440ef5d9e50b6e377a24c2a72d555c6_32.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_34.png)

使用SSH命令连接：
```bash
ssh root@<您的IP地址>
```
例如：
```bash
ssh root@192.168.220.133
```
首次连接时会提示保存服务器指纹，输入`yes`。然后输入`root`用户的密码即可成功连接。之后的所有操作都可以在这个SSH终端中进行，与在本地图形化界面中操作效果一致。

## 获取帮助信息 🆘

![](img/b440ef5d9e50b6e377a24c2a72d555c6_44.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_46.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_48.png)

当您遇到不熟悉的命令时，获取帮助信息是解决问题的第一步。Linux系统提供了多种获取帮助的方式。

![](img/b440ef5d9e50b6e377a24c2a72d555c6_50.png)

### 使用 `--help` 选项

![](img/b440ef5d9e50b6e377a24c2a72d555c6_52.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_54.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_56.png)

大多数命令都支持`--help`或`-h`选项，可以显示基本的用法和参数简介。
```bash
nmcli --help
```
但这通常只提供简略的信息。

### 使用 `man` 手册

![](img/b440ef5d9e50b6e377a24c2a72d555c6_58.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_60.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_62.png)

`man`（manual的缩写）命令提供最详细、最权威的命令手册。其语法为：
```bash
man <命令名>
```
例如，查看`nmcli`命令的完整手册：
```bash
man nmcli
```
`man`手册界面有特定的操作方法：
*   按 **空格键** 或 **Page Down** 向下翻页。
*   按 **Page Up** 向上翻页。
*   按 **/** 键进入搜索模式，输入关键词后按回车搜索。按 **n** 查找下一个，按 **N** 查找上一个。
*   按 **q** 键退出手册。

![](img/b440ef5d9e50b6e377a24c2a72d555c6_64.png)

`man`手册内容结构清晰，通常包括：
*   **NAME**：命令名称和简要说明。
*   **SYNOPSIS**：命令语法格式。
*   **DESCRIPTION**：详细描述。
*   **OPTIONS**：所有参数详解。
*   **EXAMPLES**：使用示例（非常实用）。
*   **SEE ALSO**：相关命令或文件。

![](img/b440ef5d9e50b6e377a24c2a72d555c6_66.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_68.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_70.png)

`man`手册分为9个章节，常见的包括：
*   **1**：普通用户可执行的命令。
*   **4**：设备文件说明。
*   **5**：配置文件格式。
*   **8**：系统管理员命令。

### 使用 `info` 文档

`info`是另一种格式的帮助文档，通常比`man`更结构化，支持超链接和更好的导航，但内容可能不如`man`详尽。
```bash
info <命令名>
```

![](img/b440ef5d9e50b6e377a24c2a72d555c6_91.png)

### 其他帮助资源

![](img/b440ef5d9e50b6e377a24c2a72d555c6_93.png)

除了上述命令，还有以下途径：
1.  **程序自带的文档**：通常安装在`/usr/share/doc/`目录下。
2.  **Red Hat 客户门户**：如果您购买了Red Hat订阅，可以访问 [Red Hat Customer Portal](https://access.redhat.com) 获取官方知识库和付费技术支持。

## 输出重定向与文件编辑 ✏️

在Linux中，编辑文件是一项核心操作，因为“一切皆文件”。首先，我们需要学会如何将命令的输出保存到文件中。

![](img/b440ef5d9e50b6e377a24c2a72d555c6_95.png)

### 输出重定向

![](img/b440ef5d9e50b6e377a24c2a72d555c6_97.png)

命令的输出分为**标准输出**（正常信息）和**标准错误**（错误信息）。我们可以将它们重定向到文件。

*   **将标准输出保存到文件**：使用 `>` 符号。如果文件不存在则创建，存在则覆盖。
    ```bash
    ls /etc > etc_list.txt
    ```
*   **将标准输出追加到文件**：使用 `>>` 符号。
    ```bash
    date >> log.txt
    ```
*   **将标准错误保存到文件**：使用 `2>` 符号。
    ```bash
    ls /nonexistent 2> error.log
    ```
*   **将标准输出和标准错误都保存到文件**：使用 `&>` 符号。
    ```bash
    some_command &> output_and_error.log
    ```
*   **丢弃输出**：重定向到 `/dev/null`。
    ```bash
    noisy_command > /dev/null 2>&1
    ```

### 使用 `vi`/`vim` 命令行编辑器 🖥️

![](img/b440ef5d9e50b6e377a24c2a72d555c6_110.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_112.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_114.png)

![](img/b440ef5d9e50b6e377a24c2a72d555c6_116.png)

在服务器管理、运维和开发中，`vi`或其增强版`vim`是必须掌握的编辑器。它功能强大，完全在终端内操作。

#### 三种模式

`vim`有三种主要模式，需要时刻注意当前处于哪种模式：
1.  **命令模式（Normal Mode）**：启动`vim`后的默认模式。在此模式下可以移动光标、删除、复制、粘贴、搜索等，但不能直接输入文本。
2.  **插入模式（Insert Mode）**：在此模式下可以像普通编辑器一样输入和编辑文本。按 `i`, `a`, `o` 等键进入。
3.  **底线命令模式（Command-Line Mode）**：在命令模式下按 `:` 进入。可以执行保存、退出、查找替换等操作。

**核心要点**：在插入模式或底线命令模式操作完成后，**必须按 `Esc` 键返回命令模式**。

#### 基本操作流程

1.  **打开文件**：
    ```bash
    vim filename.txt
    ```
2.  **进入插入模式编辑**：在命令模式下，按以下键之一：
    *   `i`：在光标前插入。
    *   `a`：在光标后插入。
    *   `o`：在当前行下方新建一行并插入。
    *   `I`：在行首插入。
    *   `A`：在行尾插入。
    *   `O`：在当前行上方新建一行并插入。
3.  **返回命令模式**：按 `Esc` 键。
4.  **保存和退出**：在命令模式下按 `:` 进入底线命令模式。
    *   `:w`：保存文件。
    *   `:q`：退出`vim`（如果文件已修改，会提示保存）。
    *   `:wq` 或 `:x`：保存并退出。
    *   `:q!`：不保存强制退出。

#### 常用命令模式操作

*   **移动光标**：
    *   `h`（左）, `j`（下）, `k`（上）, `l`（右）
    *   `gg`：跳到文件第一行。
    *   `G`：跳到文件最后一行。
    *   `:n`：跳到第n行（如 `:13` 跳到第13行）。
*   **删除**：
    *   `x`：删除光标处的字符。
    *   `dd`：删除当前整行。
    *   `5dd`：删除从当前行开始的5行。
*   **复制（yank）与粘贴（put）**：
    *   `yy`：复制当前行。
    *   `5yy`：复制从当前行开始的5行。
    *   `p`：在光标后粘贴。
    *   `P`：在光标前粘贴。
*   **撤销与重做**：
    *   `u`：撤销上一次操作。
    *   `Ctrl + r`：重做被撤销的操作。
*   **搜索**：
    *   `/keyword`：向前搜索“keyword”。
    *   `?keyword`：向后搜索“keyword”。
    *   按 `n` 查找下一个，`N` 查找上一个。
*   **显示行号**：在底线命令模式输入 `:set number` 或 `:set nu`。取消用 `:set nonumber`。
*   **可视化模式**：按 `v` 进入字符可视化模式，可以选中文本块进行操作（如复制`y`、删除`d`）。按 `V` 进入行可视化模式。

## 总结 📝

本节课中我们一起学习了：
1.  **远程连接**：如何使用SSH从任何地方安全地连接到Linux服务器。
2.  **获取帮助**：掌握了通过`--help`、`man`、`info`以及官方文档等多种渠道查询命令用法和解决问题的方法。
3.  **输出重定向**：学会了将命令的输出（包括标准输出和错误）保存到文件或进行其他处理。
4.  **文件编辑**：重点练习了强大的`vim`编辑器，理解了其三种模式（命令、插入、底线命令）的切换与基本操作，这是成为高效Linux管理员的必备技能。

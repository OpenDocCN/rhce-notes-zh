## 什么是man手册？📘

![](img/3a409730df9a355dbaaa072f2c287c39_6.png)

![](img/3a409730df9a355dbaaa072f2c287c39_7.png)

`man`是单词`manual`的缩写，意为手册。就像购买产品时会附带产品说明书一样，Linux系统中的大多数软件在开发完成后，都会附带一份详细的手册随软件一同发布。用户可以通过查看这些手册来了解软件的功能和使用方法。

![](img/3a409730df9a355dbaaa072f2c287c39_9.png)

![](img/3a409730df9a355dbaaa072f2c287c39_11.png)

![](img/3a409730df9a355dbaaa072f2c287c39_13.png)

## man手册的分类 📂

Linux系统中的文件类型众多，`man`手册也相应地进行了分类，以便用户更高效地查找信息。这是因为所有`man`手册都存放在同一个系统中，如果存在名称相同的条目（例如`passwd`，它既是一个命令，也是一个配置文件），就需要通过指定章节类型来区分。

![](img/3a409730df9a355dbaaa072f2c287c39_15.png)

![](img/3a409730df9a355dbaaa072f2c287c39_17.png)

以下是`man`手册的主要章节分类：

*   **1 - 用户命令**：普通用户可执行程序的说明。
*   **2 - 系统调用**：从用户空间调用内核功能的说明。
*   **3 - 库函数**：由程序库提供的函数说明。
*   **4 - 特殊文件**：通常是设备文件的说明。
*   **5 - 文件格式**：配置文件的格式和语法说明。
*   **6 - 游戏**：游戏的说明（较少使用）。
*   **7 - 杂项**：其他类别。
*   **8 - 系统管理命令**：通常为`root`用户使用的管理命令。
*   **9 - 内核例程**：非标准的内核API说明。

例如，查看`passwd`命令和`/etc/passwd`配置文件就需要指定不同的章节：
```bash
man 1 passwd   # 查看passwd命令的用法
man 5 passwd   # 查看/etc/passwd配置文件的格式
```

## 浏览man手册页面 🔍

![](img/3a409730df9a355dbaaa072f2c287c39_19.png)

![](img/3a409730df9a355dbaaa072f2c287c39_20.png)

上一节我们介绍了`man`手册的分类，本节中我们来看看如何有效地浏览一个`man`手册页面。以`man passwd`为例，页面通常包含名称、语法、描述、选项、示例等部分。

![](img/3a409730df9a355dbaaa072f2c287c39_22.png)

![](img/3a409730df9a355dbaaa072f2c287c39_23.png)

![](img/3a409730df9a355dbaaa072f2c287c39_24.png)

![](img/3a409730df9a355dbaaa072f2c287c39_25.png)

在`man`页面中，可以使用以下快捷键进行操作：

![](img/3a409730df9a355dbaaa072f2c287c39_27.png)

![](img/3a409730df9a355dbaaa072f2c287c39_29.png)

*   **上下箭头**：逐行滚动。
*   **空格键 / Page Down**：向下翻一页。
*   **Page Up**：向上翻一页。
*   **Home / gg**：跳转到页面开头。
*   **End / G**：跳转到页面末尾。
*   **/**：向前搜索。输入`/keyword`后按回车，然后按`n`查找下一个匹配项，按`N`查找上一个匹配项。
*   **?**：向后搜索。输入`?keyword`后按回车，然后按`n`向上查找，按`N`向下查找。
*   **h**：查看帮助，显示所有可用的快捷键。
*   **q**：退出`man`页面。

## 搜索man手册 🔎

![](img/3a409730df9a355dbaaa072f2c287c39_33.png)

![](img/3a409730df9a355dbaaa072f2c287c39_35.png)

除了按章节查看，我们还可以在系统中搜索包含特定关键词的`man`手册。这需要使用`man -k`命令，它会在手册的**名称**和**简短描述**中进行搜索。

![](img/3a409730df9a355dbaaa072f2c287c39_37.png)

例如，搜索与`passwd`相关的所有手册：
```bash
man -k passwd
```

有时，新安装的软件包提供的`man`手册需要更新系统数据库后才能被`man -k`搜索到。这时需要运行`mandb`命令来更新手册数据库：
```bash
sudo mandb
```
系统通常会自动每天运行一次`mandb`，但安装新软件后手动运行可以立即生效。

## 其他帮助获取方式 💡

![](img/3a409730df9a355dbaaa072f2c287c39_43.png)

![](img/3a409730df9a355dbaaa072f2c287c39_45.png)

`man`手册虽然详尽，但有时内容较多。对于只想快速查看命令基本用法和选项的情况，还有更简洁的方式。

![](img/3a409730df9a355dbaaa072f2c287c39_47.png)

![](img/3a409730df9a355dbaaa072f2c287c39_49.png)

![](img/3a409730df9a355dbaaa072f2c287c39_51.png)

以下是几种常见的获取命令帮助的方法：

*   **`command --help` 或 `command -h`**：大多数外部命令支持此选项，用于显示简洁的使用帮助。
    ```bash
    passwd --help
    ```
*   **`help` 命令**：用于查看Shell**内置命令**的帮助。例如`history`, `cd`, `echo`等是Bash Shell自带的命令。
    ```bash
    help history
    ```

![](img/3a409730df9a355dbaaa072f2c287c39_53.png)

![](img/3a409730df9a355dbaaa072f2c287c39_55.png)

![](img/3a409730df9a355dbaaa072f2c287c39_57.png)

**核心概念区分**：
*   **内部命令**：Shell程序（如Bash）自身包含的命令。使用`help`查看帮助。
*   **外部命令**：由独立软件包提供的命令。使用`man`或`--help`查看帮助。

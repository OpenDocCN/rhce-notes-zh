## VIM的安装与基本介绍

首先，我们需要了解如何获取和安装VIM。默认的最小化系统安装可能不包含VIM，需要手动安装。

**安装VIM的命令如下：**
```bash
yum install vim -y
```

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_14.png)

安装完成后，可以通过 `vimtutor` 命令启动一个交互式教程来练习VIM的基本操作。

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_16.png)

**启动教程命令：**
```bash
vimtutor
```

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_18.png)

VIM与传统的VI编辑器主要区别在于**语法高亮显示**和更丰富的**扩展命令**支持。例如，使用VIM打开文件时，关键字会以不同颜色标记，提高了可读性。

## VIM的三种核心模式

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_27.png)

VIM主要包含三种模式：**命令模式**、**插入模式**和**扩展命令模式**。理解这三种模式及其切换方式是熟练使用VIM的关键。

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_29.png)

### 命令模式 (Normal Mode)

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_31.png)

启动VIM后，默认进入命令模式。在此模式下，可以执行光标移动、复制、粘贴、删除等操作，但不能直接输入文本。

**从命令模式切换到插入模式**，可以按 `i`、`a`、`o` 等键。
**从命令模式切换到扩展命令模式**，需要按 `:` 键。

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_33.png)

### 插入模式 (Insert Mode)

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_35.png)

在插入模式下，可以像使用普通文本编辑器一样输入和编辑文本。按 `Esc` 键可以退出插入模式，返回命令模式。

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_37.png)

### 扩展命令模式 (Command-Line Mode)

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_39.png)

在扩展命令模式下，可以执行保存文件、退出编辑器、查找替换等高级操作。在命令模式下按 `:` 键即可进入。

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_41.png)

## 命令模式下的基本操作

在命令模式下，我们可以高效地移动光标和编辑文本。以下是常用的操作命令。

### 光标移动

*   **基本移动**：使用方向键或 `h`(左)、`j`(下)、`k`(上)、`l`(右) 键。
*   **单词间跳转**：
    *   `w`：移动到下一个单词的词首。
    *   `e`：移动到下一个单词的词尾。
    *   `b`：移动到上一个单词的词首。
*   **行内跳转**：
    *   `0`：移动到行首。
    *   `^`：移动到行首第一个非空字符。
    *   `$`：移动到行尾。
*   **页面跳转**：
    *   `Ctrl + f`：向下翻一页。
    *   `Ctrl + b`：向上翻一页。
    *   `H`：移动到屏幕顶部。
    *   `M`：移动到屏幕中间。
    *   `L`：移动到屏幕底部。

### 文本编辑操作

掌握了光标移动，接下来我们看看如何对文本进行复制、删除、粘贴等编辑。

**以下是常用的编辑命令：**

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_43.png)

*   **复制 (Yank)**：
    *   `yy`：复制当前行。
    *   `3yy`：复制从当前行开始的三行。
    *   `yw`：复制从光标处到词尾的内容。
*   **粘贴 (Put)**：
    *   `p`：在光标后或下方粘贴。
    *   `P`：在光标前或上方粘贴。
*   **删除/剪切 (Delete)**：
    *   `dd`：删除（剪切）当前行。
    *   `3dd`：删除（剪切）从当前行开始的三行。
    *   `dw`：删除（剪切）一个单词。
    *   `x`：删除光标处的字符。
    *   `D`：删除从光标处到行尾的内容。
*   **撤销与重做**：
    *   `u`：撤销上一次操作。
    *   `Ctrl + r`：重做被撤销的操作。

## 查找、替换与高级编辑

除了基本编辑，VIM还提供了强大的查找、替换和批量修改功能。

### 查找文本

*   `/keyword`：从光标处向文件**尾部**查找 “keyword”。按 `n` 跳转到下一个匹配项，按 `N` 跳转到上一个匹配项。
*   `?keyword`：从光标处向文件**首部**查找 “keyword”。按 `n` 和 `N` 的跳转方向与 `/` 查找相反。

### 替换文本

替换命令通常在扩展命令模式下使用。

**基本替换语法：**
```
:[range]s/old/new/[flags]
```

*   `range`：指定行范围，如 `1,10` 表示第1到10行，`%` 表示全文。
*   `s`：替换命令。
*   `old`/`new`：被替换的字符串和新的字符串。
*   `flags`：标志，常用 `g` 表示行内全局替换，`c` 表示替换前确认。

**示例：**
*   `:%s/foo/bar/g`：将全文所有的 “foo” 替换为 “bar”。
*   `:10,20s/old/new/gc`：在第10到20行中，将 “old” 替换为 “new”，每次替换前需要确认。

### 进入插入模式的多种方式

在命令模式下，有多种方式可以进入插入模式，以适应不同的插入场景：

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_49.png)

*   `i`：在光标**前**插入。
*   `a`：在光标**后**插入。
*   `I`：在**行首**插入。
*   `A`：在**行尾**插入。
*   `o`：在**当前行下方**新建一行并插入。
*   `O`：在**当前行上方**新建一行并插入。

## 扩展命令模式常用操作

扩展命令模式用于执行文件管理、退出编辑器等操作。

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_51.png)

![](img/4bab907a04d4b2f7200ff0526c5fb7ad_53.png)

**常用扩展命令如下：**

*   `:w`：保存文件。
*   `:q`：退出VIM（如果文件有修改未保存，会提示错误）。
*   `:q!`：强制退出，不保存任何修改。
*   `:wq` 或 `:x`：保存并退出。
*   `:set nu`：显示行号。
*   `:set nonu`：隐藏行号。
*   `:help [command]`：查看命令的帮助文档。

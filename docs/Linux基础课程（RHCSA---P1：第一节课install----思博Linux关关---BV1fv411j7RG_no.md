## 概述

![](img/7b3a63ec2f44aba18535facbffeab65f_5.png)

Linux是一种开源的操作系统内核，由林纳斯·托瓦兹于1991年发布。它基于更早的Unix系统，但对硬件要求更低，因此在服务器和开发领域非常流行。本节课我们将重点介绍红帽企业版Linux 7的安装过程。

## Linux版本介绍

上一节我们了解了Linux的起源，本节中我们来看看Linux有哪些主要的发行版本。

Linux发行版主要分为两大分支，它们的主要区别在于软件包管理方式。

以下是基于RPM包管理器的分支：
*   **红帽企业版Linux**：商业版本，提供官方支持，例如RHEL 7和RHEL 8。
*   **CentOS**：社区企业操作系统，是RHEL的免费克隆版，与RHEL高度兼容。
*   **Fedora**：红帽的社区版，用于测试新技术，稳定后会引入RHEL。

以下是基于DEB包管理器的分支：
*   **Ubuntu**：基于Debian，以用户友好和桌面体验出色而闻名。
*   **Debian**：以稳定著称，是许多发行版（包括Ubuntu）的基础。

## 安装前的准备

在开始安装系统之前，我们需要了解一些核心概念和准备工作。

### 分区规划

分区是将硬盘划分为逻辑部分的过程。在Linux安装中，有几个关键分区是必须创建的。

以下是三个核心分区及其作用：
*   **根分区**：类似于Windows的C盘，是系统文件和目录的根目录，必须创建。
*   **/boot分区**：存放系统启动所需的引导文件和内核。通常分配300-500MB即可。
*   **交换分区**：当物理内存不足时，用于临时存放不活跃的数据。传统建议大小是物理内存的1到2倍，但可根据实际情况调整。

### 文件系统选择

文件系统决定了数据在磁盘上的组织方式。RHEL 7默认使用**XFS**文件系统，而之前的RHEL 6默认使用**EXT4**。

### 虚拟化软件选择

对于初学者，建议在虚拟机中安装Linux，以避免影响主机系统并方便练习。

以下是两种常用的虚拟机软件：
*   **VMware Workstation**：功能强大，广泛使用。
*   **VirtualBox**：免费、轻量，是Oracle提供的开源选择。

## 安装步骤详解

现在，让我们进入具体的安装流程。我们将以VMware环境为例，演示RHEL 7的安装。

![](img/7b3a63ec2f44aba18535facbffeab65f_7.png)

### 第一步：创建虚拟机

![](img/7b3a63ec2f44aba18535facbffeab65f_9.png)

![](img/7b3a63ec2f44aba18535facbffeab65f_11.png)

![](img/7b3a63ec2f44aba18535facbffeab65f_13.png)

首先，在虚拟化软件中创建一台新的虚拟机。

以下是创建虚拟机时的关键配置项：
1.  选择安装媒介：指向你下载好的RHEL 7 ISO镜像文件。
2.  选择客户机操作系统：选择“Linux”和对应的版本（如Red Hat Enterprise Linux 7 64位）。
3.  命名虚拟机并指定存储位置。
4.  指定磁盘容量：例如20GB，选择“将虚拟磁盘拆分成多个文件”。
5.  自定义硬件（重要）：
    *   **内存**：建议至少分配1GB（RHEL 8建议2GB）。
    *   **网络适配器**：选择网络连接模式。
    *   **新CD/DVD**：确认已连接到ISO镜像文件。

![](img/7b3a63ec2f44aba18535facbffeab65f_15.png)

### 第二步：配置虚拟机网络

![](img/7b3a63ec2f44aba18535facbffeab65f_17.png)

网络配置是虚拟机设置的关键环节，它决定了虚拟机如何与外界通信。

以下是三种主要的网络连接模式：
*   **桥接模式**：虚拟机的IP地址与主机在同一局域网网段，可以被局域网内其他机器访问。
*   **NAT模式**：虚拟机通过主机的IP地址共享上网，外部网络无法直接访问虚拟机。
*   **仅主机模式**：虚拟机与主机形成一个私有网络，只能与主机通信，无法访问外网。

### 第三步：启动并安装操作系统

启动虚拟机，开始安装流程。

![](img/7b3a63ec2f44aba18535facbffeab65f_19.png)

![](img/7b3a63ec2f44aba18535facbffeab65f_20.png)

以下是安装过程中的主要配置步骤：
1.  选择启动项：在引导界面选择第一项“Install Red Hat Enterprise Linux 7.0”。
2.  选择语言：建议初学者选择“中文（简体）”。
3.  安装信息摘要：
    *   **软件选择**：默认是最小安装（无图形界面）。建议初学者选择“带GUI的服务器”，以安装图形界面。
    *   **安装位置**：即磁盘分区。初学者可以选择“自动配置分区”。
    *   **网络和主机名**：可以先不配置，安装完成后再设置。
    *   **安全策略**：可暂时不配置。
4.  开始安装：点击“开始安装”按钮。
5.  设置root密码：为超级管理员用户`root`设置一个强密码。
6.  创建用户：可以创建一个普通用户（可选）。
7.  等待安装完成，然后重启系统。

## 安装后的基本设置

系统安装完成后，首次启动需要进行一些初始配置。

![](img/7b3a63ec2f44aba18535facbffeab65f_30.png)

![](img/7b3a63ec2f44aba18535facbffeab65f_32.png)

以下是首次启动的配置步骤：
1.  接受许可证协议。
2.  完成初始设置，如注册系统（可选择跳过）、配置Kdump（可选）等。
3.  使用你创建的用户或root用户登录系统。

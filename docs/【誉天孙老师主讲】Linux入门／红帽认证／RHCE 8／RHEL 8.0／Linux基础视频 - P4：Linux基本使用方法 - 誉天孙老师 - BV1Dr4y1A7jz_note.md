图形界面并非Linux内核原生提供，它需要额外的软件组件支持：
*   **X Window**：一个提供图形显示功能的底层协议框架。
*   **桌面环境**：运行在X Window之上的用户界面工具。常见的开源桌面环境有 **GNOME** 和 **KDE**。红帽企业版Linux默认使用GNOME。

### 控制台的数量与切换

![](img/5018ee814d2d3f2724457da5b28229fc_56.png)

![](img/5018ee814d2d3f2724457da5b28229fc_58.png)

Linux系统（以红帽8为例）默认提供了6个控制台：
*   1个图形控制台（通常是第一个）。
*   5个虚拟控制台（字符界面）。

![](img/5018ee814d2d3f2724457da5b28229fc_60.png)

![](img/5018ee814d2d3f2724457da5b28229fc_62.png)

![](img/5018ee814d2d3f2724457da5b28229fc_64.png)

以下是它们之间的切换方法：
*   从图形界面切换到虚拟控制台：按下 `Ctrl + Alt + F2` 到 `F6`。
*   从虚拟控制台切换回图形界面：按下 `Ctrl + Alt + F1`。
*   在虚拟控制台中启动图形界面：执行命令 `startx`。**前提**是系统已安装X Window和桌面环境。

![](img/5018ee814d2d3f2724457da5b28229fc_66.png)

![](img/5018ee814d2d3f2724457da5b28229fc_68.png)

![](img/5018ee814d2d3f2724457da5b28229fc_70.png)

![](img/5018ee814d2d3f2724457da5b28229fc_72.png)

![](img/5018ee814d2d3f2724457da5b28229fc_73.png)

**核心概念**：`startx` 命令是在当前控制台启动图形应用程序，并不会自动切换到图形控制台（`tty1`）。切换控制台必须使用 `Ctrl + Alt + Fn` 组合键。

![](img/5018ee814d2d3f2724457da5b28229fc_77.png)

![](img/5018ee814d2d3f2724457da5b28229fc_83.png)

多个控制台是相互独立的，这提高了系统的可靠性。例如，当某个控制台无响应时，可以切换到其他控制台进行故障排查或继续工作。

### 终端窗口与标签页管理

以下是一些终端内的实用操作：

*   **打开新标签页**：快捷键 `Ctrl + Shift + T`。
*   **切换标签页**：快捷键 `Alt + 1`， `Alt + 2` ...（数字对应标签页顺序）。
*   **关闭标签页**：在标签页内输入命令 `exit`，或使用快捷键 `Ctrl + D`。
*   **清屏**：输入命令 `clear`，或使用快捷键 `Ctrl + L`。
*   **调整字体大小**：放大 `Ctrl + Shift + +`；缩小 `Ctrl + -`。

![](img/5018ee814d2d3f2724457da5b28229fc_102.png)

![](img/5018ee814d2d3f2724457da5b28229fc_104.png)

**重要提示**：尽量避免直接点击终端窗口的关闭按钮（X），这可能会异常终止正在运行的任务。建议使用 `exit` 命令正常退出。

### 复制与粘贴

在图形界面的终端中：
*   复制：用鼠标选中文本，然后使用 `Ctrl + Shift + C`。
*   粘贴：使用 `Ctrl + Shift + V`。

**注意**：在Linux终端中，`Ctrl + C` 的常见功能是中断当前正在运行的命令，而不是复制。

## 执行简单的Linux命令

熟悉了操作环境后，我们来学习一些最基本的Linux命令。

### 查看系统信息

*   `date`：显示当前的系统日期和时间。
*   `cal`：显示当前月份的日历。
*   `id`：显示当前用户的身份信息（用户ID、组ID等）。也可以查询指定用户，如 `id username`，若用户不存在则会提示 `no such user`。

![](img/5018ee814d2d3f2724457da5b28229fc_110.png)

![](img/5018ee814d2d3f2724457da5b28229fc_112.png)

### 用户身份与切换

命令行提示符（如 `[root@localhost ~]#`）包含了重要信息：
*   `root`：当前用户名。
*   `localhost`：主机名。
*   `~`：当前所在目录（`~`代表用户的家目录）。
*   `#`：提示符，`#`表示当前是root用户（超级管理员），`$`表示普通用户。

切换用户的命令是 `su` (switch user)：
*   `su - username`：切换到指定用户。**注意**：`-` 符号很重要，它表示切换用户的同时加载该用户的环境配置。
*   从root切换到任何用户都**不需要**输入目标用户的密码。
*   从普通用户切换到其他用户（包括root）都**需要**输入目标用户的密码。
*   输入 `exit` 命令可以退回之前的用户身份。

### 用户管理与密码修改

![](img/5018ee814d2d3f2724457da5b28229fc_114.png)

*   `useradd username`：创建一个新用户（需要root权限）。例如：`useradd user1`。
*   `passwd`：修改当前用户的密码。
*   `passwd username`：root用户修改指定用户的密码。

![](img/5018ee814d2d3f2724457da5b28229fc_116.png)

**密码修改规则**：
1.  root用户可以为任何用户修改密码，且无需知道原密码。
2.  普通用户只能修改自己的密码，且必须输入当前密码。
3.  密码通常有复杂性要求，例如红帽系统默认要求至少8个字符。

最后，我们来学习如何在Linux中创建和编辑文本文件。这里介绍两个常用的编辑器：`nano` 和 `vim`。

![](img/5018ee814d2d3f2724457da5b28229fc_131.png)

![](img/5018ee814d2d3f2724457da5b28229fc_133.png)

### Nano编辑器

`nano` 是一个简单易用的文本编辑器，适合初学者。
*   打开/创建文件：`nano filename`
*   编辑文件：直接输入文本即可。
*   保存并退出：按 `Ctrl + X`，然后按 `Y` 确认保存，最后回车确认文件名。

### Vim编辑器

`vim` 是一个功能强大但学习曲线稍陡的编辑器，是系统管理员和开发者的常用工具。其基本使用模式如下：

**打开文件**：`vim filename`

![](img/5018ee814d2d3f2724457da5b28229fc_135.png)

![](img/5018ee814d2d3f2724457da5b28229fc_137.png)

**三种主要模式**：
1.  **正常模式 (Normal Mode)**：打开文件后默认进入此模式，可以执行命令，但不能输入文本。
2.  **插入模式 (Insert Mode)**：在此模式下可以输入和编辑文本。从正常模式按 `i` 键进入。
3.  **命令行模式 (Command-line Mode)**：在正常模式下按 `:` 进入，可以执行保存、退出等命令。

![](img/5018ee814d2d3f2724457da5b28229fc_139.png)

**基本操作流程**：
1.  `vim test.txt` （打开文件）
2.  按 `i` 键 （进入插入模式）
3.  输入文本内容 （编辑文件）
4.  按 `Esc` 键 （退回正常模式）
5.  按 `:` 键 （进入命令行模式）
6.  输入 `wq` 然后回车 （保存并退出）

**常用命令行模式命令**：
*   `:w`：保存文件。
*   `:q`：退出vim（如果文件有修改未保存，会提示错误）。
*   `:q!`：强制退出，不保存任何修改。
*   `:wq` 或 `:x`：保存并退出。

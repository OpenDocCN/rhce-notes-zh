# RHEL 9 精通课程：04-04-007：文件与目录的复制与移动 📂➡️📂

在本节课中，我们将要学习如何在 Linux 命令行中使用 `cp` 和 `mv` 命令来复制与移动文件和目录。这是管理文件系统的基础操作。

## 复制文件与目录

上一节我们介绍了基本的命令行操作，本节中我们来看看如何使用 `cp` 命令进行复制。

![](img/3d2353477a38100e06c96518a5268b32_1.png)

![](img/3d2353477a38100e06c96518a5268b32_3.png)

### 复制单个文件

`cp` 是 copy 的缩写。其基本语法是告诉命令要复制什么以及复制到哪里。

**命令格式**：
```bash
cp [源文件] [目标文件]
```

![](img/3d2353477a38100e06c96518a5268b32_5.png)

例如，将 `file1.txt` 复制为 `file2.txt`：
```bash
cp file1.txt file2.txt
```
执行后，会创建一个内容与 `file1.txt` 完全相同的 `file2.txt` 文件。

![](img/3d2353477a38100e06c96518a5268b32_7.png)

### 复制文件到目录

将文件复制到另一个目录也很简单。只需将目标指定为目录路径。

**命令格式**：
```bash
cp [文件1] [文件2] ... [目标目录]
```

例如，将 `file1.txt` 和 `file2.txt` 复制到 `destination` 文件夹：
```bash
cp file1.txt file2.txt destination/
```
最后一个参数 `destination/` 是目标目录，之前的所有参数都是要复制的文件。

![](img/3d2353477a38100e06c96518a5268b32_9.png)

### 使用通配符复制

![](img/3d2353477a38100e06c96518a5268b32_10.png)

通配符 `*` 可以匹配多个文件，这在复制时非常高效。

![](img/3d2353477a38100e06c96518a5268b32_12.png)

例如，复制 `destination` 文件夹中的所有内容到当前目录（用 `.` 表示）：
```bash
cp destination/* .
```
这里的 `.` 代表当前工作目录。`..` 则代表父目录。

![](img/3d2353477a38100e06c96518a5268b32_14.png)

![](img/3d2353477a38100e06c96518a5268b32_16.png)

### 复制整个目录（递归复制）

![](img/3d2353477a38100e06c96518a5268b32_18.png)

要复制一个目录及其内部所有内容，需要使用 `-R`（递归）选项。

**命令格式**：
```bash
cp -R [源目录] [目标路径]
```

![](img/3d2353477a38100e06c96518a5268b32_20.png)

例如，将 `copy_me` 文件夹完整地复制到 `destination` 文件夹中：
```bash
cp -R copy_me/ destination/
```
执行后，`destination` 文件夹内会包含 `copy_me` 文件夹及其所有文件。

![](img/3d2353477a38100e06c96518a5268b32_22.png)

## 移动与重命名文件与目录

我们已经学会了复制，现在来看看如何使用 `mv` 命令进行移动和重命名。

### 重命名文件

![](img/3d2353477a38100e06c96518a5268b32_24.png)

![](img/3d2353477a38100e06c96518a5268b32_26.png)

`mv` 命令可以用于重命名文件。其逻辑是“移动”文件到同一个位置但使用新名字。

![](img/3d2353477a38100e06c96518a5268b32_28.png)

**命令格式**：
```bash
mv [旧文件名] [新文件名]
```

![](img/3d2353477a38100e06c96518a5268b32_30.png)

例如，将 `old_name.txt` 重命名为 `new_name.txt`：
```bash
mv old_name.txt new_name.txt
```

### 重命名目录

重命名目录的操作与重命名文件完全相同。

![](img/3d2353477a38100e06c96518a5268b32_32.png)

例如，将 `old_folder` 重命名为 `new_folder`：
```bash
mv old_folder new_folder
```
即使目录内有文件，重命名操作也不会影响其内容。

![](img/3d2353477a38100e06c96518a5268b32_34.png)

### 移动文件到目录

![](img/3d2353477a38100e06c96518a5268b32_36.png)

![](img/3d2353477a38100e06c96518a5268b32_38.png)

`mv` 命令的核心功能是移动。其语法与 `cp` 类似，先指定要移动的内容，最后指定目标目录。

**命令格式**：
```bash
mv [要移动的文件或目录] [目标目录]
```

![](img/3d2353477a38100e06c96518a5268b32_40.png)

例如，将 `new_folder` 目录中的所有文件移动到当前桌面目录：
```bash
mv new_folder/* .
```
移动操作完成后，源文件夹 `new_folder` 将变为空。

![](img/3d2353477a38100e06c96518a5268b32_42.png)

### 移动并重命名目录

`mv` 命令的强大之处在于可以一次性完成移动和重命名。

例如，将 `Documents` 文件夹中的 `new_folder` 移动到桌面，并同时将其重命名为 `jackpot`：
```bash
mv ~/Documents/new_folder/ ./jackpot
```
这个命令执行了两个操作：一是移动，二是将移动后的文件夹命名为 `jackpot`。

![](img/3d2353477a38100e06c96518a5268b32_44.png)

![](img/3d2353477a38100e06c96518a5268b32_46.png)

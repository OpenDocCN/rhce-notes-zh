以下是Linux系统入门必备的几个核心概念：

![](img/57a1e7122377efe71d1309de5263c053_49.png)

![](img/57a1e7122377efe71d1309de5263c053_51.png)

![](img/57a1e7122377efe71d1309de5263c053_53.png)

![](img/57a1e7122377efe71d1309de5263c053_55.png)

![](img/57a1e7122377efe71d1309de5263c053_57.png)

*   **多用户系统**：允许同时有多个用户登录并使用同一台计算机的资源。系统理论上最多支持65535个用户。
*   **多任务系统**：允许同时运行多个程序或进程，例如可以在一台服务器上同时运行网站、数据库和监控服务。
*   **严格区分大小写**：系统对文件名、目录名、命令、选项和参数都区分大小写。例如，文件 `file.txt` 和 `File.txt` 是两个不同的文件；命令 `ls` 和 `LS` 也是不同的。
*   **一切皆文件**：这是Linux的一个重要哲学。硬件设备（如CPU、内存、硬盘）、目录、普通文件等，在系统中都以文件的形式存在和管理。
*   **树形目录结构**：整个文件系统以一个倒挂的树形结构组织，起点称为 **根目录**，用单个正斜杠 `/` 表示。所有文件和目录都存在于根目录之下。
*   **无扩展名依赖**：Linux不依赖文件扩展名（如 `.txt`, `.exe`）来识别文件类型或决定如何打开它。文件类型由文件内容本身决定。用户看到的扩展名（如 `.conf`, `.sh`）主要是为了方便人类识别。
*   **无提示即成功**：大多数命令执行成功后，不会返回任何提示信息。命令行没有报错，通常就意味着操作成功。这与Windows的操作习惯不同。
*   **无回收站机制**：Linux系统默认没有类似Windows的回收站。使用删除命令（如 `rm`）删除文件后，通常无法直接恢复，因此操作需谨慎。

理解了系统的基本概念后，我们来看看与系统交互的主要界面——命令终端。屏幕上闪烁的光标所在的行，就是命令终端，我们绝大部分操作都在这里通过键盘输入命令完成。

![](img/57a1e7122377efe71d1309de5263c053_85.png)

![](img/57a1e7122377efe71d1309de5263c053_87.png)

![](img/57a1e7122377efe71d1309de5263c053_89.png)

![](img/57a1e7122377efe71d1309de5263c053_91.png)

一个典型的命令提示符如下所示：
```
[root@localhost ~]#
```
或
```
[test@localhost ~]$
```
以下是各字段的含义解析：

*   **`root` / `test`**：**当前登录的用户名**。`root` 是超级管理员账号，拥有系统最高权限；其他名称（如 `test`）均为普通用户账号，权限较小。
*   **`@`**：分隔符，用于分隔用户名和主机名。
*   **`localhost`**：**当前主机名**。系统安装后的默认主机名，后期可以修改。
*   **`~`**：**当前用户所在的目录**。波浪线 `~` 是一个特殊符号，代表当前用户的 **家目录**。用户首次登录系统时，默认进入自己的家目录。
    *   超级管理员 `root` 的家目录是 `/root`。
    *   普通用户（如 `test`）的家目录通常在 `/home/` 目录下，例如 `/home/test`。
*   **`#` 或 `$`**：**命令提示符结尾符**，用于标识当前用户的身份。
    *   `#`：表示当前用户是 **超级管理员（root）**。
    *   `$`：表示当前用户是 **普通用户**。

命令行的一般格式如下：
```
命令字 [选项]... [参数]...
```
*   **命令字**：即命令本身，是实现特定功能的指令。例如，`ls` 命令用于列出目录内容。
*   **[选项]**：用于调整命令的功能。选项是可选的（`[]`表示可选），可以同时使用多个（`...`表示多个）。
    *   **短选项**：通常以单个 `-` 开头，后接一个字符，如 `-l`。多个短选项可以合并，例如 `-l -h` 可以写成 `-lh`。
    *   **长选项**：通常以 `--` 开头，后接一个单词，如 `--help`。长选项不能合并。
*   **[参数]**：命令的操作对象，可以是文件、目录等。参数也是可选的，并且可以指定多个。

![](img/57a1e7122377efe71d1309de5263c053_130.png)

![](img/57a1e7122377efe71d1309de5263c053_132.png)

让我们通过 `ls` 命令来具体理解这个格式：

*   **基本功能**：输入 `ls` 并回车，会列出当前目录下的文件和目录。这类似于在Windows中双击打开一个文件夹。
    ```bash
    ls
    ```
*   **使用选项**：`-l` 选项可以以长格式显示详细信息（包括权限、所有者、大小、修改时间等）。
    ```bash
    ls -l
    ```
*   **合并短选项**：`-h` 选项可以以人类易读的格式（如K, M, G）显示文件大小。我们可以将 `-l` 和 `-h` 合并使用。
    ```bash
    ls -lh
    ```
*   **指定参数**：在命令后加上路径作为参数，可以查看指定目录的内容。
    ```bash
    ls -lh /opt
    ```

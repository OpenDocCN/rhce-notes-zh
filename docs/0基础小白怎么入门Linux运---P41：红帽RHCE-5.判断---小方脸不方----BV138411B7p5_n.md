## 文件状态判断 🔍

![](img/a7a2b36efca958849b19e21a8197d485_7.png)

![](img/a7a2b36efca958849b19e21a8197d485_9.png)

上一节我们介绍了Shell的基本概念，本节中我们来看看如何判断文件的状态。Shell提供了一系列单字母选项，用于快速检查文件或目录的属性。

![](img/a7a2b36efca958849b19e21a8197d485_11.png)

![](img/a7a2b36efca958849b19e21a8197d485_13.png)

![](img/a7a2b36efca958849b19e21a8197d485_15.png)

以下是常用的文件状态判断选项及其用法：

*   **`-e`**：判断文件或目录是否存在。
    ```bash
    [ -e /etc/passwd ] && echo "文件存在"
    ```
*   **`-d`**：判断指定路径是否为目录。
    ```bash
    [ -d /etc ] && echo "/etc 是一个目录"
    ```
*   **`-f`**：判断指定路径是否为普通文件。
    ```bash
    [ -f /etc/passwd ] && echo "/etc/passwd 是一个文件"
    ```
*   **`-r`**：判断当前用户对文件是否具有读权限。
    ```bash
    [ -r /etc/passwd ] && echo "可读"
    ```
*   **`-w`**：判断当前用户对文件是否具有写权限。
    ```bash
    [ -w /etc/passwd ] && echo "可写"
    ```
*   **`-x`**：判断当前用户对文件是否具有执行权限。
    ```bash
    [ -x /bin/ls ] && echo "可执行"
    ```

**注意**：使用这些判断时，必须将表达式放在方括号 `[ ]` 内，并且括号内各部分（包括括号本身）都需要用空格隔开。判断命令本身不输出结果，需要通过 `$?` 获取上一条命令的退出状态码（0表示成功/真，非0表示失败/假）。

## 整数比较 🔢

![](img/a7a2b36efca958849b19e21a8197d485_17.png)

![](img/a7a2b36efca958849b19e21a8197d485_19.png)

了解了文件判断后，我们进入数值比较领域。Shell中对整数进行比较需要使用特定的运算符，不能直接使用数学符号。

![](img/a7a2b36efca958849b19e21a8197d485_21.png)

以下是整数比较的运算符：

![](img/a7a2b36efca958849b19e21a8197d485_23.png)

*   **`-eq`**：等于（Equal）。
    ```bash
    [ 1 -eq 1 ] && echo "1等于1"
    ```
*   **`-ne`**：不等于（Not Equal）。
    ```bash
    [ 2 -ne 1 ] && echo "2不等于1"
    ```
*   **`-gt`**：大于（Greater Than）。
    ```bash
    [ 2 -gt 1 ] && echo "2大于1"
    ```
*   **`-ge`**：大于等于（Greater or Equal）。
    ```bash
    [ 1 -ge 1 ] && echo "1大于等于1"
    ```
*   **`-lt`**：小于（Less Than）。
    ```bash
    [ 1 -lt 2 ] && echo "1小于2"
    ```
*   **`-le`**：小于等于（Less or Equal）。
    ```bash
    [ 1 -le 1 ] && echo "1小于等于1"
    ```

## 字符串对比 📝

![](img/a7a2b36efca958849b19e21a8197d485_29.png)

![](img/a7a2b36efca958849b19e21a8197d485_31.png)

![](img/a7a2b36efca958849b19e21a8197d485_33.png)

与整数比较不同，字符串对比使用不同的运算符。它主要用于比较文本内容是否相同。

以下是字符串对比的常用方法：

![](img/a7a2b36efca958849b19e21a8197d485_35.png)

![](img/a7a2b36efca958849b19e21a8197d485_37.png)

*   **`=` 或 `==`**：判断两个字符串是否相等。
    ```bash
    [ "$USER" = "root" ] && echo "当前用户是root"
    ```
*   **`!=`**：判断两个字符串是否不相等。
    ```bash
    [ "$USER" != "nobody" ] && echo "当前用户不是nobody"
    ```
*   **`-z`**：判断字符串长度是否为零（即字符串为空）。
    ```bash
    [ -z "$var" ] && echo "变量var是空的"
    ```
*   **`-n`**：判断字符串长度是否非零（即字符串非空）。
    ```bash
    [ -n "$USER" ] && echo "USER变量非空"
    ```

![](img/a7a2b36efca958849b19e21a8197d485_39.png)

![](img/a7a2b36efca958849b19e21a8197d485_41.png)

![](img/a7a2b36efca958849b19e21a8197d485_43.png)

![](img/a7a2b36efca958849b19e21a8197d485_45.png)

**注意**：使用变量进行字符串比较时，建议将变量用双引号引起来，以防止变量值为空时导致语法错误。

## 常用数值运算方式 🧮

![](img/a7a2b36efca958849b19e21a8197d485_47.png)

![](img/a7a2b36efca958849b19e21a8197d485_49.png)

![](img/a7a2b36efca958849b19e21a8197d485_51.png)

![](img/a7a2b36efca958849b19e21a8197d485_53.png)

在脚本中，我们经常需要进行数学运算。Shell提供了多种数值运算的方法。

以下是几种常用的数值运算方式：

![](img/a7a2b36efca958849b19e21a8197d485_55.png)

![](img/a7a2b36efca958849b19e21a8197d485_57.png)

![](img/a7a2b36efca958849b19e21a8197d485_59.png)

1.  **`$[ ]` 方式**：这是较简洁的写法。
    ```bash
    echo $[1 + 1]   # 输出 2
    echo $[5 * 2]   # 输出 10
    echo $[10 / 3]  # 输出 3 (整数除法)
    echo $[10 % 3]  # 输出 1 (取余)
    ```

![](img/a7a2b36efca958849b19e21a8197d485_61.png)

![](img/a7a2b36efca958849b19e21a8197d485_63.png)

2.  **`$(( ))` 方式**：这是C语言风格的写法，功能与 `$[ ]` 类似。
    ```bash
    echo $((1 + 1))   # 输出 2
    echo $((5 * 2))   # 输出 10
    ```

![](img/a7a2b36efca958849b19e21a8197d485_65.png)

3.  **`expr` 命令**：一个外部命令，使用时需要注意空格和特殊符号的转义。
    ```bash
    expr 1 + 1        # 输出 2
    expr 5 \* 2       # 输出 10，乘号*需要转义
    expr 10 / 3       # 输出 3
    expr 10 % 3       # 输出 1
    ```

4.  **`let` 命令**：常用于脚本中，将运算结果赋值给变量，并支持 `++`、`+=` 等简洁写法。
    ```bash
    let a=1+1
    echo $a           # 输出 2

    let i=5
    let i++           # i 自增1，等价于 i=i+1
    echo $i           # 输出 6

    let i+=2          # i 增加2，等价于 i=i+2
    echo $i           # 输出 8

    let i*=2          # i 乘以2，等价于 i=i*2
    echo $i           # 输出 16
    ```

## 逻辑运算符连接命令 ⛓️

![](img/a7a2b36efca958849b19e21a8197d485_69.png)

![](img/a7a2b36efca958849b19e21a8197d485_71.png)

![](img/a7a2b36efca958849b19e21a8197d485_73.png)

![](img/a7a2b36efca958849b19e21a8197d485_75.png)

最后，我们学习如何将多个命令通过逻辑关系组合起来，实现简单的条件执行。这是在命令行中实现“如果...就...”逻辑的快捷方式。

以下是三种连接命令的方式：

![](img/a7a2b36efca958849b19e21a8197d485_77.png)

*   **`&&`（逻辑与）**：只有前一个命令执行成功（返回状态码为0），后一个命令才会执行。
    ```bash
    # 如果/etc/passwd文件存在，则查看它
    [ -f /etc/passwd ] && cat /etc/passwd

    # 查询软件包，如果存在则启动服务
    rpm -q vsftpd && systemctl start vsftpd
    ```

![](img/a7a2b36efca958849b19e21a8197d485_79.png)

*   **`||`（逻辑或）**：只有前一个命令执行失败（返回状态码非0），后一个命令才会执行。
    ```bash
    # 如果软件包不存在，则安装它
    rpm -q vsftpd || yum install -y vsftpd
    ```

*   **`;`（分号）**：顺序执行多个命令，无论前一个命令成功与否，后一个命令都会执行。命令之间没有逻辑依赖关系。
    ```bash
    # 依次执行三条命令：创建文件、查询包、列出目录
    touch /tmp/test.txt; rpm -q vsftpd; ls /root
    ```

![](img/a7a2b36efca958849b19e21a8197d485_81.png)

![](img/a7a2b36efca958849b19e21a8197d485_83.png)

这些运算符可以组合使用，构建更复杂的逻辑：
```bash
# 如果软件包vsftpd不存在，则安装它，安装成功后启动并设置开机自启
rpm -q vsftpd || yum install -y vsftpd && systemctl start vsftpd && systemctl enable vsftpd
```

![](img/a7a2b36efca958849b19e21a8197d485_85.png)

**逻辑解析**：`||` 优先级高于 `&&`。所以先判断 `rpm -q vsftpd`，如果失败（包不存在），则执行 `yum install -y vsftpd`。安装命令若成功，则会继续执行后面的 `&&` 连接的命令（启动和启用服务）。

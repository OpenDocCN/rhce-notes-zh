**多用户系统**
Linux允许许多用户同时登录并使用同一台计算机的资源。在企业早期，一台服务器可以连接多个终端，供不同用户使用各自的账号进行操作。

![](img/ba203c0253333edda2f04c0a402928e1_29.png)

![](img/ba203c0253333edda2f04c0a402928e1_31.png)

![](img/ba203c0253333edda2f04c0a402928e1_33.png)

![](img/ba203c0253333edda2f04c0a402928e1_35.png)

**多任务系统**
系统可以同时运行多个程序或进程。就像在Windows中你可以一边听歌一边浏览网页，Linux服务器也能同时运行网站、数据库、监控软件等多种服务。

**严格区分大小写**
在Linux中，文件名、目录名、命令及其选项和参数都严格区分大小写。例如，文件 `file.txt` 和 `FILE.TXT` 是两个完全不同的文件。

**一切皆文件**
这是Linux的一个重要哲学。硬件设备（如CPU、内存、硬盘）、目录、普通文件等，在系统中都以文件的形式存在和管理。例如，查看CPU信息就是读取 `/proc/cpuinfo` 这个特殊的文件。

![](img/ba203c0253333edda2f04c0a402928e1_49.png)

![](img/ba203c0253333edda2f04c0a402928e1_51.png)

**树形目录结构**
所有文件和目录都以一个倒置的树形结构组织，这个结构的起点称为 **根目录**，用单个正斜杠 `/` 表示。删除根目录意味着摧毁整个系统。

![](img/ba203c0253333edda2f04c0a402928e1_53.png)

![](img/ba203c0253333edda2f04c0a402928e1_55.png)

![](img/ba203c0253333edda2f04c0a402928e1_57.png)

**无扩展名依赖**
Linux不依赖文件扩展名（如 `.txt`, `.exe`）来识别文件类型或决定如何打开它。系统通过文件内容头部的元数据来识别类型。我们看到的扩展名（如 `.conf`, `.sh`）主要是为了方便用户识别。

![](img/ba203c0253333edda2f04c0a402928e1_59.png)

![](img/ba203c0253333edda2f04c0a402928e1_61.png)

![](img/ba203c0253333edda2f04c0a402928e1_63.png)

![](img/ba203c0253333edda2f04c0a402928e1_65.png)

**无回收站机制**
Linux系统默认没有类似Windows的回收站。使用 `rm` 等命令删除文件后，通常无法直接恢复，因此执行删除操作时需要格外谨慎。

**无提示即成功**
大多数Linux命令执行成功后不会给出“操作成功”的提示。如果命令执行后没有返回错误信息，通常意味着它已成功执行。报错时，系统会明确给出错误提示。

## 命令终端字段含义介绍

![](img/ba203c0253333edda2f04c0a402928e1_87.png)

![](img/ba203c0253333edda2f04c0a402928e1_89.png)

![](img/ba203c0253333edda2f04c0a402928e1_91.png)

![](img/ba203c0253333edda2f04c0a402928e1_93.png)

理解了系统的基本概念后，我们来看看与系统交互的主要界面——命令行终端。终端是输入和执行命令的地方，在Linux运维中，键盘远比鼠标重要。

以下是终端提示符 `[root@localhost ~]#` 各字段的含义解析：

*   **`root`**：**当前登录的用户名**。这里显示的是超级管理员账号 `root`。
*   **`@`**：分隔符，用于分隔用户名和主机名。
*   **`localhost`**：**当前系统的主机名**。安装后默认为 `localhost`，后期可以根据用途修改（如 `web-server`, `db-server`）。
*   **`~`**：**当前用户所在的目录**。波浪号 `~` 是用户**家目录**的简写。用户首次登录系统时，默认进入自己的家目录。
*   **`#`** 或 **`$`**：**提示符结尾**，代表当前用户的身份。
    *   `#`：表示当前用户是**超级管理员 (root)**，拥有系统最高权限。
    *   `$`：表示当前用户是**普通用户**，权限受到限制。

**关于家目录：**
*   超级管理员 `root` 的家目录是 `/root`。
*   普通用户（如 `test`）的家目录通常位于 `/home/` 目录下，并以用户名命名，例如 `/home/test`。

现在，我们开始学习如何向系统发出指令。Linux命令遵循一个通用格式，理解这个格式是高效使用命令行的基础。

命令行的一般格式如下：
```
命令字 [选项] [参数]
```

![](img/ba203c0253333edda2f04c0a402928e1_116.png)

![](img/ba203c0253333edda2f04c0a402928e1_118.png)

![](img/ba203c0253333edda2f04c0a402928e1_120.png)

![](img/ba203c0253333edda2f04c0a402928e1_122.png)

![](img/ba203c0253333edda2f04c0a402928e1_124.png)

**命令字**
即命令本身，是实现特定功能的指令（如创建文件、查看目录等）。Linux有数千条命令，入门阶段我们只需掌握最常用的几十条。

![](img/ba203c0253333edda2f04c0a402928e1_126.png)

![](img/ba203c0253333edda2f04c0a402928e1_128.png)

![](img/ba203c0253333edda2f04c0a402928e1_130.png)

**选项**
用于调整或扩展命令的功能。选项通常以 `-` 或 `--` 开头。
*   **短选项**：以单个 `-` 开头，后接单个字符（如 `-l`, `-a`）。多个短选项可以合并，例如 `-l -h` 可以写成 `-lh`。
*   **长选项**：以 `--` 开头，后接一个单词（如 `--help`, `--version`）。长选项不能合并书写。

![](img/ba203c0253333edda2f04c0a402928e1_132.png)

**参数**
命令的操作对象，可以是文件、目录或程序等。

**格式详解：**
*   `[]`：方括号内的内容表示是**可选的**。你可以根据需要使用或不使用选项和参数。
*   `...`：省略号表示前面的项目（选项或参数）可以同时有**多个**。

**示例分析：**
以 `ls` 命令为例，它用于列出目录内容。
*   `ls`：仅使用命令字，列出当前目录下的文件和目录名。
*   `ls -l`：使用命令字和短选项 `-l`，以长格式列出详细信息（权限、所有者、大小、修改时间等）。
*   `ls -lh /etc`：使用命令字、合并的短选项 `-lh`（`-l` 长格式，`-h` 人性化显示文件大小）和参数 `/etc`，以易读格式列出 `/etc` 目录的详细信息。
*   `ls --help`：使用命令字和长选项 `--help`，查看 `ls` 命令的帮助信息。

**注意事项：**
命令字、选项、参数之间**必须用空格分隔**。系统通过空格来区分它们。

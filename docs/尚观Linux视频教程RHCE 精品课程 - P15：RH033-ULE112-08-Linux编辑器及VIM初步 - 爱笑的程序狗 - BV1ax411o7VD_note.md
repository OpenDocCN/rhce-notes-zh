在本节课中，我们将要学习Linux系统中编辑器的使用，特别是VIM编辑器的基本操作。VIM是Linux环境下功能强大且高效的文本编辑器，掌握其核心用法将极大提升你的工作效率。

## 概述

Linux系统下存在多种编辑器，例如古老的ED、流编辑器SED以及可视化的VI/VIM。目前，VIM（VI Improved）是使用最广泛的可视化编辑器。它功能强大，但初学者需要理解其独特的操作模式才能熟练使用。

## 编辑器的发展历程

![](img/2ed0a1fa23b8a10466463f512bb5a60f_8.png)

最早的Linux编辑器是ED，它是一个非可视化的行编辑器，至今仍存在于最基本的系统中作为备用。后来发展出了流编辑器SED，它能够以非交互的方式批量处理文本流。例如，使用SED进行全局替换的命令是：
```bash
sed 's/old/new/g' filename
```
但SED不直接修改原文件，输出需重定向。

为了进行更直观的交互式编辑，出现了可视化的VI编辑器。而VIM是VI的增强版（VI Improved），提供了更多便捷功能。在大多数现代Linux发行版中，`vi`命令实际上是`vim`的别名。

## VIM的核心：三种模式 🔄

VIM操作的核心在于理解其三种工作模式：**命令模式**、**插入模式**和**命令行模式**。模式之间的切换是流畅使用VIM的关键。

*   **命令模式**：启动VIM后的默认模式。在此模式下，键盘输入被解释为命令，用于移动光标、删除、复制等，而不是输入文本。
*   **插入模式**：在此模式下，键盘输入的内容会作为文本插入到文件中。从命令模式可以按多个键进入此模式。
*   **命令行模式**：在命令模式下按冒号 `:` 进入。在此可以执行保存、退出、查找替换等扩展命令。

模式转换关系如下：
**命令模式** <--> (按 `i`, `a`, `o` 等键) <--> **插入模式**
**命令模式** <--> (按 `:` 键) <--> **命令行模式**
从**插入模式**或**命令行模式**返回**命令模式**，均需按 `Esc` 键。

![](img/2ed0a1fa23b8a10466463f512bb5a60f_10.png)

## 从命令模式进入插入模式

为了提高编辑效率，VIM提供了多个从命令模式进入插入模式的快捷键。以下是常用的几个：

*   `i`：在光标当前位置进入插入模式。
*   `a`：在光标下一个字符处进入插入模式。
*   `I`（大写）：移动光标到本行行首，并进入插入模式。
*   `A`（大写）：移动光标到本行行尾，并进入插入模式。
*   `o`：在当前行下方新开一行，并进入插入模式。
*   `O`（大写）：在当前行上方新开一行，并进入插入模式。

记住其中几个常用键（如 `i`, `a`, `o`）就能显著提升编辑速度。

## 命令模式下的常用操作

![](img/2ed0a1fa23b8a10466463f512bb5a60f_12.png)

在命令模式下，有大量快捷键用于快速编辑。以下是一些核心操作：

![](img/2ed0a1fa23b8a10466463f512bb5a60f_14.png)

### 光标移动
*   `0`：移动到本行行首。
*   `$`：移动到本行行尾。
*   `gg`：移动到文件第一行。
*   `G`：移动到文件最后一行。
*   `15G`：移动到第15行。
*   `w`：向后移动一个单词。
*   `b`：向前移动一个单词。
*   `Ctrl+g`：显示当前行信息和文件状态。

### 文本编辑
*   `x`：删除光标处的字符。
*   `dd`：删除（剪切）当前行。
*   `yy`：复制当前行。
*   `p`：在光标下方粘贴。
*   `u`：撤销上一次操作。
*   `Ctrl+r`：重做上一次撤销的操作。
*   `r`：替换光标处的一个字符（按 `r` 后再按新字符）。
*   `R`：进入替换模式，持续替换字符直到按 `Esc`。

![](img/2ed0a1fa23b8a10466463f512bb5a60f_16.png)

### 结合数字执行多次操作
在命令前加数字可以重复该操作，这是VIM的一个强大特性。
*   `5dd`：删除5行。
*   `3yy`：复制3行。
*   `10i`：进入插入模式，并随后输入的字符会重复10次（输入完按 `Esc`）。
*   `4x`：删除4个字符。

![](img/2ed0a1fa23b8a10466463f512bb5a60f_18.png)

![](img/2ed0a1fa23b8a10466463f512bb5a60f_19.png)

![](img/2ed0a1fa23b8a10466463f512bb5a60f_21.png)

### 查找与剪贴板
*   `/keyword`：向下查找关键词。
*   `?keyword`：向上查找关键词。
*   `n`：查找下一个匹配项。
*   `"ayy`：将当前行复制到名为 `a` 的寄存器（剪贴板）。
*   `"ap`：将寄存器 `a` 中的内容粘贴出来。

## 命令行模式下的重要命令

![](img/2ed0a1fa23b8a10466463f512bb5a60f_23.png)

![](img/2ed0a1fa23b8a10466463f512bb5a60f_25.png)

![](img/2ed0a1fa23b8a10466463f512bb5a60f_26.png)

在命令模式下按 `:` 进入命令行模式，可以执行文件操作和设置命令。

![](img/2ed0a1fa23b8a10466463f512bb5a60f_28.png)

![](img/2ed0a1fa23b8a10466463f512bb5a60f_30.png)

### 文件操作
*   `:w`：保存文件。
*   `:w filename`：另存为 `filename`。
*   `:w!`：强制保存（用于只读文件等情况）。
*   `:q`：退出VIM。
*   `:q!`：强制退出，不保存更改。
*   `:wq` 或 `:x`：保存并退出。
*   `:e filename`：编辑另一个文件。
*   `:e!`：放弃所有修改，重新加载当前文件。

![](img/2ed0a1fa23b8a10466463f512bb5a60f_32.png)

### 读入内容与执行命令
*   `:r filename`：将文件 `filename` 的内容读入到光标下方。
*   `:r !command`：将系统命令 `command` 的执行结果读入到光标下方。例如 `:r !ls`。

![](img/2ed0a1fa23b8a10466463f512bb5a60f_34.png)

![](img/2ed0a1fa23b8a10466463f512bb5a60f_35.png)

![](img/2ed0a1fa23b8a10466463f512bb5a60f_37.png)

### 全局查找与替换
替换命令格式为 `:[range]s/old/new/[flags]`。
*   `:%s/old/new/g`：将整个文件中所有的 `old` 替换为 `new`。
*   `:10,20s/old/new/g`：将第10行到第20行间的 `old` 替换为 `new`。
*   `:s/old/new/gc`：替换时每次询问确认（`c` 表示 confirm）。

![](img/2ed0a1fa23b8a10466463f512bb5a60f_39.png)

### 显示设置
*   `:set nu` 或 `:set number`：显示行号。
*   `:set nonu`：取消显示行号。
*   `:set ruler`：显示光标位置状态栏。

![](img/2ed0a1fa23b8a10466463f512bb5a60f_41.png)

![](img/2ed0a1fa23b8a10466463f512bb5a60f_43.png)

若希望VIM每次启动都自动应用某些设置（如显示行号），可以将命令写入用户主目录下的配置文件 `~/.vimrc` 中。例如，添加一行 `set nu`。

*   **分屏编辑**：使用 `vim -o file1 file2` 水平分屏打开多个文件，或 `vim -O file1 file2` 垂直分屏。在VIM内使用 `Ctrl+w` 然后按方向键在窗口间切换。
*   **执行Shell命令**：在命令行模式下输入 `:!command`（如 `:!ls`）可以暂时执行一个Shell命令。
*   **切换到Shell**：输入 `:sh` 可以临时切换到Shell环境，输入 `exit` 返回VIM。

## 磁盘的物理结构与类型

上一节我们介绍了课程概述，本节中我们来看看磁盘的物理结构。在企业环境中，如果使用云主机，通常无法直接看到物理硬盘的外观。在传统的物理服务器上，硬盘的外观类似于一块板砖，可以插入服务器的专用硬盘插口中。一个服务器可能有多个硬盘插口，例如12个或16个。

许多现代硬盘支持热插拔功能，类似于U盘，可以在系统运行时插入或移除，而无需关机。

硬盘的内部结构主要包括：
*   **主轴马达**：用于带动盘片旋转。
*   **盘片**：数据存储的介质，类似于光盘。
*   **机械手臂与读写磁头**：用于在旋转的盘片上读取和写入数据。

硬盘内部是真空的，一旦拆开进入空气或灰尘，可能导致盘片损坏和数据丢失。

## 磁盘的接口类型

了解了物理结构后，我们来看看磁盘的不同接口类型。服务器硬盘有不同的接口型号，必须与服务器插口匹配。

以下是常见的磁盘接口类型演进：
*   **IDE**：早期接口，数据传输速度慢，价格便宜，约在2009年后在服务器领域淘汰，曾用于个人计算机。
*   **SCSI**：在2013年至2017年间在服务器领域流行，数据传输速度快，并支持热插拔。
*   **SAS**：目前在服务器领域比较流行的接口。
*   **SATA**：目前主要用于个人计算机领域，服务器偶尔使用，数据传输速度快。
*   **SSD（固态硬盘）**：利用类似内存的机制存储数据，速度远快于传统机械硬盘，但价格较高。
*   **NVMe**：一种协议，通常用于固态硬盘，能提供极高的数据传输速度。

选择硬盘时，接口类型决定了兼容性和性能。通常，价格更高的硬件能提供更好的性能。

## 磁盘分区格式

在硬盘投入使用前，必须进行分区。分区格式决定了如何组织和管理硬盘空间。主要有两种分区格式：MBR和GPT。

### MBR分区格式

MBR是一种较古老的分区格式。它的主要特点如下：
*   **分区数量限制**：一块硬盘最多只能划分**4个主分区**。
*   **扩展分区解决方案**：为了突破4个分区的限制，可以将第4个分区创建为**扩展分区**。扩展分区本身不直接存储数据，而是作为一个容器，在其内部可以划分多个**逻辑分区**。
*   **容量限制**：MBR分区格式最大只支持**2.2TB**以内的硬盘容量。

以下是逻辑分区的数量限制：
*   IDE接口磁盘：最多59个逻辑分区。
*   SCSI等接口磁盘：最多11个逻辑分区。

*   **容量支持**：最大支持**18EB**的磁盘容量，这是一个极其巨大的存储空间。

### `lsblk` 命令
此命令用于列出系统中所有块设备（如硬盘）的信息，包括硬盘数量和每个硬盘的分区情况。
```bash
lsblk
```
常用选项：
*   `-d`：仅显示磁盘本身信息，不显示分区。
    ```bash
    lsblk -d /dev/sdb
    ```
*   `-f`：显示文件系统类型、UUID等详细信息。
    ```bash
    lsblk -f /dev/sda1
    ```

![](img/78003638b97760b20b360a9b04fa70f7_74.png)

![](img/78003638b97760b20b360a9b04fa70f7_76.png)

### `df` 命令
此命令用于报告文件系统的磁盘空间使用情况，显示已挂载分区的容量、已用空间、可用空间及使用百分比。
```bash
df -h
```
常用选项：
*   `-h`：以人类易读的格式（如K, M, G）显示大小。
*   `-T`：显示文件系统类型。

### `du` 命令
此命令用于估算文件和目录的磁盘使用空间。
```bash
du -sh /etc
```
常用选项：
*   `-s`：仅显示总计，不列出每个子目录的详情。
*   `-h`：以人类易读的格式显示大小。

![](img/78003638b97760b20b360a9b04fa70f7_90.png)

![](img/78003638b97760b20b360a9b04fa70f7_92.png)

![](img/78003638b97760b20b360a9b04fa70f7_94.png)

**注意**：`ls -lh`命令在统计目录时，只显示目录条目本身的大小，不统计目录内文件内容的总和；而`du`命令统计的是目录下所有文件实际占用的空间总和。

![](img/78003638b97760b20b360a9b04fa70f7_96.png)

### `blkid` 命令
此命令用于显示块设备的属性，如UUID和文件系统类型。
```bash
blkid /dev/sda1
```
其功能与`lsblk -f`类似，但默认不显示挂载点信息。

## 命令总结与回顾

![](img/78003638b97760b20b360a9b04fa70f7_110.png)

本节课中我们一起学习了磁盘管理的核心概念和常用命令。我们来简单回顾一下：
*   想查看系统中有多少块硬盘以及每个硬盘的分区情况，使用 **`lsblk`** 命令。
*   想查看各个已使用分区的空间使用率，使用 **`df -h`** 命令。
*   想统计具体文件或目录的实际磁盘占用大小，使用 **`du -sh`** 命令。
*   `blkid`命令可用于查看设备属性，但`lsblk -f`通常能提供更全面的信息。

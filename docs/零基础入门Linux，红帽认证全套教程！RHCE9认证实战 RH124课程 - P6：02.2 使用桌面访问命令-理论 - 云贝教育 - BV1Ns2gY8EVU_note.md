# Linux入门与红帽认证：P6：02.2 使用桌面访问命令-理论

在本节课中，我们将要学习Linux的图形化桌面环境，了解其与Windows桌面的区别，认识其核心组件，并掌握在桌面环境中访问命令行终端的基本方法。

## Linux桌面概述

上一节我们介绍了命令行基础，本节中我们来看看Linux的图形化桌面。首先需要明确，Linux中的桌面环境是一个独立的大型软件，并非操作系统内核的必需部分。这与Windows系统不同，Windows的桌面是操作系统核心的一部分。

![](img/337bf18e3c620c348a11c5fef8f5b270_1.png)

在Linux中，桌面环境类似于Windows中安装的独立应用程序（如微信客户端），可以选择安装或不安装。通常，为了提高性能和节约资源，Linux服务器不会安装图形化界面。

![](img/337bf18e3c620c348a11c5fef8f5b270_3.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_5.png)

## 图形界面的驱动协议

![](img/337bf18e3c620c348a11c5fef8f5b270_7.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_9.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_10.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_12.png)

要想安装Linux图形界面，必须先安装图形界面的驱动服务器。早期，这个服务基于**X协议**（一种为实现特定目标而构建的协议，类似于HTTP/HTTPS）。由X协议实现的图形化界面称为 **X Window**。

![](img/337bf18e3c620c348a11c5fef8f5b270_14.png)

在RHEL 6等早期版本中，管理员可以通过执行 `init 5` 或 `startx` 命令来启动图形界面。

![](img/337bf18e3c620c348a11c5fef8f5b270_16.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_18.png)

现在主流的图形界面驱动协议是第十一代（X11）。基于X11协议的驱动服务器软件有很多，在Linux中常见的是 **XORG**。

![](img/337bf18e3c620c348a11c5fef8f5b270_20.png)

从RHEL 8开始，红帽开始支持新一代的图形驱动服务器：**Wayland**。这意味着在RHEL 8中，旧的XORG和新的Wayland可以共存并切换。而在RHEL 9中，默认使用的就是Wayland驱动服务器。

![](img/337bf18e3c620c348a11c5fef8f5b270_22.png)

## 桌面环境

当系统安装了图形驱动服务器（如Wayland或XORG）后，就可以选择安装图形化的桌面环境。桌面环境的选择非常丰富。

以下是几种常见的桌面环境：
*   **GNOME**：RHEL等发行版的默认桌面环境。
*   **KDE Plasma**：功能强大、高度可定制的桌面环境。
*   **Cinnamon**：基于GNOME 3，但采用传统桌面布局。
*   **XFCE**：轻量级、快速且经典的桌面环境。
*   **DDE (Deepin Desktop Environment)**：国内深度（Deepin）发行版开发的桌面环境。

![](img/337bf18e3c620c348a11c5fef8f5b270_24.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_25.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_27.png)

在GNOME桌面中，有一个核心功能组件叫做 **GNOME Shell**，它提供了桌面的核心功能并支持高度自定义，例如切换不同的主题。

![](img/337bf18e3c620c348a11c5fef8f5b270_29.png)

## GNOME桌面布局与功能

![](img/337bf18e3c620c348a11c5fef8f5b270_31.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_32.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_33.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_35.png)

GNOME Shell为我们提供了直观的桌面布局。其核心区域和功能可以通过以下编号来理解：

![](img/337bf18e3c620c348a11c5fef8f5b270_1.png)

1.  **顶栏**：桌面顶部的横条区域。
2.  **活动概览**：点击顶栏左侧的“活动”或按键盘上的 `Super` 键（通常是Windows键或Command键），可以进入活动概览视图，查看所有窗口的缩略图。
3.  **系统菜单**：点击顶栏右侧区域，可以打开系统菜单，进行声音调整、网络连接查看、系统设置、锁屏、关机/重启/注销等操作。
4.  **Dash栏**：在活动概览视图的底部，是固定或常用的应用程序快捷启动栏。
5.  **窗口缩略图**：在活动概览视图的中央，显示当前工作区所有打开窗口的缩略图。
6.  **工作区**：在活动概览视图的右侧，可以创建和管理多个虚拟桌面（工作区），使用 `Super + Alt + 方向键` 或鼠标进行切换。
7.  **托盘与日期**：顶栏中间区域显示日期和时间，点击后可查看日历和通知消息托盘。

## 系统设置详解

![](img/337bf18e3c620c348a11c5fef8f5b270_43.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_45.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_47.png)

通过系统菜单打开“设置”，可以对桌面环境进行详细配置。设置界面左侧为导航栏。

![](img/337bf18e3c620c348a11c5fef8f5b270_48.png)

以下是关键设置项的简要说明：
*   **网络**：管理网络连接。**注意**：在实验环境中请勿随意修改，以免破坏定制的网络配置。
*   **背景**：更改桌面壁纸和样式。
*   **通知**：管理来自应用程序的系统托盘通知。
*   **搜索**：配置在文件、应用、终端等位置的搜索行为。
*   **多任务**：设置热角、窗口分屏等提高效率的快捷键和功能。
*   **电源**：设置屏幕自动关闭和锁屏的时间。
*   **显示**：调整屏幕分辨率、缩放比例等。例如，可以将分辨率调整为更舒适的 `1400x900`。
*   **键盘**：添加或管理输入法，例如添加中文“智能拼音”输入法。
*   **区域与语言**：更改系统语言和格式。
*   **关于**：查看设备名称、内存、处理器、磁盘容量、操作系统及GNOME版本等信息。

## 在桌面中访问命令行

![](img/337bf18e3c620c348a11c5fef8f5b270_50.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_52.png)

在GNOME桌面中，有多种方式可以打开命令行终端。

![](img/337bf18e3c620c348a11c5fef8f5b270_54.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_56.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_57.png)

以下是打开终端的两种常用方法：
1.  点击“活动”进入概览视图，在Dash栏或应用程序列表中点击“终端”图标。
2.  直接按键盘快捷键 `Alt + F2`，在弹出的运行对话框中输入 `gnome-terminal` 然后回车。

![](img/337bf18e3c620c348a11c5fef8f5b270_59.png)

![](img/337bf18e3c620c348a11c5fef8f5b270_61.png)

## 锁屏与注销

![](img/337bf18e3c620c348a11c5fef8f5b270_63.png)

通过点击顶栏右侧的系统菜单，可以找到以下选项：
*   **锁定屏幕**：选择“锁定”或按 `Super + L` 快捷键，立即进入锁屏状态。
*   **注销/关机/重启**：在系统菜单中点击用户名，选择“注销”、“关机”或“重启”。

![](img/337bf18e3c620c348a11c5fef8f5b270_65.png)

在本节课中，我们将要学习Shell脚本中非常重要的`if`条件判断语句。`if`判断用于根据条件是否成立来执行不同的操作，它比之前学习的简单条件判断功能更强大，适合实现更复杂的逻辑。我们将从最简单的单分支开始，逐步学习双分支和多分支结构。

## 概述

`if`判断的核心是根据给定的条件决定执行哪部分代码。条件通常放在中括号 `[ ]` 内，可以是文件状态检查、数值比较或字符串对比等。我们将学习三种语法结构：单分支、双分支和多分支，并通过实例理解它们的应用场景。

## if单分支判断

![](img/255b3b79d49679ba1b1320613f27dbe0_16.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_18.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_20.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_22.png)

上一节我们介绍了条件判断的基本概念，本节中我们来看看最简单的`if`单分支。单分支的特点是**只能判断条件成立的情况**，如果条件不成立，则不会执行任何操作。

它的语法结构如下：

![](img/255b3b79d49679ba1b1320613f27dbe0_24.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_26.png)

```bash
if [ 条件 ]
then
    条件成立时执行的命令
fi
```

或者另一种写法：

![](img/255b3b79d49679ba1b1320613f27dbe0_28.png)

```bash
if [ 条件 ]; then
    条件成立时执行的命令
fi
```

![](img/255b3b79d49679ba1b1320613f27dbe0_30.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_32.png)

以下是使用单分支判断文件是否存在的示例步骤：

![](img/255b3b79d49679ba1b1320613f27dbe0_34.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_36.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_38.png)

1.  创建一个脚本文件 `if01.sh`。
2.  在脚本中，使用 `-f` 选项判断 `/etc/passwd` 文件是否存在。
3.  如果文件存在，则输出“存在”。
4.  给脚本添加执行权限并运行。

![](img/255b3b79d49679ba1b1320613f27dbe0_40.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_42.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_44.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_46.png)

```bash
#!/bin/bash
if [ -f /etc/passwd ]
then
    echo "存在"
fi
```

![](img/255b3b79d49679ba1b1320613f27dbe0_48.png)

执行脚本，如果文件存在则会输出“存在”；如果判断一个不存在的文件，则不会有任何输出，这体现了单分支“只能判断对”的特点。

## if双分支判断

![](img/255b3b79d49679ba1b1320613f27dbe0_50.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_52.png)

单分支只能处理条件成立的情况，这显然不够。双分支结构则能同时处理条件成立和不成立两种情况，功能更完善。

双分支的语法结构如下：

```bash
if [ 条件 ]
then
    条件成立时执行的命令
else
    条件不成立时执行的命令
fi
```

![](img/255b3b79d49679ba1b1320613f27dbe0_54.png)

以下是使用双分支实现的一个猜数字游戏脚本步骤：

![](img/255b3b79d49679ba1b1320613f27dbe0_56.png)

1.  创建脚本 `if03.sh`。
2.  使用 `$RANDOM` 环境变量生成一个随机数，并通过取余运算 `%` 将其范围控制在0-9之间。
3.  使用 `read -p` 提示用户输入一个0-9的数字。
4.  使用 `if` 双分支判断用户输入的数字是否与随机数相等，并给出相应提示。

![](img/255b3b79d49679ba1b1320613f27dbe0_58.png)

```bash
#!/bin/bash
num1=$((RANDOM % 10))
read -p "请输入0-9之间的一个数字：" number
if [ $number -eq $num1 ]
then
    echo "恭喜中奖！奖励哇塞女孩一枚！"
else
    echo "猜错了，哇塞女孩正在等你，请继续努力！"
    echo "本次正确结果为：$num1"
fi
```

![](img/255b3b79d49679ba1b1320613f27dbe0_60.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_62.png)

这个脚本展示了双分支如何根据条件成立与否执行不同的代码块。

![](img/255b3b79d49679ba1b1320613f27dbe0_64.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_66.png)

## if多分支判断

当我们需要判断多种可能的情况时，就需要用到多分支结构。它允许我们连续检查多个条件，直到某个条件成立，并执行对应的代码块。

多分支的语法结构如下：

![](img/255b3b79d49679ba1b1320613f27dbe0_68.png)

```bash
if [ 条件1 ]
then
    条件1成立时执行的命令
elif [ 条件2 ]
then
    条件2成立时执行的命令
elif [ 条件3 ]
then
    条件3成立时执行的命令
else
    以上所有条件都不成立时执行的命令
fi
```

![](img/255b3b79d49679ba1b1320613f27dbe0_70.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_72.png)

以下是使用多分支编写的一个成绩查询脚本步骤：

![](img/255b3b79d49679ba1b1320613f27dbe0_74.png)

1.  创建脚本 `if04.sh`。
2.  使用 `read -p` 获取用户输入的成绩。
3.  使用多分支 `if-elif-else` 结构判断成绩所属的等级（优秀、良好、一般、及格、不及格），并输出不同的评语。

![](img/255b3b79d49679ba1b1320613f27dbe0_76.png)

```bash
#!/bin/bash
read -p "请输入自己的成绩进行查询：" score
if [ $score -ge 90 ]
then
    echo "你的成绩非常优秀！奖励一个哇塞女孩！"
elif [ $score -ge 80 ]
then
    echo "你的成绩比较优秀！离哇塞女孩仅差一步。"
elif [ $score -ge 70 ]
then
    echo "你的成绩比较一般，还需努力。"
elif [ $score -ge 60 ]
then
    echo "你的成绩勉强及格。"
else
    echo "成绩不及格，请继续努力。"
fi
```

执行脚本并输入不同分数，可以看到程序会根据分数区间匹配第一个成立的条件，并执行对应的语句，之后便不再检查后续条件。

## 实用案例：软件包安装与管理

![](img/255b3b79d49679ba1b1320613f27dbe0_78.png)

结合变量和`if`判断，我们可以编写更实用的脚本。以下是实现一个智能安装与管理`vsftpd`服务脚本的思路：

1.  使用 `rpm -q` 命令检查软件包是否已安装，并将其执行结果状态（通过 `$?` 获取）作为判断条件。
2.  使用双分支判断：如果已安装，则确保服务启动并设置开机自启；如果未安装，则先安装软件包，再进行同样的服务配置。
3.  使用 `> /dev/null 2>&1` 将不必要的命令输出重定向到“黑洞”，使脚本输出更简洁。

![](img/255b3b79d49679ba1b1320613f27dbe0_80.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_82.png)

```bash
#!/bin/bash
rpm -q vsftpd > /dev/null 2>&1
if [ $? -eq 0 ]
then
    systemctl start vsftpd > /dev/null 2>&1
    systemctl enable vsftpd > /dev/null 2>&1
    echo "vsftpd 服务已启动并设置开机自启。"
else
    echo "正在安装 vsftpd 软件包..."
    yum install -y vsftpd > /dev/null 2>&1
    systemctl start vsftpd > /dev/null 2>&1
    systemctl enable vsftpd > /dev/null 2>&1
    echo "vsftpd 安装完成，服务已启动并设置开机自启。"
fi
```

这个案例展示了如何将命令执行结果、条件判断和流程控制结合起来，完成一个自动化任务。

## 总结

本节课中我们一起学习了Shell脚本中`if`条件判断的三种主要结构。

*   **单分支 (`if-then-fi`)**: 用于仅当条件成立时执行特定操作。
*   **双分支 (`if-then-else-fi`)**: 用于根据条件成立与否，分别执行不同的操作。
*   **多分支 (`if-then-elif-else-fi`)**: 用于连续判断多个条件，执行第一个成立条件对应的操作。

![](img/255b3b79d49679ba1b1320613f27dbe0_84.png)

![](img/255b3b79d49679ba1b1320613f27dbe0_86.png)

核心要点包括：
1.  所有判断条件都放在 `[ ]` 中。
2.  `[ ]` 内的每一部分（包括括号本身与内容）之间都需要有**空格**。
3.  如果条件中使用了变量，变量名前一定要加 `$` 符号。
4.  脚本编写往往是在理解他人脚本的基础上进行修改和优化，以符合自己的需求。

![](img/255b3b79d49679ba1b1320613f27dbe0_88.png)

通过本节课的示例，我们看到了`if`判断如何与变量、命令替换、数值运算等知识结合，解决实际问题。熟练掌握`if`判断是编写功能强大Shell脚本的基石。

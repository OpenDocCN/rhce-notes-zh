## 1. mkdir命令：创建新目录 📁
`mkdir`命令的英文全称是“make directory”，用于创建新的目录。这类似于在Windows系统中新建一个文件夹。

### 命令格式
`mkdir [选项] 目录名`

### 常用选项
*   **`-p`**：递归创建多级目录。

### 操作演示
1.  **在当前目录创建单个目录**
    ```bash
    mkdir test
    ls
    ```
    执行后，会在当前用户的家目录下创建一个名为`test`的蓝色目录。

2.  **在当前目录同时创建多个目录**
    ```bash
    mkdir test1 test2 test3 test4
    ls
    ```
    可以一次性创建`test1`到`test4`四个目录。

3.  **在指定路径创建目录**
    ```bash
    mkdir /opt/t1
    ls /opt
    ```
    这会在`/opt`目录下创建一个名为`t1`的目录。**注意**：如果需要同时在`/opt`下创建多个目录，必须为每个目录指定完整路径，例如 `mkdir /opt/t2 /opt/t3`。

4.  **使用`-p`选项递归创建目录**
    假设我们需要在`/opt/t2`目录下创建 `x1/x2/x3/x4` 这样的层级结构。
    ```bash
    mkdir -p /opt/t2/x1/x2/x3/x4
    ls -R /opt/t2
    ```
    使用`-p`选项后，即使中间目录`x1`、`x2`等不存在，命令也会自动创建它们。

### 注意事项
*   目录或文件名不能包含 `/` 字符，因为它是路径分隔符。
*   名称长度不能超过255个字符。

## 2. cd命令：切换工作目录 🔄
`cd`命令的英文全称是“change directory”，用于切换当前的工作目录。

### 命令格式
`cd [目录名]`

![](img/40c418728044115fe97b25ff6ed9c4e7_24.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_26.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_28.png)

### 路径概念
在使用`cd`命令时，需要理解两种路径：
*   **绝对路径**：以根目录 `/` 为起点的完整路径，如 `/opt/t2/x1`。
*   **相对路径**：以当前所在目录为起点的路径，如当前在`/opt`，要进入`t2`，只需 `cd t2`。

### 快捷操作
以下是`cd`命令的一些快捷用法：
*   `cd` 或 `cd ~`：直接返回当前用户的家目录。
*   `cd .`：切换到当前目录（通常用于执行当前目录下的脚本，如 `./script.sh`）。
*   `cd ..`：切换到上一级目录。
*   `cd -`：在两个最近切换过的目录之间来回切换。

![](img/40c418728044115fe97b25ff6ed9c4e7_30.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_32.png)

### 操作演示
```bash
cd /opt/t2/x1/x2/x3/x4  # 使用绝对路径切换到深层目录
pwd                     # 显示当前路径
cd                      # 直接回家目录
cd /opt                 # 切换到/opt目录
cd t2/x1                # 使用相对路径进入子目录
cd ..                   # 返回上一级目录
cd -                    # 切换回上一次所在的目录（即/opt）
```

![](img/40c418728044115fe97b25ff6ed9c4e7_36.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_38.png)

### 命令格式
`pwd`

![](img/40c418728044115fe97b25ff6ed9c4e7_40.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_42.png)

### 操作演示
当你在复杂的目录结构中时，使用`pwd`可以快速确认自己的位置。

```bash
# 输出：/opt/t2/x1/x2/x3
```

## 4. rmdir命令：删除空目录 🗑️
`rmdir`命令用于删除**空目录**。如果目录内有内容，该命令将无法执行。

![](img/40c418728044115fe97b25ff6ed9c4e7_44.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_46.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_48.png)

### 命令格式
`rmdir 目录名`

![](img/40c418728044115fe97b25ff6ed9c4e7_50.png)

### 操作演示与局限
```bash
rmdir /opt/t2  # 尝试删除非空目录，会失败
```
由于`/opt/t2`目录下包含`x1`等子目录，所以删除失败。要删除非空目录，需要先逐级删除其内容，或者使用后面会学到的更强大的`rm`命令。

## 5. touch命令：创建空文件 📄
`touch`命令用于创建新的空白文件，类似于在Windows中新建文本文档。

### 命令格式
`touch 文件名`

![](img/40c418728044115fe97b25ff6ed9c4e7_68.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_70.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_72.png)

### 操作演示
1.  **在当前目录创建文件**
    ```bash
    touch test.txt
    ls
    ```
2.  **同时创建多个文件**
    ```bash
    touch file1.txt file2.txt file3.log
    ls
    ```
3.  **在指定路径创建文件**
    ```bash
    touch /opt/t1/newfile.txt
    ls /opt/t1
    ```
**注意**：在Linux中，文件扩展名（如`.txt`）不是必须的，系统主要通过文件属性来识别类型。添加扩展名主要是为了方便用户识别，或在跨平台使用时让Windows系统能识别。

## 6. cp命令：复制文件或目录 📋
`cp`命令的英文全称是“copy”，用于复制文件或目录，相当于Windows中的“复制-粘贴”操作。

![](img/40c418728044115fe97b25ff6ed9c4e7_80.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_82.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_84.png)

### 命令格式
`cp [选项] 源文件或目录 目标目录`

![](img/40c418728044115fe97b25ff6ed9c4e7_86.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_88.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_90.png)

### 常用选项
*   **`-r`**：递归复制，用于复制目录及其内部所有内容。
*   **`-p`**：复制时保留源文件的属性（如修改时间、权限等）不变。

### 操作演示
1.  **复制单个文件**
    ```bash
    cp test.txt /media/  # 将test.txt复制到/media目录下
    ls /media/
    ls .                 # 源文件依然存在
    ```

![](img/40c418728044115fe97b25ff6ed9c4e7_92.png)

![](img/40c418728044115fe97b25ff6ed9c4e7_94.png)

2.  **同时复制多个文件**
    ```bash
    cp file1.txt file2.txt /media/
    ```

3.  **复制目录（必须加`-r`选项）**
    ```bash
    cp -r /opt/t2 /media/
    ls /media/
    ```

4.  **复制时保留文件属性（`-p`选项）**
    ```bash
    ls -l test.txt                # 查看源文件属性和时间
    cp -p test.txt /opt/t1/       # 保留属性复制
    ls -l /opt/t1/test.txt        # 查看副本，属性时间与源文件一致
    ```

5.  **复制时重命名**
    在指定目标路径时直接使用新名称即可实现重命名。
    ```bash
    cp test.txt /media/backup_test.txt  # 复制并重命名
    ls /media/
    ```

## 总结
本节课我们一起学习了Linux中六个基础的文件和目录操作命令：
1.  **`mkdir`**：创建目录，`-p`选项可创建多级目录。
2.  **`cd`**：切换工作目录，需理解绝对路径与相对路径。
3.  **`pwd`**：显示当前所在的绝对路径。
4.  **`rmdir`**：删除空目录（使用有局限）。
5.  **`touch`**：创建空白文件。
6.  **`cp`**：复制文件或目录，`-r`用于复制目录，`-p`用于保留属性，复制时可重命名。

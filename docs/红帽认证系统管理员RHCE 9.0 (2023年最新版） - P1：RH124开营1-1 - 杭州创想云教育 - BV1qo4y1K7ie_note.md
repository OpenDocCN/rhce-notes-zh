*   环境配置符合红帽课程的最高要求（例如，64GB内存，NVMe磁盘，多网卡）。
*   学员在报名后，可在上课至考试期间免费使用该环境，无需额外付费。
*   请注意，线上环境基于嵌套虚拟化技术，性能可能略低于线下真机环境。

课程总时长为十天，采用“4-3-3”或“3-3-4”的课时分配比例，确保零基础学员有充足的时间掌握第一本书（RH124）的内容。学习红帽认证的关键在于动手实践，必须通过反复练习来巩固知识。

*   **servera / serverb**：需要管理的目标服务器。
*   **workstation**：图形化界面的管理工作站，相当于公司配发的笔记本。
*   **bastion**：堡垒机兼路由器，负责流量转发，**必须保持开机状态**，否则其他虚拟机无法正常启动。
*   **foundation**：基础主机，学员**只能**在此主机上执行特定命令或远程连接到实验虚拟机，**切勿修改其网络等配置**。

管理这些虚拟机需要使用红帽提供的专用命令 `lab`。例如，启动所有虚拟机的命令是：
```bash
lab vmctl start all
```

上一节我们介绍了实验环境的构成，本节中我们来看看如何登录并使用它。

以下是登录线上实验环境的步骤：
1.  使用Windows“远程桌面连接”软件，连接地址：`redhat.bestvbc.com:23353`。
2.  使用分配的外网跳转机账户和密码登录。
3.  登录后，在跳转机桌面使用“TigerVNC Viewer”软件，输入分配的内网实验机IP和端口进行连接。
4.  使用分配的虚拟机密码（通常为 `redhat` 或 `student`）登录实验机。

环境使用提示：
*   若界面卡顿，可通过虚拟机桌面右上角的电源菜单重启，或使用VNC Viewer菜单栏的“Send Ctrl-Alt-Del”功能强制重启。
*   建议将虚拟机显示分辨率调整为 `1440x900` 以获得最佳体验。
*   线上学员建议使用SSH命令行连接 `workstation` 进行操作，以减少图形界面带来的性能负担。连接命令示例：
    ```bash
    ssh student@workstation
    ```
*   首次使用，请在`foundation`主机上检查环境标识文件：`cat /etc/rht`，确认内容为`rh999`。

## Linux与开源基础概念 🐧

在开始动手操作之前，我们需要了解一些核心概念。Linux是一种开源操作系统。开源意味着其源代码是公开的，任何人都可以查看、使用和修改，这带来了安全、透明和自由等优势。

Linux系统由内核（Kernel）和一系列软件组成。林纳斯·托瓦兹开发了Linux内核，而GNU项目提供了大量的基础工具。两者的结合形成了GNU/Linux系统。像红帽、Ubuntu这样的公司或社区将内核、工具及各种应用软件打包在一起，就形成了不同的**Linux发行版**。

红帽（Red Hat）是开源领域的领导者。其产品线包括：
*   **Fedora**：社区版，包含最新技术，是RHEL的上游。
*   **Red Hat Enterprise Linux (RHEL)**：企业版，稳定可靠，需订阅获得支持。
*   **CentOS Stream**：位于Fedora和RHEL之间，是RHEL的持续交付上游。

常见的Linux发行版还有Ubuntu、Debian、openSUSE等，它们各有特点，适用于不同场景。

## 访问命令行终端 🖥️

与Linux系统交互的主要方式是通过**命令行界面（CLI）**。用户通过一个称为**Shell**的程序与系统内核沟通。Shell是一个命令解释器，它将用户输入的命令翻译成系统能理解的操作。最常见的Shell是Bash（Bourne-Again Shell）。

用户需要通过一个**终端（Terminal）**来调用Shell。终端可以是：
*   **物理终端**：早期的专用硬件设备。
*   **虚拟控制台**：在Linux系统本地，通过 `Ctrl+Alt+F[1-6]` 切换的多个文本界面。
*   **终端模拟器**：在图形桌面环境中运行的软件，如GNOME Terminal。
*   **远程终端**：通过SSH协议从网络另一台计算机连接的终端，如使用PuTTY、Xshell等软件。

在终端中，你会看到**命令提示符**，例如 `[student@workstation ~]$`。它显示了当前用户、主机名和当前目录（`~`代表用户家目录）。`$`表示普通用户，`#`表示超级管理员root用户。

一个基本的命令结构通常包含**命令**、**选项**和**参数**。只有命令是必须的。例如，在 `ls -l /home` 中，`ls`是命令，`-l`是选项，`/home`是参数。

以下是几个入门命令示例：
*   **`pwd`**：打印当前工作目录。
    ```bash
    pwd
    ```
*   **`cd`**：切换工作目录。
    ```bash
    cd /tmp  # 切换到/tmp目录
    cd ~     # 切换回用户家目录
    ```
*   **`whoami`**：显示当前登录的用户名。
*   **`tty`**：显示当前终端的设备文件。
*   **`exit` 或 `logout`**：退出当前登录会话。在Bash Shell中，按 `Ctrl+D` 快捷键有相同效果。

在本节课中，我们将要学习Linux系统中的两个重要概念：**重定向**和**vim编辑器**。重定向允许我们控制命令的输入和输出流向，而vim是一个功能强大的文本编辑器，是系统管理员必备的工具。掌握这两项技能对于日常运维和故障排查至关重要。

## 重定向

上一节我们回顾了基础知识，本节中我们来看看**重定向**。当我们执行一条命令时，其输出结果默认会显示在终端屏幕上。但有时我们需要将这些结果保存到文件中，或者丢弃不需要的输出，这时就需要用到重定向。

### 输出类型与重定向符号

在Linux中，命令的输出分为两种类型：**标准输出**（stdout，文件描述符为1）和**标准错误输出**（stderr，文件描述符为2）。输入则对应**标准输入**（stdin，文件描述符为0）。

![](img/b2bce10beee66ce55e6c2a36d346c79b_4.png)

以下是重定向的基本用法：

![](img/b2bce10beee66ce55e6c2a36d346c79b_6.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_8.png)

*   **`>` 或 `1>`**：将**标准输出**重定向到文件，会覆盖原文件内容。
    *   示例：`ls -l /tmp > file.txt`
*   **`>>` 或 `1>>`**：将**标准输出**重定向到文件，采用追加方式，不会覆盖原内容。
    *   示例：`ls -l /tmp >> file.txt`
*   **`2>`**：将**标准错误输出**重定向到文件，会覆盖原文件。
    *   示例：`cd /root 2> error.log`
*   **`2>>`**：将**标准错误输出**重定向到文件，采用追加方式。
    *   示例：`cd /root 2>> error.log`
*   **`&>` 或 `2>&1`**：将**标准输出**和**标准错误输出**同时重定向到同一个文件。
    *   示例：`find /home -user student &> output.log`
*   **`/dev/null`**：这是一个特殊的空设备文件，重定向到此文件意味着丢弃所有输出。
    *   示例：`cd /root 2> /dev/null`

![](img/b2bce10beee66ce55e6c2a36d346c79b_10.png)

**重要提示**：在将输出重定向到重要文件时，建议使用追加符号 `>>` 而非覆盖符号 `>`，以避免意外覆盖原有数据。

### 输入重定向

![](img/b2bce10beee66ce55e6c2a36d346c79b_12.png)

除了输出，我们也可以重定向输入。

*   **`<`**：将文件内容作为命令的输入。
    *   示例：`mysql -u root -p < database_backup.sql`

### Vim的三种模式

Vim的操作主要围绕三种模式进行，理解它们是熟练使用Vim的关键。

![](img/b2bce10beee66ce55e6c2a36d346c79b_44.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_46.png)

1.  **命令模式**：启动Vim后默认进入的模式。在此模式下可以执行复制、删除、查找等快捷操作，但不能直接输入文本。按 `Esc` 键可以从其他模式退回此模式。
2.  **插入模式**：在此模式下可以像普通编辑器一样输入和编辑文本。从命令模式可以通过以下按键进入插入模式：
    *   `i`：在光标当前位置前开始插入。
    *   `I`：在光标所在行的行首开始插入。
    *   `a`：在光标当前位置后开始插入。
    *   `A`：在光标所在行的行尾开始插入。
    *   `o`：在光标所在行的下一行新建一行并开始插入。
    *   `O`：在光标所在行的上一行新建一行并开始插入。
3.  **末行模式**：在命令模式下按 `:` 进入。在此模式下可以执行保存、退出、搜索替换等操作。

![](img/b2bce10beee66ce55e6c2a36d346c79b_48.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_50.png)

### 命令模式常用操作

![](img/b2bce10beee66ce55e6c2a36d346c79b_54.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_56.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_57.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_59.png)

在命令模式下，可以使用以下快捷键进行高效编辑：

![](img/b2bce10beee66ce55e6c2a36d346c79b_61.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_63.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_65.png)

*   **光标移动**：
    *   `^` 或 `0`：移动到行首。
    *   `$`：移动到行尾。
    *   `G`：移动到文件末尾。
    *   `gg`：移动到文件开头。
    *   `:n`：跳转到第n行（需在末行模式下输入）。
*   **复制、剪切与粘贴**：
    *   `yy`：复制当前行。
    *   `nyy`：复制从当前行开始的n行。
    *   `dd`：剪切（删除）当前行。
    *   `ndd`：剪切（删除）从当前行开始的n行。
    *   `p`：将复制或剪切的内容粘贴到光标**之后**。
    *   `P`：将复制或剪切的内容粘贴到光标**之前**。
*   **删除与撤销**：
    *   `x`：删除光标所在位置的字符。
    *   `X`：删除光标前一个字符。
    *   `u`：撤销上一次操作。
    *   `Ctrl + r`：重做被撤销的操作。
*   **查找**：
    *   `/keyword`：从光标处向后查找关键字 `keyword`，按 `n` 查找下一个，按 `N` 查找上一个。
    *   `?keyword`：从光标处向前查找关键字。

![](img/b2bce10beee66ce55e6c2a36d346c79b_67.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_69.png)

### 末行模式常用操作

![](img/b2bce10beee66ce55e6c2a36d346c79b_71.png)

在末行模式下，可以执行文件级操作：

*   **保存与退出**：
    *   `:w`：保存文件。
    *   `:q`：退出Vim（如果文件有修改未保存，会提示）。
    *   `:wq` 或 `:x`：保存并退出。
    *   `:q!`：强制退出，不保存修改。
    *   `:wq!`：强制保存并退出（常用于只读文件）。
*   **显示与跳转**：
    *   `:set nu`：显示行号。
    *   `:set nonu`：取消显示行号。
*   **替换文本**：
    *   `:%s/old/new/g`：将文件中所有的 `old` 替换为 `new`。
        *   `%` 表示整个文件范围，可以替换为 `n,m` 表示从第n行到第m行。
        *   `g` 表示全局替换（一行内所有匹配项），不加 `g` 则只替换每行的第一个匹配项。
    *   示例：将 `/bin/nologin` 替换为 `/bin/bash`：`:%s/\/bin\/nologin/\/bin\/bash/g`
        *   **注意**：因为 `/` 在替换命令中有特殊含义，所以需要使用 `\/` 进行转义。
*   **执行外部命令与读入内容**：
    *   `:!command`：暂时离开Vim执行外部命令 `command`，按回车返回。
    *   `:r filename`：将文件 `filename` 的内容读入并插入到光标下方。
    *   `:r !command`：将外部命令 `command` 的执行结果读入并插入到光标下方。

### Vim高级功能

![](img/b2bce10beee66ce55e6c2a36d346c79b_79.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_81.png)

![](img/b2bce10beee66ce55e6c2a36d346c79b_83.png)

*   **分屏编辑**：
    *   `vim -o file1 file2`：水平分屏打开多个文件。
    *   `vim -O file1 file2`：垂直分屏打开多个文件。
    *   在分屏间切换：`Ctrl + w` 然后按方向键。
*   **可视化模式**：
    *   `v`：进入字符可视化模式，按字符选择文本。
    *   `V`：进入行可视化模式，按行选择文本。
    *   `Ctrl + v`：进入块可视化模式，按矩形块选择文本。选中后可以进行复制（`y`）、删除（`d`）等操作。
*   **恢复文件**：Vim在编辑文件时会生成一个 `.swp` 的交换文件。如果编辑时异常退出（如系统崩溃），再次打开文件时Vim会提示恢复。可以使用 `vim -r filename` 或打开文件后执行 `:recover` 来尝试恢复未保存的更改。

## Linux 系统基础

![](img/3aa38c6e6120b7d5359b2b12c645db04_57.png)

在成功连接到命令行界面后，我们需要了解一些 Linux 系统的基础知识。

![](img/3aa38c6e6120b7d5359b2b12c645db04_59.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_61.png)

Red Hat Enterprise Linux (RHEL) 是一个企业级操作系统，通常用于生产环境。CentOS (Community Enterprise Operating System) 是其社区版本。两者的主要区别在于，RHEL 提供付费的技术支持服务，而 CentOS 没有。但两者的命令和操作方式几乎完全相同，因此学习 RHEL 的技能可以无缝应用到 CentOS 上。

![](img/3aa38c6e6120b7d5359b2b12c645db04_63.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_65.png)

Linux 系统主要由内核（Kernel）和外壳（Shell）组成。内核是操作系统的核心，负责管理硬件和系统资源。Shell 是一个命令行解释器，它作为用户与内核之间的中介，接收用户输入的命令并将其传递给内核执行。

常见的 Shell 类型是 Bash (Bourne-Again Shell)，它是 Red Hat 系统的默认 Shell。我们通过 Shell 来执行命令、运行脚本，从而完成系统管理工作。

## 理解命令行界面

上一节我们介绍了 Linux 的基本框架，本节中我们来看看命令行界面的具体构成。

当通过 Shell 登录系统后，会看到一个命令提示符，例如 `[root@servA ~]#`。这个提示符包含了以下信息：

*   `root`：当前登录的用户名。
*   `@`：分隔符。
*   `servA`：主机名（简短格式）。
*   `~`：当前所在的工作目录，波浪号 `~` 代表当前用户的**家目录**。
*   `#`：提示符本身，`#` 表示当前用户是超级管理员 root；如果是普通用户，则显示为 `$`。

![](img/3aa38c6e6120b7d5359b2b12c645db04_67.png)

这个提示符的格式由一个叫做 `PS1` 的系统变量控制。执行 `echo $PS1` 命令可以查看其当前值。

![](img/3aa38c6e6120b7d5359b2b12c645db04_69.png)

在 Linux 中，执行一个命令本质上是运行一个可执行文件。系统会在一个名为 `PATH` 的环境变量所定义的目录路径中查找这些可执行文件。执行 `echo $PATH` 可以查看这些路径。

如果命令文件不在 `PATH` 变量包含的目录中，系统将无法找到它。此时，需要将该命令所在的目录路径临时或永久地添加到 `PATH` 变量中。

![](img/3aa38c6e6120b7d5359b2b12c645db04_71.png)

## 命令的构成与执行

理解了命令行的基础后，我们来深入学习命令本身是如何构成和执行的。

一个完整的命令通常由三部分组成：**命令**、**选项** 和 **参数**。

![](img/3aa38c6e6120b7d5359b2b12c645db04_73.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_75.png)

1.  **命令**：决定要执行什么操作，对应一个可执行文件。
2.  **选项**：用于调节命令的行为或输出格式，通常以 `-`（短选项）或 `--`（长选项）开头。
3.  **参数**：命令操作的对象，例如文件名、目录名等。

![](img/3aa38c6e6120b7d5359b2b12c645db04_77.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_78.png)

其基本语法格式为：
```
命令 [选项] [参数]
```

![](img/3aa38c6e6120b7d5359b2b12c645db04_80.png)

例如，在命令 `ls -l /home` 中：
*   `ls` 是命令（列出目录内容）。
*   `-l` 是选项（以长格式显示详细信息）。
*   `/home` 是参数（要查看的目录）。

![](img/3aa38c6e6120b7d5359b2b12c645db04_81.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_83.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_84.png)

如果不清楚某个命令的用法，可以使用 `--help` 选项来获取帮助，例如 `ls --help`。Linux 系统内置了丰富的帮助文档，这在学习和考试中是非常有用的资源。

![](img/3aa38c6e6120b7d5359b2b12c645db04_86.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_87.png)

## 系统登录与界面切换

![](img/3aa38c6e6120b7d5359b2b12c645db04_91.png)

最后，我们来了解登录 Linux 系统的不同方式以及界面之间的切换。

![](img/3aa38c6e6120b7d5359b2b12c645db04_93.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_94.png)

Linux 是一个多用户、多任务的操作系统。用户可以通过图形界面或字符界面（命令行）登录。在字符界面下，系统提供了多个虚拟终端（通常为 tty2 到 tty6），允许同一用户在不同终端同时登录。

![](img/3aa38c6e6120b7d5359b2b12c645db04_96.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_98.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_99.png)

在已经启动图形界面的系统上，可以通过快捷键 `Ctrl+Alt+F2` 到 `F6` 切换到不同的字符界面终端。要切换回图形界面，则使用 `Ctrl+Alt+F1`。

![](img/3aa38c6e6120b7d5359b2b12c645db04_101.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_103.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_105.png)

![](img/3aa38c6e6120b7d5359b2b12c645db04_107.png)

无论是本地直接操作，还是通过 SSH 远程连接，最终都是通过 Shell 来与系统内核进行交互，完成各项运维任务。

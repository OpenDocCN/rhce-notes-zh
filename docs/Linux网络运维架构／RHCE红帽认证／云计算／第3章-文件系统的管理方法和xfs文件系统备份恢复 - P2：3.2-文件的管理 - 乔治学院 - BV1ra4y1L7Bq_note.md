## 创建文件

![](img/5d3d2537ce193df339494bfaadc170d7_7.png)

上一节我们介绍了文件系统的基本概念，本节中我们来看看如何创建文件。

### 使用 `touch` 命令
`touch` 命令最基本的功能是创建一个空文件。如果文件已存在，`touch` 会更新该文件的“修改时间”。

![](img/5d3d2537ce193df339494bfaadc170d7_9.png)

**基本语法**：
```bash
touch 文件名
```

**操作示例**：
1.  创建一个名为 `a.txt` 的空文件。
    ```bash
    touch a.txt
    ```
2.  同时创建多个文件。
    ```bash
    touch file1 file2
    ```
3.  使用序列创建一批文件（例如从 `file6` 到 `file20`）。
    ```bash
    touch file{6..20}
    ```

![](img/5d3d2537ce193df339494bfaadc170d7_11.png)

![](img/5d3d2537ce193df339494bfaadc170d7_13.png)

创建完成后，可以使用 `ls -l` 命令查看文件的详细信息，包括时间戳。

![](img/5d3d2537ce193df339494bfaadc170d7_15.png)

![](img/5d3d2537ce193df339494bfaadc170d7_17.png)

### 文件的时间属性
每个Linux文件都包含三个主要的时间戳，可以使用 `stat` 命令查看：
*   **最近访问时间 (Access Time, atime)**：文件内容最后一次被读取的时间。
*   **最近更改时间 (Modify Time, mtime)**：文件**内容**最后一次被修改的时间。
*   **最近改动时间 (Change Time, ctime)**：文件**元数据**（如权限、所有者）最后一次被改变的时间。

**注意**：`ctime` 并非“创建时间”，而是“变更时间”。理解这些时间戳有助于后期进行文件查找和系统审计。

![](img/5d3d2537ce193df339494bfaadc170d7_19.png)

### 其他创建文件的方法
除了 `touch`，还有其他方法可以创建并写入内容：
*   **使用文本编辑器**：如 `vi` 或 `vim`。
    ```bash
    vim newfile.txt
    ```
*   **使用输出重定向**：`echo` 命令配合 `>` 符号可以将输出内容写入文件。如果文件不存在，则会创建它。
    ```bash
    echo “Hello World” > newfile.txt
    ```
    *   `>` 表示覆盖写入。
    *   `>>` 表示追加写入。

### 使用 `mkdir` 命令
`mkdir` 命令用于创建目录。

![](img/5d3d2537ce193df339494bfaadc170d7_32.png)

**基本语法**：
```bash
mkdir 目录名
```

![](img/5d3d2537ce193df339494bfaadc170d7_34.png)

**操作示例**：
1.  创建单个目录。
    ```bash
    mkdir dir1
    ```
2.  同时创建多个目录。
    ```bash
    mkdir dir2 dir3
    ```
3.  创建多级目录（使用 `-p` 参数）。如果父目录不存在，此参数会一并创建。
    ```bash
    mkdir -p /tmp/aa/bb/cc
    ```

## 删除文件与目录

![](img/5d3d2537ce193df339494bfaadc170d7_40.png)

创建了文件和目录后，自然也需要知道如何删除它们。

### 使用 `rm` 命令
`rm` 命令用于删除文件或目录。**这是一个需要谨慎使用的命令**。

**核心语法与参数**：
*   `rm 文件名`：删除文件，系统会询问确认。
*   `rm -f 文件名`：强制删除文件，不询问。
*   `rm -r 目录名`：递归删除目录及其内部所有内容。
*   `rm -rf 目录名`：**强制递归删除目录，不进行任何确认。请务必确认路径无误后再执行。**

**操作示例**：
强制删除一个目录及其下所有内容。
```bash
rm -rf dirname
```

![](img/5d3d2537ce193df339494bfaadc170d7_42.png)

![](img/5d3d2537ce193df339494bfaadc170d7_44.png)

**重要提示**：在工作中，为防止误删，可以先使用 `mv` 命令将文件移动到临时目录（如 `/tmp`），确认无误后再删除。

![](img/5d3d2537ce193df339494bfaadc170d7_46.png)

### 使用 `rmdir` 命令
`rmdir` 命令用于删除**空目录**。如果目录非空，则删除失败。
```bash
rmdir emptydir
```
由于此命令只能删除空目录，在实际工作中使用场景有限，了解即可。

## 复制与移动文件

![](img/5d3d2537ce193df339494bfaadc170d7_50.png)

![](img/5d3d2537ce193df339494bfaadc170d7_52.png)

文件管理还包括复制和移动操作。

### 复制 (`cp`) 命令
`cp` 命令用于复制文件或目录。

![](img/5d3d2537ce193df339494bfaadc170d7_54.png)

**基本语法**：
```bash
cp 源文件 目标文件
cp -r 源目录 目标目录
```
参数 `-r` 代表“递归”，用于复制目录及其所有内容。

![](img/5d3d2537ce193df339494bfaadc170d7_56.png)

**操作示例**：
1.  复制文件。
    ```bash
    cp a.txt b.txt
    ```
2.  复制整个目录。
    ```bash
    cp -r /home/user/data /opt/backup/
    ```

![](img/5d3d2537ce193df339494bfaadc170d7_58.png)

### 移动 (`mv`) 命令
`mv` 命令有两个主要功能：移动文件/目录，以及重命名文件/目录。

**基本语法**：
```bash
mv 源文件 目标路径/新文件名
```

![](img/5d3d2537ce193df339494bfaadc170d7_60.png)

![](img/5d3d2537ce193df339494bfaadc170d7_62.png)

**操作示例**：
1.  移动文件到其他目录。
    ```bash
    mv file.txt /tmp/
    ```
2.  重命名文件。
    ```bash
    mv oldname.txt newname.txt
    ```
3.  移动并重命名。
    ```bash
    mv file.txt /backup/archived_file.txt
    ```

## 查看文件内容

![](img/5d3d2537ce193df339494bfaadc170d7_66.png)

最后，我们学习如何查看文件的内容。以下是几个常用命令：

![](img/5d3d2537ce193df339494bfaadc170d7_68.png)

![](img/5d3d2537ce193df339494bfaadc170d7_70.png)

以下是查看文件内容的几种方法：
*   `cat 文件名`：一次性显示整个文件内容，适合查看小文件。
*   `less 文件名`：分页显示文件内容，支持上下翻页、搜索，适合查看大文件。按 `q` 键退出。
*   `head -n 数字 文件名`：显示文件开头指定行数（默认10行）。
*   `tail -n 数字 文件名`：显示文件末尾指定行数（默认10行）。常用 `tail -f 文件名` 来实时追踪日志文件的新增内容。

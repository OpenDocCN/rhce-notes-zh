### 创建虚拟机硬件平台

![](img/e53891a1f49789183a1583fb3204bb2a_59.png)

![](img/e53891a1f49789183a1583fb3204bb2a_61.png)

![](img/e53891a1f49789183a1583fb3204bb2a_63.png)

安装虚拟机软件的过程非常简单，只需一路点击“下一步”即可完成，此处不再赘述。安装好虚拟机软件后，我们开始创建虚拟机的硬件环境。

1.  打开虚拟机软件，选择“创建新的虚拟机”。
2.  选择“自定义（高级）”安装模式，以便进行更详细的配置。
3.  硬件兼容性选择最高版本。
4.  **关键步骤**：选择“稍后安装操作系统”。如果此时选择了安装介质，系统会以最小化模式安装，这与我们后续的实验环境不符。
5.  选择客户机操作系统为“Linux”，版本为“Red Hat Enterprise Linux 8 64位”。
6.  为虚拟机命名（例如“跟老刘学习Linux第一课”），并选择安装位置（建议放在空间较大的磁盘分区）。
7.  处理器配置：根据您物理机CPU的核心数进行设置（可通过任务管理器查看），设置核心数可以让虚拟机性能更好。如果不确定，选择1个核心也可以满足实验需求。
8.  内存分配：如果物理机内存大于4GB，建议为虚拟机分配2048MB（2GB）内存。
9.  网络类型：暂时选择“仅主机模式”，具体原理将在第8章详细讲解。
10. I/O控制器类型和磁盘类型：保持默认推荐选项即可。
11. 选择磁盘：选择“创建新虚拟磁盘”。
12. 磁盘容量：分配20GB即可，选择“将虚拟磁盘拆分成多个文件”。
13. 指定磁盘文件名称，保持默认即可。
14. 完成硬件创建。

### 什么是Shell？

Shell是人与计算机硬件之间的“翻译官”。用户通过Shell输入命令，Shell将这些命令“翻译”成内核能理解的指令，指挥硬件工作，然后再将结果“翻译”成用户能看懂的信息返回。

常见的Shell解释器有多种，其中 **Bash** 是最常用、最通用的一个。它有以下优点：
*   支持使用上下方向键调用历史命令。
*   支持使用 `Tab` 键自动补全命令或文件名。
*   拥有强大的批处理脚本支持（第4章详解）。
*   具有良好的环境变量支持（第3章详解）。

![](img/e53891a1f49789183a1583fb3204bb2a_115.png)

### 命令格式与书写规范

一个完整的命令行通常包含三部分：**命令**、**参数**、**对象**。它们之间用空格进行分隔。

![](img/e53891a1f49789183a1583fb3204bb2a_117.png)

![](img/e53891a1f49789183a1583fb3204bb2a_119.png)

![](img/e53891a1f49789183a1583fb3204bb2a_121.png)

![](img/e53891a1f49789183a1583fb3204bb2a_123.png)

*   **命令**：要执行的操作，即“做什么”，如 `新建`、`删除`。
*   **参数**：对命令功能的修饰，使其更贴合工作需要，即“怎么做”，如 `快速地`、`详细地`。参数通常分为长格式（`--word`）和短格式（`-w`）。
*   **对象**：命令的操作目标，即“对谁做”，如 `文件A`。

![](img/e53891a1f49789183a1583fb3204bb2a_125.png)

**格式示例**：
`命令 [参数] [对象]`

![](img/e53891a1f49789183a1583fb3204bb2a_127.png)

![](img/e53891a1f49789183a1583fb3204bb2a_129.png)

![](img/e53891a1f49789183a1583fb3204bb2a_131.png)

**注意事项**：
1.  命令、参数、对象之间必须用**空格**分隔。
2.  命令和参数通常**区分大小写**。
3.  多个短格式参数可以合并，只保留一个 `-`，如 `-l -a` 可以写成 `-la`。
4.  参数的长格式和短格式不能合并。

### 常用基础命令实战

以下是一些最基础的命令，我们先通过它们感受一下命令行的操作。无需死记硬背，后续通过反复使用自然会熟悉。

#### 1. echo命令
用于在终端输出字符串或提取变量的值。
```bash
echo linuxprobe.com
```

#### 2. date命令
用于查看或设置系统日期和时间。
```bash
date
# 按特定格式输出日期
date "+%Y-%m-%d"
```

![](img/e53891a1f49789183a1583fb3204bb2a_163.png)

![](img/e53891a1f49789183a1583fb3204bb2a_165.png)

![](img/e53891a1f49789183a1583fb3204bb2a_167.png)

#### 3. timedatectl命令
用于设置系统时间（需要先关闭网络时间同步服务）。
```bash
timedatectl set-time “2022-06-01 08:00:00”
```

![](img/e53891a1f49789183a1583fb3204bb2a_169.png)

#### 4. reboot命令
用于重启系统。
```bash
reboot
```

#### 5. poweroff命令
用于关闭系统。
```bash
poweroff
```

#### 6. wget命令
用于从网络下载文件。
```bash
wget https://linuxprobe.com/docs/linuxprobe.pdf
```

#### 7. ps命令
用于查看系统进程状态。进程常见状态有：
*   **R（Running）**：运行中。
*   **S（Sleeping）**：休眠中，不消耗资源。
*   **D（Uninterruptible Sleep）**：不可中断进程（类似“未响应”）。
*   **Z（Zombie）**：僵尸进程，已终止但未释放资源。
*   **T（Stopped）**：已停止。

```bash
ps aux
```
`ps aux` 命令可以查看所有进程的详细信息。为了更直观、实时地查看进程（类似Windows任务管理器），我们可以使用 `top` 命令，这将在下节课详细讲解。

# 红帽认证入门：P1：账号注册与Credly平台使用指南 🚀

![](img/83fea3143060198c5ed413b8321bae17_0.png)

在本节课中，我们将学习如何为参加红帽认证考试做准备。核心任务是注册两个必需的账号：**红帽官网账户**和**Credly平台账户**。这两个账号是获取和管理你的认证证书与徽章的关键。

---

![](img/83fea3143060198c5ed413b8321bae17_2.png)

## 注册前的准备：理解两个核心账号

![](img/83fea3143060198c5ed413b8321bae17_4.png)

![](img/83fea3143060198c5ed413b8321bae17_6.png)

![](img/83fea3143060198c5ed413b8321bae17_8.png)

在开始注册前，你需要了解两个核心概念及其作用。

**红帽网络账户 (Red Hat Network Account)**：这是你的红帽官方身份标识，简称 `HN`。它有两个主要用途：
1.  **参加考试**：考试时需要输入注册时使用的邮箱和这个 `HN` 账号。
2.  **验证证书**：考后需登录红帽官网，通过邮箱和 `HN` 账号验证并管理你的证书，确保证书归属无误。曾有他人冒用账号绑定证书的案例，因此账号安全至关重要。

**Credly平台账户**：这是一个国际通用的数字证书认证与管理平台。它的作用是：
*   **接收并管理证书**：红帽考试通过后，会将电子证书和徽章发送到你的Credly账户。
*   **能力认可与展示**：类似于国内的“学信网”，它集中管理你的各类国际认证。你可以选择公开信息，有机会直接收到企业的招聘邀请。

简单来说，流程是：用红帽账户参加考试 → 红帽将成绩和证书发送到你的邮箱 → 在Credly平台领取并管理证书。

---

## 第一步：注册红帽官网账户

上一节我们介绍了两个核心账号的作用，本节中我们来看看如何注册红帽官网账户。

![](img/83fea3143060198c5ed413b8321bae17_10.png)

![](img/83fea3143060198c5ed413b8321bae17_12.png)

访问红帽官网：`redhat.com`。首次访问可能是英文界面，你可以在页面中找到语言选项，切换为中文。

![](img/83fea3143060198c5ed413b8321bae17_14.png)

![](img/83fea3143060198c5ed413b8321bae17_16.png)

在网站右上角找到“登录”按钮，由于没有账户，需要点击旁边的“立即注册”。

![](img/83fea3143060198c5ed413b8321bae17_18.png)

![](img/83fea3143060198c5ed413b8321bae17_20.png)

以下是注册时需要填写的关键信息及注意事项：

*   **用户名 (Username)**：建议仅使用**字母、数字和下划线**的组合。**切勿直接使用邮箱地址作为用户名**，否则在考试绑定和后续工作中可能带来麻烦，且用户名一旦注册无法修改。
*   **密码 (Password)**：设置一个强密码，建议至少14位字符。
*   **名字 (First Name) 和姓氏 (Last Name)**：**务必使用英文或拼音填写**。中文可能导致证书打印时显示为乱码（方块）。
*   **邮箱地址 (Email Address)**：这是最重要的信息，务必使用你长期持有且能正常访问的邮箱。所有考试通知和证书链接都将发至此邮箱。
*   **账户类型 (Account Type)**：选择“个人”。
*   **国家/地区 (Country/Region)**：根据实际情况选择。
*   **地址 (Address)**：其他信息如地址、电话、公司等，可用英文填写，也可酌情填写，对核心功能影响不大。

填写完毕后，点击“创建我的账户”。系统会向你的邮箱发送一封确认邮件，点击邮件中的链接激活账户即可完成注册。

**重要提醒**：请谨慎考虑是否使用公司邮箱注册。如果未来离职，可能无法再访问该邮箱，会导致无法接收证书等重要信息。

## 第二步：注册Credly平台账户

成功注册红帽账户后，下一步是注册Credly账户，以便未来接收和管理证书。

访问Credly官网：`credly.com`。网站加载可能较慢，请耐心等待。

![](img/83fea3143060198c5ed413b8321bae17_22.png)

![](img/83fea3143060198c5ed413b8321bae17_24.png)

在网站右上角点击“Sign In”，然后选择“创建账户”。

![](img/83fea3143060198c5ed413b8321bae17_26.png)

![](img/83fea3143060198c5ed413b8321bae17_28.png)

![](img/83fea3143060198c5ed413b8321bae17_30.png)

以下是注册Credly账户的注意事项：

![](img/83fea3143060198c5ed413b8321bae17_32.png)

![](img/83fea3143060198c5ed413b8321bae17_34.png)

![](img/83fea3143060198c5ed413b8321bae17_36.png)

*   **邮箱地址**：**必须使用你在红帽账户中绑定的同一个邮箱**进行注册，否则将无法接收到红帽发送的证书。
*   **姓名**：同样，**必须使用英文或拼音**填写，以确保生成的证书姓名显示正确。
*   **密码**：设置你的登录密码。
*   勾选同意条款后完成注册。

![](img/83fea3143060198c5ed413b8321bae17_40.png)

注册成功后，建议过一两天再次登录测试，确保账户状态正常。曾有案例因姓名与某些名单重合导致账户异常，若遇此情况，需联系平台客服解决。

## 考后操作：关联与获取证书

![](img/83fea3143060198c5ed413b8321bae17_51.png)

![](img/83fea3143060198c5ed413b8321bae17_53.png)

当你通过红帽考试后，需要按顺序完成以下操作来获取证书：

![](img/83fea3143060198c5ed413b8321bae17_55.png)

1.  **查收邮件**：红帽会向你的注册邮箱发送成绩单和证书领取通知。
2.  **登录红帽官网**：在个人中心的“证书”部分，可能需要手动添加证书，输入你的考试ID和邮箱进行验证关联。
3.  **登录Credly平台**：点击红帽邮件中关于Credly的链接，或直接登录Credly，即可在“我的徽章”中看到获得的证书。
4.  **下载与分享**：在Credly中，你可以下载带二维码的PDF证书。他人扫描二维码可在线验证证书真伪。请确保Credly个人资料中的姓名是英文，并设置好隐私选项。

![](img/83fea3143060198c5ed413b8321bae17_56.png)

![](img/83fea3143060198c5ed413b8321bae17_58.png)

![](img/83fea3143060198c5ed413b8321bae17_59.png)

![](img/83fea3143060198c5ed413b8321bae17_61.png)

如果考后长时间未收到Credly的邮件，可以检查红帽成绩邮件中是否有申诉链接，或直接联系红帽客服请求重新发送。

![](img/83fea3143060198c5ed413b8321bae17_63.png)

![](img/83fea3143060198c5ed413b8321bae17_65.png)

**关于成绩**：红帽考试成绩通常在**3个美国工作日**内公布。若超过此时间未收到，请先检查邮箱是否正确，再联系你的培训负责人查询。

## 总结与回顾

本节课中我们一起学习了红帽认证考试前的关键准备工作：

1.  **理解了两个核心账号**：红帽网络账户 (`HN`) 用于考试和身份验证；Credly账户用于接收和管理数字证书与徽章。
2.  **完成了账号注册**：逐步指导了如何在红帽官网和Credly平台进行注册，并强调了**使用英文姓名**、**不使用邮箱作为用户名**、**确保邮箱一致**等关键注意事项。
3.  **了解了考后流程**：明确了考试通过后，通过邮箱链接在Credly平台领取和下载证书的完整步骤。

![](img/83fea3143060198c5ed413b8321bae17_69.png)

![](img/83fea3143060198c5ed413b8321bae17_70.png)

请务必在考试前完成这两个账号的注册与验证，确保一切信息准确无误，为顺利获得认证扫清障碍。

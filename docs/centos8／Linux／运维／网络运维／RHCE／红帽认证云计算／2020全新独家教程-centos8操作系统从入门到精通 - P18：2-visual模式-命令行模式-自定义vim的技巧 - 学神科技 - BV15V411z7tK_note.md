### Visual 可视块模式 🧱

![](img/8b730af70855f36ab372431ea5511a23_9.png)

上一节我们介绍了 Vim 的基本操作，本节中我们来看看 Visual 可视块模式。该模式常用于编程或修改服务器配置文件时，进行多行注释或批量编辑。

**进入可视块模式**：在 Normal 模式下，按下 `Ctrl + V` 即可进入列编辑模式，也称为可视块模式。此时可以向上或向下移动光标，选中需要编辑的文本块。

![](img/8b730af70855f36ab372431ea5511a23_11.png)

![](img/8b730af70855f36ab372431ea5511a23_12.png)

![](img/8b730af70855f36ab372431ea5511a23_14.png)

以下是可视块模式的核心操作：

![](img/8b730af70855f36ab372431ea5511a23_16.png)

*   **批量删除**：选中文本块后，按下大写 `D` 即可删除选中的内容。
*   **批量注释**：选中需要注释的行的开头部分，按下大写 `I` 进入插入模式，输入注释符号（如 `#`），然后按下 `ESC` 键，选中的所有行前都会自动添加该符号。
*   **取消批量注释**：选中注释符号所在的列，按下小写 `d` 即可删除这些符号，从而取消注释。

![](img/8b730af70855f36ab372431ea5511a23_18.png)

**示例**：编辑 `/etc/ssh/sshd_config` 文件，批量取消或添加 `Port 22` 前的 `#` 注释符。

### 命令行模式 💻

![](img/8b730af70855f36ab372431ea5511a23_20.png)

掌握了可视块模式后，我们进入命令行模式的学习。该模式用于执行保存、退出、查找替换等高级命令。

**基本命令格式**：在 Normal 模式下输入 `:` 进入命令行模式。

![](img/8b730af70855f36ab372431ea5511a23_22.png)

以下是常用的命令行操作列表：

![](img/8b730af70855f36ab372431ea5511a23_24.png)

*   **文件操作**
    *   `:w` - 保存文件。
    *   `:w!` - 强制保存文件（针对只读文件）。
    *   `:q` - 退出 Vim（如果文件未修改）。
    *   `:q!` - 不保存修改，强制退出。
    *   `:wq` 或 `:x` - 保存并退出。
    *   `:wq!` - 强制保存并退出。
*   **撤销与恢复**
    *   `:e!` - 放弃所有修改，恢复到文件打开时的状态。
*   **快捷操作**
    *   在 Normal 模式下，按下大写 `ZZ` 可以快速保存并退出。

![](img/8b730af70855f36ab372431ea5511a23_26.png)

![](img/8b730af70855f36ab372431ea5511a23_28.png)

**调用外部命令与读取文件**：在命令行模式中，可以执行外部 Shell 命令或读取其他文件内容。

![](img/8b730af70855f36ab372431ea5511a23_30.png)

*   `:!<命令>` - 执行外部 Shell 命令。例如，`:!ifconfig` 可以查看网络信息，并将结果输出。
*   `:r <文件路径>` - 将指定文件的内容读取并插入到当前光标位置。例如，`:r /etc/hosts`。

![](img/8b730af70855f36ab372431ea5511a23_32.png)

![](img/8b730af70855f36ab372431ea5511a23_34.png)

### 文本查找与替换 🔍

![](img/8b730af70855f36ab372431ea5511a23_36.png)

在命令行模式中，文本替换功能非常强大。其基本语法结构如下：

`:范围 s/旧内容/新内容/选项`

*   **范围**：指定替换操作的行范围。
    *   `%` - 代表整个文件。
    *   `1,3` - 代表第1行到第3行。
    *   `3` - 仅代表第3行。
*   **分隔符**：`/` 是默认分隔符，但可以替换为其他字符，如 `#` 或 `@`。
*   **选项**：
    *   `g` - 替换一行中所有匹配的旧内容，而不仅仅是第一个。
    *   `i` - 忽略大小写进行匹配。

**示例**：
*   `:1,3 s/bin/xuegod/` - 将第1至3行中每行第一个 `bin` 替换为 `xuegod`。
*   `:1,3 s/bin/xuegod/g` - 将第1至3行中所有的 `bin` 替换为 `xuegod`。
*   `:% s/do/xuegod/gi` - 在整个文件中，忽略大小写地将所有 `do` 替换为 `xuegod`。
*   `:% s@/xuegod@do@g` - 使用 `@` 作为分隔符，将所有的 `/xuegod` 替换为 `do`。

### 自定义 Vim 环境 ⚙️

![](img/8b730af70855f36ab372431ea5511a23_38.png)

为了使 Vim 更符合个人使用习惯，我们可以对其进行临时或永久配置。

![](img/8b730af70855f36ab372431ea5511a23_40.png)

![](img/8b730af70855f36ab372431ea5511a23_42.png)

**临时设置**：在命令行模式下输入，仅对当前会话有效。
*   `:set number` 或 `:set nu` - 显示行号。
*   `:set nonumber` 或 `:set nonu` - 取消显示行号。
*   `:set hlsearch` 和 `:set nohlsearch` - 启用/取消搜索高亮。

**永久设置**：将配置写入用户主目录下的 `~/.vimrc` 文件，每次启动 Vim 时自动生效。
*   使用命令 `echo “set number” >> ~/.vimrc` 将显示行号的设置永久化。

![](img/8b730af70855f36ab372431ea5511a23_44.png)

### 多文件编辑与对比 📑

![](img/8b730af70855f36ab372431ea5511a23_46.png)

![](img/8b730af70855f36ab372431ea5511a23_48.png)

Vim 支持同时编辑多个文件，方便进行代码或配置对比。

![](img/8b730af70855f36ab372431ea5511a23_50.png)

**分屏编辑**：
*   `vim -o 文件1 文件2` - 上下分屏打开两个文件。
*   `vim -O 文件1 文件2` - 左右分屏打开两个文件。

![](img/8b730af70855f36ab372431ea5511a23_52.png)

![](img/8b730af70855f36ab372431ea5511a23_53.png)

**在分屏间切换光标**：按下 `Ctrl + W`，然后按 `W`（即按两次 `Ctrl+W`）。

![](img/8b730af70855f36ab372431ea5511a23_55.png)

![](img/8b730af70855f36ab372431ea5511a23_57.png)

![](img/8b730af70855f36ab372431ea5511a23_59.png)

**更专业的文件对比**：使用 `vimdiff` 命令可以更直观地对比文件差异。
*   `vimdiff 文件1 文件2`
*   差异处会高亮显示，同样可以使用 `Ctrl+W W` 在窗口间切换。

![](img/8b730af70855f36ab372431ea5511a23_63.png)

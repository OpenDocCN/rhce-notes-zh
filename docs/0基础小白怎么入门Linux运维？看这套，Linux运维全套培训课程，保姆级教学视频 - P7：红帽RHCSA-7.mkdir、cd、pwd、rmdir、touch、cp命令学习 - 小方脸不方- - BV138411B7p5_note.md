## mkdir命令：创建目录 📁

![](img/81d14e15a64c676f0ba9de886dd82a73_17.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_18.png)

`mkdir`命令的英文全称是`make directory`，用于创建新的目录。这类似于在Windows系统中新建一个文件夹。

![](img/81d14e15a64c676f0ba9de886dd82a73_20.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_22.png)

**命令格式**：
```bash
mkdir [选项] 目录名
```

以下是`mkdir`命令的常用操作演示：

**1. 在当前目录创建单个目录**
```bash
mkdir test
```
执行后，会在你当前所在的目录下创建一个名为`test`的目录。

**2. 在当前目录同时创建多个目录**
```bash
mkdir test1 test2 test3 test4
```
这条命令会在当前目录下一次性创建`test1`到`test4`四个目录。

**3. 在指定路径创建目录**
```bash
mkdir /opt/t1
```
这条命令会在根目录下的`/opt`目录中创建一个名为`t1`的目录。创建时，必须指定完整的路径。

**4. 递归创建多级目录（使用`-p`选项）**
有时我们需要创建具有层级结构的目录。`-p`选项可以递归地创建所有不存在的父目录。
```bash
mkdir -p /opt/t2/x1/x2/x3/x4
```
这条命令会一次性创建从`t2`到`x4`的整个目录树。如果不加`-p`选项，当上级目录不存在时，命令会报错。

**注意事项**：
*   目录名不能包含`/`字符，因为`/`在Linux中代表根目录和路径分隔符。
*   目录或文件名的长度不能超过255个字符。

## cd命令：切换目录 🔄

`cd`命令的英文全称是`change directory`，用于切换当前的工作目录。Linux系统严格区分大小写。

**命令格式**：
```bash
cd [目录名]
```

如果不指定目录名，直接输入`cd`，则会切换到当前用户的家目录。

**路径概念：绝对路径与相对路径**
*   **绝对路径**：以根目录`/`为起点，描述目标位置的完整路径。例如：`/opt/t2/x1/x2`。
*   **相对路径**：以当前所在目录为起点，描述目标位置的路径。例如，当前在`/opt`目录，要进入`t2`，只需输入`cd t2`。

以下是`cd`命令的快捷操作：

![](img/81d14e15a64c676f0ba9de886dd82a73_24.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_26.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_28.png)

**1. 切换到家目录**
```bash
cd
# 或
cd ~
```
`~`符号代表当前用户的家目录。

**2. 切换到上一级目录**
```bash
cd ..
```
`..`代表上一级目录。可以连续使用，例如`cd ../..`可以向上移动两级。

**3. 切换到当前目录**
```bash
cd .
```
`.`代表当前目录。这个用法在后续执行脚本或文件时更常见。

![](img/81d14e15a64c676f0ba9de886dd82a73_30.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_32.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_34.png)

**4. 在两个目录间快速切换**
```bash
cd -
```
`-`代表上一次所在的目录。使用此命令可以在当前目录和上一个目录之间来回切换。

## pwd命令：显示当前目录 📍

![](img/81d14e15a64c676f0ba9de886dd82a73_36.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_38.png)

`pwd`命令的英文全称是`print working directory`，用于打印（显示）当前所在工作目录的绝对路径。

![](img/81d14e15a64c676f0ba9de886dd82a73_40.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_42.png)

**命令格式**：
```bash
pwd
```
这条命令非常简单，直接执行即可。例如，如果你在一个很深的目录结构中，`pwd`可以清晰地告诉你完整的路径位置。

## rmdir命令：删除空目录 🗑️

`rmdir`命令用于删除**空目录**。如果目录内有任何文件或子目录，该命令将无法执行。

![](img/81d14e15a64c676f0ba9de886dd82a73_44.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_46.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_48.png)

**命令格式**：
```bash
rmdir 目录名
```

由于它只能删除空目录，在实际工作中使用频率较低。例如，要删除一个非空目录`/opt/t2`，你需要先逐级删除其内部的所有内容，非常麻烦。我们后续会学习更强大的`rm`命令来处理非空目录的删除。

## touch命令：创建文件 📄

![](img/81d14e15a64c676f0ba9de886dd82a73_58.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_60.png)

`touch`命令用于创建新的空白文件，类似于在Windows中“新建文本文档”。

![](img/81d14e15a64c676f0ba9de886dd82a73_62.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_64.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_66.png)

**命令格式**：
```bash
touch 文件名
```

其用法与`mkdir`命令非常相似：

![](img/81d14e15a64c676f0ba9de886dd82a73_68.png)

**1. 在当前目录创建文件**
```bash
touch test.txt
```
创建了一个名为`test.txt`的空白文件。在Linux中，文件扩展名（如`.txt`）不是必需的，主要是为了用户识别方便。

![](img/81d14e15a64c676f0ba9de886dd82a73_70.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_72.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_74.png)

**2. 同时创建多个文件**
```bash
touch file1 file2 file3
```

![](img/81d14e15a64c676f0ba9de886dd82a73_76.png)

**3. 在指定路径创建文件**
```bash
touch /opt/test1 /opt/test2
```

## cp命令：复制文件/目录 📋

![](img/81d14e15a64c676f0ba9de886dd82a73_82.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_84.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_86.png)

`cp`命令的英文全称是`copy`，用于复制文件或目录。这类似于Windows中的“复制”和“粘贴”操作。

![](img/81d14e15a64c676f0ba9de886dd82a73_88.png)

![](img/81d14e15a64c676f0ba9de886dd82a73_90.png)

**命令格式**：
```bash
cp [选项] 源文件或目录 目标文件或目录
```
*   **源**：你想要复制的文件或目录。
*   **目标**：你想要复制到的位置。

以下是`cp`命令的核心用法：

![](img/81d14e15a64c676f0ba9de886dd82a73_92.png)

**1. 复制文件**
```bash
cp test1.txt /media/
```
将当前目录下的`test1.txt`文件复制到`/media/`目录下。源文件仍然保留。

![](img/81d14e15a64c676f0ba9de886dd82a73_94.png)

**2. 复制多个文件**
```bash
cp test2.txt test3.txt /media/
```

**3. 复制时重命名**
```bash
cp test1.txt /media/abc.txt
```
将`test1.txt`复制到`/media/`目录，并重命名为`abc.txt`。

**4. 复制目录（使用`-r`选项）**
```bash
cp -r /opt/t2 /media/
```
`-r`选项代表递归复制，用于复制目录及其内部的所有子目录和文件。

**5. 保留文件属性复制（使用`-p`选项）**
```bash
cp -p test2.txt /opt/t1/
```
`-p`选项可以在复制文件时，保留原文件的属性（如修改时间、权限等）不变。这在做数据备份时非常有用。

## 总结

本节课中我们一起学习了Linux系统管理文件和目录的六个基础命令：
*   **`mkdir`**：创建目录，`-p`可递归创建。
*   **`cd`**：切换目录，理解绝对路径和相对路径是关键。
*   **`pwd`**：显示当前所在的完整路径。
*   **`rmdir`**：删除空目录（使用受限）。
*   **`touch`**：创建空白文件。
*   **`cp`**：复制文件或目录，常用选项`-r`（复制目录）和`-p`（保留属性）。

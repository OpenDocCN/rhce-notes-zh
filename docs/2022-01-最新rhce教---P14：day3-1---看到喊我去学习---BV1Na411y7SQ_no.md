## Vim可视化模式与多文件操作 📝

![](img/305b003b8bfeed4f9eca6b28946cc401_20.png)

![](img/305b003b8bfeed4f9eca6b28946cc401_22.png)

上一节我们介绍了Vim的基本操作，本节中我们来看看Vim更强大的可视化模式和多文件编辑功能。

![](img/305b003b8bfeed4f9eca6b28946cc401_24.png)

### 可视化模式

![](img/305b003b8bfeed4f9eca6b28946cc401_26.png)

可视化模式允许您像在图形界面中一样，用光标选择文本块，然后进行批量操作。

*   **进入可视化模式**：在命令模式下，按 `v` 进入字符选择模式，按 `V` 进入行选择模式，按 `Ctrl+v` 进入块选择模式。
*   **批量注释**：选中多行后，按 `I` 进入行首插入，输入注释符（如 `#`），再按 `Esc`，所选行将全部被注释。
*   **批量删除**：选中文本后，按 `d` 即可删除。
*   **批量替换**：选中文本后，按 `:` 进入命令行模式，输入 `s/旧文本/新文本/g` 进行替换。

以下是进入块选择模式并批量注释的示例：
```vim
Ctrl+v  # 进入块选择模式
j j j   # 向下选择多行
I       # 进入插入模式
#       # 输入注释符
Esc     # 退出，所选行行首均被添加#
```

### 多文件与多窗口模式

![](img/305b003b8bfeed4f9eca6b28946cc401_32.png)

![](img/305b003b8bfeed4f9eca6b28946cc401_34.png)

Vim可以同时编辑多个文件，并在多个窗口间切换。

![](img/305b003b8bfeed4f9eca6b28946cc401_36.png)

![](img/305b003b8bfeed4f9eca6b28946cc401_37.png)

![](img/305b003b8bfeed4f9eca6b28946cc401_39.png)

*   **多文件编辑**：使用 `:next` 和 `:prev` 在文件列表中切换。使用 `:args` 查看文件列表。
*   **多窗口分割**：
    *   `:sp 文件名` 水平分割窗口。
    *   `:vsp 文件名` 垂直分割窗口。
*   **窗口间切换**：按 `Ctrl+w`，然后按方向键（`h`, `j`, `k`, `l`）跳转到对应窗口。
*   **退出**：`:q` 关闭当前窗口，`:qa` 关闭所有窗口。

## 文本内容查看命令 📄

![](img/305b003b8bfeed4f9eca6b28946cc401_45.png)

![](img/305b003b8bfeed4f9eca6b28946cc401_47.png)

在Linux中，有多种命令用于查看文件内容，各有其用途。

### cat, tac, rev 命令

*   **cat**：正向连接并显示文件内容。
    *   `cat -n 文件名`：显示行号。
    *   `cat -A 文件名`：显示所有字符（包括行尾符`$`和制表符`^I`），常用于检查配置文件格式。
*   **tac**：逆向按行显示文件内容。
*   **rev**：逆向按字符显示每一行的内容。

![](img/305b003b8bfeed4f9eca6b28946cc401_49.png)

![](img/305b003b8bfeed4f9eca6b28946cc401_51.png)

以下是检查配置文件是否有多余空格的示例：
```bash
cat -A /etc/some_config.conf
# 行尾的`$`前如果有空格，将会显示出来，有助于排查问题。
```

### more 和 less 命令

用于分页查看大文件。

*   **more**：向下翻页查看。按空格键翻页，按 `q` 退出。无法向上翻页。
*   **less**：更强大的分页查看器。可上下翻页，支持搜索。
    *   空格键：向下翻页。
    *   `b` 键：向上翻页。
    *   `/关键词`：向下搜索。
    *   `?关键词`：向上搜索。
    *   `q` 键：退出。

![](img/305b003b8bfeed4f9eca6b28946cc401_53.png)

### head 和 tail 命令

![](img/305b003b8bfeed4f9eca6b28946cc401_55.png)

查看文件的开头或结尾部分。

*   **head**：查看文件开头。
    *   `head -n 20 文件名`：查看前20行。
    *   `head -c 100 文件名`：查看前100个字节。
*   **tail**：查看文件结尾，常用于监控日志。
    *   `tail -n 20 文件名`：查看末尾20行。
    *   `tail -f 文件名`：实时追踪文件新增内容（常用）。
    *   `tail -F 文件名`：实时追踪，即使文件被轮转（删除后新建同名文件）也能继续追踪（更可靠）。

以下是实时监控系统日志的示例：
```bash
tail -F /var/log/messages
```

## 管理本地用户和组 👥

![](img/305b003b8bfeed4f9eca6b28946cc401_57.png)

Linux是一个多用户系统，通过用户和组来管理权限。

### 用户和组的概念

*   **用户**：系统资源的操作者。每个用户有唯一的UID。
    *   **超级用户**：`root`，UID为0。
    *   **系统用户**：UID为1-999（RHEL 7+），用于运行系统服务。
    *   **普通用户**：UID从1000开始，由管理员创建。
*   **组**：用户的集合，用于批量分配权限。每个组有唯一的GID。
    *   **主要组**：用户创建时自动生成或指定的一个主要组。
    *   **附加组**：用户可以属于多个附加组，从而继承这些组的权限。

![](img/305b003b8bfeed4f9eca6b28946cc401_59.png)

用户信息存储在 `/etc/passwd`，密码加密信息存储在 `/etc/shadow`。组信息存储在 `/etc/group`。

![](img/305b003b8bfeed4f9eca6b28946cc401_61.png)

### 用户管理命令

以下是管理用户的核心命令：

*   **创建用户**：`useradd [选项] 用户名`
    *   `-u`：指定UID。
    *   `-g`：指定主要组。
    *   `-G`：指定附加组。
    *   `-d`：指定家目录。
    *   `-s`：指定登录shell（如 `/bin/bash` 或 `/sbin/nologin`）。
*   **修改用户属性**：`usermod [选项] 用户名` （选项与 `useradd` 类似）
*   **删除用户**：`userdel -r 用户名` （`-r` 选项同时删除家目录）
*   **设置/修改密码**：`passwd [用户名]` （不加用户名则修改当前用户密码）
*   **切换用户**：`su - 用户名` （`-` 会切换完整的环境变量）

以下是创建一个用户并指定其附加组的示例：
```bash
useradd -G wheel,developers alice  # 创建用户alice，并加入wheel和developers组
passwd alice                       # 为alice设置密码
```

### 组管理命令

*   **创建组**：`groupadd [选项] 组名`
    *   `-g`：指定GID。
*   **修改组**：`groupmod [选项] 组名`
*   **删除组**：`groupdel 组名`
*   **管理组成员**：
    *   `gpasswd -a 用户名 组名`：将用户加入组。
    *   `gpasswd -d 用户名 组名`：将用户从组中移除。

### 密码策略与账户锁定

*   **设置密码策略**：使用 `chage` 命令。
    *   `chage -l 用户名`：列出用户密码过期信息。
    *   `chage -M 90 -m 7 -W 14 用户名`：设置密码最大有效期90天，最小有效期7天，到期前14天警告。
*   **锁定与解锁用户**：
    *   `passwd -l 用户名`：锁定用户，禁止登录。
    *   `passwd -u 用户名`：解锁用户。
*   **强制用户下次登录修改密码**：`chage -d 0 用户名`

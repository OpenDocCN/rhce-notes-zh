## AWK命令结构详解

![](img/8ecb103100cb5769bb4f46425f537e41_8.png)

上一节我们介绍了AWK的基本概念，本节中我们来看看AWK命令的完整结构。一个AWK命令可以包含三个部分：`BEGIN`块、主处理块和`END`块。

**基本命令格式如下：**
```bash
awk 'BEGIN{前置命令} /模式/{主处理命令} END{结束命令}' 文件名
```
*   `BEGIN`块中的命令在处理文件**之前**执行一次。
*   主处理块中的命令会对文件中**每一行**进行匹配和处理。
*   `END`块中的命令在处理完所有行**之后**执行一次。

例如，下面的命令演示了完整的结构：
```bash
awk 'BEGIN{print "用户名\t\t解释器"} /^root/{print $1, $7} END{print "总行数:", NR}' /etc/passwd
```
这个命令会先打印表头，然后打印以`root`开头的行的第一列和第七列（用户名和解释器），最后输出文件的总行数。

## 格式化输出与注意事项

在使用AWK进行输出时，我们经常需要格式化，例如使用制表符`\t`进行对齐。

![](img/8ecb103100cb5769bb4f46425f537e41_12.png)

以下是格式化输出的一个例子：
```bash
awk 'BEGIN{print "用户名\t\t解释器"} {print $1 "\t\t" $7}' /etc/passwd
```
**需要注意**：像`\t`这样的非AWK内置语法，必须放在引号内，否则AWK无法识别。

![](img/8ecb103100cb5769bb4f46425f537e41_14.png)

AWK对命令中的空格要求不严格，但为了代码清晰，建议在适当位置添加空格。

AWK不仅仅是一个文本过滤器，它内置支持基本的数学运算。

![](img/8ecb103100cb5769bb4f46425f537e41_38.png)

![](img/8ecb103100cb5769bb4f46425f537e41_40.png)

例如，可以直接计算：
```bash
awk 'BEGIN{print 10 + 10, 10 - 5, 10 / 2, 10 % 3}'
```

更重要的是，AWK是一门完整的编程语言，它支持：
*   **条件判断**（if/else）
*   **循环**（for/while）
*   **数组**

## 实战应用：过滤系统信息

上一部分我们了解了AWK的能力，现在来看看如何用它解决`grep`难以处理的问题。以下是几个经典用例。

![](img/8ecb103100cb5769bb4f46425f537e41_58.png)

### 1. 精确提取内存信息
使用`free -h`查看内存时，如果我们只想看剩余内存（available），`grep`难以精确提取某一列。

![](img/8ecb103100cb5769bb4f46425f537e41_60.png)

![](img/8ecb103100cb5769bb4f46425f537e41_62.png)

**使用AWK可以轻松实现：**
```bash
free -h | awk '/Mem:/ {print “剩余内存：” $4}'
```
这条命令先匹配包含“Mem:”的行，然后打印其第四列，结果清晰明了。

![](img/8ecb103100cb5769bb4f46425f537e41_64.png)

### 2. 提取网卡流量信息
查看网卡`ens33`的入口流量（RX bytes）。

![](img/8ecb103100cb5769bb4f46425f537e41_66.png)

**使用`grep`过滤比较繁琐，而AWK更直接：**
```bash
ifconfig ens33 | awk '/RX packets/ {print “入口流量：” $5}'
```
通过匹配“RX packets”行并打印第五列，我们直接得到了入口流量数据。可以进一步组合命令，同时输出入口和出口流量。

### 3. 监控脚本示例
结合循环和判断，AWK可以用于简单的监控脚本。例如，监控根分区使用情况：

```bash
#!/bin/bash
while true
do
    # 提取根分区剩余空间（带单位）
    available=$(df -h | awk '/\/$/ {print $4}')
    echo “根分区剩余空间：$available”
    # 可以在此处添加判断逻辑，如空间不足时告警
    sleep 5
done
```
这个脚本每5秒检查一次根分区的剩余空间。通过`awk`精确匹配以`/`结尾的行（根分区）并提取第四列可用空间。

## 开机自启与计划任务

有时我们需要让监控脚本开机自动运行。

**一种方法是使用`/etc/rc.d/rc.local`文件：**
1.  确保`/etc/rc.d/rc.local`文件有执行权限：`chmod +x /etc/rc.d/rc.local`
2.  将你的脚本执行命令（建议使用绝对路径）添加到这个文件中。
3.  系统启动时就会自动执行该文件内的命令。

![](img/8ecb103100cb5769bb4f46425f537e41_70.png)

![](img/8ecb103100cb5769bb4f46425f537e41_72.png)

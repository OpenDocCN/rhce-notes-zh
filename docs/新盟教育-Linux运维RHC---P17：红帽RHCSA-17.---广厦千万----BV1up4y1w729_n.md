## 修改用户属性

上一节我们介绍了如何创建用户，本节中我们来看看如何修改已存在的用户信息。当用户创建后，我们可能需要更改其UID、家目录、描述信息或所属组等属性。

修改用户属性的命令是 `usermod`，其选项与 `useradd` 命令基本一致，因此无需额外记忆。

以下是 `usermod` 命令的常用操作示例：

*   **修改用户UID**：将用户 `user1` 的UID改为 `6666`。
    ```bash
    usermod -u 6666 user1
    ```
*   **修改用户描述信息**：为用户 `user1` 添加描述信息。
    ```bash
    usermod -c "邮箱 xxx@163.com" user1
    ```
*   **将用户加入附加组**：将用户 `user1` 加入到 `yunwei`、`kaifa`、`ceshi` 组。
    ```bash
    usermod -G yunwei,kaifa,ceshi user1
    ```
*   **修改用户登录Shell**：将用户 `user1` 的登录Shell改为 `/bin/sh`。
    ```bash
    usermod -s /bin/sh user1
    ```

> **注意**：修改用户属性时，通常需要该用户已退出系统登录状态。

## 删除用户

![](img/eb5058828f2891e175aa2885c527fd19_12.png)

删除操作相对简单。`userdel` 命令用于删除给定的用户。

![](img/eb5058828f2891e175aa2885c527fd19_14.png)

以下是删除用户的相关操作：

*   **仅删除用户账号**：此操作不会删除用户的家目录。
    ```bash
    userdel user1
    ```
*   **删除用户及其家目录**：使用 `-r` 选项可以同时删除用户及其家目录。
    ```bash
    userdel -r user100
    ```

> **提示**：`userdel -r` 命令只会删除用户的家目录。如果用户在系统其他位置创建了文件，需要手动清理。

## 组管理基础

![](img/eb5058828f2891e175aa2885c527fd19_21.png)

![](img/eb5058828f2891e175aa2885c527fd19_23.png)

组是用户的集合，用于简化权限管理。创建组的命令是 `groupadd`。

*   **创建新组**：创建一个名为 `student` 的组。
    ```bash
    groupadd student
    ```

组的信息存储在 `/etc/group` 文件中。每一行代表一个组，字段以冒号分隔。

以下是 `/etc/group` 文件字段的含义：
*   **组名**：组的名称。
*   **组密码占位符**：通常为 `x`，真实密码存储在 `/etc/gshadow` 文件中。
*   **组ID (GID)**：组的唯一数字标识。
*   **组成员列表**：属于该组的用户列表，用户名之间用逗号分隔。

## 修改组属性

![](img/eb5058828f2891e175aa2885c527fd19_27.png)

![](img/eb5058828f2891e175aa2885c527fd19_29.png)

对于已存在的组，我们可以使用 `groupmod` 命令修改其属性。

![](img/eb5058828f2891e175aa2885c527fd19_30.png)

以下是修改组属性的操作：

*   **修改组ID**：将 `student` 组的GID改为 `5678`。
    ```bash
    groupmod -g 5678 student
    ```
*   **修改组名**：将 `student` 组改名为 `studgrp`。
    ```bash
    groupmod -n studgrp student
    ```

## 管理组成员

`gpasswd` 命令是管理组成员的主要工具，可用于添加或删除组内用户。

![](img/eb5058828f2891e175aa2885c527fd19_36.png)

以下是管理组成员的操作：

*   **将用户加入组**：将用户 `user6` 加入到 `studgrp` 组。
    ```bash
    gpasswd -a user6 studgrp
    ```
*   **将用户移出组**：将用户 `user6` 从 `studgrp` 组中移除。
    ```bash
    gpasswd -d user6 studgrp
    ```

## 删除组

删除组的命令是 `groupdel`。删除操作很简单，但需注意，用户的主组（基本组）不允许被删除。

![](img/eb5058828f2891e175aa2885c527fd19_42.png)

![](img/eb5058828f2891e175aa2885c527fd19_44.png)

*   **删除组**：删除 `studgrp` 组。
    ```bash
    groupdel studgrp
    ```

![](img/eb5058828f2891e175aa2885c527fd19_46.png)

> **注意**：如果尝试删除某个用户的主组，系统会提示“不能移除用户的主组”。

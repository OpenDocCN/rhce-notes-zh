在本节课中，我们将要学习Linux Shell脚本中的`while`循环。`while`循环与之前学习的`for`循环功能类似，都是用于重复执行命令，但它们的应用场景和循环机制有所不同。我们将通过具体的例子，了解`while`循环的基本语法、如何控制循环次数，以及它在实际脚本中的应用。

## 概述 📋

![](img/2c4e2525564fbb9985bd3f5432936397_4.png)

![](img/2c4e2525564fbb9985bd3f5432936397_6.png)

`while`循环是一种条件循环，只要给定的条件成立，它就会重复执行循环体内的命令。这与`for`循环基于一个值列表进行迭代的方式不同。`while`循环更适用于那些循环次数未知，需要根据条件动态判断的场景。

![](img/2c4e2525564fbb9985bd3f5432936397_8.png)

![](img/2c4e2525564fbb9985bd3f5432936397_9.png)

```bash
while [ 条件 ]

    # 要执行的命令

```

只要“条件”判断为真（成立），`do`和`done`之间的命令就会被重复执行。

上一节我们介绍了`while`循环的基本概念，本节中我们来看看它的一个简单应用：创建一个无限循环。

### 无限循环示例

![](img/2c4e2525564fbb9985bd3f5432936397_23.png)

![](img/2c4e2525564fbb9985bd3f5432936397_25.png)

以下是一个最简单的`while`无限循环脚本：

![](img/2c4e2525564fbb9985bd3f5432936397_27.png)

```bash
#!/bin/bash
while [ 1 -eq 1 ]
do
    echo "Hello, baby"
done
```

![](img/2c4e2525564fbb9985bd3f5432936397_29.png)

在这个例子中，条件 `[ 1 -eq 1 ]` 永远为真，因此脚本会不停地输出“Hello, baby”，直到我们手动按下 `Ctrl+C` 来终止它。这演示了`while`循环“条件成立即执行”的核心特性。

## 控制循环次数 🔢

虽然`while`循环常与未知次数的循环关联，但我们也可以通过变量来控制其循环次数，使其像`for`循环一样执行固定的次数。

以下是实现循环5次的示例：

```bash
#!/bin/bash
number=1
while [ $number -le 5 ]
do
    echo "Hello, baby"
    let number++  # 等同于 number=$((number + 1))
done
```

**代码解释：**
1.  我们首先定义一个变量 `number` 并赋值为1。
2.  循环条件是 `$number -le 5`，即“变量`number`的值小于或等于5”。
3.  在循环体内，我们输出信息后，使用 `let number++` 命令将变量 `number` 的值增加1。
4.  当 `number` 的值增加到6时，条件 `[ 6 -le 5 ]` 不再成立，循环结束。

这样，我们就实现了让`while`循环执行5次。但需要注意的是，如果需要固定次数的循环，使用`for`循环通常更为直观和简洁。

## 特殊的无限循环符号：冒号（:） ♾️

![](img/2c4e2525564fbb9985bd3f5432936397_36.png)

![](img/2c4e2525564fbb9985bd3f5432936397_38.png)

在Shell脚本中，单个冒号 `:` 是一个特殊的命令，它什么都不做但永远返回真（退出状态码为0）。因此，它常被用来构造一个简洁的无限循环。

![](img/2c4e2525564fbb9985bd3f5432936397_40.png)

![](img/2c4e2525564fbb9985bd3f5432936397_41.png)

![](img/2c4e2525564fbb9985bd3f5432936397_43.png)

以下是使用冒号的无限循环示例：

![](img/2c4e2525564fbb9985bd3f5432936397_45.png)

![](img/2c4e2525564fbb9985bd3f5432936397_47.png)

```bash
#!/bin/bash
while :
do
    echo "这是一个无限循环"
done
```

![](img/2c4e2525564fbb9985bd3f5432936397_49.png)

这个脚本会一直输出“这是一个无限循环”。这是一种更简洁的表示无限循环的方式，无需编写具体的条件判断。

```bash

number=$((RANDOM % 10))

![](img/2c4e2525564fbb9985bd3f5432936397_66.png)

while :
do
    read -p "请输入你猜的数字（0-9）: " guess

    then
        echo "恭喜你猜对了！哇塞女孩的编号是 $number！"
        exit 0  # 猜对后退出脚本
    else
        echo "很遗憾，猜错了。离哇塞女孩还差一点点，再试试吧！"
```

**脚本逻辑：**
1.  使用 `$((RANDOM % 10))` 生成一个0到9的随机数。
2.  进入一个由 `while :` 定义的无限循环。
3.  在循环内，提示用户输入猜测的数字。
4.  使用 `if` 判断用户输入是否等于随机数。
    *   如果相等，输出恭喜信息，并使用 `exit 0` 命令退出整个脚本。
    *   如果不相等，提示猜错，循环继续，让用户再次输入。

本节课中我们一起学习了`while`循环。
*   **核心概念**：`while`循环是一种**条件循环**，只要条件成立，就会重复执行循环体内的命令。其基本公式为：`while [ condition ]; do commands; done`。
*   **与for循环的区别**：`for`循环基于一个给定的列表进行迭代，循环次数通常是已知的；而`while`循环基于条件判断，循环次数可能未知，更适用于需要持续监控或等待的场景。
*   **控制循环**：可以通过在循环体内改变条件判断所依赖的变量值（如使用 `let` 命令递增）来控制`while`循环的次数。
*   **无限循环**：可以使用永远为真的条件（如 `[ 1 -eq 1 ]`）或特殊的冒号 `:` 命令来创建无限循环。
*   **实际应用**：我们通过编写一个可连续猜数字的游戏脚本，实践了如何利用`while`循环和 `exit` 命令来构建一个交互式的、直到满足条件才退出的程序。

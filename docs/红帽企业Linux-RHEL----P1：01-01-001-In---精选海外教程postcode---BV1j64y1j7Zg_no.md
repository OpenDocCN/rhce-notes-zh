## 课程目标与认证路径 🎯

![](img/b5f8957deca52a822c4ba2f8883608a3_2.png)

本课程的核心目标是帮助学习者精通红帽企业Linux 9系统。课程内容全面覆盖了RHCSA和RHCE认证考试所需的知识与技能。

![](img/b5f8957deca52a822c4ba2f8883608a3_3.png)

以下是本课程将涵盖的主要认证路径：

![](img/b5f8957deca52a822c4ba2f8883608a3_4.png)

*   **RHCSA (红帽认证系统管理员)**：这是系统管理的基础认证，证明你具备执行核心系统管理任务的能力。
*   **RHCE (红帽认证工程师)**：这是更高级的认证，侧重于自动化、安全和服务管理，建立在RHCSA的基础之上。

![](img/b5f8957deca52a822c4ba2f8883608a3_5.png)

上一节我们介绍了课程的整体目标，本节中我们来看看课程内容的具体设计。

## 课程内容与结构 📚

![](img/b5f8957deca52a822c4ba2f8883608a3_7.png)

![](img/b5f8957deca52a822c4ba2f8883608a3_8.png)

课程内容经过精心设计，从基础到高级，循序渐进。它不仅讲解理论知识，更注重实践操作，确保学习者能够真正掌握技能。

![](img/b5f8957deca52a822c4ba2f8883608a3_9.png)

以下是课程内容的主要组成部分：

![](img/b5f8957deca52a822c4ba2f8883608a3_10.png)

*   **Linux基础**：包括命令行操作、文件系统管理和用户权限控制。
*   **系统管理**：涵盖软件包管理、进程控制、服务管理和存储配置。
*   **网络与安全**：讲解网络配置、防火墙管理和SELinux安全策略。
*   **自动化与管理**：重点介绍Ansible自动化工具，这是RHCE认证的核心内容。

## 核心概念示例：简单的命令行

![](img/b5f8957deca52a822c4ba2f8883608a3_21.png)

为了让大家对课程内容有一个直观感受，这里展示一个最基础的Linux命令，用于列出当前目录下的文件：

![](img/b5f8957deca52a822c4ba2f8883608a3_22.png)

```bash
ls -la
```

![](img/b5f8957deca52a822c4ba2f8883608a3_23.png)

这个命令中，`ls`是命令本身，`-l`选项表示以长格式显示详细信息，`-a`选项表示显示所有文件（包括隐藏文件）。

![](img/b5f8957deca52a822c4ba2f8883608a3_24.png)

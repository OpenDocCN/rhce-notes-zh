## 关于`ls -d`命令的补充说明 🔍

上一节我们介绍了`ls`命令，本节中我们来看看`ls -d`选项的具体含义。有同学问到`ls -d`是否只显示目录，这里需要明确一点：`ls -d`选项的作用是**仅显示指定参数本身的属性**，而不查看其下的内容。

**核心概念**：`ls -d` 不进行文件类型过滤，它只是“只看你给它的那个东西自己”，无论它是目录还是文件。

例如，查看目录`/a/b/c`下的所有条目（包括文件和目录）本身的属性：
```bash
ls -d /a/b/c/*
```
即使`*`通配符匹配到了非目录的文件，`-d`选项也会将它们列出，只是将其视为一个独立的条目查看其属性。

如果希望只过滤出目录，可以结合`ls -l`和管道命令（后续会学到）来实现。例如，只列出`/boot`目录下的所有子目录：
```bash
ls -l /boot | grep ^d
```
这条命令会列出`/boot`下所有以`d`（代表directory）开头的条目，即所有子目录。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_12.png)

因此，`ls -d`的准确描述是：**即使参数是一个目录，并且其下包含内容，使用`-d`选项也只会显示该目录本身的信息，而不会显示其下的内容。**

## 查看隐藏文件 👁️

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_18.png)

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_20.png)

在Linux系统中，隐藏文件是指名称以点号`.`开头的文件或目录。默认情况下，`ls`命令不会显示它们。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_22.png)

**核心概念**：使用 `ls -A` 或 `ls -a` 可以列出所有文件，包括隐藏文件。

例如，在root用户的主目录下，默认只显示一个文件。但实际上存在许多系统隐藏的配置文件。要查看它们，只需：
```bash
ls -A /root
```
你会发现许多以`.`开头的隐藏目录和文件。`-A`选项会显示除`.`（当前目录）和`..`（上级目录）之外的所有隐藏项。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_24.png)

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_26.png)

## vi/vim文本编辑器入门 ✏️

下面，我们将进入本节课的核心内容：学习使用vi/vim文本编辑器。vi是Unix/Linux系统中默认的文本编辑器，而vim（Vi IMproved）是它的增强版，功能更强大。在最小化安装的系统中，通常只预装了vi。

### vi/vim简介

*   **vi**：Visual Interface的缩写，是Unix/Linux系统的默认文本编辑器。
*   **vim**：Vi IMproved，是vi的增强版，提供了语法高亮、更好的脚本支持等特性。它需要单独安装软件包`vim-enhanced`。

在工作中，使用vim更为普遍。如果系统没有vim，使用vi即可，基本操作是相同的。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_28.png)

### 编辑器的基本目标

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_30.png)

使用vi/vim编辑器，主要需要掌握两个功能：
1.  **修改**已存在的文件。
2.  **创建**新的文件。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_32.png)

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_33.png)

### 启动编辑器

启动编辑器并指定要编辑的文件非常简单：
```bash
vi 文件名
```
或
```bash
vim 文件名
```
如果指定的“文件名”不存在，编辑器会创建一个新的临时文件；如果文件已存在，则打开它进行编辑。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_35.png)

### 三种工作模式 🎭

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_38.png)

理解vi/vim的三种工作模式是熟练使用它的关键。你可以将其想象为变形金刚的不同形态，每种形态有不同的能力。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_40.png)

以下是三种模式的介绍：

1.  **命令模式 (Command Mode)**：启动vi/vim后默认进入的模式。在此模式下，可以执行复制、粘贴、删除、查找等操作，但不能直接输入文本。
2.  **输入模式 (Insert Mode)**：在此模式下，可以像在普通文本编辑器中一样输入、删除文本。
3.  **末行模式 (Ex Mode)**：在此模式下，可以执行保存文件、退出编辑器、查找替换等需要输入命令的操作。

### 模式切换与基本操作流程 🔄

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_42.png)

现在，我们来看看如何在这三种模式间切换，并完成一次完整的文件编辑。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_44.png)

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_45.png)

**基本操作流程如下：**

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_47.png)

1.  **打开文件**：使用 `vi 文件名` 命令。
2.  **进入输入模式**：在默认的**命令模式**下，按下字母键 **`i`**，屏幕下方出现`-- INSERT --`提示，即进入**输入模式**。此时可以输入文本内容。
3.  **返回命令模式**：在**输入模式**下，按下 **`Esc`** 键，即可退回到**命令模式**。（`Esc`键可以从任何模式返回到命令模式）。
4.  **进入末行模式**：在**命令模式**下，输入英文冒号 **`:`**，光标会跳转到屏幕最底行，并显示一个`:`，即进入**末行模式**。
5.  **保存与退出**：在**末行模式**下，可以输入以下命令：
    *   **`:wq`**：保存（Write）所有修改并退出（Quit）编辑器。
    *   **`:q!`**：放弃所有修改，强制退出（Quit）编辑器。
    *   **`:q`**：如果文件没有修改，可以直接退出。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_49.png)

**流程总结公式**：
`vi 文件名` -> **(命令模式)** -> 按 `i` -> **(输入模式)** -> 编辑文本 -> 按 `Esc` -> **(命令模式)** -> 按 `:` -> **(末行模式)** -> 输入 `wq` -> **保存并退出**。

### 实践示例

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_51.png)

1.  **创建并编辑新文件**：
    ```bash
    vi myfile.txt
    ```
    按 `i`，输入“This is my first vi file.”，按 `Esc`，输入 `:wq`。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_53.png)

2.  **查看编辑结果**：
    ```bash
    cat myfile.txt
    ```

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_55.png)

3.  **修改已有文件**：
    ```bash
    vi myfile.txt
    ```
    按 `i`，在文件末尾添加新的一行“Adding a new line.”，按 `Esc`，输入 `:wq`。

![](img/e438d1c9a5a26f6db6f87aa95d93dabe_57.png)

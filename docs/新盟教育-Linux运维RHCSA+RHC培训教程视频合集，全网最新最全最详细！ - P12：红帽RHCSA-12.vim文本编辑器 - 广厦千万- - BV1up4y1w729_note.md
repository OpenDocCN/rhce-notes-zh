## 什么是文本编辑器？🖋️

![](img/b024c33972034e6b564a1ea5a9d63fe3_21.png)

在Windows系统中，我们通常使用“记事本”来创建和编辑文本文件。例如，右击桌面选择“新建文本文档”，然后双击打开它，输入内容并保存。这个“记事本”就是Windows系统中的一个文本编辑器。

在Linux系统中，最常用的文本编辑器是`vi`和`vim`。`vim`是`vi`的增强版本，两者在核心操作上基本一致。主要区别在于，`vim`支持语法高亮，更适合编程和脚本编写，而`vi`则没有颜色提示功能。对于日常的配置文件编辑，两者都可以胜任。

### 命令格式
使用vim编辑文件的基本命令格式非常简单：
```bash
vim 文件名
```
这个命令有两个特点：
1.  若文件不存在，vim会先创建该文件，然后打开进行编辑。
2.  若文件已存在，vim会直接打开该文件进行编辑。

例如，编辑一个已存在的系统主机名文件：
```bash
vim /etc/hostname
```
编辑一个不存在的文件（vim会自动创建它）：
```bash
vim /opt/xxxx.txt
```

### vim的三种模式
vim的操作逻辑基于三种模式，这是初学者需要掌握的核心概念。

上一节我们介绍了vim的基本命令格式，本节中我们来看看它的三种工作模式。

1.  **命令模式**：启动vim后，默认进入的模式。在此模式下，键盘输入会被识别为各种编辑命令，而不是文本内容。
2.  **输入模式**：在此模式下，可以向文件中输入或编辑文本内容。
3.  **底线命令模式**：在此模式下，可以执行保存、退出、搜索替换等高级操作。

## 模式详解与常用操作

### 从命令模式进入输入模式
在命令模式下，按 `i`、`a`、`o` 等键可以进入输入模式。它们的区别在于光标的位置和插入行为：
*   `i`：在**当前光标所在字符前**插入。
*   `a`：在**当前光标所在字符后**插入。
*   `o`：在**当前光标所在行的下方**插入一个新行，并开始输入。

对于初学者，记住 `i`（insert）即可，其他键的区别可以通过移动光标来弥补。

![](img/b024c33972034e6b564a1ea5a9d63fe3_45.png)

![](img/b024c33972034e6b564a1ea5a9d63fe3_46.png)

### 从输入模式返回命令模式
在输入模式下，按键盘左上角的 `Esc` 键，即可退出输入模式，返回到命令模式。

![](img/b024c33972034e6b564a1ea5a9d63fe3_48.png)

![](img/b024c33972034e6b564a1ea5a9d63fe3_50.png)

### 从命令模式进入底线命令模式
在命令模式下，按英文冒号 `:`，窗口底部会出现一个冒号提示符，此时即进入了底线命令模式，可以输入各种底线命令。

### 保存与退出操作
保存和退出操作主要在底线命令模式下完成。

以下是常用的保存与退出命令：
*   `:w`：保存文件。
*   `:q`：退出vim（如果文件有修改且未保存，会提示错误）。
*   `:wq` 或 `:x`：保存文件并退出vim。
*   `ZZ`（**命令模式下，大写Z按两次**）：快速保存并退出。
*   `:q!`：强制退出，不保存任何修改。
*   `:wq!`：强制保存并退出（通常用于修改只读文件，仅root用户可用）。

## 命令模式下的高效编辑命令
掌握了模式切换和保存退出后，我们来看看在命令模式下有哪些高效的编辑命令。

### 光标移动与跳转
*   `0`（数字零）：将光标移动到**行首**。
*   `$`：将光标移动到**行尾**。
*   `gg`：跳转到**文件第一行**。
*   `G`（大写）：跳转到**文件最后一行**。
*   `:数字`：在底线命令模式下，输入行号可直接跳转到指定行（如 `:1000` 跳转到第1000行）。

### 删除、剪切与复制
*   `x`：删除**当前光标所在**的字符。
*   `dd`：剪切（删除）**当前光标所在**的整行。它实际上是剪切操作，如果不粘贴，效果就是删除。
*   `数字dd`：剪切从当前行开始的指定行数（如 `3dd` 剪切当前行及下面两行，共三行）。
*   `p`：将剪切或复制的内容**粘贴**到当前光标所在行的下方。
*   `yy`：复制**当前光标所在**的整行。
*   `数字yy`：复制从当前行开始的指定行数（如 `2yy` 复制当前行及下一行）。
*   `C`（大写）：删除从光标位置到**行尾**的所有内容，并进入输入模式。

![](img/b024c33972034e6b564a1ea5a9d63fe3_52.png)

### 撤销与搜索
*   `u`：撤销上一次的修改。可以连续按 `u` 来撤销更早的操作，直到文件恢复到打开时的初始状态（**注意：保存退出后无法撤销**）。
*   `/关键字`：在命令模式下，输入 `/` 后跟要搜索的关键词，按回车即可进行搜索。
*   `n`：在搜索后，按 `n` 可以快速定位到下一个匹配项。

![](img/b024c33972034e6b564a1ea5a9d63fe3_54.png)

## 底线命令模式的高级功能
除了保存退出，底线命令模式还提供了一些提升编辑效率的功能。

![](img/b024c33972034e6b564a1ea5a9d63fe3_56.png)

### 显示与取消行号
*   `:set nu`：显示文件的行号。
*   `:set nonu`：取消行号显示。

### 全局搜索与替换
替换是vim中非常强大的一个功能，语法如下：
```vim
:%s/旧内容/新内容/g
```
*   `%`：代表对整个文件进行操作。
*   `s`：代表替换（substitute）。
*   `/旧内容/新内容/`：将“旧内容”替换为“新内容”。
*   `g`：代表全局（global）替换。如果不加 `g`，vim默认只替换每一行中第一个匹配到的“旧内容”。

例如，将文件中所有的“port”替换为“xxxx”：
```vim
:%s/port/xxxx/g
```

![](img/b024c33972034e6b564a1ea5a9d63fe3_58.png)

### 取消搜索高亮
执行搜索后，匹配的词汇会高亮显示。如果不希望看到高亮，可以执行：
```vim
:nohl
```

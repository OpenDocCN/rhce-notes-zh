## 命令模式、输入模式与末行模式 🔄

vi/vim编辑器有三种主要的工作模式，理解并能在它们之间切换是使用该编辑器的关键。

*   **命令模式**：这是启动vi/vim后的默认模式。在此模式下，可以执行复制、粘贴、删除、查找等操作，但不能直接输入文本。
*   **输入模式**：在此模式下，可以向文件中输入或编辑文本内容。
*   **末行模式**：在此模式下，可以执行保存文件、退出编辑器、查找替换等需要输入命令的操作。

![](img/50ee1b979b569c129693524774b03439_10.png)

这三种模式的关系可以概括为：**命令模式是核心枢纽**。从命令模式可以进入输入模式或末行模式，而从其他模式通常需要先返回命令模式。

## 基本使用流程 🛠️

![](img/50ee1b979b569c129693524774b03439_20.png)

上一节我们介绍了vi/vim的三种模式，本节中我们来看看如何使用它们完成创建或编辑文件的基本操作。

![](img/50ee1b979b569c129693524774b03439_22.png)

### 打开或创建文件

![](img/50ee1b979b569c129693524774b03439_24.png)

使用 `vi` 或 `vim` 命令后接文件名即可。如果文件不存在，编辑器会创建一个新文件；如果文件存在，则打开它进行编辑。

```bash
vi myfile.txt
# 或
vim myfile.txt
```

### 进入输入模式并编辑

打开文件后，默认处于命令模式。要输入文字，需要切换到输入模式。最常用的方法是按下字母键 **`i`**。此时编辑器底部通常会显示 `-- INSERT --` 或类似提示，表示已进入输入模式。此时，你可以像使用普通文本编辑器一样输入内容。

![](img/50ee1b979b569c129693524774b03439_26.png)

### 保存并退出文件

![](img/50ee1b979b569c129693524774b03439_28.png)

编辑完成后，需要保存更改并退出编辑器。以下是操作步骤：

![](img/50ee1b979b569c129693524774b03439_30.png)

![](img/50ee1b979b569c129693524774b03439_31.png)

1.  按 **`Esc`** 键，从当前模式（通常是输入模式）**返回命令模式**。
2.  输入英文冒号 **`:`**，进入**末行模式**。光标会移动到编辑器底部。
3.  在冒号后输入退出命令并回车。
    *   输入 **`wq`** 并回车：保存文件并退出编辑器。
    *   输入 **`q!`** 并回车：放弃所有修改，强制退出编辑器。

以下是基本操作流程的命令序列总结：

![](img/50ee1b979b569c129693524774b03439_33.png)

![](img/50ee1b979b569c129693524774b03439_34.png)

![](img/50ee1b979b569c129693524774b03439_36.png)

```
i -> 输入文本 -> Esc -> :wq -> 回车
```

![](img/50ee1b979b569c129693524774b03439_38.png)

### 查看编辑结果

退出编辑器后，可以使用 `cat` 命令查看文件内容，确认编辑是否成功。

```bash
cat myfile.txt
```

### 关于vi与vim

![](img/50ee1b979b569c129693524774b03439_49.png)

*   `vi` 是Unix/Linux系统中**默认**的文本编辑器，在最小化安装的系统上一定存在。
*   `vim` 是 `vi` 的增强版本，提供了语法高亮、更强大的搜索等特性，体验更好。但它是一个独立的软件包，可能需要手动安装。
*   在Red Hat系列系统（如RHEL、CentOS）中，可以通过以下命令安装vim增强包：
    ```bash
    yum -y install vim-enhanced
    ```

### 模式切换总结

![](img/50ee1b979b569c129693524774b03439_51.png)

为了方便记忆，以下是模式切换的核心快捷键：

![](img/50ee1b979b569c129693524774b03439_53.png)

*   **命令模式 -> 输入模式**：按 **`i`** 键。
*   **任何模式 -> 命令模式**：按 **`Esc`** 键。
*   **命令模式 -> 末行模式**：按 **`:`** 键。

![](img/50ee1b979b569c129693524774b03439_55.png)

你可以将vi/vim想象成一个“变形金刚”：**命令模式**是它的默认机器人形态；按下 `i` 咒语，它变形成**输入模式**——一辆可以“写入”文字的车；按下 `:` 咒语，它则变形成**末行模式**——一个能执行保存、退出等高级指令的形态。`Esc` 键就是让它变回默认形态的通用口令。

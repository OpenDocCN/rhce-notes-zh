### 可视块模式
可视块模式常用于编程或修改服务器配置文件时，需要进行多行注释或批量编辑。

进入可视块模式的方法如下：
在正常模式下按下 `Ctrl + V` 即可进入列编辑模式，也称为可视块模式。你可以向上或向下移动光标来选中文本块。

![](img/55662b1927f127ffb4d8b2389015ad45_20.png)

以下是可视块模式的核心操作：

![](img/55662b1927f127ffb4d8b2389015ad45_22.png)

![](img/55662b1927f127ffb4d8b2389015ad45_24.png)

*   **批量删除**：选中文本块后，按下小写 `d` 即可删除选中内容。
*   **批量插入**：选中文本块后，按下大写 `I`，进入插入模式并输入内容（如注释符`#`），然后按下 `Esc`，Vim会自动在所有选中行的行首插入你输入的内容。

![](img/55662b1927f127ffb4d8b2389015ad45_26.png)

**示例：批量注释与取消注释**
假设我们编辑SSH配置文件 `/etc/ssh/sshd_config`，想批量取消或添加行首的注释符 `#`。
1.  进入可视块模式：`Ctrl + V`。
2.  用方向键选中多行行首的 `#` 号。
3.  要删除注释（取消注释），按下 `d`。
4.  要添加注释，选中行首区域后，按下 `I`，输入 `#`，再按下 `Esc`。

![](img/55662b1927f127ffb4d8b2389015ad45_30.png)

### 命令行模式
命令行模式用于执行保存、退出、查找替换等高级操作。在正常模式下输入 `:` 即可进入。

![](img/55662b1927f127ffb4d8b2389015ad45_34.png)

以下是常用的命令行指令：

![](img/55662b1927f127ffb4d8b2389015ad45_36.png)

*   **保存与退出**：
    *   `:w` - 保存文件。
    *   `:w!` - 强制保存（例如对只读文件）。
    *   `:q` - 退出（仅当文件未修改时有效）。
    *   `:q!` - 不保存强制退出。
    *   `:wq` 或 `:x` - 保存并退出。
    *   `:wq!` - 强制保存并退出。
*   **恢复文件**：`:e!` - 放弃所有修改，恢复到文件打开时的状态。
*   **快捷保存退出**：在正常模式下，按下 `ZZ`（两个大写Z）可以快速保存并退出。

**示例：编辑只读文件**
编辑没有写入权限的文件（如 `/etc/shadow`）时，修改后需要使用 `:wq!` 来强制保存退出。

### 在Vim中执行外部命令与读取文件
在命令行模式下，可以方便地调用系统命令或读取其他文件内容。

*   **执行外部命令**：输入 `:!` 后跟上命令，例如 `:!ifconfig` 可以查看网络信息，结果会显示在Vim下方。
*   **读取文件内容**：输入 `:r` 后跟上文件路径，可以将指定文件的内容插入到当前光标位置。例如，`:r /etc/hosts`。

![](img/55662b1927f127ffb4d8b2389015ad45_38.png)

![](img/55662b1927f127ffb4d8b2389015ad45_40.png)

### 文本查找与替换
Vim提供了强大的文本替换功能，其基本语法格式为：
`:[范围]s/旧内容/新内容/[选项]`

![](img/55662b1927f127ffb4d8b2389015ad45_42.png)

*   **范围**：`%` 表示全文，`1,10` 表示第1到10行。
*   **选项**：
    *   `g` - 替换行内所有匹配项（默认只替换每行第一个）。
    *   `i` - 忽略大小写。

![](img/55662b1927f127ffb4d8b2389015ad45_44.png)

**示例：**
*   `:1,3 s/bin/xuegao/` - 将第1到3行中每行第一个 `bin` 替换为 `xuegao`。
*   `:1,3 s/bin/xuegao/g` - 将第1到3行中所有的 `bin` 替换为 `xuegao`。
*   `:% s/do/xuegao/gi` - 全文忽略大小写，将所有 `do` 和 `DO` 替换为 `xuegao`。
*   分隔符可以自定义，例如 `:% s#/old#/new#g`。

### 自定义Vim环境
你可以临时或永久地设置Vim的工作环境以提升效率。

![](img/55662b1927f127ffb4d8b2389015ad45_52.png)

![](img/55662b1927f127ffb4d8b2389015ad45_53.png)

![](img/55662b1927f127ffb4d8b2389015ad45_55.png)

*   **临时设置（仅当前会话有效）**：
    *   `:set number` 或 `:set nu` - 显示行号。
    *   `:set nonumber` 或 `:set nonu` - 取消显示行号。
*   **永久设置（对用户生效）**：
    在用户家目录下创建或编辑配置文件 `~/.vimrc`，将设置命令写入其中。例如，添加 `set number` 这一行，以后每次用Vim打开文件都会显示行号。
    ```bash
    echo ‘set number‘ >> ~/.vimrc
    ```

![](img/55662b1927f127ffb4d8b2389015ad45_57.png)

![](img/55662b1927f127ffb4d8b2389015ad45_59.png)

### 分屏编辑多个文件
Vim允许同时编辑多个文件，方便对比和参考。

![](img/55662b1927f127ffb4d8b2389015ad45_61.png)

![](img/55662b1927f127ffb4d8b2389015ad45_63.png)

*   **上下分屏**：使用 `vim -o 文件1 文件2` 命令打开。
*   **左右分屏**：使用 `vim -O 文件1 文件2` 命令打开。
*   **在分屏间切换光标**：按下 `Ctrl + W`，然后按 `W`（即按两次 `Ctrl+W`）。
*   **退出所有分屏**：在命令行模式下输入 `:qa`。
*   **更直观的文件对比**：使用 `vimdiff 文件1 文件2` 命令，它会高亮显示两个文件的不同之处，是进行代码或配置对比的利器。

![](img/55662b1927f127ffb4d8b2389015ad45_65.png)

### 其他编辑器简介
Linux系统下还有其他图形化或简单的文本编辑器，例如：
*   `gedit`：类似于Windows记事本的图形化编辑器。
*   `nano`：一个简单易用的终端文本编辑器，操作提示更友好。
对于运维工作而言，熟练掌握Vim是核心要求，其他编辑器可以作为辅助工具了解。

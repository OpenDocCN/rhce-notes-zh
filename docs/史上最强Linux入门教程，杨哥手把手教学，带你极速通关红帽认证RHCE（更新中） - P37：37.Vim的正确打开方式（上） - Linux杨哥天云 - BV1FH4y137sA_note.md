上一节我们介绍了Vim的基本概念和模式切换。本节中，我们来看看如何在命令模式下高效地编辑文本。

首先，我们需要一个用于练习的文本文件。以下命令将创建一个包含系统文件列表前12行的新文件 `file.ts`。

![](img/3ca532cd9e6016964ce538a80b7dba6d_19.png)

```bash
ls -a | head -n 12 > file.ts
```

![](img/3ca532cd9e6016964ce538a80b7dba6d_21.png)

现在，使用Vim打开这个文件进行编辑：

```bash
vim file.ts
```

打开文件后，默认进入的是**命令模式**。在此模式下，我们可以执行各种编辑命令，而无需进入插入模式。

![](img/3ca532cd9e6016964ce538a80b7dba6d_23.png)

在命令模式下，**光标移动**是高效操作的基础。虽然可以使用键盘方向键，但Vim设计了一套更高效的键位，让你的手无需离开主键盘区。

![](img/3ca532cd9e6016964ce538a80b7dba6d_29.png)

![](img/3ca532cd9e6016964ce538a80b7dba6d_31.png)

以下是替代方向键的四个键位：
*   **h**：向左移动光标。
*   **j**：向下移动光标。
*   **k**：向上移动光标。
*   **l**：向右移动光标。

![](img/3ca532cd9e6016964ce538a80b7dba6d_33.png)

记住，`j` 键的底部是向下的，这有助于联想记忆。多练习几次就能熟悉。

掌握了光标移动，我们就可以开始编辑文本了。以下是命令模式下最常用的几个操作。

**删除（剪切）命令**
*   **dd**：删除（剪切）光标所在的整行。
*   **3dd**：从光标所在行开始，删除（剪切）3行。
*   **dw**：删除（剪切）从光标位置开始到单词结尾的部分。
*   **x**：删除（剪切）光标所在位置的单个字符。连续按 `x` 可以快速删除多个字符。

![](img/3ca532cd9e6016964ce538a80b7dba6d_45.png)

![](img/3ca532cd9e6016964ce538a80b7dba6d_47.png)

**复制命令**
*   **yy**：复制光标所在的整行。
*   **3yy**：从光标所在行开始，复制3行。

![](img/3ca532cd9e6016964ce538a80b7dba6d_49.png)

**粘贴命令**
*   **p**：将剪切或复制的内容粘贴到光标所在行的**下一行**。

![](img/3ca532cd9e6016964ce538a80b7dba6d_51.png)

![](img/3ca532cd9e6016964ce538a80b7dba6d_53.png)

**撤销与重做**
*   **u**：撤销上一次操作。可以连续按 `u` 进行多次撤销。
*   **Ctrl + r**：重做被撤销的操作（不常用，但需要知道）。

![](img/3ca532cd9e6016964ce538a80b7dba6d_55.png)

![](img/3ca532cd9e6016964ce538a80b7dba6d_57.png)

![](img/3ca532cd9e6016964ce538a80b7dba6d_59.png)

**替换字符**
*   **r**：替换光标所在位置的一个字符。例如，输入 `ry` 会将当前字符替换为 `y`。
*   **Shift + r (R)**：进入**替换模式**，可以连续替换多个字符，按 `Esc` 键返回命令模式。

![](img/3ca532cd9e6016964ce538a80b7dba6d_61.png)

**快速跳转**
*   **gg**：跳转到文件**第一行**。
*   **Shift + g (G)**：跳转到文件**最后一行**。

本节课中我们一起学习了Vim命令模式下的核心编辑技巧。我们掌握了如何使用 `h`、`j`、`k`、`l` 高效移动光标，并学习了**复制(yy)**、**粘贴(p)**、**删除/剪切(dd)**、**撤销(u)** 以及**快速跳转(gg/G)** 等关键命令。这些是提升在Linux环境下编辑效率的基石，请务必多加练习。下一节，我们将继续探索Vim的更多强大功能。

## 使用点（.）配合cp命令

上一节我们介绍了`cp`命令，本节中我们来看看一个使用点（`.`）的小技巧。点（`.`）代表当前目录。

例如，我想把网卡配置文件拷贝到当前所在目录。
```bash
cp /etc/sysconfig/network-scripts/ifcfg-ens33 .
```
这条命令将`ifcfg-ens33`文件拷贝到了当前目录。点（`.`）在这里就代表了当前目录的位置。

## 理解cp命令的改名功能

![](img/b993bcbca6a8625375533a0021f648db_21.png)

接下来，我们深入理解`cp`命令的改名功能。这个功能在移动和拷贝时都可能用到。

![](img/b993bcbca6a8625375533a0021f648db_23.png)

![](img/b993bcbca6a8625375533a0021f648db_25.png)

以下是关于拷贝目录时改名功能的说明：
*   第一次执行 `cp -r test /opt/xxxx` 时，由于`/opt`目录下没有`xxxx`目录，`test`目录会被拷贝过去并改名为`xxxx`。
*   第二次再执行 `cp -r test /opt` 时，因为`/opt`目录下已经存在`xxxx`目录，这条命令会将`test`目录拷贝到`/opt/xxxx`目录里面去，而不是覆盖它。

![](img/b993bcbca6a8625375533a0021f648db_27.png)

![](img/b993bcbca6a8625375533a0021f648db_29.png)

![](img/b993bcbca6a8625375533a0021f648db_31.png)

![](img/b993bcbca6a8625375533a0021f648db_33.png)

![](img/b993bcbca6a8625375533a0021f648db_34.png)

![](img/b993bcbca6a8625375533a0021f648db_35.png)

主要需要理解它的改名功能。一旦在目标路径指定了新名字，后续的拷贝操作就会基于这个新名字进行。

## mv命令：移动与重命名

现在，我们来看看`mv`命令。`mv`命令用于移动文件或目录，在Windows中的对应操作是“剪切”。

![](img/b993bcbca6a8625375533a0021f648db_39.png)

命令格式与`cp`命令基本一致：`mv [选项] 源文件 目标文件`。

![](img/b993bcbca6a8625375533a0021f648db_41.png)

![](img/b993bcbca6a8625375533a0021f648db_43.png)

![](img/b993bcbca6a8625375533a0021f648db_45.png)

移动文件很简单。例如，将`hello`文件移动到`/opt/t1`目录：
```bash
mv hello /opt/t1
```
`mv`命令同样具有改名功能。例如，将`test`目录移动到根目录`/`下并重命名为`xxxxxx`：
```bash
mv test /xxxxxx
```
如果不想改名，直接指定目标目录即可，文件会保留原名。

![](img/b993bcbca6a8625375533a0021f648db_47.png)

## cat命令：查看文件内容

![](img/b993bcbca6a8625375533a0021f648db_49.png)

![](img/b993bcbca6a8625375533a0021f648db_51.png)

![](img/b993bcbca6a8625375533a0021f648db_53.png)

下面我们学习查看文件内容的命令。在Linux中，我们使用`cat`命令来查看文件内容。

命令格式为：`cat [选项] 文件名`。常用的选项是`-n`，用于显示行号。

以下是使用`cat`命令查看一些常见系统文件的例子：
*   `cat /etc/passwd`：查看系统所有用户信息。
*   `cat -n /etc/fstab`：查看系统挂载配置文件，并显示行号。
*   `cat /etc/hostname`：查看主机名。
*   `cat /etc/os-release` 或 `cat /etc/redhat-release`：查看系统版本信息。
*   `cat /etc/group`：查看系统所有组信息。
*   `cat /etc/sysconfig/network-scripts/ifcfg-ens33`：查看网卡配置文件。

![](img/b993bcbca6a8625375533a0021f648db_55.png)

`cat`命令适合查看内容较少的文件。如果文件内容很长，它会将所有内容一次性输出到屏幕，导致前面的内容无法回溯查看。

## less命令：分页查看大文件

为了解决查看大文件的问题，我们使用`less`命令。`less`是一个分页显示工具，适合查看内容量大的文件。

![](img/b993bcbca6a8625375533a0021f648db_62.png)

![](img/b993bcbca6a8625375533a0021f648db_63.png)

命令格式为：`less [选项] 文件名`。使用大写的`-N`可以显示行号。

![](img/b993bcbca6a8625375533a0021f648db_65.png)

使用`less`查看文件时，就像看书一样可以分页浏览：
*   上下方向键：逐行滚动。
*   Page Down（或Fn+下方向键）：向下翻一页。
*   Page Up（或Fn+上方向键）：向上翻一页。
*   大写`G`：跳转到文件末尾。
*   小写`gg`：跳转到文件开头。
*   `:1000`然后回车：精准跳转到第1000行。
*   `q`：退出`less`视图。

## head与tail命令：查看文件首尾

有时我们只需要查看文件的开头或结尾部分，这时可以使用`head`和`tail`命令。

![](img/b993bcbca6a8625375533a0021f648db_67.png)

![](img/b993bcbca6a8625375533a0021f648db_69.png)

![](img/b993bcbca6a8625375533a0021f648db_71.png)

![](img/b993bcbca6a8625375533a0021f648db_73.png)

`head`命令用于显示文件开头的内容，默认显示前10行。
使用`-n`选项可以指定显示的行数，例如`head -n 20 文件名`显示前20行。也可以简写为`head -20 文件名`。

![](img/b993bcbca6a8625375533a0021f648db_75.png)

`tail`命令用于显示文件末尾的内容，默认显示后10行。
同样可以使用`-n`选项，例如`tail -20 文件名`显示末尾20行。
`tail`命令还有一个非常实用的`-f`选项，用于动态追踪文件末尾的新增内容，常用于监控日志文件。按`Ctrl+C`可以终止追踪。

## rm命令：删除文件与目录 ⚠️

![](img/b993bcbca6a8625375533a0021f648db_81.png)

![](img/b993bcbca6a8625375533a0021f648db_83.png)

最后，我们来学习需要谨慎使用的`rm`命令，它用于删除文件或目录。

![](img/b993bcbca6a8625375533a0021f648db_85.png)

![](img/b993bcbca6a8625375533a0021f648db_86.png)

命令格式为：`rm [选项] 文件或目录`。

在开始练习删除前，请务必了解虚拟机的“快照”功能。它可以保存系统当前状态，万一误操作导致系统问题，可以通过快照恢复。

**删除目录**需要加上`-r`（递归）选项，系统会询问确认。
```bash
rm -r 目录名
```

**删除文件**则不需要`-r`选项。
```bash
rm 文件名
```

**强制删除**使用`-f`选项，它将跳过确认提示。
```bash
rm -f 文件名
```

通常，删除目录及其内部所有内容会组合使用`-rf`选项：
```bash
rm -rf 目录名
```

**星号（*）** 是一个通配符，代表“任意所有”。例如，`rm -rf /opt/*`会删除`/opt`目录下的所有内容。
**请极度谨慎使用形如`rm -rf /*`的命令，这会删除根目录下的所有文件，导致系统崩溃。**

另外请注意路径中的斜线（`/`）：
*   对于目录，路径末尾的`/`可有可无（如`/opt`和`/opt/`等效）。
*   对于文件，路径末尾不能有`/`。
*   在路径中间，`/`是必要的目录分隔符（如`/opt/test/file.txt`）。

以下是Linux系统的几个核心概念：

![](img/2211dbdd246d74480e6ab7c17a7cf884_42.png)

1.  **多用户系统**：允许同时有多个用户登录并使用同一台计算机的资源。系统理论上支持最多65535个用户。
2.  **多任务系统**：允许同时运行多个程序或进程，例如可以同时运行网站服务、数据库服务和监控程序。
3.  **严格区分大小写**：系统对文件名、目录名、命令、选项和参数都严格区分大小写。例如，文件 `fileA` 和 `FileA` 是两个不同的文件。
4.  **一切皆文件**：这是Linux的一个重要哲学。硬件设备（如CPU、内存、硬盘）、目录、普通文件等，在系统中都以文件的形式存在和访问。
5.  **树形目录结构**：系统中的所有文件和目录都以倒挂的树形结构组织，起点称为“根目录”，用单个斜杠 `/` 表示。删除根目录意味着删除整个系统。
6.  **无扩展名要求**：Linux系统不依赖文件扩展名（如 `.txt`, `.exe`）来识别文件类型或决定如何打开它。扩展名（如 `.sh`, `.conf`, `.log`）主要是为了方便用户识别。
7.  **无提示即成功**：大多数命令执行成功后，系统不会返回“成功”提示。如果命令没有报错信息，通常意味着它已成功执行。
8.  **无回收站**：Linux系统默认没有类似Windows的回收站功能。使用删除命令（如 `rm`）删除文件后，通常难以恢复，操作时需格外谨慎。

当我们登录系统后，会看到类似这样的提示符：
`[root@localhost ~]#`

![](img/2211dbdd246d74480e6ab7c17a7cf884_74.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_76.png)

以下是各字段的详细解释：

![](img/2211dbdd246d74480e6ab7c17a7cf884_83.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_85.png)

*   **`root`**：代表**当前登录的用户名**。这里使用的是超级管理员账号 `root`。
*   **`@`**：分隔符，用于分隔用户名和主机名。
*   **`localhost`**：代表**当前计算机的主机名**。安装后默认为 `localhost`，后期可以修改。
*   **`~`**：波浪号，代表**当前用户所在的目录**。`~` 是用户“家目录”（home directory）的简写。用户首次登录系统时，默认进入自己的家目录。
    *   超级管理员 `root` 的家目录是 `/root`。
    *   普通用户（如 `test`）的家目录通常是 `/home/用户名`，例如 `/home/test`。
*   **`#` 或 `$`**：代表**当前用户的身份**。
    *   **`#`**：提示符以井号结尾，表示当前用户是**超级管理员（root）**，拥有最高权限。
    *   **`$`**：提示符以美元符号结尾，表示当前用户是**普通用户**，权限有限。

`命令字 [选项]... [参数]...`

下面我们以 `ls` 命令为例，详细解释每个部分：

![](img/2211dbdd246d74480e6ab7c17a7cf884_108.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_110.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_111.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_113.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_115.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_117.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_119.png)

*   **命令字**：即命令本身，是实现特定功能的指令。例如 `ls` 命令用于列出目录内容。
    *   示例：`ls`
*   **选项**：用于调整命令的功能。选项分为短选项和长选项。
    *   **短选项**：通常以单个短横线 `-` 开头，后接单个字符，如 `-l`。多个短选项可以合并，例如 `-l -h` 可以写成 `-lh`。
        *   示例：`ls -l` （以长格式显示详细信息）
        *   示例：`ls -lh` （以长格式并人性化显示文件大小）
    *   **长选项**：通常以两个短横线 `--` 开头，后接一个单词，如 `--help`。长选项不能像短选项那样合并书写。
        *   示例：`ls --help` （显示 `ls` 命令的帮助信息）
*   **参数**：命令的执行对象，可以是文件、目录等。
    *   示例：`ls /opt` （列出 `/opt` 目录下的内容）
    *   示例：`ls file1 file2` （列出 `file1` 和 `file2` 两个文件的信息）

![](img/2211dbdd246d74480e6ab7c17a7cf884_121.png)

![](img/2211dbdd246d74480e6ab7c17a7cf884_123.png)

**格式符号说明：**
*   `[ ]`：方括号内的内容表示是**可选的**。命令可以不带选项或参数执行。
*   `...`：省略号表示前面的元素（选项或参数）可以同时有**多个**。

**命令执行**：输入完命令后，按下 **Enter** 键（回车键）来执行它。

**清屏技巧**：如果终端显示内容杂乱，可以使用 `clear` 命令或快捷键 **Ctrl + L** 来清空屏幕。

## 什么是文本编辑器？🤔

![](img/bf8baa9efaa223bbd51940d372b0a1d8_11.png)

上一节我们介绍了Linux系统的基本信息查看命令，本节中我们来看看如何创建和编辑文件。在Windows系统中，我们通过“记事本”来创建和编辑文本文档。例如，右击新建一个文本文档，双击打开后写入内容并保存。这个“记事本”就是Windows中的文本编辑器，它具备创建文件和对文件内容进行增删改查的功能。

![](img/bf8baa9efaa223bbd51940d372b0a1d8_13.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_15.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_17.png)

在Linux系统中，对应的工具是 **vi** 或 **vim**。vim是从vi发展而来的一个功能更强大的文本编辑器。它们的主要区别在于：
*   **vi**：基础的文本编辑器，没有语法高亮等功能。
*   **vim**：是vi的增强版，具备**编程能力**和**语法高亮**。例如，在编写Shell脚本时，vim会用不同颜色提示语法是否正确。

![](img/bf8baa9efaa223bbd51940d372b0a1d8_19.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_21.png)

对于日常修改配置文件，使用vi或vim均可。但鉴于vim功能更强大，我们主要学习vim。

## vim的基本使用 🚀

vim的命令格式非常简单：

![](img/bf8baa9efaa223bbd51940d372b0a1d8_23.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_25.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_27.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_29.png)

```bash
vim [文件名]
```

![](img/bf8baa9efaa223bbd51940d372b0a1d8_31.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_33.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_35.png)

它的工作逻辑是：
*   **若文件不存在**：则创建新文件并进入编辑状态。
*   **若文件存在**：则直接打开该文件进行编辑。

例如，编辑一个已存在的系统主机名文件：
```bash
vim /etc/hostname
```
编辑一个不存在的文件（vim会先创建它）：
```bash
vim /opt/xxxx.tt
```

## vim的三种模式 🔄

vim的操作比图形化的记事本复杂，它分为三种模式，这是掌握vim的关键。

1.  **命令模式**：启动vim后，默认进入的模式。在此模式下，键盘输入会被识别为命令，而非文本内容。
2.  **输入模式**：在此模式下，可以向文件中输入或编辑文本内容。
3.  **底线命令模式**：在此模式下，可以执行保存、退出、搜索替换等高级操作。

三种模式的转换关系如下图所示：

```
启动vim
    ↓
[命令模式] ←--- (按 `Esc` 键)
    |  (按 `i`, `a`, `o` 键)
    ↓
[输入模式] (进行文本编辑)
    |
(按 `Esc` 键)
    ↓
[命令模式]
    |  (按 `:` 键)
    ↓
[底线命令模式] (执行 `:w`, `:q` 等命令)
```

![](img/bf8baa9efaa223bbd51940d372b0a1d8_37.png)

### 1. 从命令模式进入输入模式

![](img/bf8baa9efaa223bbd51940d372b0a1d8_39.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_41.png)

在命令模式下，按以下键可以进入输入模式：
*   **`i`**：在**当前光标所在字符前**插入。
*   **`a`**：在**当前光标所在字符后**插入。
*   **`o`**：在**当前光标所在行的下方**插入一个新行。

初学者只需记住 `i` 即可，进入输入模式后，可以像使用普通编辑器一样输入文本、换行、移动光标。

### 2. 保存与退出（底线命令模式）

编辑完成后，需要先按 `Esc` 键退回**命令模式**，然后按 `:` 键进入**底线命令模式**。

![](img/bf8baa9efaa223bbd51940d372b0a1d8_43.png)

以下是底线命令模式下常用的保存与退出操作：

![](img/bf8baa9efaa223bbd51940d372b0a1d8_45.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_47.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_49.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_51.png)

*   **`:w`**：保存文件。
*   **`:q`**：退出vim（如果文件有修改且未保存，会提示失败）。
*   **`:wq`** 或 **`:x`**：保存并退出。
*   **`ZZ`**（**命令模式**下，大写）：快速保存并退出。
*   **`:q!`**：强制退出，**不保存**所有修改。
*   **`:wq!`**：强制保存并退出（常用于修改只读文件，仅root用户有效）。

![](img/bf8baa9efaa223bbd51940d372b0a1d8_53.png)

### 3. 命令模式下的常用操作

在命令模式下，有大量高效的光标移动和文本操作命令。以下是核心命令列表：

**光标移动**
*   `0`：光标移动到**行首**。
*   `$`：光标移动到**行尾**。
*   `gg`：光标移动到**文件第一行**。
*   `G`：光标移动到**文件最后一行**。
*   `:[行号]`：在底线命令模式下，快速跳转到指定行，例如 `:1000`。

**删除/剪切**
*   `x`：删除**当前光标所在**的一个字符。
*   `dd`：剪切（删除）**当前光标所在**的整行。
*   `[数字]dd`：剪切从当前行开始的指定行数，例如 `3dd` 剪切3行。
*   `C`（大写）：删除从光标到行尾的内容，并进入输入模式。

**复制/粘贴**
*   `yy`：复制**当前光标所在**的整行。
*   `[数字]yy`：复制从当前行开始的指定行数，例如 `2yy`。
*   `p`：将剪切或复制的内容**粘贴到当前光标所在行的下方**。

**撤销与重做**
*   `u`：撤销上一次操作（可多次使用，恢复到文件打开时的状态，但**保存后无效**）。
*   `Ctrl + r`：重做，即撤销“撤销”操作（恢复被 `u` 撤销的操作）。

**搜索与替换**
*   `/关键字`：在命令模式下，从当前光标位置**向下**搜索指定关键字，按 `n` 跳转到下一个匹配项。
*   `:%s/旧内容/新内容/g`：在底线命令模式下，**全局替换**文件中的“旧内容”为“新内容”。
    *   去掉末尾的 `g`，则只替换每行中**第一个**匹配项。

![](img/bf8baa9efaa223bbd51940d372b0a1d8_55.png)

**显示设置**
*   `:set nu`：在底线命令模式下，**显示行号**。
*   `:set nonu`：**取消显示行号**。
*   `:nohl`：**取消搜索关键字的高亮显示**。

## 实战练习与技巧 💡

![](img/bf8baa9efaa223bbd51940d372b0a1d8_57.png)

为了熟练掌握vim，请务必进行大量练习。你可以复制一个系统文件来练习，避免误操作：

![](img/bf8baa9efaa223bbd51940d372b0a1d8_59.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_61.png)

![](img/bf8baa9efaa223bbd51940d372b0a1d8_63.png)

```bash
cp /etc/services /opt/
vim /opt/services
```

在这个内容丰富的文件中，你可以练习：
*   使用 `gg`、`G`、`:100` 进行快速跳转。
*   使用 `/port` 搜索关键词。
*   使用 `dd`、`yy`、`p` 进行剪切、复制、粘贴。
*   使用 `:%s/port/XXXX/g` 进行全局替换。
*   使用 `:set nu` 显示行号，方便定位。

**重要提示**：vim的学习曲线较陡，初期可能会感到不适应。但请坚持练习，这些命令一旦熟练，将极大提升你在Linux环境下的工作效率。记住核心循环：`i` 进入编辑 -> `Esc` 退出编辑 -> `:` 执行保存等操作。

Linux是一个多用户、多任务的操作系统。这意味着它允许多个用户同时登录并使用系统资源，也允许系统同时运行多个程序。

![](img/0d7d302ed08cb042a805f4a7b8dda579_39.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_41.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_43.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_45.png)

以下是Linux系统的一些核心特点：

![](img/0d7d302ed08cb042a805f4a7b8dda579_47.png)

*   **多用户系统**：允许同时有多个用户登录并使用同一台计算机。系统理论上最多可支持65，535个用户。
*   **多任务系统**：允许系统同时执行多个程序或任务，例如同时运行网站服务、数据库服务和监控程序。
*   **严格区分大小写**：系统对命令、文件名、目录名等都严格区分大小写。例如，文件 `a.txt` 和 `A.txt` 是两个不同的文件。
*   **一切皆文件**：这是Linux的一个重要哲学。硬件设备（如CPU、内存、硬盘）、目录、普通文件等，在系统中都以文件的形式存在和管理。查看硬件信息即查看对应的文件。
*   **树形目录结构**：系统中的所有文件和目录都以一个倒置的树形结构组织，这个结构的起点称为 **根目录**，用单个斜杠 **`/`** 表示。删除根目录意味着删除整个系统的数据。
*   **没有扩展名限制**：Linux不依赖文件扩展名（如 `.txt`， `.exe`）来识别文件类型。文件是否有扩展名，系统都能识别。常见的扩展名（如 `.conf`， `.sh`， `.log`）主要是为了方便用户识别文件用途。
*   **没有提示就是最好的提示**：大多数命令执行成功后，系统不会返回“成功”的提示信息。命令执行后若无报错信息，通常意味着操作成功。
*   **没有回收站**：Linux系统默认没有类似Windows的回收站功能。文件一旦被删除，通常无法直接恢复，因此执行删除操作时需要格外谨慎。

我们通过远程工具连接后看到的黑色窗口就是命令终端，它是我们输入和执行命令的地方。在Linux中，绝大多数操作都通过键盘输入命令完成，鼠标的作用相对较小。

![](img/0d7d302ed08cb042a805f4a7b8dda579_81.png)

一个典型的命令提示符如下所示：
```
[root@localhost ~]#
```
以下是各个字段的含义解析：

![](img/0d7d302ed08cb042a805f4a7b8dda579_89.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_91.png)

*   **`root`**：代表 **当前登录的用户名**。这里显示的是超级管理员账号 `root`。
*   **`@`**：是一个分隔符，用于分隔用户名和主机名。
*   **`localhost`**：代表 **当前主机的主机名**。安装系统后默认为 `localhost`，后期可以根据需要修改。
*   **`~`**：波浪号代表 **当前用户所在的目录**。`~` 有特殊含义，它表示当前用户的 **家目录**。
    *   用户首次登录系统时，默认进入自己的家目录。
    *   超级管理员 `root` 的家目录是 **`/root`**。
    *   普通用户（如 `test`）的家目录通常在 **`/home/用户名`** 下，例如 `/home/test`。
*   **`#`** 或 **`$`**：代表 **当前用户的身份**。
    *   **`#`**：表示当前用户是 **超级管理员（root）**，拥有系统的最高权限。
    *   **`$`**：表示当前用户是 **普通用户**，权限受到限制。

命令行不是随意输入的，它遵循特定的格式：`命令字 [选项]... [参数]...`

下面我们以 `ls` 命令为例进行拆解，`ls` 命令的基本功能是列出目录下的内容。

![](img/0d7d302ed08cb042a805f4a7b8dda579_116.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_118.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_120.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_122.png)

*   **命令字**：即命令本身，是实现特定功能的指令。例如 `ls` 就是“列出”的命令。
    *   命令：`ls`
    *   作用：查看当前目录下的文件和目录。

![](img/0d7d302ed08cb042a805f4a7b8dda579_124.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_126.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_128.png)

![](img/0d7d302ed08cb042a805f4a7b8dda579_130.png)

*   **选项**：用于调整或增强命令的功能。选项前通常带有短横线 `-`。
    *   **短选项**：由一个 `-` 后接单个字母组成，例如 `-l`。多个短选项可以合并，如 `-lh`。
    *   **长选项**：由两个 `-` 后接一个单词组成，例如 `--help`。长选项不能合并。
    *   示例：`ls -l`
    *   作用：`-l` 选项让 `ls` 以长格式显示，列出文件和目录的详细信息（权限、所有者、大小、修改时间等）。

![](img/0d7d302ed08cb042a805f4a7b8dda579_132.png)

*   **参数**：命令的操作对象，可以是文件、目录等。
    *   示例：`ls -l /etc`
    *   作用：命令 `ls` 作用于参数 `/etc`，即列出 `/etc` 目录下的详细内容。

*   **格式符号说明**：
    *   `[]`：方括号内的内容是可选的。例如 `[选项]` 表示你可以不加选项，直接使用命令。
    *   `...`：省略号表示前面的元素（选项或参数）可以同时出现多个。
        *   示例：`ls -l -h /etc /root` 使用了两个选项 `-l` 和 `-h`，以及两个参数 `/etc` 和 `/root`。

**一个完整的命令示例**：
`ls -lh /etc/passwd`
*   `ls`：命令字。
*   `-lh`：合并使用的两个短选项（`-l` 长格式显示，`-h` 以人类易读的格式显示文件大小）。
*   `/etc/passwd`：命令的参数，即要查看的文件。

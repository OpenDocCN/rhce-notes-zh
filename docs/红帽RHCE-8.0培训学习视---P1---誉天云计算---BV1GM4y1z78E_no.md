### 复制、粘贴与删除

![](img/b875cddc438ee544190f7778e01a5577_68.png)

![](img/b875cddc438ee544190f7778e01a5577_70.png)

![](img/b875cddc438ee544190f7778e01a5577_72.png)

在命令模式下，vim提供了一系列快捷键来快速操作文本。

![](img/b875cddc438ee544190f7778e01a5577_74.png)

![](img/b875cddc438ee544190f7778e01a5577_76.png)

![](img/b875cddc438ee544190f7778e01a5577_78.png)

![](img/b875cddc438ee544190f7778e01a5577_80.png)

以下是常用的删除与复制命令：
*   **`x` 或 `delete`**：删除光标处的单个字符。
*   **`dd`**：删除光标所在的整行。
*   **`[n]dd`**：删除从光标行开始的向下`n`行（例如 `10dd` 删除10行）。
*   **`cc`**：删除整行并进入插入模式，以便重新编辑。
*   **`yy`**：复制光标所在的整行。
*   **`[n]yy`**：复制从光标行开始的向下`n`行（例如 `5yy` 复制5行）。
*   **`p`**：将复制或删除（剪切）的内容粘贴到光标**之后**。
*   **`P`**：将复制或删除（剪切）的内容粘贴到光标**之前**。
*   **`u`**：撤销上一次操作。

![](img/b875cddc438ee544190f7778e01a5577_82.png)

![](img/b875cddc438ee544190f7778e01a5577_83.png)

![](img/b875cddc438ee544190f7778e01a5577_85.png)

![](img/b875cddc438ee544190f7778e01a5577_87.png)

在末行模式下，也可以进行跨行复制与移动：
*   **`:n1,n2 copy n3`**：将第`n1`行到第`n2`行的内容复制到第`n3`行的下方。
*   **`:n1,n2 move n3`**：将第`n1`行到第`n2`行的内容移动到第`n3`行的下方。

### 查找与替换

![](img/b875cddc438ee544190f7778e01a5577_95.png)

![](img/b875cddc438ee544190f7778e01a5577_97.png)

![](img/b875cddc438ee544190f7778e01a5577_99.png)

![](img/b875cddc438ee544190f7778e01a5577_101.png)

在命令模式下，可以快速查找文件内的字符串。

![](img/b875cddc438ee544190f7778e01a5577_103.png)

![](img/b875cddc438ee544190f7778e01a5577_105.png)

![](img/b875cddc438ee544190f7778e01a5577_107.png)

*   **`/keyword`**：从光标位置**向下**搜索`keyword`。
*   **`?keyword`**：从光标位置**向上**搜索`keyword`。
*   找到后，按 **`n`** 跳转到下一个匹配项，按 **`N`** 跳转到上一个匹配项。

![](img/b875cddc438ee544190f7778e01a5577_109.png)

![](img/b875cddc438ee544190f7778e01a5577_111.png)

![](img/b875cddc438ee544190f7778e01a5577_113.png)

![](img/b875cddc438ee544190f7778e01a5577_115.png)

![](img/b875cddc438ee544190f7778e01a5577_116.png)

在末行模式下，可以进行更强大的替换操作。

![](img/b875cddc438ee544190f7778e01a5577_118.png)

![](img/b875cddc438ee544190f7778e01a5577_120.png)

![](img/b875cddc438ee544190f7778e01a5577_122.png)

*   **`:%s/old/new/g`**：将文件中所有的`old`替换为`new`。
    *   `%` 表示全文范围。
    *   `s` 表示替换（substitute）。
    *   `g` 表示行内全局替换（一行内所有匹配项）。
*   **`:%s/old/new/gc`**：替换时逐个确认。
*   **`:n1,n2s/old/new/g`**：仅替换第`n1`行到第`n2`行之间的内容。

vim支持分割窗口，方便同时查看或编辑多个文件。

![](img/b875cddc438ee544190f7778e01a5577_136.png)

*   **`:vsp [filename]`**：纵向分割窗口。
*   在窗口间切换：先按 `Ctrl` + `w`，然后按方向键（`h`, `j`, `k`, `l`）。
*   **`vim -o file1 file2`**：启动vim并横向打开多个文件。
*   **`vim -O file1 file2`**：启动vim并纵向打开多个文件。

利用块可视化模式可以快速添加或删除多行注释：

![](img/b875cddc438ee544190f7778e01a5577_178.png)

![](img/b875cddc438ee544190f7778e01a5577_180.png)

1.  将光标移动到要注释的第一行行首。
2.  按 `Ctrl` + `v` 进入块可视化模式。
3.  用方向键向下选择要注释的行。
4.  按 `Shift` + `i` 进入插入模式。
5.  输入注释符（如 `#`）。
6.  按 `Esc`，所有选中的行行首都会自动添加注释符。

以下是用于查看文件内容的常用命令：

*   **`cat`**：连接文件并打印到标准输出。常用于查看整个文件内容。
    *   示例：`cat /etc/passwd`
*   **`more` / `less`**：分页浏览文件内容，适用于查看长文件。
    *   按空格键翻页，按 `q` 键退出。
    *   `less` 比 `more` 功能更多，支持向前翻页。
*   **`head`**：显示文件开头部分，默认显示前10行。
    *   示例：`head -n 20 file.txt` 显示前20行。
*   **`tail`**：显示文件末尾部分，默认显示后10行。
    *   示例：`tail -n 15 file.log` 显示后15行。
    *   **`tail -f file.log`**：实时追踪文件末尾新增的内容，常用于监控日志，按 `Ctrl` + `c` 退出。

以下是`grep`的常用选项：
*   **`-i`**：忽略大小写进行匹配。
*   **`-n`**：显示匹配行的行号。
*   **`-v`**：反向选择，显示**不包含**模式的行。
*   **`-c`**：只输出匹配行的总数。
*   **`-r`** 或 **`-R`**：递归搜索目录下的所有文件。
*   **`-E`**：使用扩展正则表达式（等同于 `egrep`）。

示例：
*   `grep -i ‘error’ /var/log/messages`：在日志中查找所有“error”（不区分大小写）。
*   `grep -v ‘^#’ /etc/ssh/sshd_config | grep -v ‘^$’`：查看配置文件的有效行（去掉注释和空行）。
*   `grep -r ‘TODO’ ~/projects/`：递归查找家目录下projects文件夹中所有包含“TODO”的文件。

#### `cut` - 按列提取文本

![](img/b875cddc438ee544190f7778e01a5577_294.png)

![](img/b875cddc438ee544190f7778e01a5577_296.png)

`cut` 命令用于从文件的每一行中提取部分内容（列）。

![](img/b875cddc438ee544190f7778e01a5577_298.png)

![](img/b875cddc438ee544190f7778e01a5577_299.png)

![](img/b875cddc438ee544190f7778e01a5577_301.png)

常用选项：
*   **`-d‘分隔符’`**：指定字段的分隔符，默认为制表符。
*   **`-f 字段列表`**：指定要提取的字段编号（从1开始）。
*   **`-c 字符范围`**：按字符位置提取。

![](img/b875cddc438ee544190f7778e01a5577_303.png)

![](img/b875cddc438ee544190f7778e01a5577_305.png)

示例：
*   `cut -d‘:’ -f1 /etc/passwd`：提取`/etc/passwd`文件中以冒号分隔的第一列（用户名）。
*   `ifconfig ens160 | grep inet | head -1 | cut -d‘ ’ -f10`：提取IP地址（需要根据实际空格数调整`-f`参数）。

#### `sort` - 文本排序

![](img/b875cddc438ee544190f7778e01a5577_337.png)

![](img/b875cddc438ee544190f7778e01a5577_339.png)

`sort` 命令用于对文本行进行排序。

![](img/b875cddc438ee544190f7778e01a5577_341.png)

![](img/b875cddc438ee544190f7778e01a5577_343.png)

![](img/b875cddc438ee544190f7778e01a5577_345.png)

常用选项：
*   **`-n`**：按数值大小排序。
*   **`-r`**：反向排序（降序）。
*   **`-u`**：排序并去重（等同于 `sort | uniq`）。
*   **`-t‘分隔符’ -k 字段编号`**：指定分隔符并按特定字段排序。

![](img/b875cddc438ee544190f7778e01a5577_347.png)

![](img/b875cddc438ee544190f7778e01a5577_349.png)

![](img/b875cddc438ee544190f7778e01a5577_351.png)

示例：
*   `sort -nr file.txt`：按数值降序排列。
*   `cut -d‘:’ -f3 /etc/passwd | sort -n`：提取用户ID并排序。

## 培训环境介绍

培训环境分为线上环境和线下环境。

线上环境在机房服务器上，我们通过VNC软件远程连接。你需要知道服务器的IP和端口号，这些信息记录在一个Excel文件中。VNC软件版本不要超过5.2，否则可能显示异常。

线下环境建议使用自己的笔记本，安装VMware Workstation或Fusion软件。硬件要求是CPU支持虚拟化技术，内存4G以上。软件需要从VMware官网下载。

![](img/04a46a48b0596729c8bdfe675d420057_14.png)

培训环境的虚拟机文件在百度网盘中，名为 `foundation0`。这套环境包含三台主要的KVM虚拟机：
*   `classroom`： 提供DHCP、DNS等服务，尽量不要修改。
*   `server`： 实验用的服务器。
*   `desktop`： 实验用的客户端。

![](img/04a46a48b0596729c8bdfe675d420057_16.png)

在培训环境中，可以使用 `rht-vmctl` 命令控制 `server` 和 `desktop` 虚拟机。

![](img/04a46a48b0596729c8bdfe675d420057_18.png)

![](img/04a46a48b0596729c8bdfe675d420057_20.png)

![](img/04a46a48b0596729c8bdfe675d420057_22.png)

**虚拟机控制命令示例：**
```bash
rht-vmctl status server      # 查看server虚拟机状态
rht-vmctl start server       # 启动server虚拟机
rht-vmctl reset server       # 重置server虚拟机（恢复快照）
rht-vmctl view server        # 打开server虚拟机的控制台视图
```

## 访问命令行

在Linux系统中，大多数管理操作都使用命令。图形界面虽然存在，但远程维护管理时命令更常用。

![](img/04a46a48b0596729c8bdfe675d420057_26.png)

![](img/04a46a48b0596729c8bdfe675d420057_28.png)

### 打开终端

有多种方式可以打开终端（命令行界面）：
1.  在桌面空白处点击鼠标右键，选择 `Open in Terminal`。
2.  点击屏幕左上角的 `Applications` -> `Favorites` -> `Terminal`。
3.  按 `Alt+F2`，输入 `gnome-terminal` 后回车。

![](img/04a46a48b0596729c8bdfe675d420057_30.png)

![](img/04a46a48b0596729c8bdfe675d420057_32.png)

### 终端操作技巧

*   **新建标签页**： `Ctrl+Shift+T`
*   **放大字体**： `Ctrl+Shift+=` 或 `Ctrl++`
*   **缩小字体**： `Ctrl+-`
*   **永久设置字体**： 点击终端菜单栏的 `Edit` -> `Profile Preferences`，修改字体。
*   **正常结束命令/退出登录**： `Ctrl+D`
*   **强制终止命令**： `Ctrl+C`

### 使用SSH远程连接

![](img/04a46a48b0596729c8bdfe675d420057_34.png)

![](img/04a46a48b0596729c8bdfe675d420057_36.png)

![](img/04a46a48b0596729c8bdfe675d420057_38.png)

![](img/04a46a48b0596729c8bdfe675d420057_40.png)

可以使用SSH命令从 `foundation0` 远程登录到 `server` 虚拟机进行管理。培训环境中已配置密钥认证，首次连接可能不需要密码。

**SSH连接命令示例：**
```bash
ssh root@server0     # 以root用户身份登录server0
ssh student@server0  # 以student用户身份登录server0
```

登录后，注意观察命令提示符（`[root@server0 ~]#` 或 `[student@server0 ~]$`），它显示了当前用户名、主机名和所在目录。`#` 代表管理员（root）权限，`$` 代表普通用户权限。

### 虚拟控制台切换

Linux默认开启了6个虚拟控制台（`tty1`-`tty6`），用于在图形界面崩溃时进行字符界面操作。

*   **从图形切换到字符终端**： `Ctrl+Alt+F2` 到 `F6`
*   **从字符终端切回图形**： `Ctrl+Alt+F1` 或 `Alt+F1`

在字符终端下，可以直接使用 `Alt+F1` 到 `F6` 进行切换。

## Linux命令基础

![](img/04a46a48b0596729c8bdfe675d420057_42.png)

![](img/04a46a48b0596729c8bdfe675d420057_44.png)

### 命令的组成

Linux命令通常由三部分组成：**命令**、**选项**和**参数**。
*   **命令**： 要执行的操作。
*   **选项**： 用于调整命令行为，通常以 `-`（缩写）或 `--`（完整单词）开头。
*   **参数**： 命令作用的对象，如文件名、目录名。

![](img/04a46a48b0596729c8bdfe675d420057_46.png)

![](img/04a46a48b0596729c8bdfe675d420057_48.png)

**基本格式：**
```
命令 [选项] [参数]
```

### 常用命令示例

![](img/04a46a48b0596729c8bdfe675d420057_50.png)

![](img/04a46a48b0596729c8bdfe675d420057_52.png)

**1. `ls` 命令**
`ls` 是 `list` 的缩写，用于列出目录内容。

```bash
ls          # 列出当前目录内容
ls /var     # 列出/var目录内容
ls -l       # 以长格式列出详细信息
ls -lh      # 以人类可读的格式（如K， M， G）显示文件大小
ls --help   # 查看ls命令的帮助信息
```
在输出中，蓝色代表目录，黑色代表普通文件，青色代表链接（快捷方式）。

**2. `date` 命令**
`date` 命令用于显示或设置系统日期和时间。

```bash
date                # 显示当前日期和时间
date -u             # 显示UTC（协调世界时）时间
date +%Y-%m-%d      # 以“年-月-日”格式显示日期
date +%H:%M:%S      # 以“时:分:秒”格式显示时间
```

**3. `passwd` 命令**
`passwd` 是 `password` 的缩写，用于修改用户密码。

```bash
passwd              # 修改当前用户的密码
```
执行后，会提示输入当前密码，然后输入两次新密码。红帽系统对密码强度有要求，过于简单的密码会被拒绝。

![](img/04a46a48b0596729c8bdfe675d420057_54.png)

### 获取命令帮助

![](img/04a46a48b0596729c8bdfe675d420057_56.png)

**1. `--help` 选项**
大多数命令都支持 `--help` 选项来查看简要帮助。

```bash
ls --help
```

![](img/04a46a48b0596729c8bdfe675d420057_58.png)

![](img/04a46a48b0596729c8bdfe675d420057_60.png)

**2. `man` 命令**
`man` 是 `manual` 的缩写，提供更详细的手册页。

![](img/04a46a48b0596729c8bdfe675d420057_62.png)

![](img/04a46a48b0596729c8bdfe675d420057_64.png)

![](img/04a46a48b0596729c8bdfe675d420057_66.png)

![](img/04a46a48b0596729c8bdfe675d420057_68.png)

![](img/04a46a48b0596729c8bdfe675d420057_70.png)

![](img/04a46a48b0596729c8bdfe675d420057_72.png)

```bash
man ls      # 查看ls命令的完整手册
```
在 `man` 页面中：
*   按 `/` 键进入搜索模式，输入关键词后回车查找。
*   按 `n` 键查找下一个匹配项。
*   按 `N` 键查找上一个匹配项。
*   按 `q` 键退出手册页。

![](img/04a46a48b0596729c8bdfe675d420057_74.png)

### 命令记忆方法

对于初学者，可以按照以下步骤记忆命令：
1.  **查单词**： 使用 `man` 或 `--help` 找到命令对应的英文单词（如 `ls` -> `list`）。
2.  **记含义**： 记住单词在命令上下文中的核心含义（如 `list` -> 列出）。
3.  **懂规则**： 了解常见的缩写规则，如省略元音字母（`list` -> `ls`）。
4.  **多练习**： 通过反复使用来形成肌肉记忆。

### 重要注意事项

*   **严格区分大小写**： 命令、选项、参数、文件名在Linux中都区分大小写。
*   **注意命令回显**： 执行命令后，务必查看输出信息。没有错误信息通常表示执行成功。养成“敲命令，看回显”的习惯，对后续排错至关重要。

## 图形界面简介

红帽7的图形界面（GNOME）与Windows有相似之处，但布局略有不同：
*   **菜单栏**： 位于屏幕顶部，包含 `Applications`（应用菜单，类似开始菜单）、`Places`（常用位置）、`System` 等。
*   **系统状态区**： 位于屏幕右上角，包含时间、声音、网络、用户设置等。
*   **活动概述**： 按键盘上的 `Windows` 键或点击屏幕左上角 `Activities`，可以查看所有窗口、应用程序和多个桌面工作区。
*   **多桌面切换**： 可以使用 `Ctrl+Alt+上/下箭头` 在多个桌面间切换。
*   **系统设置**： 点击右上角齿轮图标或打开 `Applications` -> `System Tools` -> `Settings`，可以进行类似Windows控制面板的各项设置，如显示、电源、用户、网络等。

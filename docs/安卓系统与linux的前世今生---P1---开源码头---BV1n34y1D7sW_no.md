# 安卓系统与Linux的前世今生：P1：核心关系解析 🔍

在本节课中，我们将要学习安卓系统与Linux内核之间的核心关系，理解安卓如何基于Linux构建，以及它做了哪些关键的定制。

---

众所周知，谷歌的安卓系统与Linux系统有着千丝万缕的联系。那么，它们之间究竟存在什么关系呢？

![](img/d68ac8948b20bb6f01f9fb51eaf3e144_3.png)

上一段提出了核心问题，本节中我们来看看具体的答案。

安卓本质上仍然是一个Linux系统。只不过，谷歌对它进行了**裁剪**和**定制**。

从技术架构图可以看到，谷歌自己编写了一个名为 **Dalvik** 的程序，运行在Linux系统之上。这个程序就是安卓的Java虚拟机（JVM）。

因此，使用Java语言开发的安卓应用程序，可以通过这个Java虚拟机运行在安卓系统之上。

Linux内核的优势在于其**大内存管理**、**进程管理**、基于权限的**安全模型**、统一的**驱动模型**、**共享库**支持以及**代码开源**等特性。

所以，相较于大多数标准的Linux发行版，安卓是一个使用了**Linux内核**，并进行了深度裁剪与定制的、特殊的Linux发行版。

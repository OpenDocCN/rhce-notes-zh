## 回顾：RHEL 8安装与初始化设置 🔄

上一节我们介绍了Linux的起源，本节我们来回顾RHEL 8的具体安装与初始化设置过程。

### 安装过程回顾

以下是RHEL 8安装过程中的关键步骤：

1.  **选择安装语言与模式**：安装界面首先要求选择语言，随后可选择安装模式，例如带GUI的图形界面安装或最小化安装。
2.  **磁盘分区**：可以选择自动分区或自定义分区。自定义分区时，通常需要创建以下分区：
    *   **/boot分区**：用于存放启动文件，建议分配1024MB。
    *   **swap分区**：交换分区，大小通常设置为物理内存的两倍。
    *   **/ 分区**：根目录，分配剩余的大部分磁盘空间。
3.  **网络与主机名配置**：在此处可以配置网络连接，可以选择DHCP自动获取IP或手动设置静态IP地址。
4.  **时区设置**：选择所在地区的时区。
5.  **软件包选择**：选择需要安装的软件包组，例如带GUI的服务器或最小安装。
6.  **设置root密码与创建用户**：为系统管理员root设置密码，并创建一个普通用户（例如 `student`）。
7.  **完成安装与许可协议**：等待安装完成，重启后需要接受许可协议才能进入系统。

### 配置方法

在RHEL 8中，配置网络主要有两种方式：

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_53.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_55.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_56.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_58.png)

1.  **图形化界面配置**：
    *   点击顶部栏网络图标 -> `Wired Settings`。
    *   在设置窗口中，点击齿轮图标进入详细设置。
    *   在`IPv4`标签页，将方法从`Automatic (DHCP)`改为`Manual`。
    *   手动填写IP地址、子网掩码、网关和DNS服务器地址。
    *   点击`Apply`使配置生效。

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_60.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_62.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_64.png)

2.  **命令行配置（修改配置文件）**：
    *   网络配置文件通常位于 `/etc/sysconfig/network-scripts/` 目录下，文件名类似 `ifcfg-ens160`。
    *   使用文本编辑器（如`vi`或`nano`）以root权限编辑此文件。
    *   关键配置项示例：
        ```bash
        BOOTPROTO=static        # 将DHCP改为static
        ONBOOT=yes              # 确保开机自启
        IPADDR=192.168.181.130  # 你的IP地址
        NETMASK=255.255.255.0   # 子网掩码
        GATEWAY=192.168.181.2   # 网关地址
        DNS1=8.8.8.8            # DNS服务器
        ```
    *   保存退出后，重启网络服务使配置生效：
        ```bash
        nmcli connection reload
        nmcli connection up ens160
        ```
    *   使用 `ip addr show` 或 `ifconfig` 命令检查IP是否配置成功。

### 验证与排错

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_78.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_79.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_81.png)

*   **验证连通性**：在Windows主机上打开命令提示符，使用 `ping <你的Linux_IP>` 命令测试网络是否通畅。
*   **虚拟机网络模式**：确保虚拟机网络适配器设置为`NAT模式`或`桥接模式`。在VMware中，可通过 `编辑` -> `虚拟网络编辑器` 查看和配置子网信息。
*   **权限问题**：编辑系统网络配置文件需要`root`权限。如果使用普通用户操作，请使用 `sudo` 命令或切换到root用户。

*   **用户类型**：

*   **命令提示符**：`#` 表示root用户，`$` 表示普通用户。

### 命令基础

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_108.png)

*   **命令别名**：可以为常用命令创建简短的别名。
    *   临时设置：`alias ll='ls -l'`
    *   永久设置：将别名定义添加到用户家目录的 `~/.bashrc` 文件中。
    *   查看别名：`alias`
    *   取消别名：`unalias ll`
*   **命令格式**：`命令 [选项] [参数]`
    *   短选项：`-l`， `-h`
    *   长选项：`--all`， `--help`
*   **多命令执行**：
    *   `cmd1 ; cmd2`：顺序执行，无论cmd1成功与否。
    *   `cmd1 && cmd2`：仅当cmd1成功，才执行cmd2。
    *   `cmd1 || cmd2`：仅当cmd1失败，才执行cmd2。
*   **命令历史与补全**：
    *   `history`：查看命令历史记录。
    *   `!n`：执行历史记录中第n条命令。
    *   `Tab`键：命令或路径补全。按一次补全，按两次显示所有可能选项。

### 文件与目录管理

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_116.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_118.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_120.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_122.png)

![](img/d5d20fc4a9ff49b0e1505afa6fb95555_124.png)

*   **路径**：
    *   **绝对路径**：从根目录`/`开始的完整路径，如 `/home/student/file.txt`。
    *   **相对路径**：相对于当前目录的路径，如 `./file.txt` 或 `../doc/file.txt`。
*   **常用命令**：
    *   `pwd`：显示当前工作目录。
    *   `ls`：列出目录内容。
    *   `cd`：切换目录。
    *   `cp`：复制文件或目录。
    *   `mv`：移动或重命名文件或目录。
    *   `rm`：删除文件或目录（`-r` 选项删除目录）。
    *   `mkdir`：创建目录。
    *   `rmdir`：删除空目录。
*   **通配符**：用于匹配文件名。
    *   `*`：匹配任意多个字符。
    *   `?`：匹配任意单个字符。
    *   `[abc]`：匹配a、b、c中的任意一个字符。

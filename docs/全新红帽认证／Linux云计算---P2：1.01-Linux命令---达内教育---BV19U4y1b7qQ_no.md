# Linux命令基础：1.01：命令行概述

在本节课中，我们将要学习Linux命令行的基础概念，包括什么是命令行、解释器的作用、Linux的目录结构以及命令行的基本格式和常用快捷键。掌握这些基础知识是后续深入学习Linux系统管理的基石。

## 什么是命令行？💻

命令行是管理员输入的一串字符，用于完成特定的系统任务。用户通过终端输入这串字符并按回车键提交，系统便会执行相应的操作。例如，输入 `ip address list` 可以查看网络接口的地址信息。

## 解释器与内核的关系 🔧

Linux系统能够理解并执行命令，依赖于一个特殊的程序——**解释器**，通常被称为 **Shell**（外壳）。它的核心作用是充当用户与操作系统内核之间的翻译官。

![](img/64eaf5a6e60e8bfcfbf64c8abaf725c3_3.png)

*   **内核**：是操作系统的核心，直接管理计算机的硬件资源（如CPU、内存、磁盘）。
*   **Shell**：包裹在内核之外，负责将用户输入的、易于理解的命令（如 `ls`）翻译成内核能够识别和执行的二进制指令。

![](img/64eaf5a6e60e8bfcfbf64c8abaf725c3_9.png)

![](img/64eaf5a6e60e8bfcfbf64c8abaf725c3_10.png)

在大多数Linux系统中，默认的解释器程序是 `/bin/bash`。如果没有这个解释器，用户将无法与系统进行交互。

## Linux目录结构 🌳

Linux采用树形结构来组织文件和目录，这与Windows类似，但路径分隔符和顶级目录名称不同。

![](img/64eaf5a6e60e8bfcfbf64c8abaf725c3_24.png)

![](img/64eaf5a6e60e8bfcfbf64c8abaf725c3_26.png)

*   **根目录**：整个文件系统的起点，用单个斜杠 `/` 表示。
*   **路径分隔**：使用斜杠 `/` 来分隔不同层级的目录，例如 `/boot/grub2`。

以下是常见的一级目录及其用途：

![](img/64eaf5a6e60e8bfcfbf64c8abaf725c3_28.png)

![](img/64eaf5a6e60e8bfcfbf64c8abaf725c3_30.png)

*   **`/bin`** 与 **`/sbin`**：存放可执行程序文件。`/sbin` 下的程序通常需要管理员权限才能执行。
*   **`/dev`**：存放各种硬件设备文件（如磁盘、光盘）。
*   **`/home`**：存放所有普通用户的主目录（例如用户 `zhangsan` 的主目录是 `/home/zhangsan`）。
*   **`/root`**：系统管理员（root用户）的专属主目录。
*   **`/boot`**：存放系统启动所需的文件，包括内核。**切勿随意删除此目录**。
*   **`/etc`**：存放系统的各种配置文件。
*   **`/var`**：存放经常变化的（Variable）数据，如系统日志、邮件等。
*   **`/tmp`**：存放临时文件。
*   **`/mnt`** 与 **`/media`**：用于挂载外部存储设备（如U盘、光盘）的目录。`/media` 通常用于系统自动挂载。

## 命令行的基本格式 📝

一条完整的Linux命令通常由三部分组成，其基本格式为：
```
命令字 [选项] [参数]
```
*   **命令字**：即命令的名称，是必须的部分（如 `ls`）。
*   **选项**：用于调节命令的执行方式或效果，通常以 `-` 开头。多个单字母选项可以合并，例如 `-l -h` 可以写成 `-lh`。
    *   例如，`ls -l` 表示以长格式（显示详细信息）列出文件。
    *   例如，`ls -h` 表示以人类易读的格式（如KB、MB）显示文件大小。
*   **参数**：指定命令操作的对象，通常是文件或目录路径。
    *   例如，`ls /` 表示列出根目录下的内容。
    *   例如，`ls -lh /boot /etc` 表示以详细且易读的格式同时列出 `/boot` 和 `/etc` 目录的内容。

选项和参数都可以有多个，并且顺序有时可以灵活调整。

## 常用快捷键 ⚡

熟练使用快捷键可以极大提高命令行操作效率。

*   **`Tab` 键**：**自动补全**。输入命令、文件或目录名的前几个字母后按 `Tab` 键，系统会自动补全或列出所有可能的选项。
*   **`Ctrl + L`**：**清屏**。快速清除当前终端屏幕内容，效果等同于执行 `clear` 命令。
*   **`Ctrl + C`**：**终止当前任务**。放弃正在前台运行的命令。
*   **`Esc + .`**：**快速粘贴上一条命令的最后一个参数**。在输入新命令时，可以快速引用上一个命令的参数，无需重新输入。

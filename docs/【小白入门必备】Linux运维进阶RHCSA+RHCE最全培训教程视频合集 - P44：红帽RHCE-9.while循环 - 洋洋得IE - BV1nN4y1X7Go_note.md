## while循环的基本概念

![](img/c03e1d52ebf373189f725d89978b1074_10.png)

![](img/c03e1d52ebf373189f725d89978b1074_12.png)

![](img/c03e1d52ebf373189f725d89978b1074_14.png)

上一节我们介绍了`for`循环，它主要用于根据一个已知的列表或范围来重复执行命令。本节中我们来看看`while`循环。

![](img/c03e1d52ebf373189f725d89978b1074_16.png)

`while`循环的核心逻辑是：**只要给定的条件成立（为真），就会重复执行循环体内的命令**。这与`for`循环基于“值列表”进行循环的机制不同。

其基本语法结构如下：
```bash
while [ 条件 ]
do
    # 要执行的命令
done
```

![](img/c03e1d52ebf373189f725d89978b1074_18.png)

## while循环与死循环

`while`循环最常见的应用之一是创建“死循环”。所谓死循环，就是条件永远为真，循环会一直执行下去，直到被手动中断（例如按 `Ctrl+C`）。

以下是一个最简单的死循环示例，它使用一个特殊的冒号 `:` 作为条件，这个符号在Shell中代表“永远为真”：
```bash
while :
do
    echo "Hello"
done
```
运行这个脚本，它会不停地输出“Hello”，因为条件 `:` 永远成立。

## 控制循环次数

虽然`while`循环常用于死循环，但我们也可以通过变量来控制它的执行次数，使其像`for`循环一样有明确的终点。

以下是实现一个执行5次循环的示例：
```bash
#!/bin/bash
number=1
while [ $number -le 5 ]
do
    echo "循环第 $number 次"
    let number++  # 等同于 number=$((number+1))
done
```
在这个脚本中：
1.  我们初始化变量 `number=1`。
2.  循环条件是 `$number -le 5`（小于等于5）。
3.  每次循环后，使用 `let number++` 将变量 `number` 的值加1。
4.  当 `number` 变为6时，条件不再成立，循环结束。

**请注意**：如果忘记在循环体内增加变量的值（如 `let number++`），条件将永远成立，脚本就会陷入真正的死循环。

## 实践：编写猜数字游戏脚本

理解了`while`循环的基本用法后，我们来实践一个经典案例：改进之前的猜数字游戏，使其可以持续猜测，直到猜对为止。

```bash

# 生成一个1-10之间的随机数
target=$((RANDOM % 10 + 1))

# 使用while循环实现持续猜测
while :
do
    read -p "请输入你猜的数字：" guess
    if [ $guess -eq $target ]
    then
        echo "恭喜你！猜对了！"
        exit 0  # 猜对后退出脚本
    else
        echo "很遗憾，猜错了，再试一次吧！"
    fi
done
```

**脚本解析**：
1.  `target=$((RANDOM % 10 + 1))` 生成目标随机数。
2.  `while :` 开启一个无限循环。
3.  在循环体内，使用 `read` 命令获取用户输入。
4.  使用 `if` 判断用户是否猜对。
5.  如果猜对，输出成功信息并使用 `exit 0` 正常退出整个脚本，从而结束循环。
6.  如果猜错，提示用户并继续下一次循环。

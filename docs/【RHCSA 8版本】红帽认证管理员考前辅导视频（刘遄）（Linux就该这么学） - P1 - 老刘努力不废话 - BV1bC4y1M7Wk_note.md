## 红帽认证简介

在开始具体操作前，我们简要了解红帽认证的价值：

1.  **行业地位**：红帽是领先的开源解决方案供应商，其认证在Linux领域具有很高的权威性和认可度。
2.  **认证选择**：在Linux认证体系中，红帽认证（如RHCSA、RHCE）是业界普遍认可的主流选择，其含金量远超其他同类认证。
3.  **企业影响**：红帽公司本身也是科技行业的重要参与者，这从侧面印证了其技术实力与认证的价值。

1.  将所有压缩包解压到当前目录。
2.  将解压出的虚拟机文件（如 `.vmdk` 文件）复制到您本地VMware Workstation软件的虚拟机存储目录中。
3.  打开VMware Workstation，通过 **文件 -> 打开** 菜单，选择复制过来的虚拟机文件（例如 `rhel8`）来加载虚拟机。
    *   **注意**：由于镜像文件较大，加载时请稍作等待，避免因操作过快导致文件识别错误。
4.  加载成功后，您可以在虚拟机的 **管理快照** 中找到两个预设的快照：`RHCSA 8` 和 `RHCE 8`。本课程使用 `RHCSA 8` 快照。右键点击该快照，选择 **转到**，即可将虚拟机状态恢复至考试初始环境。

初始信息通常包括：
*   虚拟机的主机名与IP地址（如 `servera.lab.example.com` / `172.25.250.100`）。
*   统一的默认密码：**`flectrag`**。在考试中，除非题目特别指明，否则所有需要设置或输入密码的地方均使用此密码。
*   预配置的YUM软件仓库地址，您需要将其正确指向。
*   题目要求您完成配置后，必须保证系统重启后所有设置依然生效，否则将无法得分。

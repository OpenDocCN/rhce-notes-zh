## 🔑 Tab键自动补齐

![](img/8949abe064c5a80b59d6094fa37861b1_10.png)

![](img/8949abe064c5a80b59d6094fa37861b1_11.png)

![](img/8949abe064c5a80b59d6094fa37861b1_13.png)

![](img/8949abe064c5a80b59d6094fa37861b1_14.png)

Tab键的主要作用是**自动补齐**。当你不确定命令、文件名或路径的全称时，可以使用Tab键来帮助你快速完成输入。

### 补齐命令
如果你只记得命令的开头部分，可以输入开头字母后按Tab键。例如，输入 `pas` 后按Tab，系统会自动补全为 `passwd`。

![](img/8949abe064c5a80b59d6094fa37861b1_16.png)

![](img/8949abe064c5a80b59d6094fa37861b1_17.png)

**注意**：按一次Tab键没有反应，通常有两种情况：
1.  以你输入的开头字母为起始的命令有多个。
2.  你输入的开头字母有误，系统找不到匹配项。

![](img/8949abe064c5a80b59d6094fa37861b1_19.png)

对于第一种情况，**连续按两次Tab键**，系统会列出所有可能的命令供你选择。例如，输入 `p` 后连按两次Tab，会显示所有以 `p` 开头的命令。

### 补齐路径和参数
Tab键同样可以用于补齐文件路径和命令参数。例如：
*   输入 `ls /etc/netw` 后按Tab，可能会自动补全为 `ls /etc/network`。
*   输入 `passwd ad` 后按Tab，可能会自动补全用户名 `admin`。

**重要提示**：Tab键的自动补齐功能依赖于一个名为 `bash-completion` 的软件包。如果你安装的是最小化（无图形界面）的Linux系统，可能默认没有此功能，需要手动安装该包。

## 📜 History历史命令调用

![](img/8949abe064c5a80b59d6094fa37861b1_21.png)

![](img/8949abe064c5a80b59d6094fa37861b1_23.png)

在命令行中执行过的命令会被系统记录下来，我们可以通过 `history` 命令查看和调用这些历史记录，避免重复输入。

### 查看历史记录
输入 `history` 命令，会按序号列出之前执行过的所有命令。
```bash
history
```
如果只想查看最近的N条记录，可以使用：
```bash
history N
```

### 调用历史命令
有多种方式可以快速调用历史命令：

1.  **通过序号调用**：使用 `!序号` 的形式。例如，`!58` 会执行历史记录中序号为58的命令。
    ```bash
    !58
    ```

2.  **通过开头字符串调用**：使用 `!字符串` 的形式，会执行**最近一条**以该字符串开头的命令。例如，`!sys` 会执行最近一条以 `sys` 开头的命令。
    ```bash
    !sys
    ```

![](img/8949abe064c5a80b59d6094fa37861b1_25.png)

![](img/8949abe064c5a80b59d6094fa37861b1_27.png)

3.  **调用上一条命令**：使用 `!!` 可以快速重新执行刚刚执行过的上一条命令。
    ```bash
    !!
    ```

![](img/8949abe064c5a80b59d6094fa37861b1_29.png)

![](img/8949abe064c5a80b59d6094fa37861b1_31.png)

![](img/8949abe064c5a80b59d6094fa37861b1_33.png)

![](img/8949abe064c5a80b59d6094fa37861b1_35.png)

4.  **搜索历史命令**：按下 `Ctrl + R` 组合键，进入反向搜索模式。此时输入命令中的任意关键词，系统会动态搜索并显示匹配的历史命令，按回车即可执行。

![](img/8949abe064c5a80b59d6094fa37861b1_37.png)

![](img/8949abe064c5a80b59d6094fa37861b1_39.png)

5.  **使用方向键**：按键盘的 **上箭头 (↑)** 和 **下箭头 (↓)** 可以逐条翻阅历史命令。

![](img/8949abe064c5a80b59d6094fa37861b1_41.png)

6.  **调用上一条命令的最后一个参数**：这是一个非常实用的小技巧。按下 `Alt + .` (点) 或 `Esc + .` (点)，可以快速输入上一条命令的最后一个参数（通常是文件路径或长参数），节省输入时间。

![](img/8949abe064c5a80b59d6094fa37861b1_43.png)

## ⚠️ 关于Root用户的特别提醒

![](img/8949abe064c5a80b59d6094fa37861b1_45.png)

在学习和实验过程中，我们可能会使用 `root`（超级管理员）用户，因为它拥有系统的最高权限，操作不受限制。

**但是，请务必牢记**：在生产环境或工作环境中，**严禁直接使用root用户进行日常操作**。因为其权限过高，一旦发生误操作（如误删系统文件），可能导致灾难性后果，如服务中断、数据丢失等。日常工作中应使用普通用户账户，仅在需要时通过 `su` 或 `sudo` 命令临时获取管理员权限。

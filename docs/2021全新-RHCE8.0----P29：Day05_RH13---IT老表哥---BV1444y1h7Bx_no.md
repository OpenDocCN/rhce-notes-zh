### 注释符 (#)

![](img/78ed9048384295792cc6a9957d8b6ad7_11.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_13.png)

井号 `#` 是注释的开头。该字符及其同一行后面的所有内容都会被 Shell 忽略。在编写脚本时，建议多加注释以提高代码可读性。

![](img/78ed9048384295792cc6a9957d8b6ad7_15.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_17.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_18.png)

**示例：**
```bash
# 这是一行注释
echo "Hello World" # 这也是注释的一部分
```

### 转义与引号

![](img/78ed9048384295792cc6a9957d8b6ad7_20.png)

为了禁用字符的特殊含义，我们可以使用反斜杠 `\`、单引号 `'` 或双引号 `"`。

![](img/78ed9048384295792cc6a9957d8b6ad7_21.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_22.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_24.png)

*   **反斜杠 `\`**：取消紧随其后的单个字符的特殊含义。
    ```bash
    echo \# This is not a comment
    ```
*   **单引号 `'`**：保留括号内所有字符的字面含义，是“绝对引用”。
    ```bash
    echo '$PATH `pwd` "Hello"'
    ```
*   **双引号 `"`**：会保留美元符号 `$`、反引号 `` ` `` 和反斜杠 `\` 的特殊含义（反斜杠仅在特定字符前保留含义），其他字符按字面处理。
    ```bash
    echo "$PATH `pwd` \"Hello\""
    ```

![](img/78ed9048384295792cc6a9957d8b6ad7_25.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_27.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_29.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_31.png)

**核心区别：**
*   单引号内所有内容原样输出。
*   双引号内会进行**变量扩展**和**命令替换**。

![](img/78ed9048384295792cc6a9957d8b6ad7_33.png)

### 变量赋值与引用

![](img/78ed9048384295792cc6a9957d8b6ad7_35.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_37.png)

为变量赋值字符串时，建议用引号括起来，否则空格会被解释为单词分隔符。

通过 `$变量名` 可以引用变量的值，这是 `${变量名}` 的简化形式。但在某些易产生歧义的情况下，必须使用花括号形式。

**示例：**
```bash
first="Zhang"
last="San"
# 错误示例：试图引用 first_san 变量（未定义）
fullname_wrong=$first_san
# 正确示例：明确引用 first 变量
fullname_right=${first}_san
echo $fullname_right # 输出：Zhang_san
```

上一节我们介绍了如何通过引号和转义来控制字符的特殊含义，本节中我们来看看 Shell 的扩展功能，它能让脚本更加强大和灵活。

## Shell 扩展功能

### 命令替换

![](img/78ed9048384295792cc6a9957d8b6ad7_39.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_41.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_43.png)

命令替换可以将命令的输出结果替换到当前位置。有两种语法：

![](img/78ed9048384295792cc6a9957d8b6ad7_44.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_45.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_47.png)

1.  反引号 `` `command` ``
2.  更推荐：`$(command)`

`$()` 支持嵌套，且更易于阅读。

![](img/78ed9048384295792cc6a9957d8b6ad7_49.png)

**示例：**
```bash
# 获取当前日期
today=$(date)
echo "Today is: $today"

![](img/78ed9048384295792cc6a9957d8b6ad7_50.png)

# 嵌套示例：获取登录用户数
user_count=$(who | wc -l)
echo "Logged in users: $user_count"
```

### 算术扩展

![](img/78ed9048384295792cc6a9957d8b6ad7_52.png)

Bash 可以执行简单的整数算术运算，使用 `$((表达式))` 语法。表达式中的变量名不需要加 `$`。

![](img/78ed9048384295792cc6a9957d8b6ad7_54.png)

**示例：**
```bash
a=5
b=3
# 基本运算
sum=$((a + b))
echo "Sum: $sum" # 输出：Sum: 8

![](img/78ed9048384295792cc6a9957d8b6ad7_56.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_57.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_59.png)

# 直接使用
echo $((a * b + 2)) # 输出：17

![](img/78ed9048384295792cc6a9957d8b6ad7_61.png)

# 支持嵌套和自增/自减
i=1
echo $((i++)) # 输出：1 (先取值，后加1)
echo $i       # 输出：2
echo $((++i)) # 输出：3 (先加1，后取值)
```

**常用运算符：**
*   `+`, `-`, `*`, `/` (整除), `%` (取余)
*   `**` (幂运算)
*   `++`, `--` (自增/自减)
*   `()` (改变优先级)

![](img/78ed9048384295792cc6a9957d8b6ad7_63.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_65.png)

### for 循环

![](img/78ed9048384295792cc6a9957d8b6ad7_78.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_79.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_81.png)

`for` 循环用于遍历一个项目列表。

![](img/78ed9048384295792cc6a9957d8b6ad7_84.png)

**基本语法：**
```bash
for variable in item1 item2 ... itemN
do
    # 循环体命令
    echo "Processing $variable"
done
```

![](img/78ed9048384295792cc6a9957d8b6ad7_86.png)

**项目列表的来源非常灵活：**

以下是几种常见的 `for` 循环用法：

![](img/78ed9048384295792cc6a9957d8b6ad7_88.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_90.png)

*   **直接列出：**
    ```bash
    for file in f1.txt f2.txt f3.txt; do echo $file; done
    ```
*   **通配符扩展：**
    ```bash
    for script in /etc/*.sh; do echo $script; done
    ```
*   **命令替换：**
    ```bash
    for user in $(cat userlist.txt); do echo "User: $user"; done
    ```
*   **花括号扩展：**
    ```bash
    for num in {1..5}; do echo $num; done
    ```
*   **seq 命令：**
    ```bash
    for i in $(seq 1 2 10); do echo $i; done # 从1到10，步长为2
    ```
*   **位置参数：**
    ```bash
    # 脚本中遍历所有传入的参数
    for param in "$@"; do echo "Arg: $param"; done
    # 简写形式
    for param; do echo "Arg: $param"; done
    ```

### while 循环

`while` 循环在测试条件为真（退出状态码为 0）时，持续执行循环体。

![](img/78ed9048384295792cc6a9957d8b6ad7_92.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_94.png)

**基本语法：**
```bash
while [ 测试条件 ]
do
    # 循环体命令
done
```

**示例：**
```bash
count=3
while [ $count -gt 0 ]
do
    echo "Countdown: $count"
    count=$((count - 1))
done
echo "Blast off!"
```
**注意：** 循环体内必须有改变条件的语句，否则会成为死循环。

### until 循环

![](img/78ed9048384295792cc6a9957d8b6ad7_96.png)

`until` 循环与 `while` 相反，它在测试条件为假（退出状态码非 0）时执行，条件为真时停止。

![](img/78ed9048384295792cc6a9957d8b6ad7_98.png)

**基本语法：**
```bash
until [ 测试条件 ]
do
    # 循环体命令
done
```

**示例：**
```bash
number=1
until [ $number -eq 6 ]
do
    echo "Number is: $number"
    number=$((number + 1))
done
echo "Reached 6."
```

![](img/78ed9048384295792cc6a9957d8b6ad7_100.png)

![](img/78ed9048384295792cc6a9957d8b6ad7_102.png)

`while` 和 `until` 都可以使用多个测试命令，但**以最后一个命令的退出状态码**作为循环是否继续的判断依据。

## 总结

![](img/78ed9048384295792cc6a9957d8b6ad7_106.png)

本节课中我们一起学习了 Bash Shell 的核心扩展功能和循环控制结构。

1.  **特殊字符与引用**：理解了 `#`, `\`, `'`, `"` 的作用，以及 `${变量}` 在消除歧义时的重要性。
2.  **扩展功能**：
    *   **命令替换**：使用 `$(command)` 将命令输出作为值使用。
    *   **算术扩展**：使用 `$((expression))` 进行整数运算。
3.  **循环控制**：
    *   **for 循环**：擅长遍历已知的列表（文件、数字序列、命令输出等）。
    *   **while 循环**：在条件为真时重复执行，适用于次数不定的循环。
    *   **until 循环**：在条件为假时重复执行，是 `while` 的逻辑反面。

![](img/78ed9048384295792cc6a9957d8b6ad7_108.png)

这些知识是编写自动化脚本、处理批量任务的基础。请务必通过实践练习来巩固理解，例如尝试用 `for` 循环批量重命名文件，或用 `while` 循环创建一个简单的交互式菜单。

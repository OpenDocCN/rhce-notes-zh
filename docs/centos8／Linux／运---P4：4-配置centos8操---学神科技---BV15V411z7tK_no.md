在本节课中，我们将学习如何配置 CentOS 8 操作系统的安装参数，并完成自定义磁盘分区。这是安装过程中至关重要的一步，直接关系到系统未来的稳定性和可扩展性。

## 语言与时区设置 🌍

安装程序启动后，首先进入的是语言选择界面。默认选项是 English，我们需要选择 **Chinese** 以使用中文界面。在中文选项中，选择 **简体中文** 即可。

![](img/54d1a1f66f9aaca53601a4b7666fae25_7.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_9.png)

选择完成后，点击“继续”按钮。接下来需要配置键盘布局和时区。键盘布局保持默认的“汉语”即可。时区设置非常重要，因为它会影响系统日志记录的时间戳。请选择 **亚洲/上海** 时区。

![](img/54d1a1f66f9aaca53601a4b7666fae25_13.png)

## 磁盘分区方案 🗂️

![](img/54d1a1f66f9aaca53601a4b7666fae25_25.png)

上一节我们选择了安装源，本节中我们来看看如何规划磁盘空间。在“安装目的地”界面，选择要进行安装的磁盘，然后点击“自定义”进行手动分区。

![](img/54d1a1f66f9aaca53601a4b7666fae25_27.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_29.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_31.png)

系统默认可能使用 LVM（逻辑卷管理），但对于初学者，我们选择更直观的 **标准分区**。

![](img/54d1a1f66f9aaca53601a4b7666fae25_33.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_35.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_36.png)

以下是创建分区的步骤和推荐大小：

![](img/54d1a1f66f9aaca53601a4b7666fae25_38.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_40.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_42.png)

1.  **引导分区 (/boot)**：这个分区存放系统内核和引导程序。在 CentOS 8 中，建议分配 **1024 MB (1 GB)** 或更大空间，以确保未来系统升级时有足够空间存放新内核。文件系统类型选择 **xfs**。
    *   **公式**：`/boot` 分区大小 >= 1024 MB
2.  **交换分区 (swap)**：当物理内存不足时，系统会使用这部分空间。对于学习环境，分配 **2048 MB (2 GB)** 即可。其文件系统类型固定为 **swap**。
    *   **注意**：传统上 swap 大小是内存的 1.5-2 倍，但当物理内存很大（如超过 16GB）时，此规则不再适用。
3.  **根分区 (/)**: 这是系统的主要分区，存放操作系统文件和日志。建议分配 **50 GB**。文件系统类型选择 **xfs**。
    *   **公式**：`/` 分区大小 ≈ 50 GB
4.  **数据分区 (可选，如 /data)**：可以将剩余的所有空间分配给一个独立的数据分区（例如 `/data`）。这样做的好处是，即使系统盘（根分区）出现问题需要重装，数据分区里的文件也不会丢失。
    *   **代码示例（分区规划）**:
        ```
        /boot   -> 1024M, xfs
        swap    -> 2048M, swap
        /       -> 50G, xfs
        /data   -> 剩余所有空间, xfs
        ```

![](img/54d1a1f66f9aaca53601a4b7666fae25_44.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_46.png)

分区设置完成后，点击“完成”并接受更改。

## 内核转储与网络配置 🌐

接下来，关闭 **Kdump** 功能。这是一个内核崩溃转储机制，用于诊断系统严重错误。对于学习和一般用途，可以关闭它以节省内存资源。

![](img/54d1a1f66f9aaca53601a4b7666fae25_48.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_50.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_52.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_53.png)

网络配置是关键步骤。打开网络连接，并建议配置为 **静态 IP 地址**，这样便于后续通过 SSH 等工具进行远程管理。

![](img/54d1a1f66f9aaca53601a4b7666fae25_55.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_57.png)

以下是配置静态 IP 的步骤：
1.  进入网络设置，找到你的网卡（常见名称如 `ens33` 或 `ens160`）。
2.  关闭“自动连接”，选择“手动”配置。
3.  点击“添加”按钮，填写以下信息：
    *   **地址**：例如 `192.168.1.63`
    *   **子网掩码**：例如 `24` (相当于 `255.255.255.0`)
    *   **网关**：例如 `192.168.1.1`
    *   **DNS 服务器**：例如 `8.8.8.8` 或你的网关地址
4.  点击“保存”。

![](img/54d1a1f66f9aaca53601a4b7666fae25_59.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_61.png)

同时，在此界面设置 **主机名**，例如 `xuegod63`。保持主机名与 IP 地址的对应关系，有助于后续学习和实验。

![](img/54d1a1f66f9aaca53601a4b7666fae25_63.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_65.png)

## 用户设置与安装启动 👤

![](img/54d1a1f66f9aaca53601a4b7666fae25_67.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_69.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_71.png)

![](img/54d1a1f66f9aaca53601a4b7666fae25_73.png)

最后，为系统管理员 `root` 账户设置一个密码。为了课程实验方便，可以设置为简单的 `123456`。**请注意，在生产环境中必须使用高强度密码。**

目前阶段可以不创建额外的普通用户，等待系统安装完成后再创建。

确认所有配置（语言、时区、分区、网络、root 密码）无误后，点击 **“开始安装”**。系统将开始从安装介质复制文件并安装软件包，这个过程需要一些时间，请耐心等待。

![](img/54d1a1f66f9aaca53601a4b7666fae25_75.png)

安装完成后，点击 **“重启”** 即可进入新安装的 CentOS 8 系统。

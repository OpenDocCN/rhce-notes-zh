## 获取帮助的常用方法

上一节我们介绍了各种命令的基本操作，本节中我们来看看如何探索命令的更多选项。例如，使用 `ls -lh` 命令时，`-h` 选项能以更人性化的方式显示文件大小。那么，如何自行发现这些选项呢？

![](img/fcd2c1f7147d35a04f6fc493281a4e61_9.png)

![](img/fcd2c1f7147d35a04f6fc493281a4e61_11.png)

Linux系统提供了几种自我帮助的工具。

![](img/fcd2c1f7147d35a04f6fc493281a4e61_13.png)

![](img/fcd2c1f7147d35a04f6fc493281a4e61_15.png)

### 1. 使用 `--help` 选项

大多数命令都支持 `--help` 选项（或其简写 `-h`），它能快速显示该命令的基本用法和选项说明。

以下是使用 `df` 命令获取帮助的示例：
```bash
df --help
```
执行此命令后，你会看到关于 `df` 命令的详细说明，包括所有可用选项及其含义。例如，`-h` 选项的解释是“以易读方式显示大小（例如 1K, 234M, 2G）”。

### 2. 查阅手册页：`man` 命令

`--help` 提供的信息通常比较简洁。若需要更详细的文档，包括命令描述、参数详解、示例甚至相关文件，可以使用 `man`（manual的缩写）命令。

例如，查看 `df` 命令的完整手册：
```bash
man df
```
这将打开一个分页显示的详细手册。手册内容通常比 `--help` 更全面。

#### 在 `man` 页面中导航

进入 `man` 页面后，可以使用以下快捷键进行操作：

![](img/fcd2c1f7147d35a04f6fc493281a4e61_17.png)

![](img/fcd2c1f7147d35a04f6fc493281a4e61_19.png)

*   **退出**：按 `q` 键。
*   **向下翻页**：按 `空格键` 或 `Page Down` 键。
*   **向上翻页**：按 `b` 键或 `Page Up` 键。
*   **向下翻半页**：按 `d` 键。
*   **向上翻半页**：按 `u` 键。
*   **跳转到页面顶部**：按 `g` 键。
*   **跳转到页面底部**：按 `Shift + g`（即大写 `G`）键。
*   **搜索关键字**：按 `/` 键，然后输入要查找的字符串（如 `/sort`），按回车。搜索到的结果会高亮显示。
*   **查找下一个匹配项**：按 `n` 键。
*   **查找上一个匹配项**：按 `Shift + n`（即大写 `N`）键。

![](img/fcd2c1f7147d35a04f6fc493281a4e61_21.png)

### 3. 理解手册章节

`man` 手册根据内容类型分为多个章节。例如，`passwd` 既是一个命令（修改密码），也是一个配置文件（`/etc/passwd`）。

![](img/fcd2c1f7147d35a04f6fc493281a4e61_23.png)

![](img/fcd2c1f7147d35a04f6fc493281a4e61_25.png)

![](img/fcd2c1f7147d35a04f6fc493281a4e61_27.png)

*   查看命令 `passwd` 的手册（第1章）：
    ```bash
    man passwd
    ```
*   查看配置文件 `passwd` 的格式说明（第5章）：
    ```bash
    man 5 passwd
    ```

![](img/fcd2c1f7147d35a04f6fc493281a4e61_29.png)

如果不指定章节号，`man` 默认显示编号最小的章节内容。

### 4. 通过关键字查找命令：`man -k`

如果你不记得完整的命令名，只知道其功能涉及某个关键词，可以使用 `man -k` 进行搜索。

![](img/fcd2c1f7147d35a04f6fc493281a4e61_31.png)

例如，查找所有手册页标题或描述中包含 “password” 的命令：
```bash
man -k password
```
或者使用 `apropos` 命令，它与 `man -k` 功能相同：
```bash
apropos password
```

![](img/fcd2c1f7147d35a04f6fc493281a4e61_33.png)

### 5. 更详细的文档：`pinfo` 与 `/usr/share/doc`

*   **`pinfo` 命令**：它以超链接文本的形式提供信息，比 `man` 页面更详细，但操作也稍复杂（使用方向键选择链接，回车进入，`u` 返回上级）。使用频率相对较低。
    ```bash
    pinfo df
    ```
*   **`/usr/share/doc` 目录**：这里存放着大量软件包附带的详细文档、示例配置文件（example）和版权信息。这是获取深度信息的最佳官方来源之一。你可以根据软件包名称在此目录下查找相关文档。

## 总结

本节课中我们一起学习了在Linux系统中进行自我帮助的多种方法。从最快速的 `--help` 选项，到最常用的 `man` 手册，再到通过关键字搜索命令 (`man -k`)，以及查阅更详细的 `pinfo` 和 `/usr/share/doc` 文档，这些工具构成了强大的自我学习体系。熟练掌握这些方法，尤其是 `man` 命令的导航与搜索，将使你能够在Linux世界中独立探索，快速解决遇到的问题。

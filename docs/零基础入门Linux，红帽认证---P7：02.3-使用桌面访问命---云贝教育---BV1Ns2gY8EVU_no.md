# Linux入门与红帽认证：RH124：P7：02.3 使用桌面访问命令-实验 🖥️

在本节课程中，我们将通过一个实验来学习如何在Linux图形化桌面环境中访问和使用命令行终端。这个实验旨在帮助你熟悉课程实验的基本流程，包括如何准备实验环境、执行命令以及完成练习。

## 概述

本实验将指导你完成以下操作：登录工作站、修改用户密码、锁定与解锁屏幕、了解图形界面的关机操作，以及最终完成实验。我们将使用`student`用户身份在桌面环境中进行操作。

上一节我们介绍了命令行基础，本节中我们来看看如何在图形化界面中完成一个完整的实验流程。

### 实验准备

首先，我们需要登录到工作站并准备实验环境。

1.  登录到`workstation`的本地控制台。
    ![](img/e999a8d9406ee81be4432317640339d7_1.png)

2.  以`student`用户身份登录。
    ![](img/e999a8d9406ee81be4432317640339d7_3.png)

![](img/e999a8d9406ee81be4432317640339d7_5.png)

3.  登录后，在终端中执行以下命令来准备本实验所需的桌面环境：
    ```bash
    lab start cli-desktop
    ```
    *提示：输入命令时，可以使用 `Tab` 键进行自动补全。*
    ![](img/e999a8d9406ee81be4432317640339d7_5.png)

![](img/e999a8d9406ee81be4432317640339d7_22.png)

![](img/e999a8d9406ee81be4432317640339d7_24.png)

1.  点击桌面右上角的系统菜单，选择“锁定”。
    ![](img/e999a8d9406ee81be4432317640339d7_26.png)

![](img/e999a8d9406ee81be4432317640339d7_26.png)

2.  屏幕锁定后，输入新密码 `55TURNk3y` 进行解锁。

### 图形界面关机操作

本步骤旨在让你了解如何从图形界面发起关机，并在过程中取消操作。

![](img/e999a8d9406ee81be4432317640339d7_29.png)

1.  点击桌面右上角的系统菜单，选择“电源关闭 / Power Off”。
    ![](img/e999a8d9406ee81be4432317640339d7_29.png)
    ![](img/e999a8d9406ee81be4432317640339d7_30.png)

![](img/e999a8d9406ee81be4432317640339d7_30.png)

2.  在弹出的确认对话框中，选择“取消 / Cancel”，而不真正关闭系统。
    ![](img/e999a8d9406ee81be4432317640339d7_32.png)

![](img/e999a8d9406ee81be4432317640339d7_32.png)

### 完成实验

所有操作完成后，需要执行命令来标记本实验结束。

1.  打开一个新的终端窗口。新终端默认会位于当前用户的主目录（`~` 符号代表主目录）。
    ![](img/e999a8d9406ee81be4432317640339d7_34.png)

![](img/e999a8d9406ee81be4432317640339d7_34.png)

2.  在终端中执行以下命令来完成本实验：
    ```bash
    lab finish cli-desktop
    ```
    ![](img/e999a8d9406ee81be4432317640339d7_36.png)
    命令成功执行后，实验即告完成。
    ![](img/e999a8d9406ee81be4432317640339d7_37.png)

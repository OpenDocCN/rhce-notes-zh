## 安装引导菜单选择

上一节我们介绍了如何启动安装介质，本节我们来看看启动后出现的引导菜单选项。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_12.png)

启动后，你会看到包含三个选项的菜单。

以下是三个选项的含义：

1.  **Install Red Hat Enterprise Linux 7.0**：这是正常安装操作系统的选项。
2.  **Test this media & install Red Hat Enterprise Linux 7.0**：此选项会先检测安装介质的完整性，然后再开始安装。这个过程非常耗时。
3.  **Troubleshooting**：此选项用于进入救援模式以修复系统。在后续课程中可能会讲到。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_14.png)

如果目的是安装操作系统，请选择第一个选项。

## 安装语言设置

![](img/b3ec2d819c1d58dec813f624b9f1eb66_16.png)

选择第一个选项后，系统会开始加载并进入图形化安装界面。首先看到的是语言选择界面。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_18.png)

在这个安装过程中，你需要选择安装时使用的语言。我的原则是建议大家使用英文界面。使用中文界面可能会导致部分内容显示不习惯，且某些目录或设置项的名称可能仍是英文，造成混淆。为了后续学习的统一和避免不必要的麻烦，请选择英文。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_20.png)

请选择“English (United States)”，然后点击右下角的“Continue”按钮继续。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_22.png)

## 安装信息总览与配置

RHEL7的安装界面与早期版本（如RHEL6）不同，它不是一步步的向导，而是将所有配置项集中在一个总览界面中，需要你手动点开每一项进行设置。

如果最终安装结果与预期不符，问题很可能就出在这个界面的配置上。

## 配置日期与时间

首先，我们配置“DATE & TIME”（日期和时间）。日期和时间对于系统至关重要。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_30.png)

![](img/b3ec2d819c1d58dec813f624b9f1eb66_31.png)

如果系统日期和时间不正确，可能会导致严重问题。例如，在极端情况下，你甚至可能无法访问网络。这是因为现代许多网站都使用HTTPS协议进行加密通信，而加密过程依赖于精确的时间戳。如果客户端（你的电脑）与服务器的时间差异过大，服务器可能会拒绝你的访问请求。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_33.png)

你可能会有疑问：为什么我们的个人电脑和手机很少需要手动调整时间？这是因为设备在联网后，会自动与网络时间协议（NTP）服务器进行时间同步。例如，Windows系统默认与 `time.windows.com` 同步。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_35.png)

在安装界面中，你需要设置正确的时区以获得本地时间。全球使用“协调世界时（UTC）”作为统一的时间标准。中国位于东八区，因此本地时间应为 **UTC+8**。

在安装界面中，你还可以启用网络时间同步（NTP）。以下是配置步骤：

![](img/b3ec2d819c1d58dec813f624b9f1eb66_37.png)

1.  在地图上点击区域或从列表中选择“Asia/Shanghai”。
2.  将“Network Time”开关设置为“ON”。
3.  系统会使用红帽官方的NTP服务器。在企业环境中，为了更高的可靠性和精度，通常会配置内部或付费的专用时间服务器。你可以通过点击“+”号添加指定的NTP服务器地址，例如 `172.25.254.250`。

![](img/b3ec2d819c1d58dec813f624b9f1eb66_39.png)

配置完成后，点击左上角的“Done”返回主界面。

## 文件路径：定位文件的基石

在Linux系统中，一切皆文件，包括目录、设备、链接等。要访问一个文件，首先需要知道它的路径。路径分为两种：绝对路径和相对路径。

![](img/286544f679d254ba11dbe417c8d76698_10.png)

![](img/286544f679d254ba11dbe417c8d76698_12.png)

![](img/286544f679d254ba11dbe417c8d76698_14.png)

**绝对路径**是从根目录 `/` 开始的完整路径。例如，网卡配置文件的绝对路径是：
```
/etc/sysconfig/network-scripts/ifcfg-ens160
```

![](img/286544f679d254ba11dbe417c8d76698_17.png)

![](img/286544f679d254ba11dbe417c8d76698_22.png)

![](img/286544f679d254ba11dbe417c8d76698_24.png)

**相对路径**是相对于当前工作目录的路径。当前工作目录可以通过 `pwd` 命令查看。以下是一些特殊的路径表示法：
*   `~` 代表当前用户的家目录。
*   `.` 代表当前目录。
*   `..` 代表上一级目录。

## 查看目录内容：`ls` 命令详解

`ls` 命令用于列出目录中的文件和子目录。它是我们最常用的命令之一。

以下是 `ls` 命令的一些常用选项：

*   `ls -l`：以长格式列出详细信息。
*   `ls -a`：显示所有文件，包括以 `.` 开头的隐藏文件。
*   `ls -d`：显示目录本身的信息，而不是其内容。
*   `ls -h`：以人类可读的格式（如K, M, G）显示文件大小。
*   `ls -t`：按修改时间排序，最新的在前。
*   `ls -r`：反向排序。

![](img/286544f679d254ba11dbe417c8d76698_44.png)

当我们使用 `ls -l`（或其别名 `ll`）时，会看到类似下面的输出：
```
-rw-r--r--. 1 root root 1584 Jun 20 23:09 ifcfg-ens160
```
上一节我们介绍了如何定位文件，本节中我们来看看如何详细查看文件的属性。`ls -l` 输出的每一列含义如下：

![](img/286544f679d254ba11dbe417c8d76698_46.png)

![](img/286544f679d254ba11dbe417c8d76698_47.png)

1.  **文件类型与权限**：第一个字符表示文件类型，后续9个字符表示权限。
    *   `-`：普通文件
    *   `d`：目录
    *   `l`：符号链接（软链接）
    *   `b`：块设备文件（如硬盘）
    *   `c`：字符设备文件（如键盘）
    *   `s`：套接字文件
    *   `p`：管道文件
2.  **硬链接数**：指向此文件inode的链接数量。
3.  **文件所有者**：拥有该文件的用户。
4.  **文件所属组**：拥有该文件的用户组。
5.  **文件大小**：默认以字节为单位，`-h` 选项可转换为易读格式。
6.  **最后修改时间**：文件内容最后一次被修改的时间。
7.  **文件名**。

## 切换工作目录：`cd` 命令

![](img/286544f679d254ba11dbe417c8d76698_55.png)

`cd` 命令用于切换当前工作目录，其用法与Windows系统类似。

以下是 `cd` 命令的常用用法：
*   `cd ~` 或 `cd`：切换到当前用户的家目录。
*   `cd ..`：切换到上一级目录。
*   `cd -`：切换到上一次所在的目录。
*   `cd /path/to/dir`：使用绝对路径切换到指定目录。
*   `cd dir`：使用相对路径切换到当前目录下的子目录。

## 目录管理：创建与删除

![](img/286544f679d254ba11dbe417c8d76698_57.png)

了解了如何查看和切换目录后，接下来我们学习如何创建和删除目录。

![](img/286544f679d254ba11dbe417c8d76698_59.png)

![](img/286544f679d254ba11dbe417c8d76698_60.png)

![](img/286544f679d254ba11dbe417c8d76698_62.png)

### 创建目录：`mkdir`

使用 `mkdir` 命令创建新目录。
```
mkdir new_folder      # 在当前目录创建名为 new_folder 的目录
mkdir -p a/b/c        # 递归创建目录结构，即使父目录不存在
```

### 删除目录：`rmdir` 与 `rm`

![](img/286544f679d254ba11dbe417c8d76698_64.png)

![](img/286544f679d254ba11dbe417c8d76698_66.png)

`rmdir` 命令用于删除**空目录**。
```
rmdir empty_folder    # 删除空目录 empty_folder
```

![](img/286544f679d254ba11dbe417c8d76698_68.png)

![](img/286544f679d254ba11dbe417c8d76698_70.png)

![](img/286544f679d254ba11dbe417c8d76698_72.png)

![](img/286544f679d254ba11dbe417c8d76698_74.png)

![](img/286544f679d254ba11dbe417c8d76698_75.png)

![](img/286544f679d254ba11dbe417c8d76698_77.png)

![](img/286544f679d254ba11dbe417c8d76698_78.png)

要删除非空目录，需要使用 `rm` 命令并配合 `-r`（递归）选项。**此操作非常危险，请谨慎使用**。
```
rm -r folder_name     # 递归删除目录及其内部所有内容
rm -rf folder_name    # 强制递归删除，不进行任何确认提示（极度危险！）
```

![](img/286544f679d254ba11dbe417c8d76698_79.png)

**警告**：生产环境中，应尽量避免使用 `rm -rf` 命令，尤其是在根目录 `/` 下。误操作可能导致系统数据丢失。

## 文件管理：创建、查看、复制与移动

![](img/286544f679d254ba11dbe417c8d76698_81.png)

### 创建文件：`touch` 与 `echo`

`touch` 命令主要用来修改文件时间戳，但常被用于创建新的空文件。
```
touch new_file.txt    # 创建一个名为 new_file.txt 的空文件
```

`echo` 命令结合重定向 `>` 也可以创建文件，并写入内容。
```
echo “Hello World” > hello.txt  # 创建 hello.txt 并写入 “Hello World”
```

### 查看文件内容

以下是查看文件内容的几个常用命令：

*   `cat`：一次性显示整个文件内容。
    ```
    cat filename
    cat -n filename   # 显示行号
    ```
*   `more`：分页显示文件内容，只能向下翻页。
    ```
    more filename
    ```
*   `less`：分页显示文件内容，可以上下翻页、搜索，功能比 `more` 更强大。
    ```
    less filename
    ```
    *   在 `less` 视图中：空格键向下翻页，`b` 键向上翻页，`/关键词` 向下搜索，`?关键词` 向上搜索，`q` 键退出。
*   `head`：显示文件开头部分，默认10行。
    ```
    head filename
    head -n 5 filename # 显示前5行
    ```
*   `tail`：显示文件末尾部分，默认10行。`-f` 选项常用于实时追踪日志文件。
    ```
    tail filename
    tail -n 5 filename # 显示最后5行
    tail -f logfile    # 实时监控 logfile 的新增内容
    ```

![](img/286544f679d254ba11dbe417c8d76698_83.png)

### 复制与移动文件：`cp` 和 `mv`

![](img/286544f679d254ba11dbe417c8d76698_87.png)

`cp` 命令用于复制文件或目录。
```
cp source.txt dest.txt           # 复制文件
cp -r source_dir/ dest_dir/      # 递归复制目录
cp -a source_dir/ dest_dir/      # 归档复制，保留所有属性（权限、时间等）
```

![](img/286544f679d254ba11dbe417c8d76698_89.png)

`mv` 命令用于移动或重命名文件/目录。
```
mv old_name.txt new_name.txt     # 重命名文件
mv file.txt /target/directory/   # 移动文件到指定目录
```

## 课程体系介绍

红帽认证体系从7版本升级到了8版本。在RHCE 7版本中，需要学习三门课程：RH124、RH134和RH254。其中，RH124和RH134构成了初级认证RHCSA，而RH254是中级认证RHCE的课程。

![](img/d5fe66a6527cf30c7cde86216fc0207b_8.png)

在RHCE 8版本中，前两门课程没有变化，依然是RH124和RH134。但第三门课程变更为RH294。RH294主要教授Ansible自动化运维工具，这与之前的RH254课程（主要讲解服务搭建，如Web、DNS等）有较大不同。

## 配置网络与远程登录

![](img/d5fe66a6527cf30c7cde86216fc0207b_46.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_48.png)

为了方便操作，我们通常使用远程登录工具连接到Linux服务器。首先，需要确保Linux系统已获取IP地址。

![](img/d5fe66a6527cf30c7cde86216fc0207b_50.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_52.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_54.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_56.png)

以下是查看IP地址的命令：
```bash
ifconfig
```
执行该命令后，在输出信息中找到`inet`一行，即可看到本机的IP地址（例如`192.168.127.128`）。

![](img/d5fe66a6527cf30c7cde86216fc0207b_58.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_60.png)

获取IP地址后，可以使用如MobaXterm、Xshell等远程连接工具，通过SSH协议（主机地址填获取到的IP，用户名填`root`）登录到系统。

## 命令行基本结构

![](img/d5fe66a6527cf30c7cde86216fc0207b_68.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_70.png)

在终端中，命令通常遵循几种基本结构。理解这些结构是学习命令行的关键。

### 1. 单独命令
这种结构只有一个命令，执行一个特定的动作。
例如，查看当前所在目录：
```bash
pwd
```
例如，列出当前目录下的文件和文件夹：
```bash
ls
```

### 2. 命令 + 执行对象
这种结构在命令后指定了操作的对象。
例如，查看指定目录（如`/root/Desktop`）下的内容：
```bash
ls /root/Desktop
```
例如，切换到指定目录：
```bash
cd /root/Desktop
```
命令与执行对象之间必须用空格分隔。

### 3. 命令 + 选项
选项用于修改命令的行为，通常以短横线`-`开头。
例如，以长列表格式显示详细信息：
```bash
ls -l
```
例如，以人类可读的格式（带单位）显示文件大小（常与`-l`结合使用）：
```bash
ls -lh
```
多个选项可以合并，例如`ls -lh`等同于`ls -l -h`。

### 4. 命令 + 选项 + 执行对象
这是最完整的命令形式。
例如，以详细且人类可读的格式查看`/root`目录：
```bash
ls -lh /root
```

## 获取命令帮助

![](img/d5fe66a6527cf30c7cde86216fc0207b_72.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_74.png)

当不清楚一个命令的用法时，可以使用帮助功能。最常用的方法是使用`--help`选项。
例如，查看`ls`命令的帮助信息：
```bash
ls --help
```
帮助信息会显示命令的用法（Usage）、可用的选项及其说明。选项前的方括号`[]`表示该部分是可选的。

![](img/d5fe66a6527cf30c7cde86216fc0207b_76.png)

以下是一些常用`ls`选项的说明：
*   `-a` 或 `--all`：显示所有文件，包括以点`.`开头的隐藏文件。
*   `-l`：使用长列表格式显示详细信息。
*   `-h`：与`-l`一起使用时，以人类可读格式（如K， M， G）显示文件大小。
*   `-d`：显示目录本身的信息，而不是其内容。

![](img/d5fe66a6527cf30c7cde86216fc0207b_78.png)

## Linux目录结构

![](img/d5fe66a6527cf30c7cde86216fc0207b_80.png)

Linux采用一种“倒挂树”的目录结构。最顶层的目录称为根目录，用单个斜线`/`表示。

![](img/d5fe66a6527cf30c7cde86216fc0207b_82.png)

在根目录下，存在许多标准子目录，例如：
*   `/bin`：存放基本命令程序。
*   `/home`：普通用户的家目录。
*   `/root`：系统管理员root的家目录。
*   `/etc`：存放系统配置文件。

每个目录下又可以包含更多的子目录和文件，形成层级结构。使用`cd ..`可以返回上一级目录，一个点`.`代表当前目录，两个点`..`代表上级目录。

![](img/d5fe66a6527cf30c7cde86216fc0207b_84.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_86.png)

![](img/d5fe66a6527cf30c7cde86216fc0207b_88.png)

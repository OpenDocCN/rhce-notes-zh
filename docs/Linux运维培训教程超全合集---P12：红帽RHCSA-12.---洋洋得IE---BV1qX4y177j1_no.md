## 什么是文本编辑器？

![](img/924d63461ab902c7fa5e251ae62468db_13.png)

![](img/924d63461ab902c7fa5e251ae62468db_15.png)

![](img/924d63461ab902c7fa5e251ae62468db_17.png)

在Windows系统中，我们通过“记事本”来创建和编辑文本文档。例如，新建一个文本文档，写入“你好美女”，然后保存。这个“记事本”就是Windows中的文本编辑器，它具备创建文件和对文件内容进行增删改查的功能。

![](img/924d63461ab902c7fa5e251ae62468db_19.png)

在Linux系统中，对应的工具是 `vi` 或 `vim`。`vim` 是从 `vi` 发展而来的一个功能更强大的文本编辑器。主要区别在于，`vim` 具备编程能力，例如在编写Shell脚本时，它能通过语法高亮来提示语法错误，而 `vi` 则没有颜色提示功能。因此，`vim` 更适合编程，而 `vi` 则常用于修改配置文件。不过，对于基础的文件编辑，两者区别不大。

![](img/924d63461ab902c7fa5e251ae62468db_21.png)

## vim的基本使用

### 命令格式

![](img/924d63461ab902c7fa5e251ae62468db_23.png)

使用vim的命令格式非常简单：
```bash
vim 文件名
```
该命令有两个特点：
1.  若文件不存在，则创建新文件并进入编辑状态。
2.  若文件已存在，则直接打开该文件进行编辑。

![](img/924d63461ab902c7fa5e251ae62468db_25.png)

![](img/924d63461ab902c7fa5e251ae62468db_27.png)

![](img/924d63461ab902c7fa5e251ae62468db_29.png)

例如，编辑一个已存在的系统主机名文件：
```bash
vim /etc/hostname
```
编辑一个不存在的文件（vim会先创建它）：
```bash
vim /opt/xxxx.tt
```

### vim的三种模式

vim的操作分为三种模式，这是初学者需要掌握的核心概念：
1.  **命令模式**：启动vim后默认进入的模式。在此模式下，键盘输入会被识别为命令，用于移动光标、删除、复制等操作，不能直接输入文本。
2.  **输入模式**：在此模式下，可以向文件中输入文本内容。
3.  **底线命令模式**：在此模式下，可以执行保存、退出、查找替换等高级操作。

三种模式的转换关系如下图所示：
```
启动vim
   |
   V
命令模式 <--(按 ESC 键)-- 输入模式
   |                           ^
   | (按 : 键)                 | (按 a, i, o 键)
   V                           |
底线命令模式
```

## 模式切换与基础编辑

上一节我们介绍了vim的三种模式，本节中我们来看看如何在它们之间切换，并进行最基础的文本编辑。

### 从命令模式进入输入模式

在命令模式下，按以下任意键可进入输入模式：
*   **i**：在**当前光标所在字符前**插入。
*   **a**：在**当前光标所在字符后**插入。
*   **o**：在**当前光标所在行的下方**插入一个新行。

![](img/924d63461ab902c7fa5e251ae62468db_37.png)

虽然 `i` 和 `a` 有细微差别，但通过移动光标都能达到相同效果，初学者记住一个即可（通常用 `i`）。

![](img/924d63461ab902c7fa5e251ae62468db_39.png)

### 输入文本与返回命令模式

进入输入模式后，即可像使用普通编辑器一样输入文本。按 `Enter` 键换行，使用方向键移动光标。
完成输入后，按 `Esc` 键即可从输入模式**返回命令模式**。

### 保存与退出（底线命令模式）

编辑完成后，需要保存并退出。以下是操作步骤：
1.  确保处于**命令模式**（按 `Esc` 键）。
2.  输入英文冒号 `:`，进入**底线命令模式**，光标会跳转到窗口底部。
3.  输入对应的保存或退出命令。

![](img/924d63461ab902c7fa5e251ae62468db_41.png)

以下是底线命令模式下常用的保存与退出命令：

![](img/924d63461ab902c7fa5e251ae62468db_43.png)

![](img/924d63461ab902c7fa5e251ae62468db_45.png)

![](img/924d63461ab902c7fa5e251ae62468db_47.png)

![](img/924d63461ab902c7fa5e251ae62468db_49.png)

*   **:w**：保存文件。
*   **:q**：退出vim（如果文件有修改未保存，会提示失败）。
*   **:wq** 或 **:x**：保存文件并退出。
*   **ZZ**（大写，在命令模式下直接按）：保存文件并退出（最快捷的方式）。
*   **:q!**：强制退出，**不保存**任何修改。
*   **:wq!**：强制保存并退出（常用于修改只读文件，需要root权限）。

## 命令模式下的高效操作

上一节我们学会了如何进入编辑状态并保存文件，本节中我们来看看在命令模式下有哪些高效的操作命令，可以让我们在不进入输入模式的情况下快速编辑文本。

在命令模式下，我们可以执行以下操作：

### 光标移动
*   **0**（数字零）：将光标移动到**当前行的行首**。
*   **$**：将光标移动到**当前行的行尾**。
*   **gg**：将光标移动到**文件的第一行**。
*   **G**（大写）：将光标移动到**文件的最后一行**。
*   **:n**（在底线命令模式）：跳转到文件的第 `n` 行。例如 `:1000` 跳转到第1000行。

### 删除、剪切与复制
*   **x**：删除**当前光标所在**的一个字符。
*   **dd**：剪切（删除）**当前光标所在的整行**。如果之后不粘贴，效果等同于删除。
*   **ndd**：剪切从当前行开始的 `n` 行。例如 `3dd` 剪切3行。
*   **yy**：复制**当前光标所在的整行**。
*   **nyy**：复制从当前行开始的 `n` 行。例如 `2yy` 复制2行。
*   **p**（小写）：将剪切或复制的内容**粘贴到当前光标所在行的下方**。
*   **C**（大写）：删除从光标位置到行尾的所有字符，并**进入输入模式**。

### 撤销与搜索
*   **u**：撤销上一次的修改操作。可以连续按 `u` 来撤销多次操作，直到文件打开时的初始状态。
*   **Ctrl + r**：重做，即撤销“撤销”操作（恢复被 `u` 撤销的内容）。
*   **/**：进入搜索状态。输入 `/` 后，再输入要查找的关键词，按 `Enter` 开始搜索。
*   **n**：在搜索后，按 `n` 键可以快速定位到**下一个**匹配的关键词。

## 底线命令模式进阶技巧

上一节我们掌握了命令模式下的核心编辑命令，本节中我们来看看底线命令模式还有哪些提升效率的进阶技巧。

![](img/924d63461ab902c7fa5e251ae62468db_53.png)

![](img/924d63461ab902c7fa5e251ae62468db_55.png)

### 显示与取消行号
在编辑大型配置文件时，显示行号非常有用。
*   **:set nu**：显示行号。
*   **:set nonu**：取消显示行号。

### 全局内容替换
`:s` 命令用于替换文本内容，功能非常强大。
*   **:%s/旧文本/新文本/**：将文件中**每一行匹配到的第一个“旧文本”** 替换为“新文本”。
*   **:%s/旧文本/新文本/g**：在全文范围内进行替换，**一行中所有匹配的“旧文本”** 都会被替换（`g` 表示全局）。

例如，将文件中所有的 `port` 替换为 `xxxx`：
```vim
:%s/port/xxxx/g
```

### 取消搜索高亮
执行搜索 (`/`) 后，所有匹配项会高亮显示。如果不希望看到高亮，可以执行：
```vim
:nohl
```

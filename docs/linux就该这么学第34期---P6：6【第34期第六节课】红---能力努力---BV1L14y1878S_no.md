在本节课中，我们将要学习Shell脚本编程的基础知识。Shell脚本是Linux系统管理和自动化运维的强大工具。我们将从最简单的脚本结构开始，逐步学习如何接收用户输入、进行条件判断，并最终编写出能够处理复杂逻辑的脚本。

## 概述

![](img/8b715f03d72b38ea6eb320a4827b0798_5.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_7.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_9.png)

一个完整的Shell脚本主要由三部分组成：脚本声明、脚本注释和脚本命令。脚本声明告诉系统使用哪个解释器来执行脚本；脚本注释是对脚本功能的说明，仅供阅读；脚本命令则是实现具体功能的指令集合。

![](img/8b715f03d72b38ea6eb320a4827b0798_11.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_13.png)

## 4.2.1 Shell脚本的组成

![](img/8b715f03d72b38ea6eb320a4827b0798_15.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_17.png)

一个简单的Shell脚本由以下三部分构成：

![](img/8b715f03d72b38ea6eb320a4827b0798_19.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_21.png)

1.  **脚本声明**：指定脚本的解释器。
    *   格式：`#!/bin/bash`
    *   作用：告诉系统使用 `/bin/bash` 这个解释器来执行脚本。

![](img/8b715f03d72b38ea6eb320a4827b0798_23.png)

2.  **脚本注释**：对脚本功能或参数进行说明。
    *   格式：以 `#` 开头的行。
    *   作用：程序不会读取这些行，仅供用户阅读和理解。

![](img/8b715f03d72b38ea6eb320a4827b0798_25.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_27.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_29.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_31.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_33.png)

3.  **脚本命令**：实现具体功能的命令序列。
    *   格式：一系列可执行的Linux命令。
    *   作用：按照从上到下的顺序执行，完成特定任务。

### 创建第一个Shell脚本

![](img/8b715f03d72b38ea6eb320a4827b0798_35.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_37.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_39.png)

让我们通过一个简单的例子来理解这三部分。

![](img/8b715f03d72b38ea6eb320a4827b0798_41.png)

```bash
#!/bin/bash
# 这是一个测试脚本
# 创建于2022年4月15日
date
ls -al
free -h
uname -r
```

![](img/8b715f03d72b38ea6eb320a4827b0798_45.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_47.png)

**脚本说明**：
*   第一行 `#!/bin/bash` 是脚本声明。
*   第二、三行以 `#` 开头，是脚本注释。
*   第四到七行是脚本命令，依次执行 `date`、`ls -al`、`free -h` 和 `uname -r`。

![](img/8b715f03d72b38ea6eb320a4827b0798_49.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_51.png)

**执行脚本**：
由于脚本文件默认没有执行权限，我们需要使用 `bash` 命令来调用解释器执行它。
```bash
bash example.sh
```

## 接收用户参数

![](img/8b715f03d72b38ea6eb320a4827b0798_61.png)

上一节我们介绍了Shell脚本的基本结构，本节中我们来看看如何让脚本变得更“智能”——即能够接收并根据用户输入的参数做出不同的响应。

![](img/8b715f03d72b38ea6eb320a4827b0798_63.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_65.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_67.png)

Shell脚本内置了一些特殊的变量，可以直接用来接收用户在执行脚本时传递的参数。

![](img/8b715f03d72b38ea6eb320a4827b0798_69.png)

*   `$0`：当前脚本的文件名。
*   `$1`、`$2`、`$3`...：第1、2、3...个参数。
*   `$#`：总共传递的参数个数。
*   `$*`：所有参数作为一个整体。
*   `$@`：所有参数，但每个参数独立。

![](img/8b715f03d72b38ea6eb320a4827b0798_71.png)

### 示例：使用内置变量

![](img/8b715f03d72b38ea6eb320a4827b0798_73.png)

创建一个脚本 `test_args.sh`：

![](img/8b715f03d72b38ea6eb320a4827b0798_75.png)

```bash
#!/bin/bash
echo "脚本名称是：$0"
echo "第一个参数是：$1"
echo "第三个参数是：$3"
echo "第五个参数是：$5"
echo "总共传递了 $# 个参数"
echo "所有参数分别是：$*"
```

![](img/8b715f03d72b38ea6eb320a4827b0798_77.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_79.png)

执行脚本并传递参数：
```bash
bash test_args.sh A B C D E F G
```
**输出示例**：
```
脚本名称是：test_args.sh
第一个参数是：A
第三个参数是：C
第五个参数是：E
总共传递了 7 个参数
所有参数分别是：A B C D E F G
```

## 条件测试语句

![](img/8b715f03d72b38ea6eb320a4827b0798_87.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_89.png)

仅仅接收参数还不够，我们还需要能够根据参数或特定条件进行判断，从而执行不同的操作。这就需要用到条件测试语句。

![](img/8b715f03d72b38ea6eb320a4827b0798_91.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_93.png)

在Shell中，测试语句使用方括号 `[ ]` 或 `test` 命令，并且**方括号内侧必须留有空格**。

条件测试主要分为四类：文件测试、逻辑测试、整数比较和字符串比较。

![](img/8b715f03d72b38ea6eb320a4827b0798_95.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_97.png)

### 1. 文件测试

![](img/8b715f03d72b38ea6eb320a4827b0798_99.png)

用于判断文件的类型、状态和权限。

![](img/8b715f03d72b38ea6eb320a4827b0798_101.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_103.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_105.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_107.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_109.png)

*   `-e 文件路径`：判断文件是否存在。
*   `-d 文件路径`：判断是否为目录。
*   `-f 文件路径`：判断是否为一般文件。
*   `-r 文件路径`：判断是否有读权限。
*   `-w 文件路径`：判断是否有写权限。
*   `-x 文件路径`：判断是否有执行权限。

![](img/8b715f03d72b38ea6eb320a4827b0798_111.png)

**示例**：
```bash
# 判断 /etc/passwd 文件是否存在且可读
[ -e /etc/passwd ] && echo "文件存在"
[ -r /etc/passwd ] && echo "文件可读"
```

### 2. 逻辑测试

![](img/8b715f03d72b38ea6eb320a4827b0798_113.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_115.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_117.png)

用于连接多个测试条件，实现复杂的逻辑判断。

![](img/8b715f03d72b38ea6eb320a4827b0798_119.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_121.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_123.png)

*   `&&`：逻辑“与”。仅当前一个命令执行成功（返回值为0）时，才执行后面的命令。
    *   格式：`命令1 && 命令2`
*   `||`：逻辑“或”。仅当前一个命令执行失败（返回值为非0）时，才执行后面的命令。
    *   格式：`命令1 || 命令2`
*   `!`：逻辑“非”。对测试条件的结果取反。
    *   格式：`[ ! 条件 ]`

![](img/8b715f03d72b38ea6eb320a4827b0798_125.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_127.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_129.png)

**示例**：
```bash
# 如果用户是root，则输出“管理员”，否则输出“普通用户”
[ $USER = "root" ] && echo "管理员" || echo "普通用户"
```

![](img/8b715f03d72b38ea6eb320a4827b0798_131.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_133.png)

### 3. 整数比较

![](img/8b715f03d72b38ea6eb320a4827b0798_135.png)

用于比较两个整数值的大小关系。**切记不要使用数学符号（>， <， =），而要用专用操作符**。

![](img/8b715f03d72b38ea6eb320a4827b0798_137.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_139.png)

*   `-eq`：等于（Equal）
*   `-ne`：不等于（Not Equal）
*   `-gt`：大于（Greater Than）
*   `-lt`：小于（Less Than）
*   `-ge`：大于或等于（Greater or Equal）
*   `-le`：小于或等于（Less or Equal）

**示例**：
```bash
# 判断10是否大于5
[ 10 -gt 5 ] && echo "10大于5"
```

![](img/8b715f03d72b38ea6eb320a4827b0798_141.png)

### 4. 字符串比较

![](img/8b715f03d72b38ea6eb320a4827b0798_145.png)

用于比较两个字符串。

![](img/8b715f03d72b38ea6eb320a4827b0798_149.png)

*   `=` 或 `==`：字符串相等。
*   `!=`：字符串不相等。
*   `-z 字符串`：判断字符串是否为空。
*   `-n 字符串`：判断字符串是否非空。

![](img/8b715f03d72b38ea6eb320a4827b0798_151.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_153.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_159.png)

**示例**：
```bash
# 判断变量是否已定义（非空）
```

## 流程控制语句

掌握了条件测试，我们就可以使用流程控制语句来编写具有判断和循环能力的复杂脚本了。本节我们重点学习 `if` 条件语句。

![](img/8b715f03d72b38ea6eb320a4827b0798_187.png)

### if 条件语句

![](img/8b715f03d72b38ea6eb320a4827b0798_189.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_191.png)

`if` 语句有三种基本结构：单分支、双分支和多分支。

![](img/8b715f03d72b38ea6eb320a4827b0798_193.png)

#### 1. 单分支 if 语句
若条件成立，则执行预设操作；否则不执行任何操作。

![](img/8b715f03d72b38ea6eb320a4827b0798_195.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_197.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_199.png)

**语法**：
```bash
if 条件测试命令; then
    命令序列
fi
```

![](img/8b715f03d72b38ea6eb320a4827b0798_201.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_203.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_205.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_207.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_209.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_211.png)

**示例**：如果目录不存在，则创建它。
```bash
#!/bin/bash
DIR="/tmp/testdir"
if [ ! -d $DIR ]; then
    mkdir -p $DIR
    echo "目录 $DIR 创建成功。"
fi
```

![](img/8b715f03d72b38ea6eb320a4827b0798_213.png)

#### 2. 双分支 if 语句
若条件成立，则执行操作A；否则执行操作B。

![](img/8b715f03d72b38ea6eb320a4827b0798_215.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_217.png)

**语法**：
```bash
if 条件测试命令; then
    命令序列A
else
    命令序列B
fi
```

**示例**：判断主机是否在线。
```bash
#!/bin/bash
ping -c 3 -i 0.2 -W 3 $1 &> /dev/null
if [ $? -eq 0 ]; then
    echo "主机 $1 在线。"
else
    echo "主机 $1 离线。"
fi
```
**执行**：`bash ping_host.sh 192.168.1.1`

![](img/8b715f03d72b38ea6eb320a4827b0798_219.png)

#### 3. 多分支 if 语句
允许进行多次条件判断，直到匹配成功为止。

**语法**：
```bash
if 条件测试命令1; then
    命令序列1
elif 条件测试命令2; then
    命令序列2
elif ...
then
    ...
else
    命令序列N
fi
```

**示例**：成绩等级判断。
```bash
#!/bin/bash
read -p "请输入您的分数（0-100）：" GRADE
if [ $GRADE -ge 85 ] && [ $GRADE -le 100 ]; then
    echo "优秀！"
elif [ $GRADE -ge 70 ] && [ $GRADE -le 84 ]; then
    echo "良好。"
else
    echo "继续努力。"
fi
```

### for 循环语句

`for` 循环语句用于根据一个列表或范围，重复执行一组命令。它非常适合进行批量操作。

**语法**：
```bash
for 变量名 in 取值列表
do
    命令序列
done
```

![](img/8b715f03d72b38ea6eb320a4827b0798_221.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_223.png)

![](img/8b715f03d72b38ea6eb320a4827b0798_225.png)

**示例1**：批量创建用户。
假设我们有一个用户列表文件 `user_list.txt`，内容如下：
```
zhangsan
lisi
wangwu
```
创建脚本 `add_users.sh`：
```bash
#!/bin/bash
read -s -p "请输入初始密码：" PASSWD
for UNAME in `cat user_list.txt`
do
    id $UNAME &> /dev/null
    if [ $? -eq 0 ]; then
        echo "$UNAME 用户已存在。"
    else
        useradd $UNAME
        echo $PASSWD | passwd --stdin $UNAME &> /dev/null
        echo "用户 $UNAME 创建成功。"
    fi
done
```

**示例2**：批量检测主机状态。
假设我们有一个IP地址列表文件 `ip_list.txt`。
创建脚本 `scan_hosts.sh`：
```bash
#!/bin/bash
for IP in `cat ip_list.txt`
do
    ping -c 3 -i 0.2 -W 3 $IP &> /dev/null
    if [ $? -eq 0 ]; then
        echo "主机 $IP 在线。"
    else
        echo "主机 $IP 离线。"
    fi
done
```

## 总结

![](img/8b715f03d72b38ea6eb320a4827b0798_227.png)

本节课中我们一起学习了Shell脚本编程的核心基础。

1.  **脚本结构**：我们了解了Shell脚本由声明(`#!`)、注释(`#`)和命令三部分组成。
2.  **参数传递**：学会了使用 `$0`, `$1`, `$#`, `$*` 等内置变量来接收和处理用户输入的参数。
3.  **条件测试**：掌握了文件测试、逻辑测试、整数比较和字符串比较的方法，这是脚本拥有“判断力”的关键。
4.  **流程控制**：
    *   使用 `if` 语句让脚本能够根据不同的条件执行不同的分支。
    *   使用 `for` 循环语句对列表中的项目进行批量、重复的操作，极大地提升了自动化运维的效率。

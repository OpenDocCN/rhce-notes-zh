## 移动与重命名：mv命令

![](img/5075da74c6984c046f31631b91024b3a_9.png)

![](img/5075da74c6984c046f31631b91024b3a_11.png)

![](img/5075da74c6984c046f31631b91024b3a_13.png)

上一节我们介绍了`cp`命令用于复制文件，本节中我们来看看`mv`命令。`mv`命令用于**移动**文件或目录到其他位置，其功能类似于Windows系统中的“剪切”。同时，它也可以用来修改文件或目录的名称。

![](img/5075da74c6984c046f31631b91024b3a_15.png)

命令的基本格式与`cp`命令类似：
```bash
mv [选项] 源文件或目录 目标文件或目录
```

![](img/5075da74c6984c046f31631b91024b3a_17.png)

![](img/5075da74c6984c046f31631b91024b3a_19.png)

![](img/5075da74c6984c046f31631b91024b3a_21.png)

以下是`mv`命令的一些常见用法：
*   **移动文件**：将文件`hello`移动到`/opt/t1/`目录下。
    ```bash
    mv hello /opt/t1/
    ```
*   **移动多个项目**：将当前目录下的`t6`、`t7`、`t8`目录移动到`/opt/test/`目录下。
    ```bash
    mv t6 t7 t8 /opt/test/
    ```
*   **移动并重命名**：将`test`目录移动到根目录`/`下，并重命名为`xxxxx`。
    ```bash
    mv test /xxxxx
    ```
*   **仅重命名**：如果源和目标在同一目录下，则实现重命名效果。例如，将当前目录下的`test1`重命名为`newname`。
    ```bash
    mv test1 newname
    ```

![](img/5075da74c6984c046f31631b91024b3a_23.png)

**核心概念**：`mv`操作会使**源文件消失**，出现在目标位置，这与`cp`（复制，源文件保留）有本质区别。

## 查看文件内容：cat命令

![](img/5075da74c6984c046f31631b91024b3a_30.png)

![](img/5075da74c6984c046f31631b91024b3a_32.png)

![](img/5075da74c6984c046f31631b91024b3a_34.png)

![](img/5075da74c6984c046f31631b91024b3a_36.png)

![](img/5075da74c6984c046f31631b91024b3a_38.png)

在图形界面中，我们双击文件即可查看内容。在命令行中，我们需要使用`cat`命令来查看文件内容。

命令格式非常简单：
```bash
cat [选项] 文件名
```

最常用的选项是`-n`，它可以显示行号。
```bash
cat -n /etc/passwd
```

![](img/5075da74c6984c046f31631b91024b3a_40.png)

![](img/5075da74c6984c046f31631b91024b3a_42.png)

以下是使用`cat`命令查看系统一些常见配置文件的方法：
*   查看系统所有用户信息：`cat /etc/passwd`
*   查看系统主机名：`cat /etc/hostname`
*   查看系统版本：`cat /etc/centos-release` 或 `cat /etc/redhat-release`
*   查看系统所有组信息：`cat /etc/group`
*   查看网卡配置：`cat /etc/sysconfig/network-scripts/ifcfg-ens33`

![](img/5075da74c6984c046f31631b91024b3a_44.png)

![](img/5075da74c6984c046f31631b91024b3a_46.png)

**注意**：`cat`命令会一次性将文件的全部内容输出到屏幕。对于内容较少的文件这很便捷，但如果文件内容很长（例如上万行），屏幕会被快速刷过，不便查看。

## 分页查看大文件：less命令

![](img/5075da74c6984c046f31631b91024b3a_54.png)

为了解决`cat`查看大文件的不便，我们可以使用`less`命令。`less`可以将文件内容分页显示，方便我们逐页浏览，就像看书一样。

命令格式如下：
```bash
less [选项] 文件名
```
使用`-N`选项可以显示行号。
```bash
less -N /etc/services
```

进入`less`视图后，可以使用以下快捷键进行操作：
*   **上下箭头键**：向上/向下滚动一行。
*   **Page Down / Page Up** 或 **空格键 / b键**：向下/向上翻一页。
*   **g**：跳转到文件**第一行**。
*   **G**：跳转到文件**最后一行**。
*   **/关键词**：向下搜索指定关键词。
*   **?关键词**：向上搜索指定关键词。
*   **:行号**：跳转到指定行号（例如 `:1000` 跳转到第1000行）。
*   **q**：退出`less`视图。

## 查看文件首尾：head与tail命令

有时我们只需要查看文件的开头或结尾部分，这时`head`和`tail`命令就非常有用。

![](img/5075da74c6984c046f31631b91024b3a_64.png)

![](img/5075da74c6984c046f31631b91024b3a_66.png)

### head命令
`head`命令用于显示文件的开头若干行，**默认显示前10行**。

![](img/5075da74c6984c046f31631b91024b3a_68.png)

命令格式：
```bash
head [选项] 文件名
```
常用选项`-n`用于指定显示的行数。
```bash
head -20 /etc/passwd  # 查看文件前20行
head -5 /etc/hostname # 查看文件前5行
```
也可以简写为`-数字`的形式：
```bash
head -30 /etc/services
```

### tail命令
`tail`命令与`head`相反，用于显示文件的末尾若干行，**默认显示后10行**。

命令格式：
```bash
tail [选项] 文件名
```
同样使用`-n`选项指定行数。
```bash
tail -20 /etc/services  # 查看文件末尾20行
tail -5 /etc/sysconfig/network-scripts/ifcfg-ens33 # 查看网卡配置关键信息
```

![](img/5075da74c6984c046f31631b91024b3a_70.png)

![](img/5075da74c6984c046f31631b91024b3a_72.png)

![](img/5075da74c6984c046f31631b91024b3a_74.png)

![](img/5075da74c6984c046f31631b91024b3a_76.png)

`tail`命令还有一个非常实用的选项 `-f`，它可以**动态追踪**文件的更新。常用于实时监控日志文件的变化。
```bash
tail -f /var/log/messages
```
执行此命令后，终端会持续显示该文件新增的内容，按 `Ctrl+C` 可终止追踪。

## 删除文件与目录：rm命令 ⚠️

![](img/5075da74c6984c046f31631b91024b3a_80.png)

![](img/5075da74c6984c046f31631b91024b3a_82.png)

`rm`命令用于删除文件或目录。**请务必谨慎使用此命令**，因为Linux系统默认没有回收站，删除操作通常是不可逆的。

![](img/5075da74c6984c046f31631b91024b3a_84.png)

![](img/5075da74c6984c046f31631b91024b3a_85.png)

命令基本格式：
```bash
rm [选项] 文件或目录
```

以下是`rm`命令的关键选项和用法：
*   **删除普通文件**：直接使用`rm`命令，系统会询问确认。
    ```bash
    rm file.txt
    ```
*   **强制删除文件（不询问）**：使用`-f`选项。
    ```bash
    rm -f file.txt
    ```
*   **删除目录**：必须使用`-r`选项（递归删除）。
    ```bash
    rm -r directory_name/
    ```
*   **强制删除目录及其下所有内容**：组合使用`-rf`选项。**这是最常用但也最危险的删除方式**。
    ```bash
    rm -rf directory_name/
    ```
*   **使用通配符`*`**：`*`代表“任意所有字符”，用于批量操作。
    ```bash
    rm *.txt       # 删除当前目录下所有.txt文件
    rm -rf /opt/*  # 强制删除/opt目录下的所有内容
    ```

**⚠️ 严重警告**：`rm -rf /` 或 `rm -rf /*` 命令会强制删除根目录下的所有文件，导致系统崩溃，切勿在生产环境或自己的主系统中尝试！

**路径分隔符`/`的注意事项**：在路径中，`/`用作目录间的分隔符。对于目录，路径末尾的`/`是可选的（如`/opt/`和`/opt`等效）；但对于文件，路径末尾绝不能加`/`，否则系统会误认为它是一个目录。

## 虚拟机快照功能 💾

在学习过程中，尤其是练习`rm`等危险命令前，强烈建议使用虚拟机的**快照**功能。快照可以保存虚拟机在某个时间点的完整状态（相当于系统备份）。

*   **创建快照**：在虚拟机处于稳定状态时（例如刚安装好系统或完成重要配置后），通过虚拟机软件（如VMware）创建快照。
*   **恢复快照**：当系统被误操作破坏时，可以恢复到之前创建的快照点。**请注意**，恢复快照会丢失从创建快照到恢复时刻之间所做的所有更改。

这是一个极其重要的安全网，允许你大胆练习而无需担心损坏系统环境。

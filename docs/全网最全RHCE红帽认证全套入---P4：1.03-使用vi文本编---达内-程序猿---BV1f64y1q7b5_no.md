vi是Unix/Linux系统中默认的文本编辑器。Vim是Vi的增强版本，提供了更多功能，例如语法高亮。在最小化安装的系统中，通常只包含vi命令。Vim需要额外安装`vim-enhanced`软件包。

**基本流程**是使用`vi`或`vim`命令后跟文件名来打开或创建文件：
```bash
vi myfile.txt
```

![](img/d6e062c949df647781dc86a2db500e6a_26.png)

### vi编辑器的三种工作模式 🎭
vi编辑器有三种不同的工作模式，理解它们是熟练使用vi的关键。

![](img/d6e062c949df647781dc86a2db500e6a_28.png)

![](img/d6e062c949df647781dc86a2db500e6a_30.png)

![](img/d6e062c949df647781dc86a2db500e6a_31.png)

以下是三种模式的简要说明：
*   **命令模式**：打开文件后默认进入的模式。在此模式下可以执行复制、粘贴、删除、查找等操作，但不能直接输入文本。
*   **输入模式**：在此模式下，可以向文件中输入或编辑文本内容。
*   **末行模式**：在此模式下，可以执行保存文件、退出编辑器等操作。

### 模式切换与基本操作 🔄
掌握模式之间的切换是使用vi编辑器的核心。

以下是模式切换的常用方法：
1.  **从命令模式进入输入模式**：按下字母键 `i`。
2.  **从任何模式返回命令模式**：按下键盘上的 `Esc` 键。
3.  **从命令模式进入末行模式**：输入英文冒号 `:`。
4.  **在末行模式下保存与退出**：
    *   输入 `:wq` 并回车：保存修改并退出编辑器。
    *   输入 `:q!` 并回车：放弃所有修改并强制退出编辑器。

### 创建与编辑文件实践 ✍️
让我们通过一个完整的例子来实践创建和编辑文件。

![](img/d6e062c949df647781dc86a2db500e6a_45.png)

![](img/d6e062c949df647781dc86a2db500e6a_47.png)

![](img/d6e062c949df647781dc86a2db500e6a_49.png)

以下是创建一个新文件并写入内容的步骤：
1.  打开终端，输入 `vi practice.txt` 命令。
2.  按下 `i` 键，进入输入模式。
3.  输入任意文本内容，例如 `Hello, this is my first vi file.`。
4.  按下 `Esc` 键，返回命令模式。
5.  输入 `:` 进入末行模式，再输入 `wq` 并回车，保存文件并退出。
6.  使用 `cat practice.txt` 命令查看文件内容，确认编辑成功。

![](img/d6e062c949df647781dc86a2db500e6a_51.png)

要修改已存在的文件，只需将第一步中的文件名改为目标文件即可，后续操作完全相同。

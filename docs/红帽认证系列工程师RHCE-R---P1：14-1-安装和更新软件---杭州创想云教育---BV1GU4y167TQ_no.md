# 红帽认证系列工程师RHCE RH124-Chapter14：14-1：安装和更新软件包-注册系统以获取红帽支持 🖥️

在本节课中，我们将学习如何将系统注册到红帽网络，以获得官方的软件包支持和技术订阅。这是使用红帽企业版Linux获取软件更新和安全补丁的基础步骤。

上一节我们介绍了本章的整体结构，本节中我们来看看如何注册系统并获取支持。

## 系统注册概述

![](img/9e383e14ed0386fccc818887c9edad82_1.png)

![](img/9e383e14ed0386fccc818887c9edad82_3.png)

红帽的产品需要订阅才能获得完整的软件仓库访问和技术支持。注册系统主要分为四个步骤。

![](img/9e383e14ed0386fccc818887c9edad82_5.png)

![](img/9e383e14ed0386fccc818887c9edad82_6.png)

以下是注册系统的四个主要步骤：
1.  **注册系统**：在红帽官网创建个人或组织账户。
2.  **选择订阅**：根据需求选择合适的支持级别（自助支持、标准支持或高级支持）。
3.  **启用仓库**：在已订阅的系统上启用对应的软件仓库。
4.  **审核与跟踪**：在红帽门户中管理订阅和跟踪系统。

对于学习者而言，教室内的训练机通常已预配置好软件仓库。此部分内容主要针对在实际生产环境中需要使用红帽官方订阅的用户。

## 创建红帽账户

![](img/9e383e14ed0386fccc818887c9edad82_8.png)

![](img/9e383e14ed0386fccc818887c9edad82_9.png)

要使用订阅，首先需要拥有一个红帽账户（也称为RHN账户）。注册过程主要通过浏览器完成。

![](img/9e383e14ed0386fccc818887c9edad82_11.png)

![](img/9e383e14ed0386fccc818887c9edad82_12.png)

以下是创建红帽账户的步骤：
1.  访问红帽官方网站。
2.  点击页面右上角的“Log In”，然后选择“注册红帽账号”。
3.  在注册页面，选择“个人”账户类型。
4.  设置登录名（建议使用5位以上英文字母、数字或下划线组合，避免使用邮箱地址）。
5.  填写姓名、密码、邮箱地址、电话号码及联系地址等信息。
6.  提交注册后，查收激活邮件并完成账号激活。

账户激活后，即可登录红帽客户门户网站。

## 管理订阅

![](img/9e383e14ed0386fccc818887c9edad82_14.png)

![](img/9e383e14ed0386fccc818887c9edad82_15.png)

![](img/9e383e14ed0386fccc818887c9edad82_17.png)

登录红帽门户后，可以在“订阅”部分查看和管理可用的订阅级别。对于开发者或学习者，可以申请免费的开发者订阅，有效期通常为一年。

![](img/9e383e14ed0386fccc818887c9edad82_19.png)

![](img/9e383e14ed0386fccc818887c9edad82_21.png)

在系统上激活订阅有多种方式，我们将介绍图形界面和命令行两种方法。

![](img/9e383e14ed0386fccc818887c9edad82_23.png)

### 通过图形界面注册

![](img/9e383e14ed0386fccc818887c9edad82_25.png)

![](img/9e383e14ed0386fccc818887c9edad82_26.png)

![](img/9e383e14ed0386fccc818887c9edad82_28.png)

![](img/9e383e14ed0386fccc818887c9edad82_29.png)

如果系统安装了图形桌面环境，可以通过系统工具进行注册。

![](img/9e383e14ed0386fccc818887c9edad82_31.png)

1.  在应用程序菜单中找到并打开“Red Hat Subscription Manager”。
2.  输入管理员密码。
3.  在界面中点击“注册”按钮。
4.  在弹出的窗口中，填入您的红帽账户名、密码以及可能的组织ID和激活密钥。
5.  点击“注册”完成操作。注册成功后，相应的软件仓库会自动启用。

![](img/9e383e14ed0386fccc818887c9edad82_33.png)

![](img/9e383e14ed0386fccc818887c9edad82_35.png)

![](img/9e383e14ed0386fccc818887c9edad82_37.png)

### 通过命令行注册

![](img/9e383e14ed0386fccc818887c9edad82_39.png)

对于没有图形界面的服务器，可以使用 `subscription-manager` 命令进行注册。

![](img/9e383e14ed0386fccc818887c9edad82_41.png)

![](img/9e383e14ed0386fccc818887c9edad82_42.png)

注册系统的基本命令格式如下：
```bash
subscription-manager register --username=您的账户名 --password=您的密码
```
您还可以在命令中指定其他参数，如组织ID (`--org`) 和激活密钥 (`--activationkey`)。

![](img/9e383e14ed0386fccc818887c9edad82_44.png)

![](img/9e383e14ed0386fccc818887c9edad82_46.png)

![](img/9e383e14ed0386fccc818887c9edad82_47.png)

注册成功后，可以运行以下命令查看可用的订阅并附加一个到系统：
```bash
subscription-manager list --available
subscription-manager attach --pool=订阅池ID
```
要查看当前系统已启用的软件仓库，可以检查 `/etc/yum.repos.d/redhat.repo` 文件。

![](img/9e383e14ed0386fccc818887c9edad82_49.png)

## 总结

![](img/9e383e14ed0386fccc818887c9edad82_51.png)

本节课中我们一起学习了为红帽企业版Linux系统获取官方支持的核心步骤。我们了解了注册红帽账户、选择订阅级别的重要性，并掌握了通过图形界面和命令行两种方式将系统注册到红帽网络的具体操作方法。成功注册并附加订阅后，系统便可以访问红帽官方的软件仓库，为后续的软件安装、更新和管理打下基础。

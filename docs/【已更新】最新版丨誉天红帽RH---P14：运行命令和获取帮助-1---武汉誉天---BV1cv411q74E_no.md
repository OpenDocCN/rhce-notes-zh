## 命令、选项与参数

![](img/470e20aa5afb205dfd43c167fc7cd35d_5.png)

![](img/470e20aa5afb205dfd43c167fc7cd35d_6.png)

上一节我们介绍了命令行的基本概念，本节中我们来看看命令的具体构成。一个完整的命令行通常由命令、选项和参数三部分组成。

![](img/470e20aa5afb205dfd43c167fc7cd35d_8.png)

**命令**是您想要执行的操作。例如，`ls`用于列出目录内容，`passwd`用于修改用户密码。如果输入的命令不存在，系统会提示“命令未找到”。

**选项**用于修饰命令的行为，改变命令的执行方式。选项通常以短横线`-`（短选项）或双短横线`--`（长选项）开头。

![](img/470e20aa5afb205dfd43c167fc7cd35d_10.png)

**参数**是命令操作的具体对象，例如文件名、目录名或用户名。

以下是命令行的基本语法格式：
```
命令 [选项] [参数]
```
其中，`[选项]`和`[参数]`是可选的，但命令本身是必须的。

![](img/470e20aa5afb205dfd43c167fc7cd35d_12.png)

![](img/470e20aa5afb205dfd43c167fc7cd35d_14.png)

### 命令与选项

命令和选项之间必须用空格隔开。空格的数量没有严格要求，但至少需要一个。

以下是`ls`命令使用不同选项的例子：
*   `ls`：列出当前目录下的非隐藏文件和目录。
*   `ls -l`：以长格式列出详细信息，包括权限、所有者、大小和修改时间。
*   `ls -a`：列出所有文件和目录，包括以`.`开头的隐藏文件。

![](img/470e20aa5afb205dfd43c167fc7cd35d_16.png)

选项可以联合使用。多个短选项可以合并写在一个`-`后面，也可以分开写。

![](img/470e20aa5afb205dfd43c167fc7cd35d_18.png)

以下是多个选项的联合使用示例：
```
ls -l -a
ls -la
ls -al
```
以上三条命令的执行效果是相同的。

长选项通常是一个完整的单词，前面需要加`--`。长选项一般不能像短选项那样合并。

![](img/470e20aa5afb205dfd43c167fc7cd35d_20.png)

以下是长选项的使用示例：
```
ls --all
```
这条命令的效果与`ls -a`相同。

![](img/470e20aa5afb205dfd43c167fc7cd35d_22.png)

### 参数的使用

![](img/470e20aa5afb205dfd43c167fc7cd35d_24.png)

参数指定了命令操作的具体目标。例如，`ls /home`命令中的`/home`就是一个参数，表示要列出`/home`目录的内容。

![](img/470e20aa5afb205dfd43c167fc7cd35d_26.png)

您可以同时指定多个参数，参数之间也需要用空格隔开。

以下是使用多个参数的示例：
```
ls file1.txt file2.txt
```
这条命令会同时列出`file1.txt`和`file2.txt`文件的信息。

![](img/470e20aa5afb205dfd43c167fc7cd35d_28.png)

![](img/470e20aa5afb205dfd43c167fc7cd35d_29.png)

**重要规则**：命令、选项、参数之间都必须用空格隔开。这是Linux命令行解析的基础。

![](img/470e20aa5afb205dfd43c167fc7cd35d_31.png)

![](img/470e20aa5afb205dfd43c167fc7cd35d_33.png)

### 执行多个命令

![](img/470e20aa5afb205dfd43c167fc7cd35d_35.png)

在同一行中执行多个命令，可以使用分号`;`连接。分号连接的命令会按顺序执行，且彼此之间没有逻辑关联。

![](img/470e20aa5afb205dfd43c167fc7cd35d_37.png)

![](img/470e20aa5afb205dfd43c167fc7cd35d_39.png)

以下是使用分号连接命令的示例：
```
date; cal
```
这条命令会先执行`date`显示日期，然后执行`cal`显示日历。

![](img/470e20aa5afb205dfd43c167fc7cd35d_41.png)

## 获取命令帮助

我们不可能记住所有命令的用法，因此学会获取帮助是一项关键技能。做好学习笔记也同样重要。

![](img/470e20aa5afb205dfd43c167fc7cd35d_43.png)

![](img/470e20aa5afb205dfd43c167fc7cd35d_45.png)

### 查找命令路径：`which`

![](img/470e20aa5afb205dfd43c167fc7cd35d_47.png)

`which`命令用于查找某个命令对应的可执行文件的具体位置。

以下是`which`命令的使用示例：
```
which ls
```
输出可能为：`/usr/bin/ls`。如果`which`找不到对应的文件，则说明该命令在系统中不存在。

### 查看命令简要描述：`whatis`

![](img/470e20aa5afb205dfd43c167fc7cd35d_49.png)

![](img/470e20aa5afb205dfd43c167fc7cd35d_50.png)

![](img/470e20aa5afb205dfd43c167fc7cd35d_52.png)

`whatis`命令可以显示一个命令的简短功能描述。

以下是`whatis`命令的使用示例：
```
whatis ls
```
输出为：`ls (1) - list directory contents`，即“列出目录内容”。

![](img/470e20aa5afb205dfd43c167fc7cd35d_54.png)

**注意**：新安装的系统可能需要先运行`mandb`命令来初始化帮助数据库，之后`whatis`才能正常使用。

![](img/470e20aa5afb205dfd43c167fc7cd35d_56.png)

### 系统帮助资源

Linux系统提供了不同级别的帮助资源，从简短的描述到完整的手册页。在后续课程中，我们将学习更强大的帮助命令`man`，它可以提供命令的完整使用手册。

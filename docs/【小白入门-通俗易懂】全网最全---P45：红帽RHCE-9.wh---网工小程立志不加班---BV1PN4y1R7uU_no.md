在本节课中，我们将要学习Shell脚本中的`while`循环。`while`循环与之前学过的`for`循环功能相似，都是用于重复执行命令，但它们的应用场景和循环机制有所不同。`while`循环的核心在于“条件判断”，只要条件成立，就会一直执行循环体内的命令。我们将通过实例来理解其语法、特性以及如何控制循环。

## 概述 📋

![](img/c4a5c11f0699e0b92160b3547776f5b3_6.png)

上一节我们介绍了`for`循环，它基于一个给定的值列表进行循环。本节中我们来看看`while`循环。`while`循环是基于条件判断的循环结构。只要指定的条件为真（成立），它就会重复执行`do`和`done`之间的命令。这使得它非常适合用于需要持续监控或等待某个条件变化的场景。

## while循环的基本语法

![](img/c4a5c11f0699e0b92160b3547776f5b3_10.png)

![](img/c4a5c11f0699e0b92160b3547776f5b3_12.png)

![](img/c4a5c11f0699e0b92160b3547776f5b3_14.png)

`while`循环的基本语法结构如下：
```bash
while [ 条件 ]
do
    # 要执行的命令
done
```
只要“条件”成立，`do`和`done`之间的命令就会被反复执行。

![](img/c4a5c11f0699e0b92160b3547776f5b3_16.png)

### 一个简单的死循环示例

我们来创建一个简单的脚本`while.sh`，演示一个最基本的`while`循环：
```bash
#!/bin/bash
while [ 1 -eq 1 ]
do
    echo "hello baby"
done
```
执行这个脚本，你会发现终端会不停地输出“hello baby”。这是因为条件`[ 1 -eq 1 ]`（即1等于1）永远为真，所以循环永远不会停止。这便是一个“死循环”。你可以使用 `Ctrl + C` 来强制终止脚本。

## while循环与for循环的区别

![](img/c4a5c11f0699e0b92160b3547776f5b3_18.png)

为了更好地理解`while`循环，我们来对比一下它和`for`循环：
*   **`for`循环**：循环的次数通常由你给出的一个明确的值列表（例如1到254）决定。它有一个明确的终点。
*   **`while`循环**：循环的次数取决于“条件”何时变为假。只要条件为真，循环就会继续，它可能没有预设的终点，形成“死循环”。

简单来说，`for`循环是“我给你一个范围，你按次数执行”；而`while`循环是“你帮我盯着这个条件，只要条件成立你就执行”。

## 控制while循环的次数

虽然`while`循环常用于条件触发，但我们也可以通过变量来控制它的循环次数，使其像`for`循环一样执行固定的次数。

以下是实现循环5次的示例：
```bash
#!/bin/bash
number=1
while [ $number -le 5 ]
do
    echo "这是第 $number 次循环"
    let number++  # 等同于 number=$((number + 1))
done
```
**代码解释：**
1.  我们定义了一个变量`number`并赋值为1。
2.  循环条件是：当`$number`的值**小于等于5**（`-le`）时，执行循环。
3.  在循环体内，我们使用`let number++`让变量`number`的值每次增加1。
4.  当`number`增加到6时，条件`[ 6 -le 5 ]`不再成立，循环结束。

这个脚本会精确地输出5次信息。当然，如果需要固定次数的循环，使用`for`循环通常更加直观和方便。

## while循环的特殊用法：无限循环

在脚本中，有时我们需要一个真正的、无需复杂条件的无限循环（死循环）。`while`循环提供了一个简洁的写法：使用冒号`:`作为条件。

冒号`:`在Shell中是一个特殊的命令，它什么都不做，但返回的退出状态总是“真”（0）。因此，`while :` 就构成了一个无限循环。

以下是使用冒号的无限循环示例：
```bash
#!/bin/bash
while :
do
    echo "这是一个无限循环"
    sleep 1 # 暂停1秒，避免输出过快
done
```

#!/bin/bash

    *   如果相等，则输出恭喜信息，并使用 **`exit 0`** 命令**退出整个脚本**，从而结束无限循环。
    *   如果不相等，则提示猜错，循环继续，让用户再次输入。

本节课中我们一起学习了Shell脚本中的`while`循环。
*   `while`循环基于条件进行判断，条件为真则持续循环。
*   它与`for`循环的主要区别在于循环的控制机制：`for`基于列表，`while`基于条件。
*   我们可以通过修改变量值来控制`while`循环的次数。
*   使用 `while :` 可以创建一个简单的无限循环（死循环）。
*   在实际应用中，无限循环常与`exit`命令结合，在满足特定条件时（如猜对数字、完成备份）退出脚本，从而构建出灵活实用的交互式或监控式脚本。

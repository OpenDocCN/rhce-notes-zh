Visio 2016版本带来了多项新功能：
*   **丰富的图表和形状**：新增了大量美观的图表模板和超过200个预设计形状（如交换机、路由器、服务器），使图表更加形象。
*   **统一的主题风格**：提供了多种主题，可以一键应用统一的颜色、字体和效果，方便批量修改图表样式。
*   **增强的协作功能**：支持多人协同编辑和分享同一个绘图文件，便于团队合作。
*   **现代化的界面与格式**：采用了更美观的界面设计，默认文件格式更新为 `.vsdx`。

## 软件安装与激活

上一节我们介绍了Visio的基本概念，本节中我们来看看如何获取并安装这款软件。

![](img/08065309783d0fa39dbc85afdb553610_54.png)

以下是安装与激活步骤：
1.  **下载软件**：从可靠来源获取Visio 2016安装包。
2.  **运行安装**：解压下载的文件，找到并双击 `setup.exe` 可执行文件，程序将自动开始安装。
3.  **激活软件**：
    *   安装完成后，启动Visio。
    *   如果软件提示需要激活，请转到“文件”>“账户”>“产品信息”。
    *   点击“更改产品密钥”，输入有效的序列号。
    *   某些情况下，可能需要使用专门的激活工具来完成激活。请注意，安全软件可能会将激活工具误报为病毒，这是常见情况，请根据经验判断。

## Visio基础界面与操作

![](img/08065309783d0fa39dbc85afdb553610_64.png)

安装好软件后，我们正式进入Visio的操作界面。它的布局与Office系列软件（如Word）相似，非常容易上手。

![](img/08065309783d0fa39dbc85afdb553610_66.png)

**核心界面区域介绍：**
*   **顶部菜单栏与功能区**：包含“开始”、“插入”、“设计”等选项卡，每个选项卡下是相应的工具组，用于处理文本、形状、线条、样式等。
*   **左侧形状窗格**：这里是Visio的“素材库”，存放了所有可用的形状（也称为模具）。形状按类别组织，如“网络”、“计算机和显示器”、“基本形状”等。
*   **中部绘图区**：主要的作图区域，在此进行拖拽、连接、编辑等所有绘图操作。

![](img/08065309783d0fa39dbc85afdb553610_68.png)

**基本图形操作：**
*   **添加形状**：从左侧窗格中，直接将需要的形状（如服务器、云图标）拖拽到绘图区。
*   **编辑形状**：双击形状上的文字区域，即可输入或修改标签（如“Web服务器”）。
*   **调整形状**：选中形状后，拖动其边框的控制点可以缩放大小。Visio是矢量软件，缩放不会导致图像模糊。
*   **复制形状**：选中一个形状，按住 `Ctrl` 键的同时拖动鼠标，即可快速复制一个相同的形状。
*   **连接形状**：在“开始”选项卡的“工具”组中，选择“连接线”工具，点击并从一个形状拖拽到另一个形状，即可建立连接。
*   **缩放视图**：按住 `Ctrl` 键并滚动鼠标滚轮，可以快速放大或缩小绘图区的显示比例。

## 绘制一个简单的网络拓扑图

了解了基本操作后，我们通过一个实战案例来巩固所学知识。本节我们将绘制一个“中小型企业网站项目”的基础网络拓扑图。

**绘图目标：** 展示一个包含互联网、客户端、防火墙、交换机、Web服务器和数据库服务器的基本网络结构。

以下是绘图步骤：
1.  **创建标题**：使用“文本框”工具，在绘图区顶部添加标题“中小型企业网站项目”，并设置合适的字体和大小。
2.  **添加核心元素**：
    *   从“网络位置”形状中，拖出“云”形状代表互联网。
    *   从“计算机和显示器”形状中，拖出“PC”形状代表客户端。
    *   使用“矩形”和“圆角矩形”等基本形状，组合绘制一个“北京电信IDC机房”的框图。
3.  **布置网络设备**：
    *   从“网络符号”形状中，拖出“路由器”和“交换机”形状，放入机房框图内。
    *   从“服务器”形状中，分别拖出“数据库服务器”和“Web服务器”形状（建议使用3D样式，更形象），放入机房框图内。使用 `Ctrl`+拖拽 来复制一台数据库服务器，实现主从概念。
4.  **添加标注**：双击每个设备形状，为其添加标签，如“路由器 192.168.1.1”、“MySQL主从”、“Web服务器”等。
5.  **连接设备**：
    *   使用“连接线”工具，将客户端连接到云，云连接到路由器，路由器连接到交换机，交换机再分别连接到Web服务器和数据库服务器。
    *   要绘制备份链路（冗余线路），可以先画一条标准连接线，然后选中它，在“开始”选项卡的“形状样式”组中，将“线条”样式改为“虚线”。
6.  **添加说明框**：在图表下方，可以添加一个服务器形状和文本框，用于列出项目要点，例如：
    *   Web服务器定时备份
    *   使用Rsync进行实时同步
    *   部署LAMP环境
    *   使用Nagios进行监控
    *   数据库服务器定时备份

**复杂图表的绘制思路：** 更复杂的架构图（如包含负载均衡LVS、Keepalived高可用、多台服务器的集群），都是由这些基础图形和连接线层层叠加、组合而成的。只要掌握了基本元素的画法，就能构建出任何复杂的拓扑图。

## 图表的导出与使用

![](img/08065309783d0fa39dbc85afdb553610_70.png)

绘制完成的图表可以方便地用于其他场合：
*   **复制粘贴**：可以直接在Visio中选中图表，复制后粘贴到PowerPoint或Word文档中。建议在Visio中调整好最终大小后再粘贴。
*   **另存为图片**：点击“文件”>“另存为”，选择保存类型为“JPEG文件”或“PNG文件”，即可将图表导出为通用图片格式，方便插入任何文档或网页。
*   **其他格式**：Visio还支持导出为PDF、网页、甚至AutoCAD格式，兼容性很强。

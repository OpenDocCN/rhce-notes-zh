GNOME是RHEL 8默认的桌面环境。它是一个独立的组件，可以根据需要安装或移除。GNOME 3桌面集成了多种工具和开发平台，为初学者访问和操作系统提供了便利。

在RHEL 8中，GNOME桌面主要有两种显示主题：
*   **标准主题**：RHEL 8的默认主题，设计理念是让用户专注于当前任务。
*   **经典主题**：类似于RHEL 7的界面风格，提供了更传统的“开始菜单”式布局。

![](img/65f3577da7f6ea6b039e4d3b22706f4f_43.png)

在虚拟化环境（如VMware）中使用图形界面时，为了获得更好的兼容性和性能，建议选择使用 **Wayland** 显示服务器（无论是标准版还是经典版），而不是X11。

## 标准主题界面详解

现在，让我们登录到标准主题的GNOME桌面，并详细了解一下它的各个部分。

成功登录后，你可能需要调整屏幕分辨率以获得最佳显示效果。你可以在系统设置的“设备”->“显示”选项中进行调整。

以下是标准GNOME桌面主要区域的介绍：

![](img/65f3577da7f6ea6b039e4d3b22706f4f_66.png)

*   **左上角“活动”按钮**：这个按钮的功能类似于Windows系统中的“超级按钮”或“开始菜单”。点击它会显示活动概览界面。
*   **活动概览界面**：在这个界面左侧，你可以找到常用应用程序的快捷方式。顶部有一个搜索框，可以快速查找并启动应用程序。右侧会显示当前打开的应用程序窗口和虚拟桌面。
*   **虚拟桌面**：GNOME支持多个虚拟桌面，这允许你将不同的工作任务（例如，管理不同的服务器）组织到不同的桌面空间中。你可以轻松地将应用程序窗口拖放到不同的虚拟桌面。
*   **右上角系统菜单**：屏幕右上角区域包含了一系列系统控件。
    *   **音量与网络**：小喇叭图标控制声音，其下方的图标用于管理网络连接（在虚拟机中可能显示不全）。
    *   **用户菜单**：点击用户名区域，可以进行账户设置、锁定屏幕或注销。
    *   **设置（扳手图标）**：点击后打开系统设置中心，可以配置网络、背景、通知、电源管理等。
    *   **电源按钮**：点击后可以选择关机、重启或取消操作。

## 实践练习：熟悉桌面操作

理论介绍之后，我们通过一个教材上的指导练习来巩固所学知识。这个练习将引导你完成登录、修改密码、注销、锁定屏幕和关机等基本操作。

**练习目标**：使用GNOME桌面环境登录Linux系统，并在终端中运行命令。

**准备工作**：确保你的`workstation`虚拟机正在运行。你可以使用以下命令检查并启动虚拟机：
```bash
rht-vmctl status workstation
rht-vmctl start workstation
```

1.  **登录系统**：在登录界面，选择用户 `student`，输入密码 `student` 进行登录。
2.  **修改密码**：
    *   按下 `Super` 键（或点击“活动”按钮），进入活动概览。
    *   在搜索框中输入 `terminal` 或 `tmin` 来查找并打开终端应用程序。
    *   在终端中，输入命令 `passwd`。
    *   根据提示，先输入当前密码 `student`（输入时无显示），然后输入新密码 `55TURNk3y`，并再次确认。
    *   看到“密码更新成功”的提示即表示修改完成。
3.  **注销并重新登录**：
    *   点击右上角的系统菜单，选择 `student` -> `Log Out`。
    *   在确认对话框中，再次点击 `Log Out`。
    *   重新使用用户 `student` 和新密码 `55TURNk3y` 登录，验证密码修改成功。
4.  **锁定屏幕**：
    *   点击右上角系统菜单，选择锁定图标（小锁）来锁定屏幕。
    *   按空格键或点击屏幕唤醒，输入新密码 `55TURNk3y` 解锁。
5.  **关机操作（练习，不执行）**：
    *   点击右上角系统菜单底部的电源按钮。
    *   在弹出的对话框中，了解 `Power Off`（关机）、`Restart`（重启）和 `Cancel`（取消）选项。本次练习点击 `Cancel`。
6.  **清理练习环境**：练习完成后，需要运行一个脚本来重置环境。
    *   打开终端。
    *   输入命令 `lab cli-desktop finish` 并执行。
    *   这个脚本会清理实验痕迹，并将 `student` 用户的密码重置回 `student`。

## 经典主题界面浏览

之前我们介绍了标准主题，本节中我们快速浏览一下经典主题，看看它有何不同。

![](img/65f3577da7f6ea6b039e4d3b22706f4f_86.png)

注销当前会话，在登录界面选择 **`GNOME Classic on Wayland`** 会话，然后使用密码 `redhat` 登录。

![](img/65f3577da7f6ea6b039e4d3b22706f4f_88.png)

![](img/65f3577da7f6ea6b039e4d3b22706f4f_90.png)

经典主题的界面布局更接近传统的桌面。其特点包括：
*   **顶部面板**：包含“应用程序”菜单（类似开始菜单）、快速启动栏和系统状态区（时间、音量、网络等）。
*   **“应用程序”菜单**：点击后可以清晰地分类浏览所有已安装的程序，例如“附件”、“教育”等，查找软件更为直观。
*   **系统工具**：例如，在“教育”分类下，你可以找到图形化的虚拟机管理器 (`virt-manager`)，它提供了管理虚拟机的另一种方式。

无论是标准主题还是经典主题，最终高效的系统管理和配置工作，往往还是需要通过命令行终端来完成。

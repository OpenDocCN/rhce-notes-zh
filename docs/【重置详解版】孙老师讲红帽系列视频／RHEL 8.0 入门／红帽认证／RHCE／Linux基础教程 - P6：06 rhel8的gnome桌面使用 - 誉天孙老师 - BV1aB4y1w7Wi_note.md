## 系统登录方式

![](img/900d03c851c7fab30cec313cf7b267f6_5.png)

![](img/900d03c851c7fab30cec313cf7b267f6_6.png)

![](img/900d03c851c7fab30cec313cf7b267f6_8.png)

Linux系统提供了两种主要的登录方式。

### 图形界面登录

![](img/900d03c851c7fab30cec313cf7b267f6_10.png)

如果系统安装时选择了“Server with GUI”选项，启动后将进入图形登录界面。在此界面输入用户名（如 `root`）和密码即可登录。输入密码时，屏幕上不会显示输入内容（无回显），这是正常的安全机制。

![](img/900d03c851c7fab30cec313cf7b267f6_12.png)

**注意**：使用外接键盘时，请确认小键盘的输入状态，避免因小键盘未开启导致密码输入错误。

![](img/900d03c851c7fab30cec313cf7b267f6_13.png)

![](img/900d03c851c7fab30cec313cf7b267f6_15.png)

### 字符界面登录

![](img/900d03c851c7fab30cec313cf7b267f6_17.png)

![](img/900d03c851c7fab30cec313cf7b267f6_19.png)

系统启动后，也可能显示一个纯文本的登录界面。界面上会显示操作系统版本和内核信息。在此界面同样需要输入用户名和密码进行登录。登录成功后，将进入命令行操作环境。

## 图形界面的构成

![](img/900d03c851c7fab30cec313cf7b267f6_25.png)

图形界面并非所有Linux系统默认安装。它的运行依赖于两个核心组件：

![](img/900d03c851c7fab30cec313cf7b267f6_27.png)

![](img/900d03c851c7fab30cec313cf7b267f6_29.png)

1.  **X Window系统**：一个提供图形显示服务的底层协议框架。
2.  **桌面环境**：运行在X Window之上的用户界面工具。常见的开源桌面环境有：
    *   **GNOME**：红帽企业版Linux默认使用的桌面环境。
    *   **KDE**：另一个流行的桌面环境。

因此，若要在未安装图形界面的系统上启用图形功能，需要先安装X Window系统和相应的桌面环境（如GNOME）。

上一节我们介绍了两种登录界面，本节中我们来看看如何在不同的操作界面之间切换。

![](img/900d03c851c7fab30cec313cf7b267f6_37.png)

### 控制台类型与切换

RHEL 8系统提供了多个“控制台”（即操作界面），包括：
*   1个图形控制台（通常是启动后默认进入的界面）。
*   5个虚拟控制台（即纯字符命令行界面）。

它们之间可以通过快捷键进行切换：
*   `Ctrl` + `Alt` + `F1`：切换到**图形控制台**（如果已启动）。
*   `Ctrl` + `Alt` + `F2` 到 `F6`：切换到第2到第6个**虚拟控制台**。

![](img/900d03c851c7fab30cec313cf7b267f6_39.png)

**重要概念**：`startx` 命令用于在**当前**的虚拟控制台中启动图形界面应用程序，但它**不会**切换到图形控制台（即 `Ctrl` + `Alt` + `F1` 所对应的界面）。切换控制台只能使用上述快捷键。

![](img/900d03c851c7fab30cec313cf7b267f6_41.png)

![](img/900d03c851c7fab30cec313cf7b267f6_43.png)

### 多控制台的作用

![](img/900d03c851c7fab30cec313cf7b267f6_44.png)

![](img/900d03c851c7fab30cec313cf7b267f6_46.png)

提供多个独立的控制台主要是为了系统稳定性。如果某一个控制台（例如图形界面）出现故障或无响应，用户可以切换到另一个虚拟控制台进行故障排查或继续工作，各个控制台之间的操作互不影响。

## GNOME桌面与终端使用

![](img/900d03c851c7fab30cec313cf7b267f6_54.png)

![](img/900d03c851c7fab30cec313cf7b267f6_55.png)

![](img/900d03c851c7fab30cec313cf7b267f6_56.png)

现在，我们切换到图形界面，学习GNOME桌面环境的基本操作，特别是如何打开和使用终端。

在GNOME桌面中，所有应用程序都可以通过点击屏幕左上角的“活动”按钮或屏幕左下角的“应用程序菜单”来找到。

### 打开终端

在图形界面中执行命令，需要通过“终端”（Terminal）应用程序。以下是打开终端的步骤：
1.  点击“活动”或“应用程序菜单”。
2.  在应用程序列表中，找到并点击 **Terminal**（终端）。
3.  点击后，会弹出一个黑色的命令行窗口，在此可以输入并执行Linux命令。

![](img/900d03c851c7fab30cec313cf7b267f6_58.png)

### 终端多标签与窗口管理

![](img/900d03c851c7fab30cec313cf7b267f6_60.png)

![](img/900d03c851c7fab30cec313cf7b267f6_61.png)

![](img/900d03c851c7fab30cec313cf7b267f6_63.png)

为了提高效率，我们可以在终端中打开多个标签页或窗口。

以下是管理终端标签页和窗口的常用操作：
*   **新建标签页**：快捷键 `Ctrl` + `Shift` + `T`。
*   **切换标签页**：快捷键 `Alt` + `1`、`Alt` + `2` ...（数字对应标签页顺序）。
*   **新建窗口**：在终端菜单栏选择 `File` -> `New Window`，或使用快捷键 `Ctrl` + `Shift` + `N`。
*   **关闭标签页/退出终端**：在对应的标签页中输入命令 `exit`。**建议养成使用 `exit` 命令退出的习惯**，而非直接点击窗口的关闭按钮，以避免意外中断正在运行的任务。
*   **清屏**：
    *   输入命令 `clear`。
    *   或使用快捷键 `Ctrl` + `L`。

### 终端内容缩放与复制

![](img/900d03c851c7fab30cec313cf7b267f6_73.png)

![](img/900d03c851c7fab30cec313cf7b267f6_74.png)

为了方便阅读和操作，可以调整终端内容的显示大小。

![](img/900d03c851c7fab30cec313cf7b267f6_76.png)

以下是调整和复制终端内容的操作：
*   **放大字体**：`Ctrl` + `Shift` + `+` （加号）。
*   **缩小字体**：`Ctrl` + `-` （减号）。
*   **复制与粘贴**：
    *   鼠标选中文本后，右键点击选择 `Copy` 和 `Paste`。
    *   或使用快捷键：`Ctrl` + `Shift` + `C` 复制，`Ctrl` + `Shift` + `V` 粘贴。
    *   **注意**：在Linux终端中，`Ctrl` + `C` 的通用含义是**中断当前运行的程序**，而不是复制。

## 其他桌面功能简介

![](img/900d03c851c7fab30cec313cf7b267f6_78.png)

GNOME桌面右上角通常有一个系统状态区域，点击后可以快速进行一些系统设置：
*   **设置**：进入系统设置面板，可以调整网络、声音、显示、区域语言等。
*   **锁屏**：立即锁定屏幕，需要密码才能解锁。
*   **电源选项**：进行关机、重启、挂起（休眠）等操作。

![](img/900d03c851c7fab30cec313cf7b267f6_82.png)

这些功能相对直观，大家可以自行探索。

## 回顾：vim的三种工作模式

vim主要有三种工作模式：命令模式、插入模式和退出模式。

*   **命令模式**：打开文件后默认进入的模式。在此模式下，可以移动光标、剪切/粘贴文本和执行简短命令。按 `ESC` 键可以从任何其他模式返回命令模式。
*   **插入模式**：用于正常输入和编辑文本。在命令模式下按 `i`, `a`, `o` 等键可进入此模式。
*   **退出模式**：用于保存文件、退出编辑器或执行更复杂的操作（如查找替换）。在命令模式下按 `:` 键可进入此模式。

## 退出模式下的查找与替换 🔍

![](img/43674bdf09171d72c692213bf4c38d1c_19.png)

在退出模式下，我们可以对文件内容进行查找和替换操作。以下是具体的语法和示例。

![](img/43674bdf09171d72c692213bf4c38d1c_21.png)

替换操作的基本格式是：`:[范围]s/旧文本/新文本/[标志]`

![](img/43674bdf09171d72c692213bf4c38d1c_23.png)

*   **`s`**：代表 `substitute`，即替换命令。
*   **[范围]**：指定替换操作生效的行范围。
    *   `%`：表示整个文件。
    *   `m,n`：表示从第 `m` 行到第 `n` 行。
    *   不指定范围时，默认只替换**光标所在行**的第一个匹配项。
*   **[标志]**：控制替换行为。
    *   `g`：替换一行中**所有**匹配的旧文本（默认只替换第一个）。
    *   `i`：忽略大小写进行匹配。
    *   `c`：在每次替换前进行确认，输入 `y` 确认替换，`n` 跳过。

![](img/43674bdf09171d72c692213bf4c38d1c_25.png)

![](img/43674bdf09171d72c692213bf4c38d1c_27.png)

**示例：**
1.  将整个文件中所有的 `user` 替换为 `AAAA`，并忽略大小写：
    ```
    :%s/user/AAAA/gi
    ```
2.  仅替换第5行到第10行中所有的 `test` 为 `demo`：
    ```
    :5,10s/test/demo/g
    ```
3.  替换光标所在行的第一个 `hello` 为 `world`，并在替换前确认：
    ```
    :s/hello/world/c
    ```

## 可视化模式 👁️

![](img/43674bdf09171d72c692213bf4c38d1c_34.png)

![](img/43674bdf09171d72c692213bf4c38d1c_36.png)

![](img/43674bdf09171d72c692213bf4c38d1c_39.png)

![](img/43674bdf09171d72c692213bf4c38d1c_41.png)

可视化模式允许我们像使用鼠标一样，高亮选择文本块，然后对选中的内容进行操作。这比单纯的行操作更加灵活。

进入可视化模式有以下三种方式，选择后可以按 `ESC` 键退出：

![](img/43674bdf09171d72c692213bf4c38d1c_43.png)

![](img/43674bdf09171d72c692213bf4c38d1c_44.png)

*   **字符可视化模式 (按 `v`)**：以字符为单位高亮选择连续的文本区域。
*   **行可视化模式 (按 `V`，即 `Shift+v`)**：以整行为单位进行高亮选择。
*   **块可视化模式 (按 `Ctrl+v`)**：以列为单位进行矩形区域的高亮选择，非常适合处理对齐的文本（如批量注释）。

在选中文本后，可以执行以下操作：
*   `d`：删除（剪切）选中内容。
*   `y`：复制选中内容。
*   `p`：在光标后粘贴已复制/剪切的内容。
*   输入 `:` 后执行命令（如 `:s/old/new/`），则命令仅对选中区域生效。

![](img/43674bdf09171d72c692213bf4c38d1c_46.png)

![](img/43674bdf09171d72c692213bf4c38d1c_48.png)

**实用案例：批量添加/删除行首注释**
以下是使用块可视化模式 (`Ctrl+v`) 批量添加行首注释 `#` 的步骤：
1.  将光标移动到要注释的第一行的行首。
2.  按 `Ctrl+v` 进入块可视化模式。
3.  使用方向键向下选择要注释的所有行。
4.  按 `Shift+i` (即大写的 `I`) 进入插入模式。
5.  输入注释符号，例如 `#`。
6.  按 `ESC` 键，所有选中的行首将同时添加 `#`。

要批量删除行首的注释，只需在步骤3选中注释符号所在的列块后，直接按 `d` 键删除即可。

## 多窗口模式 🪟

![](img/43674bdf09171d72c692213bf4c38d1c_54.png)

多窗口模式允许我们在同一个vim界面中同时编辑和查看多个文件或同一个文件的不同部分。

![](img/43674bdf09171d72c692213bf4c38d1c_56.png)

![](img/43674bdf09171d72c692213bf4c38d1c_58.png)

以下是管理多窗口的常用命令：

![](img/43674bdf09171d72c692213bf4c38d1c_60.png)

![](img/43674bdf09171d72c692213bf4c38d1c_62.png)

*   **水平分割窗口**：在命令模式下按 `Ctrl+w` 然后按 `s`。
*   **垂直分割窗口**：在命令模式下按 `Ctrl+w` 然后按 `v`。
*   **在新窗口打开指定文件**：在退出模式下输入 `:sp 文件名` (水平分割) 或 `:vsp 文件名` (垂直分割)。
*   **在窗口间切换光标**：按 `Ctrl+w`，然后按方向键 (`h`, `j`, `k`, `l`) 或 `w` 循环切换。
*   **关闭当前窗口**：确保光标在该窗口内，然后在退出模式下输入 `:q`。

![](img/43674bdf09171d72c692213bf4c38d1c_64.png)

![](img/43674bdf09171d72c692213bf4c38d1c_66.png)

![](img/43674bdf09171d72c692213bf4c38d1c_68.png)

![](img/43674bdf09171d72c692213bf4c38d1c_70.png)

**示例操作流程：**
1.  打开一个文件。
2.  按 `Ctrl+w` `s` 水平分割窗口，上下同时显示同一文件。
3.  按 `Ctrl+w` `j` 将光标移动到下方窗口。
4.  在退出模式下输入 `:sp /etc/passwd`，在下方的窗口中打开另一个文件。
5.  现在可以在上方和下方的窗口中分别编辑和对照两个不同的文件。

## 移动与重命名：mv命令 🚚

![](img/51ba49ae8df25e56d36c27391c0883d6_11.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_13.png)

上一节我们介绍了`cp`命令用于复制文件，本节中我们来看看`mv`命令。`mv`命令用于**移动**文件或目录到其他位置，其功能类似于Windows系统中的“剪切”。同时，它也可以用来修改文件或目录的名称。

![](img/51ba49ae8df25e56d36c27391c0883d6_15.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_17.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_19.png)

命令的基本格式与`cp`命令类似：
```bash
mv [选项] 源文件或目录 目标文件或目录
```

![](img/51ba49ae8df25e56d36c27391c0883d6_21.png)

### 移动文件
例如，将当前目录下的`hello`文件移动到`/opt/t1`目录中：
```bash
mv hello /opt/t1
```
执行后，`hello`文件将从原位置消失，出现在`/opt/t1`目录下。

![](img/51ba49ae8df25e56d36c27391c0883d6_23.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_25.png)

### 重命名功能
`mv`命令在移动的同时可以完成重命名。例如，将`test`目录移动到根目录`/`下，并重命名为`xxxxx`：
```bash
mv test /xxxxx
```
如果不指定新名字，文件移动后名称不变。

## 查看文件内容：cat命令 👀

![](img/51ba49ae8df25e56d36c27391c0883d6_38.png)

在Linux中，我们使用命令来查看文件内容。`cat`命令是最基本的查看工具，适合查看内容较少的文件。

命令格式非常简单：
```bash
cat [选项] 文件名
```
常用选项是`-n`，可以显示行号。

![](img/51ba49ae8df25e56d36c27391c0883d6_40.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_42.png)

以下是查看系统一些关键配置文件的示例：
*   **查看系统所有用户信息**：`cat /etc/passwd`
*   **查看系统版本**：`cat /etc/centos-release` 或 `cat /etc/redhat-release`
*   **查看网卡配置**：`cat /etc/sysconfig/network-scripts/ifcfg-ens33`
*   **带行号查看**：`cat -n /etc/fstab`

![](img/51ba49ae8df25e56d36c27391c0883d6_44.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_46.png)

`cat`命令的缺点是，当文件内容过多时（例如`/etc/services`文件有上万行），它会将所有内容一次性输出到屏幕，导致无法向上滚动查看开头部分的内容。

## 分页查看大文件：less命令 📄

![](img/51ba49ae8df25e56d36c27391c0883d6_54.png)

为了解决查看大文件的问题，我们可以使用`less`命令。它像看书一样，支持分页显示，方便浏览。

命令格式：
```bash
less [选项] 文件名
```
常用选项`-N`可以显示行号。

![](img/51ba49ae8df25e56d36c27391c0883d6_56.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_58.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_60.png)

进入`less`视图后，可以使用以下快捷键进行操作：
*   **上下箭头键**：逐行滚动。
*   **Page Down / Page Up** 或 **空格键 / b键**：向下/向上翻页。
*   **g** 和 **G**：跳转到文件首行和末行。
*   **/关键词**：向下搜索关键词。
*   **?关键词**：向上搜索关键词。
*   **:行号**：跳转到指定行，例如 `:1000` 跳转到第1000行。
*   **q**：退出`less`视图。

![](img/51ba49ae8df25e56d36c27391c0883d6_62.png)

例如，分页查看系统服务文件：
```bash
less -N /etc/services
```

![](img/51ba49ae8df25e56d36c27391c0883d6_64.png)

## 查看文件首尾：head与tail命令 🔍

![](img/51ba49ae8df25e56d36c27391c0883d6_66.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_68.png)

有时我们只需要查看文件的开头或结尾部分，这时`head`和`tail`命令就非常有用。

### head命令
`head`命令默认显示文件开头10行内容。

命令格式及示例：
```bash
head 文件名          # 显示前10行
head -n 20 文件名    # 显示前20行
head -20 文件名      # 显示前20行（简写形式）
```

![](img/51ba49ae8df25e56d36c27391c0883d6_70.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_72.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_74.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_76.png)

### tail命令
`tail`命令默认显示文件末尾10行内容。

![](img/51ba49ae8df25e56d36c27391c0883d6_78.png)

命令格式及示例：
```bash
tail 文件名          # 显示末尾10行
tail -n 20 文件名    # 显示末尾20行
tail -20 文件名      # 显示末尾20行（简写形式）
```
`tail`命令还有一个非常实用的选项`-f`，用于**动态追踪**文件末尾内容的变化，常用于监控日志文件。
```bash
tail -f /var/log/messages
```
执行此命令后，终端会持续显示该日志文件新增的内容，按 `Ctrl+C` 可退出追踪。

## 删除文件与目录：rm命令 🗑️

![](img/51ba49ae8df25e56d36c27391c0883d6_84.png)

![](img/51ba49ae8df25e56d36c27391c0883d6_85.png)

`rm`命令用于删除文件或目录。**请注意，Linux系统默认没有回收站，删除操作需谨慎。**

命令格式：
```bash
rm [选项] 文件或目录
```

### 删除文件
删除普通文件不需要特殊选项：
```bash
rm filename.txt
```
系统会询问是否确认删除，输入`y`确认，`n`取消。

### 删除目录
删除目录必须使用`-r`（递归）选项，因为目录可能包含子文件和子目录：
```bash
rm -r directory_name
```

### 强制删除
使用`-f`（强制）选项可以跳过确认提示，直接删除：
```bash
rm -f filename.txt      # 强制删除文件，无提示
rm -rf directory_name   # 强制递归删除目录，无提示
```
**警告**：`rm -rf /` 或 `rm -rf /*` 命令会强制删除根目录下的所有文件，导致系统崩溃，**绝对禁止在生产环境或自己的学习系统中尝试**。

### 使用通配符`*`
`*`（星号）是通配符，代表“任意所有”字符。它可以用来批量操作。
```bash
rm *.log        # 删除当前目录下所有.log后缀的文件
rm -rf /opt/*   # 删除/opt目录下的所有内容（目录本身保留）
```

### 关于路径斜线`/`的补充
在操作目录时，路径末尾的`/`是可选的，系统都能识别：
```bash
ls /opt
ls /opt/
```
两者效果相同。但对于文件，后面不能加`/`，系统会将其误判为目录。在按`Tab`键自动补全时，系统会自动为目录名加上`/`，而文件名则没有。

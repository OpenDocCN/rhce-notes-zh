# Linux从入门到精通：P25：Vim编辑器常用命令详解 🖥️

在本节课中，我们将学习Vim文本编辑器的常用命令。Vim是Linux系统中功能强大的文本编辑器，掌握其基本操作对系统管理和编程至关重要。我们将从三种模式入手，详细介绍命令模式和末行模式下的核心操作。

## 命令模式常用操作

上一节我们了解了Vim的三种模式。本节中，我们来看看在默认进入的**命令模式**下，有哪些基础且高效的操作。

首先，我们打开一个文件进行演示。在终端输入以下命令进入Vim并编辑文件 `aa.txt`：
```bash
vim aa.txt
```
进入后，Vim默认处于命令模式。

以下是命令模式下的一些核心操作：

*   **删除/剪切行**：按两次 `d` 键，即 `dd`，可以删除当前光标所在的一行。更准确地说，这个操作是剪切，因为被删除的内容会被保存到缓冲区。
*   **粘贴内容**：使用小写 `p` 可以将缓冲区的内容粘贴到光标所在行的下方。
*   **删除多行**：在 `dd` 前加上数字，可以删除指定行数。例如，输入 `3dd` 会删除从当前行开始的三行。
*   **复制一行**：按两次 `y` 键，即 `yy`，可以复制当前行。
*   **复制多行**：在 `yy` 前加上数字，可以复制指定行数。例如，输入 `5yy` 会复制从当前行开始的五行。
*   **撤销操作**：按 `u` 键可以撤销上一次的操作。连续按 `u` 会按照操作顺序依次撤销。

## 末行模式常用命令

掌握了命令模式的基础编辑后，我们进入功能更丰富的**末行模式**。在命令模式下按 `:` 键即可进入末行模式。

以下是末行模式下的一些常用命令：

*   **保存文件**：输入 `w` 然后按回车，即可保存对文件的修改。
*   **退出Vim**：输入 `q` 然后按回车，即可退出Vim。
*   **强制退出（不保存）**：如果文件已被修改但不想保存，使用 `q!` 可以强制退出，所有修改将被丢弃。
*   **保存并退出**：输入 `wq` 可以保存修改并退出Vim。
*   **强制保存并退出**：在某些情况下（如文件只读），保存可能报错。此时使用 `wq!` 可以强制保存并退出。
*   **显示行号**：输入 `set number` 或简写 `set nu` 可以为文本显示行号，便于定位。
*   **隐藏行号**：输入 `set nonumber` 或简写 `set nonu` 可以隐藏行号。
*   **执行Shell命令**：在末行模式下，输入 `!` 后接命令可以在不退出Vim的情况下执行Shell命令。例如，输入 `!date` 会显示系统时间，按回车后返回编辑器。
*   **搜索字符串**：
    *   输入 `/字符串` 可以从光标位置**向下**搜索指定字符串。
    *   输入 `?字符串` 可以从光标位置**向上**搜索指定字符串。
*   **替换字符串**：
    *   **替换当前行第一个匹配项**：`:s/旧文本/新文本`
    *   **替换当前行所有匹配项**：`:s/旧文本/新文本/g`
    *   **替换全文所有匹配项**：`:%s/旧文本/新文本/g`

例如，将全文所有的 `two` 替换为 `five`，命令为：
```vim
:%s/two/five/g
```

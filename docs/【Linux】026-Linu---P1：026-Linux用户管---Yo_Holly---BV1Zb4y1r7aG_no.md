# Linux用户管理：026：用户身份与基础命令 🐧

在本节课中，我们将要学习Linux系统中用户管理的基础知识。Linux是一个多用户、多任务的操作系统，其安全性和稳定性很大程度上依赖于对用户和用户组的有效管理。我们将从理解用户身份开始，并学习创建用户、用户组以及设置密码的核心命令。

## 用户身份介绍

上一节我们提到了Linux的多用户特性，本节中我们来看看系统中具体的用户身份是如何划分的。

Linux系统中的用户身份主要分为三类：

*   **管理员用户**：确切地说，管理员是**UID（用户ID）为0**的用户。通常，系统会将`root`用户的UID设置为0，因此`root`用户就是默认的系统管理员。
*   **系统用户**：这类用户的**UID范围是1到999**。系统为每个服务程序（如Web服务器、数据库）创建独立的系统用户来运行它。这样做的好处是，即使某个服务程序出现安全漏洞，黑客的权限也被限制在该系统用户内，从而有效控制破坏范围。
*   **普通用户**：这类用户的**UID默认从1000开始**。UID 1000之前的号码是系统保留给系统用户使用的。普通用户由管理员创建，用于完成日常工作。

为了便于管理具有相同权限或任务的多个用户，Linux引入了**用户组（GID）**的概念。通过将用户加入同一个组，可以方便地为整个组统一设置权限或分配任务。例如，可以设置只有“技术部”组的成员才能访问特定的数据库。

## 常用用户管理命令

理解了用户身份后，我们来看看管理它们需要用到哪些基础命令。首先，我们需要知道如何查看用户信息。

### 查看用户信息：`id` 命令

`id`命令用于显示指定用户的详细信息，包括用户ID（UID）、主要组ID（GID）以及所属的所有组。

其基本格式为：
```bash
id [用户名]
```
例如，查看`root`用户的信息：
```bash
id root
```
输出会显示类似 `uid=0(root) gid=0(root) groups=0(root)` 的信息，表明`root`用户的UID和GID都是0。

### 创建新用户：`useradd` 命令

当需要创建新用户时，我们使用`useradd`命令。如果不指定任何选项，系统会使用默认设置创建用户，包括：
*   自动分配一个从1000开始递增的UID。
*   自动创建一个与用户名同名的用户组作为该用户的基本组。

以下是创建用户时可以指定的一些常用选项：

*   **`-e`**：设置账户的过期日期。格式为 `YYYY-MM-DD`。适用于创建有固定期限的账户（如实习生账户）。
    ```bash
    useradd -e 2021-08-06 bbb
    ```

*   **`-u`**：指定用户的UID。可以为特殊用户（如核心部门成员）分配特定的UID。
    ```bash
    useradd -u 888 pro
    ```

*   **`-g`**：指定用户的基本组（主组）。每个用户有且只有一个基本组，类似于用户的“原生家庭”。
    ```bash
    useradd -g proadv -u 111 cccc
    ```

*   **`-G`**：指定用户的扩展组（附加组）。一个用户可以属于多个扩展组，这类似于一个人可以在公司同时属于“研发部”、“项目A组”等多个团队。
    ```bash
    useradd -G pro1,pro2,pro3 dddd
    ```

### 创建用户组：`groupadd` 命令

在将用户加入特定组之前，我们需要先创建这个组。`groupadd`命令用于创建新的用户组。

```bash
groupadd [组名]
```
例如，创建一个名为 `proadv` 的组：
```bash
groupadd proadv
```

### 设置或修改密码：`passwd` 命令

使用`useradd`创建的用户初始时没有密码（或被锁定），需要使用`passwd`命令为其设置密码。密码应尽可能复杂，通常建议在8位以上，包含字母、数字和符号。

命令格式为：
```bash
passwd [用户名]
```
执行后，系统会提示你输入并确认新的密码。如果是普通用户，使用不带用户名的`passwd`可以修改自己的密码。

## 总结

本节课中我们一起学习了Linux用户管理的基础。我们首先了解了三种用户身份：**管理员（UID=0）**、**系统用户（UID 1-999）** 和**普通用户（UID从1000开始）**，并理解了**用户组（GID）** 对于集中管理权限的重要性。

接着，我们掌握了四个核心命令：
1.  使用 **`id`** 查看用户身份信息。
2.  使用 **`useradd`** 创建用户，并可以通过 `-e`、`-u`、`-g`、`-G` 等选项精细控制账户属性。
3.  使用 **`groupadd`** 创建新的用户组。
4.  使用 **`passwd`** 为用户设置或更改登录密码。

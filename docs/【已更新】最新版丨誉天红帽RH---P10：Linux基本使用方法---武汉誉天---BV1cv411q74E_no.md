## 多窗口与标签页管理 📑

![](img/7eedc979907d22593e9693185e800498_22.png)

![](img/7eedc979907d22593e9693185e800498_24.png)

如果当前终端窗口正在执行任务，而我们想同时执行其他任务，可以打开多个终端窗口或标签页。

![](img/7eedc979907d22593e9693185e800498_26.png)

![](img/7eedc979907d22593e9693185e800498_28.png)

![](img/7eedc979907d22593e9693185e800498_30.png)

以下是管理终端窗口和标签页的方法：
*   通过菜单栏的 **New Window**（新建窗口）可以打开多个独立的终端窗口。
*   通过菜单栏的 **New Tab**（新建标签页）或快捷键 **Ctrl+Shift+T** 可以在当前窗口内创建新的标签页。
*   使用快捷键 **Alt+1, Alt+2, Alt+3...** 可以在不同的标签页之间切换。
*   关闭标签页或窗口时，建议使用 `exit` 命令退出，而不是直接点击窗口右上角的关闭按钮，以避免异常中断正在运行的任务。

## 常用快捷键操作 ⚡

熟练使用快捷键可以极大提高在终端中工作的效率。

![](img/7eedc979907d22593e9693185e800498_34.png)

以下是几个常用的终端快捷键：
*   **放大字体**：`Ctrl+Shift++`（加号）
*   **缩小字体**：`Ctrl+-`（减号）
*   **清屏**：`Ctrl+L` 或输入 `clear` 命令
*   **复制选中文本**：`Ctrl+Shift+C`
*   **粘贴文本**：`Ctrl+Shift+V`
*   （注意：在Linux终端中，`Ctrl+C` 通常用于中断当前运行的命令，而不是复制。）

### 显示时间与日历

![](img/7eedc979907d22593e9693185e800498_49.png)

![](img/7eedc979907d22593e9693185e800498_51.png)

`date` 命令用于显示系统的当前日期和时间。
```bash
date
```
`cal` 命令用于显示当前月份的日历。
```bash
cal
```

### 查看用户信息

在终端提示符中，`@` 符号前的部分是**当前用户名**，后面是**主机名**，`~` 符号代表当前所在的目录（家目录）。提示符本身也有含义：`#` 表示当前是超级管理员root用户，`$` 表示当前是普通用户。

`id` 命令用于显示当前用户的身份信息。
```bash
id
```
`id` 命令后接用户名，可以查看该系统用户是否存在。
```bash
id 用户名
```
例如，`id zhangsan` 如果返回 “no such user”（无此用户），则说明用户不存在。

### 切换用户

![](img/7eedc979907d22593e9693185e800498_53.png)

![](img/7eedc979907d22593e9693185e800498_55.png)

![](img/7eedc979907d22593e9693185e800498_56.png)

Linux是多用户操作系统，可以在不注销的情况下直接切换用户。

![](img/7eedc979907d22593e9693185e800498_58.png)

![](img/7eedc979907d22593e9693185e800498_60.png)

`su` 命令用于切换用户。`su - 用户名` 会切换到目标用户并加载其环境配置。
```bash
su - root
```
输入此命令并输入root用户的密码后，提示符会从 `$` 变为 `#`，表示已切换到root管理员身份。

## 概述

文件查看是Linux运维中最常见的操作之一。不同的命令适用于不同的场景，例如快速预览、分页浏览或实时监控日志。本节将详细介绍 `cat`、`more`、`less`、`head` 和 `tail` 命令的用法与区别。

## cat命令：一次性显示全部内容

`cat` 命令的语法非常简单，其作用是**一次性显示文件的全部内容**。

**基本语法：**
```bash
cat [文件名]
```

![](img/b6d23c6735de24d13295bfde4774b1ea_7.png)

例如，查看 `/etc/passwd` 文件：
```bash
cat /etc/passwd
```

然而，如果文件内容很长，使用 `cat` 命令会导致信息快速滚过屏幕，难以阅读开头部分。因此，它更适合查看内容较少的文件。

![](img/b6d23c6735de24d13295bfde4774b1ea_9.png)

## more与less命令：分页查看文件

为了解决长文件查看的问题，我们可以使用 `more` 或 `less` 命令进行分页显示。

上一节我们介绍了`cat`命令的局限性，本节中我们来看看如何进行分页查看。

![](img/b6d23c6735de24d13295bfde4774b1ea_11.png)

### more命令

`more` 命令以分页形式显示内容，但**只能向前翻页，不支持后退**。

![](img/b6d23c6735de24d13295bfde4774b1ea_13.png)

**基本语法：**
```bash
more [文件名]
```

**操作方式：**
*   按下 **回车键**：向下滚动一行。
*   按下 **空格键**：向下滚动一页。
*   输入 **q**：退出查看。

![](img/b6d23c6735de24d13295bfde4774b1ea_15.png)

![](img/b6d23c6735de24d13295bfde4774b1ea_17.png)

### less命令

![](img/b6d23c6735de24d13295bfde4774b1ea_19.png)

`less` 命令功能更为强大，它**支持前后翻页和搜索**。

**基本语法：**
```bash
less [文件名]
```

![](img/b6d23c6735de24d13295bfde4774b1ea_21.png)

**操作方式：**
*   **空格键** 或 **PageDown**：向下翻一页。
*   **PageUp**：向上翻一页。
*   **回车键** 或 **向下箭头**：向下滚动一行。
*   **向上箭头**：向上滚动一行。
*   **/**：进入搜索模式，输入内容后按回车可高亮查找。
*   **q**：退出查看。

在实际工作中，`more` 命令常与管道符 `|` 结合使用，用于查看其他命令产生的大量输出。而 `less` 则更适合需要仔细阅读、来回翻阅文件内容的场景。

![](img/b6d23c6735de24d13295bfde4774b1ea_23.png)

![](img/b6d23c6735de24d13295bfde4774b1ea_25.png)

**管道符示例：**
```bash
ifconfig | more
```
这条命令的含义是：将 `ifconfig` 命令的输出结果，作为 `more` 命令的输入，从而实现分页查看网卡信息。

## head与tail命令：查看文件首尾部分

![](img/b6d23c6735de24d13295bfde4774b1ea_31.png)

有时我们只需要查看文件的开头或结尾部分，这时 `head` 和 `tail` 命令就非常实用。

### head命令

`head` 命令用于**显示文件的开头若干行**，默认显示前10行。

**基本语法与示例：**
```bash
head [文件名]          # 显示文件前10行
head -n 5 [文件名]     # 显示文件前5行
```
例如，查看 `/etc/passwd` 文件的前3行：
```bash
head -n 3 /etc/passwd
```

![](img/b6d23c6735de24d13295bfde4774b1ea_33.png)

### tail命令

`tail` 命令用于**显示文件的尾部若干行**，默认显示最后10行。它在监控日志文件时尤其重要。

![](img/b6d23c6735de24d13295bfde4774b1ea_35.png)

**基本语法与示例：**
```bash
tail [文件名]           # 显示文件最后10行
tail -n 5 [文件名]      # 显示文件最后5行
tail -f [文件名]        # 动态跟踪并显示文件新增内容
```

![](img/b6d23c6735de24d13295bfde4774b1ea_37.png)

![](img/b6d23c6735de24d13295bfde4774b1ea_39.png)

`tail -f` 的 `-f` 参数代表 `follow`，可以实时显示文件尾部新写入的内容，是监控日志文件变化的利器。

![](img/b6d23c6735de24d13295bfde4774b1ea_41.png)

**动态监控日志示例：**
1.  在终端A中执行以下命令，开始动态监控系统安全日志：
    ```bash
    tail -f /var/log/secure
    ```
2.  打开终端B，尝试用SSH登录本机。
3.  此时，在终端A中就会实时显示出登录行为产生的日志条目。

## 总结

本节课中我们一起学习了Linux下五个核心的文件查看命令：
1.  **`cat`**：一次性显示全部文件内容，适合小文件。
2.  **`more`**：分页显示文件内容，仅支持向前翻页，常与管道符联用。
3.  **`less`**：分页显示文件内容，功能强大，支持前后翻页和搜索。
4.  **`head`**：查看文件开头部分，默认显示前10行。
5.  **`tail`**：查看文件结尾部分，默认显示后10行，其 `-f` 参数可用于实时监控日志。

![](img/b6d23c6735de24d13295bfde4774b1ea_43.png)

![](img/b6d23c6735de24d13295bfde4774b1ea_45.png)

熟练掌握这些命令，能够帮助你在日常运维工作中高效地查看和分析文件及日志信息。

## 系统调优服务 (tuned) ⚙️

系统调优服务 `tuned` 是一个守护进程，它可以根据特定的工作负载要求，应用预定义的或自定义的调优配置文件。管理员可以以静态或动态两种方式进行调优调整。

![](img/e955ca2987b6fb68d1b1017a91342f6e_5.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_7.png)

### 预定义的调优配置文件

`tuned` 提供了一系列预定义的调优方案，适用于不同场景：

以下是 `tuned` 服务内置的一些主要配置文件：

*   **`balanced`**：默认的平衡模式，在性能和功耗间取得均衡。
*   **`desktop`**：从 `balanced` 衍生，优化交互式桌面应用的响应速度。
*   **`throughput-performance`**：优化系统吞吐量，偏向高性能。
*   **`latency-performance`**：优化服务器低延迟，牺牲部分功耗。
*   **`network-latency`**：在 `throughput-performance` 基础上，进一步优化网络延迟。
*   **`network-throughput`**：优化网络吞吐量。
*   **`powersave`**：最大化节能，偏向低功耗。
*   **`virtual-guest`**：针对虚拟化客户机优化的方案。
*   **`virtual-host`**：针对虚拟化宿主机优化的方案。

## 进程优先级管理 (nice值) 📊

![](img/e955ca2987b6fb68d1b1017a91342f6e_67.png)

进程优先级决定了系统资源（特别是CPU时间）分配给进程的顺序。我们主要关注非实时进程的优先级，它通过 **nice值** 来调整。

![](img/e955ca2987b6fb68d1b1017a91342f6e_69.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_71.png)

### 理解 nice 值

![](img/e955ca2987b6fb68d1b1017a91342f6e_73.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_75.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_77.png)

*   进程优先级范围是 **0 到 139**。
*   **0-99**：实时进程优先级，由系统控制。
*   **100-139**：非实时进程优先级，用户可调。对应的 **nice值** 范围是 **-20 到 19**。
    *   `nice = -20` 对应优先级 `100`（最高优先级）。
    *   `nice = 0` 对应优先级 `120`（默认优先级）。
    *   `nice = 19` 对应优先级 `139`（最低优先级）。

### 管理进程优先级

以下是调整进程优先级的方法：

![](img/e955ca2987b6fb68d1b1017a91342f6e_79.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_81.png)

*   **启动时指定优先级**：使用 `nice` 命令。
    ```bash
    # 以最高优先级(-20)启动 ping 命令
    nice -n -20 ping localhost
    ```

![](img/e955ca2987b6fb68d1b1017a91342f6e_83.png)

*   **调整运行中进程的优先级**：使用 `renice` 命令。
    ```bash
    # 将PID为1234的进程的nice值改为10（较低优先级）
    renice -n 10 1234
    ```

![](img/e955ca2987b6fb68d1b1017a91342f6e_85.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_87.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_89.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_91.png)

*   **使用 top 命令交互式调整**：
    1.  运行 `top` 命令。
    2.  按 `r` 键。
    3.  输入要调整的进程PID。
    4.  输入新的nice值。

![](img/e955ca2987b6fb68d1b1017a91342f6e_93.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_95.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_97.png)

*   **查看进程的nice值**：
    ```bash
    # 使用 ps 命令查看，NI列即为nice值
    ps -eo pid,comm,ni --sort=-ni | head
    # 或者在 top 命令中，NI列显示nice值
    top
    ```

![](img/e955ca2987b6fb68d1b1017a91342f6e_99.png)

![](img/e955ca2987b6fb68d1b1017a91342f6e_101.png)

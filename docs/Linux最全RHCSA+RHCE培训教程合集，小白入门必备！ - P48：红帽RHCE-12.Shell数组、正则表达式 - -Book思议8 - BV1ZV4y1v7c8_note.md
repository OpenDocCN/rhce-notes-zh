### 数组的定义与取值

定义数组的语法与定义变量类似，但需要将多个值放在小括号 `()` 内，并用空格隔开。

**代码示例：定义数组**
```bash
B=(小方 小威 九妹 阿联)
```

取值时，需要通过下标来访问数组中的特定元素。**Shell数组的下标从0开始**。

![](img/611658c683a8a5749d06d610945aa93b_16.png)

![](img/611658c683a8a5749d06d610945aa93b_18.png)

**公式：数组取值**
`${数组名[下标]}`

**代码示例：访问数组元素**
```bash
echo ${B[0]}  # 输出：小方
echo ${B[1]}  # 输出：小威
echo ${B[3]}  # 输出：阿联
```

如果想获取数组中的所有值，可以使用 `@` 或 `*` 符号。

**代码示例：获取所有值**
```bash
echo ${B[@]}
```

### 数组的实用场景

数组常与循环结合使用，以批量处理数据。以下是使用数组和for循环批量安装软件包的示例。

![](img/611658c683a8a5749d06d610945aa93b_24.png)

**代码示例：使用数组批量安装软件**
```bash
#!/bin/bash
# 定义需要安装的软件包数组
SOFTWARE=(gcc vsftpd httpd)
# 使用for循环遍历数组并安装
for pkg in ${SOFTWARE[@]}
do
    yum install -y $pkg
done
```

上一节我们了解了数组的基本操作，接下来我们看看如何向已存在的数组中添加新元素。

### 向数组添加元素

可以通过指定新的下标来向数组中添加值。

**代码示例：向数组追加元素**
```bash
B[4]="波多"
echo ${B[@]}  # 输出所有值，包含新添加的"波多"
```

## 正则表达式

正则表达式使用一系列特殊符号来描述字符串的匹配模式。它在文本搜索、替换和数据提取中极为常用，且语法在各种编程语言中基本通用。

### 常用正则表达式符号

以下是几个最核心且常用的正则表达式符号及其功能。

**1. 匹配行首 (`^`)**
`^` 符号用于匹配一行的开头。

**代码示例：匹配以root开头的行**
```bash
grep "^root" /etc/passwd
```

**2. 匹配行尾 (`$`)**
`$` 符号用于匹配一行的结尾。

**代码示例：匹配以/bin/bash结尾的行**
```bash
grep "/bin/bash$" /etc/passwd
```

**3. 匹配集合中任意单个字符 (`[ ]`)**
`[ ]` 用于匹配方括号内列出的任意一个字符。

**代码示例：匹配包含a、b或c中任意一个字符的行**
```bash
grep "[abc]" filename
```

可以在集合开头使用 `^` 进行取反，匹配不在集合中的字符。

**代码示例：匹配不包含a、b或c的行**
```bash
grep "[^abc]" filename
```

**4. 匹配任意单个字符 (`.`)**
`.` 符号可以匹配任意一个字符（除了换行符）。

**代码示例：匹配像"r.t"这样的模式，如"rat"、"rot"等**
```bash
grep "r.t" filename
```

![](img/611658c683a8a5749d06d610945aa93b_29.png)

**5. 匹配前一个字符零次或多次 (`*`)**
`*` 符号表示其前面的字符可以出现零次或多次。

**代码示例：匹配"gooood"、"god"等**
```bash
grep "go*d" filename
```

**6. 扩展正则表达式 (`grep -E` 或 `egrep`)**
某些更强大的元字符（如 `+`, `?`, `|`, `()`）需要在扩展正则表达式模式下使用。

![](img/611658c683a8a5749d06d610945aa93b_31.png)

**代码示例：使用`|`匹配多个整体模式**
```bash
egrep "(root|test)" /etc/passwd
# 或使用 grep -E
grep -E "(root|test)" /etc/passwd
```
`( )` 可以将模式组合为一个整体，`|` 表示“或”的关系。

## 核心概念总结

本节课中我们一起学习了Shell数组和正则表达式的核心知识：
1.  **Shell数组**：一种可以存储多个值的变量。通过 `${数组名[下标]}` 访问元素，下标从0开始。`${数组名[@]}` 可获取所有值。数组常用于与循环结合进行批量操作。
2.  **正则表达式**：用于文本匹配的强大工具。我们掌握了几个最实用的符号：
    *   `^` 匹配行首。
    *   `$` 匹配行尾。
    *   `[ ]` 匹配字符集合。
    *   `.` 匹配任意单个字符。
    *   `*` 匹配前一个字符零次或多次。
    *   使用 `grep -E` 或 `egrep` 来启用扩展功能，如 `( )` 和 `|`。

![](img/611658c683a8a5749d06d610945aa93b_41.png)

理解这些基础概念和常用符号，足以应对日常运维脚本中的大多数文本处理需求。在后续学习中，我们将看到它们在`sed`、`awk`等工具中的进一步应用。

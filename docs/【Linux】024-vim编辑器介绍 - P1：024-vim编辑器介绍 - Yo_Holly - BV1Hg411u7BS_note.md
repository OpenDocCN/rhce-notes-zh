# Linux文本编辑：1：vim编辑器介绍 🖥️

在本节课中，我们将要学习Linux系统中一个强大且必不可少的工具——vim文本编辑器。我们将了解它的起源、核心设计理念以及三种基本工作模式，为后续深入学习其具体操作打下基础。

## 概述

几乎所有Linux系统都内置了一个名为`vi`的文本编辑器。虽然`vi`功能已经相当完善，但仍有优化空间。因此，一个增强版本的`vi`编辑器被开发出来，它就是`vim`。`vim`的一个实用改进是能够通过字体颜色来辨别语法正确性，这极大地方便了程序设计，因此深受程序员喜爱。

在Linux系统中，“一切皆文件”。无论是配置服务还是修改系统参数，本质上都是在编辑各种配置文件。因此，一个高效的文本编辑器是必不可少的。`vim`因其强大的功能和独特的设计，得到了众多厂商和用户的广泛认可。

## vim的三种工作模式

`vim`之所以高效，核心在于它设置了三种不同的工作模式。这三种模式各司其职，并能方便地切换，从而简化了编辑工作。以下是这三种模式的介绍：

*   **命令模式**：这是启动`vim`后进入的默认模式。在此模式下，可以控制光标移动，并对文本进行复制、粘贴、删除和查找等工作。
*   **输入模式**：在此模式下，可以进行正常的文本录入和编辑工作，就像使用普通的记事本一样。
*   **末行模式**：在此模式下，可以执行保存文件、退出编辑器以及设置编辑环境等操作。

## 模式间的切换关系

理解了三种模式的功能后，我们来看看它们之间是如何切换的。模式切换是使用`vim`的关键，其关系如下图所示：

![](img/11de7f683dc007f4fe63cb8f3f3a556a_1.png)

具体切换方法如下：

*   **命令模式 ↔ 输入模式**：从命令模式按下 `a`、`i` 或 `o` 等键即可进入输入模式。从输入模式退回命令模式，只需按下 `Esc` 键。
*   **命令模式 ↔ 末行模式**：在命令模式下输入英文冒号 `:` 即可进入末行模式。从末行模式退回命令模式，同样按下 `Esc` 键。

## 为什么需要学习命令

上一节我们介绍了`vim`的多种模式，本节我们来看看为什么在`vim`中需要记忆和使用各种命令。

在Windows系统中，我们使用的文本编辑器（如Notepad++）通常提供了丰富的图形按钮，通过点击就能完成大多数操作。然而，在Linux的字符界面中，`vim`是一个纯文本编辑器，没有这些快捷图标。因此，掌握各种键盘命令就扮演了至关重要的角色。学习`vim`，很大程度上就是学习如何高效地使用这些命令来编辑文件。

## 总结

![](img/11de7f683dc007f4fe63cb8f3f3a556a_2.png)

本节课中，我们一起学习了`vim`编辑器的基本概念。我们了解到`vim`是`vi`的增强版，因其语法高亮等功能而流行。其核心设计是三种工作模式：**命令模式**、**输入模式**和**末行模式**，它们通过特定的按键（如 `Esc`、`:`、`a/i/o`）进行切换。正是由于在字符界面下缺乏图形按钮，熟练使用键盘命令成为了高效操作`vim`的关键。掌握这些基础知识，是后续深入学习`vim`各种编辑技巧的前提。

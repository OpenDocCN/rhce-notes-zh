## Linux 文件类型

![](img/6ad4a2aec2718a14df6b3990d15a36c2_11.png)

![](img/6ad4a2aec2718a14df6b3990d15a36c2_13.png)

上一节我们了解了 Linux 的目录结构，本节中我们来看看 Linux 系统中的文件类型。Linux 主要有以下七种文件类型，这是必须掌握的基础知识。

![](img/6ad4a2aec2718a14df6b3990d15a36c2_15.png)

*   **普通文件**：使用符号 `-` 表示。最常见的文件类型，例如文本文件、可执行文件。
*   **目录文件**：使用符号 `d` 表示。即目录（directory）。
*   **软链接文件**：使用符号 `l` 表示。类似于 Windows 的快捷方式。
*   **块设备文件**：使用符号 `b` 表示。对应物理设备（如硬盘）的索引文件。
*   **字符设备文件**：使用符号 `c` 表示。对应字符设备（如终端 `tty`）。
*   **管道文件**：使用符号 `p` 表示。用于进程间通信，实现一个命令的输出作为另一个命令的输入。
*   **套接字文件**：使用符号 `s` 表示。用于网络服务间通信。

> **提示**：使用 `ls -l` 或 `ll` 命令可以查看文件详细信息，第一个字符即代表文件类型。

## 路径概念

![](img/6ad4a2aec2718a14df6b3990d15a36c2_17.png)

![](img/6ad4a2aec2718a14df6b3990d15a36c2_19.png)

![](img/6ad4a2aec2718a14df6b3990d15a36c2_21.png)

在操作文件之前，我们需要理解路径的概念。路径分为绝对路径和相对路径。

*   **绝对路径**：从根目录 `/` 开始的完整路径，在任何位置都指向同一个目标。例如 `/home/student`。
*   **相对路径**：相对于当前工作目录的路径。使用 `.` 表示当前目录，`..` 表示上一级目录。

## 基础文件操作命令

![](img/6ad4a2aec2718a14df6b3990d15a36c2_23.png)

理解了路径之后，我们来看看几个最基础的文件操作命令。

![](img/6ad4a2aec2718a14df6b3990d15a36c2_25.png)

### 显示当前工作目录

使用 `pwd` 命令可以打印当前工作目录的绝对路径。

```bash
pwd
```

### 切换工作目录

使用 `cd` 命令可以更改当前工作目录。

![](img/6ad4a2aec2718a14df6b3990d15a36c2_27.png)

![](img/6ad4a2aec2718a14df6b3990d15a36c2_29.png)

以下是 `cd` 命令的一些常用用法：

![](img/6ad4a2aec2718a14df6b3990d15a36c2_31.png)

![](img/6ad4a2aec2718a14df6b3990d15a36c2_33.png)

*   `cd` 或 `cd ~`：切换到当前用户的家目录。
*   `cd -`：切换到上一个工作目录。
*   `cd ..`：切换到上一级目录。
*   `cd /path/to/dir`：使用绝对路径切换到指定目录。
*   `cd relative/path`：使用相对路径切换到指定目录。

![](img/6ad4a2aec2718a14df6b3990d15a36c2_35.png)

> **注意**：`cd` 命令的目标必须是一个目录，不能是文件。

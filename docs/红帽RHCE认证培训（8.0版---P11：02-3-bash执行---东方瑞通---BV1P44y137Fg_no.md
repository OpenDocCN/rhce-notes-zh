## Tab键补全功能 🔍

上一节我们介绍了如何在命令行中执行基本命令。本节中，我们来看看如何利用Tab键的补全功能来简化命令输入。

![](img/af1cd5d4869ccb298a777342475f083e_3.png)

在Bash中，一个完整的命令通常由**命令名**、**选项**和**参数**三部分组成。Tab键补全功能可以帮助我们自动补全这三部分内容。

![](img/af1cd5d4869ccb298a777342475f083e_5.png)

**工作原理**：当你输入命令的一部分后，按下Tab键，Bash会尝试根据已输入的内容查找并补全唯一的匹配项。如果按一次Tab键没有反应，通常意味着有多个匹配项，此时再按一次Tab键，Bash会列出所有可能的匹配项供你选择。

![](img/af1cd5d4869ccb298a777342475f083e_7.png)

![](img/af1cd5d4869ccb298a777342475f083e_9.png)

以下是Tab键补全的具体应用示例：

*   **补全命令名**：输入命令的开头几个字母（如 `pa`），然后按两次Tab键，会列出所有以 `pa` 开头的命令。
*   **补全选项**：输入命令（如 `passwd`）和一个空格后，输入短横线 `-` 并按Tab键，会列出该命令所有可用的选项。
*   **补全文件路径参数**：输入命令和部分路径（如 `cat /etc/pass`）后按Tab键，会自动补全文件或目录名。

![](img/af1cd5d4869ccb298a777342475f083e_11.png)

![](img/af1cd5d4869ccb298a777342475f083e_13.png)

**核心技巧**：在执行命令时，应养成多按Tab键的习惯。这不仅能减少输入错误，还能帮助你熟悉系统中有哪些命令和文件。

## 命令换行符 \\

有时，一个命令会非常长，写在一行里不便于阅读和检查。Bash提供了换行符 `\`（反斜杠）来解决这个问题。

![](img/af1cd5d4869ccb298a777342475f083e_15.png)

![](img/af1cd5d4869ccb298a777342475f083e_17.png)

**作用**：在命令末尾输入一个反斜杠 `\` 然后回车，Bash不会立即执行命令，而是允许你在下一行继续输入该命令的剩余部分。系统会自动在下一行行首添加一个 `>` 提示符。

![](img/af1cd5d4869ccb298a777342475f083e_19.png)

![](img/af1cd5d4869ccb298a777342475f083e_21.png)

**示例对比**：
*   单行命令：`head -n 3 /usr/share/dict/words /usr/share/dict/linux.words`
*   使用换行符的多行命令：
    ```bash
    head -n 3 \
    > /usr/share/dict/words \
    > /usr/share/dict/linux.words
    ```
    这两个命令的执行效果完全相同。使用换行符后，每个参数独立成行，使命令结构更清晰，尤其适用于编写复杂的脚本或管道命令时。

![](img/af1cd5d4869ccb298a777342475f083e_23.png)

![](img/af1cd5d4869ccb298a777342475f083e_25.png)

**注意**：`\` 符号后直接按回车，不要输入空格。

## 历史命令管理 history 📜

![](img/af1cd5d4869ccb298a777342475f083e_27.png)

![](img/af1cd5d4869ccb298a777342475f083e_29.png)

Bash会记录你在当前会话中执行过的命令。`history` 命令就是用来查看和管理这些历史记录的强大工具。

![](img/af1cd5d4869ccb298a777342475f083e_31.png)

![](img/af1cd5d4869ccb298a777342475f083e_33.png)

执行 `history` 命令（不带任何选项）会按序号列出所有记录的历史命令。

![](img/af1cd5d4869ccb298a777342475f083e_35.png)

![](img/af1cd5d4869ccb298a777342475f083e_37.png)

![](img/af1cd5d4869ccb298a777342475f083e_39.png)

以下是 `history` 命令的几个常用选项和用法：

![](img/af1cd5d4869ccb298a777342475f083e_41.png)

![](img/af1cd5d4869ccb298a777342475f083e_43.png)

*   **`history`**：列出所有历史命令。
*   **`!n`**：重新执行历史记录中第 `n` 条命令。例如 `!22` 会执行序号为22的命令。
*   **`!string`**：重新执行最近一条以指定字符串 `string` 开头的命令。例如 `!cat` 会执行最近一条以 `cat` 开头的命令。搜索顺序是**从历史列表的底部向上**查找。
*   **`history -c`**：**清空当前会话的内存历史记录**。这个命令常用于需要保密操作步骤的场景。
*   **`history -w [文件名]`**：将当前内存中的历史命令**写入**到历史记录文件（默认为用户家目录下的 `.bash_history`）中。可以指定其他文件名。
*   **`history -a [文件名]`**：将当前会话中新增的历史命令**追加**到历史记录文件中。

![](img/af1cd5d4869ccb298a777342475f083e_45.png)

**重要提示**：历史命令默认保存在内存中，只在退出会话时才会自动写入 `~/.bash_history` 文件。使用 `-w` 选项可以手动立即保存。

![](img/af1cd5d4869ccb298a777342475f083e_47.png)

![](img/af1cd5d4869ccb298a777342475f083e_49.png)

![](img/af1cd5d4869ccb298a777342475f083e_51.png)

## Bash常用快捷键 ⌨️

![](img/af1cd5d4869ccb298a777342475f083e_53.png)

![](img/af1cd5d4869ccb298a777342475f083e_55.png)

熟练使用快捷键是成为命令行高手的关键。以下是一些最实用、能显著提升效率的Bash快捷键：

![](img/af1cd5d4869ccb298a777342475f083e_57.png)

*   **`Ctrl + L`** 或 **`clear` 命令**：快速清空当前终端屏幕，效果等同于 `clear` 命令。
*   **`Ctrl + D`**：发送EOF（文件结束符），常用于退出当前终端会话或结束输入。
*   **`!!`**：执行上一条命令。
*   **`!$`** 或 **`Esc + .`**：获取上一条命令的最后一个参数。例如，执行 `ls /etc/hosts` 后，输入 `cat !$` 相当于输入 `cat /etc/hosts`。
*   **`Ctrl + A`**：将光标移动到**行首**（Home键功能）。
*   **`Ctrl + E`**：将光标移动到**行尾**（End键功能）。
*   **`Ctrl + U`**：删除从光标位置到**行首**的所有字符。
*   **`Ctrl + K`**：删除从光标位置到**行尾**的所有字符。
*   **`Ctrl + W`**：删除光标前的一个单词（以空格为分隔符）。
*   **`Alt + B`** 或 **`Ctrl + ←`**：将光标向后移动一个单词。
*   **`Alt + F`** 或 **`Ctrl + →`**：将光标向前移动一个单词。
*   **`Ctrl + R`**：**反向搜索历史命令**。按下后输入关键词，Bash会动态搜索并显示匹配的历史命令。多次按 `Ctrl + R` 可以向前翻找更早的匹配项。找到后按回车即可执行，或按左右方向键进行编辑。

![](img/af1cd5d4869ccb298a777342475f083e_59.png)

![](img/af1cd5d4869ccb298a777342475f083e_61.png)

**练习建议**：尝试在命令行中逐一使用上述快捷键，感受它们带来的便利。尤其是 `Ctrl + R`（历史搜索）、`Ctrl + A/E`（光标移动）和 `Ctrl + U/K`（快速删除），它们是最常用的效率工具。

![](img/af1cd5d4869ccb298a777342475f083e_63.png)

![](img/af1cd5d4869ccb298a777342475f083e_65.png)

## 课程总结 🎯

本节课中我们一起学习了Bash Shell的四个高效功能：
1.  **Tab键补全**：用于自动补全命令、选项和参数，减少输入量和错误。
2.  **命令换行符 `\`**：用于将长命令分成多行编写，提高可读性。
3.  **历史命令管理 `history`**：用于查看、重复执行和管理命令历史记录，是追溯操作和快速重用命令的利器。
4.  **Bash快捷键**：一系列键盘组合，用于快速清屏、移动光标、编辑命令和搜索历史，能极大提升命令行操作速度。

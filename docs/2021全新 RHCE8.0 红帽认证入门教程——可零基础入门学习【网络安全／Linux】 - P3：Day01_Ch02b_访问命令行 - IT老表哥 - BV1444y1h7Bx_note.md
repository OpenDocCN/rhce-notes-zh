## 系统登录与终端类型 🖥️

![](img/2c7504a202f0688327900cc74ab029e3_5.png)

![](img/2c7504a202f0688327900cc74ab029e3_6.png)

对于个人电脑，我们可以直接连接键盘、鼠标和显示器进行操作。但对于服务器，除了直接连接外，通常还会使用远程控制口进行管理。无论是直接连接还是远程连接，我们都将其视为一个“控制台”。

![](img/2c7504a202f0688327900cc74ab029e3_8.png)

![](img/2c7504a202f0688327900cc74ab029e3_10.png)

在 Linux 系统中，主要有两种用户界面：
*   **图形界面 (GUI)**：提供可视化的桌面环境。
*   **文本界面 (TTY)**：纯字符命令行界面。

![](img/2c7504a202f0688327900cc74ab029e3_12.png)

![](img/2c7504a202f0688327900cc74ab029e3_14.png)

在文本界面中，`TTY` 代表直接连接到系统的物理或虚拟终端。我们可以通过快捷键在多个文本终端之间切换。

![](img/2c7504a202f0688327900cc74ab029e3_16.png)

### 终端切换方法

![](img/2c7504a202f0688327900cc74ab029e3_18.png)

以下是切换终端的方法：

![](img/2c7504a202f0688327900cc74ab029e3_20.png)

![](img/2c7504a202f0688327900cc74ab029e3_22.png)

*   **从图形界面切换到文本终端**：使用 `Ctrl + Alt + F2` 到 `F6`。
*   **从文本终端切换回图形界面**：使用 `Ctrl + Alt + F1`。
*   **在纯文本环境下的终端切换**：使用 `Alt + F1` 到 `F6`（共6个虚拟终端）。

![](img/2c7504a202f0688327900cc74ab029e3_24.png)

> **注意**：某些笔记本电脑可能需要配合 `Fn` 键来使用 `F1`-`F12` 功能键。

![](img/2c7504a202f0688327900cc74ab029e3_26.png)

### 设置默认启动目标

Linux 系统使用 `systemctl` 管理启动目标（相当于旧版本的运行级别）。我们可以查看和修改默认的启动界面。

![](img/2c7504a202f0688327900cc74ab029e3_28.png)

![](img/2c7504a202f0688327900cc74ab029e3_29.png)

*   **查看当前默认启动目标**：
    ```bash
    systemctl get-default
    ```
    此命令会显示当前系统默认启动的目标，例如 `graphical.target`（图形界面）或 `multi-user.target`（多用户文本界面）。

![](img/2c7504a202f0688327900cc74ab029e3_31.png)

*   **永久切换到文本界面**：
    ```bash
    sudo systemctl set-default multi-user.target
    ```
    执行此命令并重启后，系统将默认进入文本界面。

![](img/2c7504a202f0688327900cc74ab029e3_32.png)

![](img/2c7504a202f0688327900cc74ab029e3_34.png)

![](img/2c7504a202f0688327900cc74ab029e3_36.png)

*   **永久切换回图形界面**：
    ```bash
    sudo systemctl set-default graphical.target
    ```

![](img/2c7504a202f0688327900cc74ab029e3_38.png)

![](img/2c7504a202f0688327900cc74ab029e3_40.png)

*   **临时切换界面（无需重启）**：
    ```bash
    # 从当前界面临时切换到图形界面
    sudo systemctl isolate graphical.target
    # 从当前界面临时切换到多用户文本界面
    sudo systemctl isolate multi-user.target
    ```

![](img/2c7504a202f0688327900cc74ab029e3_42.png)

![](img/2c7504a202f0688327900cc74ab029e3_44.png)

## 远程登录与注销 🔗

![](img/2c7504a202f0688327900cc74ab029e3_52.png)

在实际工作场景中，管理员经常需要从一台计算机远程连接到另一台 Linux 服务器进行操作。

*   **从 Windows 系统连接**：Windows 系统通常不自带 SSH 客户端，需要使用第三方工具，如 Xshell、PuTTY、SecureCRT 等。
*   **从 Linux 或 macOS 系统连接**：系统终端直接支持 SSH 命令。

![](img/2c7504a202f0688327900cc74ab029e3_54.png)

远程连接的通用命令格式如下：
```bash
ssh 用户名@服务器IP地址或主机名
```
例如：`ssh root@192.168.1.100`。执行后会提示输入对应用户的密码。

连接成功后，可以通过以下命令查看当前的 SSH 连接信息：
```bash
sudo lsof -i :22
```

![](img/2c7504a202f0688327900cc74ab029e3_56.png)

![](img/2c7504a202f0688327900cc74ab029e3_58.png)

![](img/2c7504a202f0688327900cc74ab029e3_59.png)

要结束远程会话并注销，有两种常用方法：
1.  输入命令：`exit`
2.  使用快捷键：`Ctrl + D`

![](img/2c7504a202f0688327900cc74ab029e3_61.png)

![](img/2c7504a202f0688327900cc74ab029e3_63.png)

## 执行命令行指令 ⚙️

![](img/2c7504a202f0688327900cc74ab029e3_68.png)

![](img/2c7504a202f0688327900cc74ab029e3_70.png)

在命令行中，输入命令后按 `Enter` 键即可执行。例如，`pwd` 命令用于显示当前工作目录。

### 命令组合执行

有时我们需要依次执行多个命令，或者根据前一个命令的执行结果来决定是否执行下一个命令。

以下是命令组合的几种方式：

*   **顺序执行（分号 `;`）**：无论前一个命令是否成功，都会继续执行下一个命令。
    ```bash
    cd /tmp ; ls
    ```

*   **逻辑“或”执行（双竖线 `||`）**：只有当前一个命令**执行失败**时，才会执行下一个命令。
    ```bash
    cd /nonexistent_dir || echo “目录不存在”
    ```

*   **逻辑“与”执行（双和号 `&&`）**：只有当前一个命令**执行成功**时，才会执行下一个命令。
    ```bash
    cd /tmp && ls -l
    ```

### 长命令分行输入

![](img/2c7504a202f0688327900cc74ab029e3_72.png)

当命令非常长时，可以使用反斜杠 `\` 将命令拆分成多行输入，以提高可读性。
```bash
tail -n 1 \
/usr/share/doc/kernel-doc-*/README \
/usr/share/doc/gnu-*/README
```
输入反斜杠后按 `Enter`，光标会跳到下一行，表示命令继续，直到输入完成并按下 `Enter` 键执行。

![](img/2c7504a202f0688327900cc74ab029e3_74.png)

## 命令行快捷键与技巧 🚀

![](img/2c7504a202f0688327900cc74ab029e3_78.png)

熟练使用快捷键可以极大提升命令行操作效率。

### 命令补全 (Tab)

![](img/2c7504a202f0688327900cc74ab029e3_80.png)

`Tab` 键是命令行中最常用的补全工具。

![](img/2c7504a202f0688327900cc74ab029e3_82.png)

*   **命令补全**：输入命令的前几个字母，按一次 `Tab`。如果该前缀能唯一确定一个命令，系统会自动补全；如果存在多个可能，按两次 `Tab` 会列出所有匹配的命令。
*   **选项/参数补全**：输入命令后，在输入选项（如 `-` 或 `--` 开头）或文件路径时，也可以使用 `Tab` 键进行补全。

### 编辑快捷键

![](img/2c7504a202f0688327900cc74ab029e3_84.png)

![](img/2c7504a202f0688327900cc74ab029e3_85.png)

以下快捷键能帮助你在命令行中快速移动光标和编辑文本：

*   `Ctrl + A`：将光标移动到行首。
*   `Ctrl + E`：将光标移动到行尾。
*   `Ctrl + U`：删除从光标处到行首的所有内容。
*   `Ctrl + K`：删除从光标处到行尾的所有内容。
*   `Ctrl + ←` / `Ctrl + →`：以单词为单位向前或向后移动光标。

![](img/2c7504a202f0688327900cc74ab029e3_87.png)

### 历史命令搜索与调用

![](img/2c7504a202f0688327900cc74ab029e3_89.png)

![](img/2c7504a202f0688327900cc74ab029e3_91.png)

系统会记录你执行过的命令（默认保存最近的1000条），方便重复调用。

![](img/2c7504a202f0688327900cc74ab029e3_93.png)

![](img/2c7504a202f0688327900cc74ab029e3_95.png)

*   **查看历史命令**：
    ```bash
    history
    ```

![](img/2c7504a202f0688327900cc74ab029e3_97.png)

![](img/2c7504a202f0688327900cc74ab029e3_99.png)

*   **执行历史命令**：使用 `!` 加历史命令的编号。
    ```bash
    !93
    ```
    这将执行历史记录中编号为 93 的命令。

*   **搜索历史命令**：按 `Ctrl + R`，然后输入关键词，可以反向搜索历史命令。连续按 `Ctrl + R` 可以在多个匹配结果间切换。

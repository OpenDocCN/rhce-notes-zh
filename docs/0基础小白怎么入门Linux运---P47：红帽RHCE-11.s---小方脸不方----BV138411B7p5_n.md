## 字符串截取 ✂️

![](img/18945c10979289f33f661bb87874c5a7_5.png)

字符串截取是指从一个字符串中提取指定部分。在Shell中，我们可以使用特定的语法来实现这一功能。

![](img/18945c10979289f33f661bb87874c5a7_7.png)

以下是字符串截取的基本语法：
```bash
${变量名:起始位置:长度}
```
例如，我们有一个变量 `phone="13812345678"`，想要截取后四位（即“5678”）。我们需要从第7位开始（索引从0开始计算），截取4位。
```bash
phone="13812345678"
echo ${phone:7:4}
```
执行上述命令，将输出“5678”。这种方法不仅适用于数字，也适用于文件内容等其他字符串。

## 字符串替换 🔄

![](img/18945c10979289f33f661bb87874c5a7_13.png)

![](img/18945c10979289f33f661bb87874c5a7_15.png)

字符串替换是指将字符串中的指定部分替换为新的内容。其语法与Vim编辑器中的替换命令类似。

![](img/18945c10979289f33f661bb87874c5a7_17.png)

以下是字符串替换的基本语法：
```bash
${变量名/旧字符串/新字符串}
```
例如，我们有一个变量 `str="12345678"`，想要将中间的“1234”替换为“****”。
```bash
str="12345678"
echo ${str/1234/****}
```
执行上述命令，将输出“****5678”。需要注意的是，如果字符串中有多个匹配项，默认只替换第一个。若要替换全部，需使用双斜线：
```bash
echo ${str//3/*}
```

## 字符串掐头去尾 ✂️

![](img/18945c10979289f33f661bb87874c5a7_21.png)

![](img/18945c10979289f33f661bb87874c5a7_23.png)

字符串掐头去尾是指删除字符串开头或结尾的指定部分。这在批量修改文件扩展名等场景中非常有用。

![](img/18945c10979289f33f661bb87874c5a7_25.png)

以下是字符串掐头去尾的基本语法：
- **掐头（从左删除）**：`${变量名#关键词}` 或 `${变量名##关键词}`（最长匹配）
- **去尾（从右删除）**：`${变量名%关键词}` 或 `${变量名%%关键词}`（最长匹配）

例如，我们有一个变量 `file="test.txt"`，想要删除其扩展名“.txt”。
```bash
file="test.txt"
echo ${file%.txt}
```
执行上述命令，将输出“test”。这种方法可以用于批量修改文件扩展名。

## 实践：生成随机密码 🔐

在实际工作中，我们可能需要生成随机密码。虽然我们可以自己编写脚本，但更高效的方法是借鉴他人的代码。

以下是一个生成随机8位密码的脚本示例：
```bash
#!/bin/bash
# 定义包含大小写字母和数字的字符串
chars="abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789"
password=""
# 循环8次，每次随机选取一个字符
for i in {1..8}; do
    rand=$((RANDOM % 62))
    char=${chars:$rand:1}
    password="${password}${char}"
done
echo $password
```
执行上述脚本，每次都会生成一个随机的8位密码。这种方法利用了字符串截取和随机数生成，实现了密码的随机性。

## 实践：批量修改文件扩展名 📁

有时我们需要批量修改文件的扩展名。虽然可以手动操作，但使用脚本可以大大提高效率。

以下是一个批量将`.txt`文件扩展名修改为`.doc`的脚本示例：
```bash
#!/bin/bash
for file in *.txt; do
    mv "$file" "${file%.txt}.doc"
done
```
执行上述脚本，当前目录下所有`.txt`文件的扩展名都将被修改为`.doc`。这种方法利用了字符串去尾和重命名命令，实现了批量修改。

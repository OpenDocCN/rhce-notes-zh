## 概述
在本节课中，我们将学习Vim编辑器的基础知识。Vim是Linux系统中一个功能强大的命令行文本编辑器，尤其适用于在无图形界面的服务器环境中修改配置文件。掌握Vim是系统管理员和开发人员的一项核心技能。

## 为什么需要学习Vim？
在生产环境的服务器中，配置文件通常以文本形式存在。由于服务器通常没有图形界面，直接编辑这些文件需要命令行工具。虽然可以通过下载、编辑再上传的方式修改文件，但这可能导致编码问题（如乱码）且效率低下，特别是对于微小的修改。因此，学习一个在命令行下高效编辑文件的工具是必要的。

## Vim简介
Linux系统中有多种文本编辑工具，如图形界面的`gedit`，命令行的`nano`、`emacs`和`sed`等。我们推荐学习Vim，因为它功能强大、学习曲线相对平缓，且几乎所有Linux发行版都默认安装。

Vim有三个主要版本：
1.  **Vim-minimal**：提供最基本的`vi`工具，功能较少。
2.  **Vim-enhanced**：增强版，提供`vim`命令，支持语法高亮、文件类型检测、拼写检查和插件扩展。
3.  **gVim**：提供图形界面的Vim，适合初学者熟悉操作。

通常我们安装和使用的是`vim-enhanced`版本。

## 安装与启动Vim
在RHEL/CentOS系统中，可以使用`yum`命令安装Vim。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_20.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_22.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_24.png)

以下是安装增强版和图形界面版的命令：
```bash
yum install -y vim-enhanced vim-X11
```
安装完成后，可以通过以下命令启动Vim并编辑文件：
```bash
vim 文件名
```
例如，编辑`/etc/hosts`文件：
```bash
vim /etc/hosts
```
注意：系统中`vi`命令可能是`vim`的别名，执行`alias`命令可以查看。

## Vim的四种模式 🎛️
Vim的操作围绕四种模式展开，理解这些模式是熟练使用Vim的关键。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_31.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_33.png)

### 1. 命令模式 (Normal Mode)
启动Vim后，默认进入命令模式。在此模式下，按键被解释为命令，而不是输入字符。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_35.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_37.png)

以下是命令模式下的一些基本操作：
*   **移动光标**：使用方向键或 `h`(左)、`j`(下)、`k`(上)、`l`(右)。
*   **单词移动**：按 `w` 移动到下一个单词开头。
*   **跳转行首/行尾**：按 `0` 跳转到行首，按 `$` 跳转到行尾。
*   **跳转文件首尾**：按 `gg` 跳转到文件第一行，按 `G` 跳转到文件最后一行。

### 2. 插入模式 (Insert Mode)
在插入模式下，可以像使用普通文本编辑器一样输入和删除文本。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_39.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_41.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_43.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_45.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_47.png)

从命令模式进入插入模式的方法：
*   **`i`**：在光标前插入。
*   **`a`**：在光标后插入。
*   **`I`**：在当前行首插入。
*   **`A`**：在当前行尾插入。
*   **`o`**：在当前行下方新建一行并插入。
*   **`O`**：在当前行上方新建一行并插入。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_49.png)

按 `Esc` 键可以从插入模式返回命令模式。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_51.png)

### 3. 可视模式 (Visual Mode)
可视模式用于选择文本块，以便进行复制、删除等操作。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_53.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_54.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_56.png)

从命令模式进入可视模式的方法：
*   **`v`**：字符可视模式，按字符选择文本。
*   **`V`**：行可视模式，按行选择文本。
*   **`Ctrl+v`**：块可视模式，按矩形块选择文本。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_58.png)

选择文本后，可以执行命令，例如按 `d` 删除选中内容，操作完成后自动回到命令模式。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_60.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_62.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_64.png)

### 4. 命令行模式 (Command-line Mode)
在命令模式下按 `:` 进入命令行模式，可以执行保存、退出、搜索等命令。

以下是常用的命令行模式命令：
*   **`:w`**：保存文件。
*   **`:q`**：退出Vim（如果文件有修改且未保存，会提示错误）。
*   **`:q!`**：强制退出，不保存修改。
*   **`:wq`** 或 **`:x`**：保存并退出。
*   **`:`!命令`**：执行外部Shell命令，例如 `:!ls` 列出当前目录文件。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_66.png)

## 基础编辑操作示例 ✍️
上一节我们介绍了Vim的四种模式，本节中我们通过具体例子来看看如何进行基础编辑。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_68.png)

### 示例1：插入、删除与撤销
1.  用Vim打开一个文件。
2.  按 `i` 进入插入模式，输入文本。
3.  按 `Esc` 返回命令模式。
4.  将光标移动到要删除的字符上，按 `x` 删除该字符。
5.  如果误操作，在命令模式下按 `u` 撤销上一次操作，按 `Ctrl+r` 重做被撤销的操作。
6.  输入 `:wq` 保存并退出。

### 示例2：复制与粘贴
1.  在命令模式下，将光标移动到要复制的文本起始处。
2.  按 `v` 进入可视模式，移动光标选择文本。
3.  按 `y` 复制选中的文本。
4.  移动光标到想要粘贴的位置。
5.  按 `p` 将复制的文本粘贴到光标后。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_70.png)

![](img/9752c964ff3d3c40736ea0b4d50c5c49_72.png)

**进阶**：在命令模式下，`yy` 复制当前行，`4yy` 复制从当前行开始的4行，然后使用 `p` 粘贴。

## 获取帮助与练习 📚
Vim功能繁多，无需一次性记住所有命令。内置的教程是很好的学习资源。

![](img/9752c964ff3d3c40736ea0b4d50c5c49_74.png)

在终端中输入以下命令可以启动Vim教程：
```bash
vimtutor
```
这个交互式教程涵盖了光标移动、文本编辑、保存退出等基本操作，非常适合初学者循序渐进地学习。

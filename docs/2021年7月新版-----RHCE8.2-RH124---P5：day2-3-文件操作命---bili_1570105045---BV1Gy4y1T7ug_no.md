## 复制文件和目录

![](img/fac55e6ea3da74e9a8237d85cfae2100_3.png)

上一节我们介绍了文件元数据，本节中我们来看看如何使用`cp`命令复制文件和目录。`cp`命令可以将文件或目录复制到指定位置，并可以选择是否保留原文件的元数据。

![](img/fac55e6ea3da74e9a8237d85cfae2100_5.png)

**基本复制命令格式**：
```bash
cp [选项] 源文件 目标路径
```

![](img/fac55e6ea3da74e9a8237d85cfae2100_7.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_9.png)

以下是`cp`命令的常用操作示例：

![](img/fac55e6ea3da74e9a8237d85cfae2100_11.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_13.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_15.png)

*   **简单复制**：`cp /data/hello.sh /home/` 将文件复制到目标目录，保持原名。
*   **复制并重命名**：`cp /data/hello.sh /home/test.sh` 将文件复制到目标目录并重命名。
*   **保留元数据复制**：`cp -a /data/hello.sh /home/hello1.sh` 使用 `-a` 选项可以保留文件的所有元数据（如时间戳、权限等）。`-a` 选项已包含递归操作 `-r`。
*   **递归复制目录**：`cp -r /etc/selinux /home/` 使用 `-r` 或 `-R` 选项可以递归复制整个目录及其内容。

![](img/fac55e6ea3da74e9a8237d85cfae2100_17.png)

**重要提示**：默认的`cp`命令会创建具有新元数据的新文件。使用 `-a` 选项可以完整复制源文件的元数据。在复制目录时，如果目标目录已存在，`cp`命令的行为是将源目录放入目标目录内，而非直接覆盖。

![](img/fac55e6ea3da74e9a8237d85cfae2100_19.png)

## 移动和重命名文件

![](img/fac55e6ea3da74e9a8237d85cfae2100_21.png)

在Linux中，移动和重命名文件都使用 `mv` 命令。这是因为移动操作的本质是将文件从一个路径“移动”到另一个路径，如果目标路径的文件名不同，就实现了重命名。

![](img/fac55e6ea3da74e9a8237d85cfae2100_23.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_25.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_27.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_29.png)

**基本移动/重命名命令格式**：
```bash
mv [选项] 源文件 目标路径
```

![](img/fac55e6ea3da74e9a8237d85cfae2100_31.png)

以下是`mv`命令的常用操作示例：

![](img/fac55e6ea3da74e9a8237d85cfae2100_33.png)

*   **移动文件**：`mv /data/passwd /home/test/` 将文件移动到目标目录。
*   **重命名文件**：`mv passwd new_passwd` 在当前目录下将文件重命名。
*   **交互式移动**：`mv -i file1 file2` 使用 `-i` 选项会在覆盖文件前进行确认提示。
*   **强制移动**：`mv -f file1 file2` 使用 `-f` 选项会强制覆盖，不进行提示。
*   **备份后移动**：`mv -b file1 file2` 使用 `-b` 选项会在覆盖目标文件前对其进行备份（通常添加`~`后缀）。

**核心概念**：`mv`命令操作的是文件的路径索引（inode），通常不会改变文件的原始数据块，因此执行速度很快。但跨文件系统移动时，其行为等同于“复制+删除”。

![](img/fac55e6ea3da74e9a8237d85cfae2100_35.png)

## 删除文件和目录

![](img/fac55e6ea3da74e9a8237d85cfae2100_37.png)

删除操作使用 `rm` 命令。这是一个需要谨慎使用的危险命令，因为删除的文件通常难以恢复。

**基本删除命令格式**：
```bash
rm [选项] 文件或目录
```

以下是`rm`命令的常用操作示例：

![](img/fac55e6ea3da74e9a8237d85cfae2100_39.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_41.png)

*   **删除文件**：`rm file.txt` 删除单个文件。系统别名通常已设置为 `rm -i` 以进行提示。
*   **交互式删除**：`rm -i file.txt` 删除前确认。
*   **强制删除**：`rm -f file.txt` 强制删除，忽略不存在的文件和不提示。
*   **递归删除目录**：`rm -rf directory/` **（慎用！）** 递归强制删除目录及其下所有内容。

![](img/fac55e6ea3da74e9a8237d85cfae2100_43.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_45.png)

**安全建议**：生产环境中，建议使用`mv`命令将文件移动到临时回收站目录来代替直接`rm`删除，或确保有完善的备份机制。`rm`命令并非真正擦除磁盘数据，只是标记inode为可重用，在数据被覆盖前，仍有专业工具恢复的可能。

## 目录树与目录操作

### 显示目录树结构

`tree`命令可以以树状图形式列出目录的内容结构，直观清晰。

![](img/fac55e6ea3da74e9a8237d85cfae2100_47.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_49.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_51.png)

**基本命令格式**：
```bash
tree [选项] [目录]
```

以下是`tree`命令的常用操作示例：

![](img/fac55e6ea3da74e9a8237d85cfae2100_53.png)

*   **显示当前目录树**：`tree`
*   **显示指定目录树**：`tree /home/test`
*   **只显示目录**：`tree -d` 使用 `-d` 选项仅列出目录。
*   **限制显示层级**：`tree -L 2` 使用 `-L` 选项指定要显示的目录深度。

### 创建目录

![](img/fac55e6ea3da74e9a8237d85cfae2100_55.png)

创建目录使用 `mkdir` 命令。

**基本命令格式**：
```bash
mkdir [选项] 目录名
```

![](img/fac55e6ea3da74e9a8237d85cfae2100_57.png)

以下是`mkdir`命令的常用操作示例：

*   **创建单个目录**：`mkdir new_dir`
*   **递归创建多级目录**：`mkdir -p a/b/c/d` 使用 `-p` 选项可以自动创建路径中不存在的父目录。
*   **显示创建过程**：`mkdir -pv a/b/c` 使用 `-v` 选项显示详细的创建信息。

![](img/fac55e6ea3da74e9a8237d85cfae2100_59.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_61.png)

### 删除空目录

![](img/fac55e6ea3da74e9a8237d85cfae2100_67.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_69.png)

删除空目录使用 `rmdir` 命令。该命令只能删除**空目录**。

![](img/fac55e6ea3da74e9a8237d85cfae2100_71.png)

**基本命令格式**：
```bash
rmdir [选项] 目录名
```

![](img/fac55e6ea3da74e9a8237d85cfae2100_73.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_75.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_76.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_78.png)

以下是`rmdir`命令的常用操作示例：

![](img/fac55e6ea3da74e9a8237d85cfae2100_80.png)

![](img/fac55e6ea3da74e9a8237d85cfae2100_82.png)

*   **删除空目录**：`rmdir empty_dir`
*   **递归删除空目录**：`rmdir -p a/b/c` 使用 `-p` 选项会尝试删除指定路径上的所有空目录。例如，如果 `a/b/c`、`a/b`、`a` 都为空，则会被依次删除。

![](img/fac55e6ea3da74e9a8237d85cfae2100_84.png)

**注意**：如果目录非空，`rmdir` 命令会报错。要删除非空目录，必须使用 `rm -r` 命令。

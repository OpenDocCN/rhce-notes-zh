## 什么是文本编辑器？🤔
文本编辑器是用于创建和修改纯文本文件的程序。在Windows系统中，我们常用的“记事本”就是一个文本编辑器。它可以创建文件，并允许我们对文件内容进行增、删、改、查等操作。

![](img/3bfbffdb361908f27e29247c96bad071_15.png)

![](img/3bfbffdb361908f27e29247c96bad071_17.png)

![](img/3bfbffdb361908f27e29247c96bad071_19.png)

在Linux系统中，最著名的文本编辑器是`vi`和`vim`。`vim`是`vi`的增强版本，主要区别在于`vim`支持语法高亮，更适合编程和脚本编写，而`vi`则没有颜色提示功能。对于日常修改配置文件，两者都可以使用。

## vim的基本使用 🚀
### 命令格式
使用vim打开或编辑文件的基本命令格式非常简单：
```bash
vim 文件名
```
*   若文件**不存在**，vim会创建该新文件并进入编辑状态。
*   若文件**已存在**，vim会直接打开该文件并进入编辑状态。

### vim的三种模式
vim的操作主要围绕三种模式进行，理解这三种模式是熟练使用vim的关键。

![](img/3bfbffdb361908f27e29247c96bad071_25.png)

![](img/3bfbffdb361908f27e29247c96bad071_27.png)

![](img/3bfbffdb361908f27e29247c96bad071_29.png)

![](img/3bfbffdb361908f27e29247c96bad071_31.png)

1.  **命令模式**：启动vim后默认进入的模式。在此模式下，键盘输入会被识别为各种编辑命令，而不是文本内容。
2.  **输入模式**：在此模式下，可以向文件中输入或编辑文本内容。
3.  **底线命令模式**：在此模式下，可以执行保存、退出、搜索、替换等高级操作。

## 模式切换与基础编辑 ✍️
上一节我们介绍了vim的三种模式，本节中我们来看看如何在它们之间切换并进行最基础的文本编辑。

### 从命令模式进入输入模式
在命令模式下，有多个按键可以切换到输入模式，最常用的是 `i`。
*   `i`：在**当前光标所在字符前**进入输入模式。
*   `a`：在**当前光标所在字符后**进入输入模式。
*   `o`：在**当前光标所在行的下方**插入一个新行，并进入输入模式。

**提示**：`i` 和 `a` 的区别仅在于光标插入位置，通过移动光标可以轻松弥补，因此记住其中一个即可。`o` 用于快速换行输入。

进入输入模式后，底部会显示 `-- INSERT --` 标识。此时可以像使用普通编辑器一样输入文本，使用方向键移动光标，按回车键换行。

![](img/3bfbffdb361908f27e29247c96bad071_39.png)

### 从输入模式返回命令模式
在输入模式下，按键盘左上角的 `Esc` 键，即可退出输入模式，返回命令模式。此时底部的 `-- INSERT --` 标识会消失。

![](img/3bfbffdb361908f27e29247c96bad071_41.png)

![](img/3bfbffdb361908f27e29247c96bad071_43.png)

### 到底线命令模式并保存退出
编辑完成后，我们通常需要保存文件并退出vim。

1.  首先，按 `Esc` 确保回到**命令模式**。
2.  然后，输入英文冒号 `:`，进入**底线命令模式**，光标会跳转到窗口底部。
3.  在底线命令模式下，输入以下命令之一并回车：
    *   `w`：保存文件。
    *   `q`：退出vim。
    *   `wq` 或 `x`：保存文件并退出vim。
    *   `q!`：**强制退出**，不保存任何修改。
    *   `wq!`：**强制保存并退出**，常用于修改只读文件（仅root用户可用）。
4.  也可以直接在命令模式下按 `Shift + zz`（连续按两次大写的Z），快速保存并退出。

**核心流程总结**：编辑完成后，`Esc` -> `:` -> `wq` -> `Enter`。

## 命令模式下的高效操作 ⚡

以下是命令模式下常用的操作指令列表：

![](img/3bfbffdb361908f27e29247c96bad071_21.png)

### 光标移动
*   `0`（数字零）：将光标快速移动到**当前行的行首**。
*   `$`：将光标快速移动到**当前行的行尾**。
*   `gg`：将光标快速移动到**文件的第一行**。
*   `G`：将光标快速移动到**文件的最后一行**。
*   `:数字`：在底线命令模式下，输入行号可直接跳转到指定行，例如 `:100` 跳转到第100行。

### 删除、剪切与复制
*   `x`：删除**当前光标所在位置**的一个字符。
*   `dd`：剪切（删除）**当前光标所在的一整行**。如果之后不粘贴，效果等同于删除。
    *   `数字dd`：剪切从当前行开始的指定数字行，例如 `3dd` 剪切当前行及下面两行。
*   `p`：将剪切或复制的内容**粘贴**到当前光标所在行的**下方**。
*   `yy`：复制**当前光标所在的一整行**。
    *   `数字yy`：复制从当前行开始的指定数字行，例如 `2yy` 复制当前行及下一行。
*   `C`（大写）：删除从当前光标到**行尾**的所有内容，并**立即进入输入模式**。

### 撤销与恢复
*   `u`：撤销上一次的修改操作。可以连续按 `u` 来撤销更早的操作，直到文件打开时的初始状态。
*   `Ctrl + r`：恢复被撤销的操作（即“重做”）。如果你按 `u` 撤销多了，可以用此命令恢复回来。

### 文本搜索
*   `/关键字`：在文件中从当前光标位置**向下**搜索指定的关键字。按 `n` 跳转到下一个匹配项，按 `N` 跳转到上一个匹配项。
*   `?关键字`：在文件中从当前光标位置**向上**搜索指定的关键字。

**注意**：`vim` 会高亮显示所有搜索匹配项，而 `vi` 可能不会。

## 底线命令模式进阶技巧 🛠️
除了保存退出，底线命令模式还提供了更多强大功能来管理编辑会话和批量处理文本。

以下是底线命令模式下常用的操作指令列表：

![](img/3bfbffdb361908f27e29247c96bad071_57.png)

![](img/3bfbffdb361908f27e29247c96bad071_59.png)

![](img/3bfbffdb361908f27e29247c96bad071_25.png)

### 显示与取消行号
*   `:set nu`：在编辑器中显示**行号**。
*   `:set nonu`：取消行号显示。

### 替换文件内容
替换命令是批量修改文本的利器，格式为：`:范围 s/旧内容/新内容/选项`。
*   `:%s/old/new/g`：
    *   `%` 代表整个文件范围。
    *   `s` 表示替换操作。
    *   `old` 是要被替换的旧文本。
    *   `new` 是替换后的新文本。
    *   `g` 表示全局替换（一行内所有匹配项都替换），如果没有 `g`，则只替换每行第一个匹配项。
*   **示例**：`:1,50s/foo/bar/g` 将第1行到第50行中所有的 `foo` 替换为 `bar`。

### 取消搜索高亮
执行搜索后，匹配项会高亮显示。如需取消，可以输入：
*   `:noh`：取消当前的高亮显示。

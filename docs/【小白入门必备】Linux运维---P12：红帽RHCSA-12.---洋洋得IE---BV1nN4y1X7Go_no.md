## 什么是文本编辑器？🤔

![](img/61f9f03bccf6fefc870a995daf360a77_19.png)

![](img/61f9f03bccf6fefc870a995daf360a77_21.png)

![](img/61f9f03bccf6fefc870a995daf360a77_23.png)

在Windows系统中，当你新建一个文本文档并打开它进行编辑时，所使用的工具（如“记事本”）就是一个文本编辑器。它的功能是创建文件，并对文件内容进行增、删、改、查。

在Linux系统中，最常用的文本编辑器是 `vi` 和 `vim`。`vim` 是 `vi` 的增强版本，主要区别在于 `vim` 支持语法高亮，更适合编程（例如编写Shell脚本），而 `vi` 则没有颜色提示。对于日常修改配置文件，两者都可以使用。

## vim的基本使用 🚀

![](img/61f9f03bccf6fefc870a995daf360a77_33.png)

![](img/61f9f03bccf6fefc870a995daf360a77_35.png)

![](img/61f9f03bccf6fefc870a995daf360a77_37.png)

vim的命令格式非常简单：
```bash
vim 文件名
```
它的特点是：
*   若文件不存在，则创建新文件并进入编辑状态。
*   若文件已存在，则直接打开该文件进行编辑。

例如，编辑一个已存在的系统文件（如主机名文件）：
```bash
vim /etc/hostname
```
编辑一个不存在的文件（vim会先创建它）：
```bash
vim /opt/xxxx.tt
```

## vim的三种模式 🔄

vim的操作分为三种模式，这是初学者需要理解的核心概念。

上一节我们介绍了vim的基本命令格式，本节中我们来看看它的三种工作模式。

![](img/61f9f03bccf6fefc870a995daf360a77_39.png)

![](img/61f9f03bccf6fefc870a995daf360a77_41.png)

![](img/61f9f03bccf6fefc870a995daf360a77_43.png)

### 1. 命令模式 (Command Mode)
首次使用 `vim` 打开文件时，默认进入的就是命令模式。在此模式下，键盘输入会被识别为命令，而不是文本内容。命令模式是执行复制、删除、粘贴、搜索等操作的起点。

### 2. 输入模式 (Insert Mode)
在命令模式下，按下特定键（如 `i`, `a`, `o`）即可进入输入模式。此时，你可以像使用普通记事本一样，自由地向文件中输入或修改文本内容。按 `Esc` 键可以退出输入模式，返回命令模式。

### 3. 底线命令模式 (Last Line Mode)
在命令模式下，按下英文冒号 `:` 即可进入底线命令模式。此时光标会移动到屏幕底部，可以输入保存文件、退出编辑器、搜索替换等高级命令。

## 模式切换与核心操作 🛠️

![](img/61f9f03bccf6fefc870a995daf360a77_51.png)

![](img/61f9f03bccf6fefc870a995daf360a77_53.png)

理解了三种模式后，我们来学习如何在它们之间切换，并掌握最常用的操作命令。

### 从命令模式进入输入模式
在命令模式下，有多个键可以进入输入模式，区别在于光标插入的位置：
*   **`i`**：在**当前光标所在字符前**插入。
*   **`a`**：在**当前光标所在字符后**插入。
*   **`o`**：在**当前光标所在行的下方**插入一个新行。

对于初学者，记住 `i` 键即可，通过移动光标可以灵活定位。

### 保存与退出（底线命令模式操作）
编辑完成后，需要先按 `Esc` 键确保回到命令模式，然后按 `:` 进入底线命令模式，才能执行保存或退出。

以下是底线命令模式下常用的保存与退出命令：
*   **`:w`**：保存文件。
*   **`:q`**：退出vim（如果文件有修改未保存，会提示错误）。
*   **`:wq`** 或 **`:x`**：保存文件并退出。
*   **`ZZ`**（**命令模式下**，大写Z按两次）：快速保存并退出。
*   **`:q!`**：强制退出，**不保存**任何修改。
*   **`:wq!`**：强制保存并退出（常用于修改只读文件，需要root权限）。

### 命令模式下的常用编辑命令
在命令模式下，可以直接对文本进行高效编辑。

![](img/61f9f03bccf6fefc870a995daf360a77_57.png)

![](img/61f9f03bccf6fefc870a995daf360a77_59.png)

以下是命令模式下的一些核心编辑命令：
*   **移动光标**：
    *   `0`：移动到**行首**。
    *   `$`：移动到**行尾**。
    *   `gg`：移动到**文件第一行**。
    *   `G`：移动到**文件最后一行**。
    *   `:数字`：在底线命令模式下，输入行号可快速跳转到指定行（如 `:1000` 跳转到第1000行）。
*   **删除/剪切**：
    *   `x`：删除**当前光标所在**的一个字符。
    *   `dd`：删除（剪切）**当前光标所在**的整行。
    *   `数字dd`：删除（剪切）从当前行开始的指定**数字**行（如 `3dd` 删除3行）。
    *   `D` 或 `d$`：删除从光标到**行尾**的内容。
    *   `C`：删除从光标到行尾的内容，并**进入输入模式**。
*   **复制/粘贴**：
    *   `yy`：复制**当前光标所在**的整行。
    *   `数字yy`：复制从当前行开始的指定**数字**行。
    *   `p`：将复制或剪切的内容**粘贴**到当前光标所在行的**下方**。
*   **撤销与恢复**：
    *   `u`：撤销上一次操作。
    *   `Ctrl + r`：恢复被撤销的操作（即反撤销）。
*   **搜索**：
    *   `/关键词`：在文件中**向下搜索**指定关键词，按 `n` 跳转到下一个匹配项。
    *   `?关键词`：在文件中**向上搜索**指定关键词。
    *   `:nohl`：在底线命令模式下，**取消**搜索关键词的高亮显示。

### 进阶功能（底线命令模式）
以下是底线命令模式下的一些实用设置与替换功能：
*   **显示行号**：`:set nu`
*   **取消行号**：`:set nonu`
*   **替换文本**：
    *   `:%s/旧文本/新文本/g`：将文件中**所有**匹配的“旧文本”替换为“新文本”。
    *   `:%s/旧文本/新文本/`：将每行中**第一个**匹配的“旧文本”替换为“新文本”。
    *   `:起始行,结束行s/旧文本/新文本/g`：在指定行范围内进行替换。

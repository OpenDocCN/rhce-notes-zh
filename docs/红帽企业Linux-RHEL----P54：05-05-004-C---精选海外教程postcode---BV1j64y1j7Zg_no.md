# 红帽企业Linux RHEL 9精通课程：P54：05-05-004 条件语句

在本节课中，我们将学习如何在Bash脚本中使用条件语句。条件语句是编程的基础，它允许脚本根据不同的情况做出决策。我们将重点介绍`if`条件语句的语法、算术比较运算符、逻辑运算符以及与字符串的比较方法。

## 算术关系运算符

上一节我们介绍了条件语句的基本概念，本节中我们来看看可以与条件一起使用的算术关系运算符。这些运算符主要用于比较数字的大小或相等性。

以下是主要的算术关系运算符及其在脚本中的写法：

*   **-lt**：小于
*   **-gt**：大于
*   **-le**：小于或等于
*   **-ge**：大于或等于
*   **-eq**：等于
*   **-ne**：不等于

## 编写第一个if条件脚本

理解了运算符后，让我们通过一个例子来学习如何编写一个完整的`if`条件脚本。

我们将创建一个名为`if.sh`的脚本文件。

```bash
#!/bin/bash
a=4
b=3
if [ $a -gt $b ]; then
    echo “A比B大”
    echo “这是另一行，也将在条件满足时打印”
fi
```

**代码解释**：
1.  `#!/bin/bash` 指定脚本使用Bash解释器。
2.  `a=4` 和 `b=3` 定义了两个变量。
3.  `if [ $a -gt $b ]; then` 是条件判断的开始。`-gt` 表示“大于”。**注意：方括号 `[ ]` 内的空格是必需的**。
4.  如果条件为真，则执行 `then` 和 `fi` 之间的所有命令。
5.  `fi` 表示 `if` 语句的结束。

保存并运行脚本，你会看到输出“A比B大”。如果将变量`a`的值改为2，则条件不满足，脚本不会有任何输出。

## 使用else和elif分支

如果条件不满足时我们想执行其他操作，可以使用`else`。如果有多个条件需要判断，则可以使用`elif`。

以下是使用`else`和`elif`的示例：

```bash
#!/bin/bash
a=2
b=3
if [ $a -gt $b ]; then
    echo “A大于B”
elif [ $a -lt $b ]; then
    echo “A小于B”
else
    echo “A等于B”
fi
```

**代码解释**：
1.  首先检查 `a` 是否大于 `b`。
2.  如果不满足，则进入 `elif` 检查 `a` 是否小于 `b`。
3.  如果以上两个条件都不满足，则执行 `else` 部分的命令。

运行此脚本，会根据`a`和`b`的值输出相应的结果。

## 逻辑运算符：AND与OR

有时我们需要同时满足多个条件，或者满足多个条件之一。这时就需要用到逻辑运算符`&&`（AND）和`||`（OR）。

以下是使用逻辑运算符的示例脚本：

```bash
#!/bin/bash
echo “请输入第一个名字：”
read name1
echo “请输入第二个名字：”
read name2

if [ “$name1” = “John” ] && [ “$name2” = “John” ]; then
    echo “两个名字都是John”
elif [ “$name1” = “Mark” ] || [ “$name2” = “Mark” ]; then
    echo “其中一个名字是Mark”
else
    echo “未找到匹配条件”
fi
```

**代码解释**：
1.  `read` 命令用于从用户输入中读取值并存入变量。
2.  `[ “$name1” = “John” ] && [ “$name2” = “John” ]` 使用 `&&` 连接两个条件，表示**两个条件必须同时为真**。
3.  `[ “$name1” = “Mark” ] || [ “$name2” = “Mark” ]` 使用 `||` 连接两个条件，表示**两个条件中有一个为真即可**。

## 字符串比较

之前我们使用 `-eq`、`-gt` 等运算符比较数字。对于字符串的比较，我们使用不同的符号。

以下是字符串比较的示例：

```bash
#!/bin/bash
name1=“Ibrahim”
name2=“John”

if [ “$name1” = “$name2” ]; then
    echo “两个名字相同”
else
    echo “两个名字不同”
fi
```

**代码解释**：
1.  字符串相等性比较使用 **`=`** 运算符。
2.  字符串不相等比较使用 **`!=`** 运算符（例如 `[ “$name1” != “$name2” ]`）。

**重要区别**：
*   **数字比较**：使用 `-eq`， `-ne`， `-gt`， `-lt`， `-ge`， `-le`。
*   **字符串比较**：使用 `=` 和 `!=`。

### 命令组合：`pwd`、`cd`、`ls`

这三个命令通常联合使用，用于定位、切换和查看目录内容。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_6.png)

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_8.png)

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_9.png)

*   **`pwd` (Print Working Directory)**：用于显示当前所在的工作目录路径。
    ```bash
    pwd
    ```
*   **`cd` (Change Directory)**：用于切换当前的工作目录。
    ```bash
    cd /目标路径
    ```
*   **`ls` (List)**：用于列出指定目录下的文件和子目录。
    ```bash
    ls
    ```

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_11.png)

以下是`cd`命令的一些常用技巧：

*   `cd` 或 `cd ~`：直接切换到当前用户的**家目录**。
*   `cd -`：切换到**上一次**所在的目录。
*   `cd ~用户名`：切换到指定用户的家目录（如 `cd ~student`）。

通过组合使用这些命令，你可以像在图形界面中一样，在文件树中自由探索：用`pwd`确认位置，用`ls`查看内容，再用`cd`进入感兴趣的目录。

## 理解路径：绝对路径与相对路径

在访问文件时，我们需要用路径来指明文件的位置。路径有两种表示方法：绝对路径和相对路径。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_13.png)

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_15.png)

*   **绝对路径**：从文件系统的根目录（`/`）开始写起的完整路径。每个文件的绝对路径是**唯一**的。
    *   例如：`/home/student/Desktop`
*   **相对路径**：相对于**当前工作目录**的路径。不以斜杠（`/`）开头。
    *   例如：如果当前在`/home/student`，那么`Desktop`就是一个相对路径，指向`/home/student/Desktop`。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_17.png)

路径中还有两个特殊的符号：
*   `.` （一个点）：代表**当前目录**。
*   `..` （两个点）：代表**上一级目录**。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_19.png)

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_21.png)

**为什么需要两种路径？**
两者各有优劣。绝对路径明确唯一，但书写可能较长；相对路径书写简洁，但其有效性依赖于当前目录。在编写脚本或生产环境中，为了可靠性，通常更倾向于使用绝对路径。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_23.png)

## 深入 `ls` 命令

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_28.png)

`ls`命令是查看目录内容的核心工具，它拥有许多选项来定制输出格式和内容。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_30.png)

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_32.png)

以下是`ls`命令的一些常用选项：

*   `ls -a` 或 `ls --all`：显示所有文件，包括以点（`.`）开头的**隐藏文件**。
*   `ls -l`：以**长列表格式**显示详细信息，包括文件类型、权限、所有者、大小和修改时间。
    *   例如：`-rw-r--r--. 1 root root 1234 Dec 5 10:00 file.txt`
*   `ls -R`：**递归**显示目录及其所有子目录的内容。
*   `ls -1`：每行只显示一个文件或目录名（竖排显示）。
*   `ls -r`：**反向**排序显示结果。
*   `ls -t`：按**修改时间**排序，最新的排在最前面。
*   `ls -ltr`：一个常用组合，表示“以长格式、按修改时间反向排序（最新的在最下面）”。

另一个有用的目录树查看命令是 `tree`，它可以直观地显示目录结构：
```bash
tree /home
tree -L 2 /home # 只显示2层深度
```

## 文件时间与 `touch` 命令

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_38.png)

`touch`命令主要有两个用途：1) 更新文件的时间戳；2) 创建空文件。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_40.png)

一个文件通常包含三个时间属性（可用 `stat 文件名` 查看）：
*   **Access Time (atime)**：最后访问时间。
*   **Modify Time (mtime)**：最后**内容**修改时间。
*   **Change Time (ctime)**：最后**状态**修改时间（如权限、所有者变更）。

`touch`命令的基本用法：

*   `touch 文件名`：如果文件**存在**，则将其所有时间戳更新为当前时间；如果文件**不存在**，则创建一个新的空文件。
*   `touch -c 文件名`：仅更新时间戳，如果文件不存在，则**不创建**它。
*   `touch -a 文件名`：只更新文件的访问时间（atime）。
*   `touch -m 文件名`：只更新文件的修改时间（mtime）。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_42.png)

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_43.png)

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_45.png)

需要注意的是，更改 `atime` 或 `mtime` 通常会导致 `ctime` 也被更新为当前时间。

![](img/9e5bf41d1e69a5f07c6435a12f2314c7_47.png)

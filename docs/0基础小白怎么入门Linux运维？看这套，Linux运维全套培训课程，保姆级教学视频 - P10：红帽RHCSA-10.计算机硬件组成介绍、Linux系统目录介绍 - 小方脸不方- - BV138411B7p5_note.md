## 计算机硬件组成介绍 🧩

上一节我们复习了基础命令，本节中我们来看看计算机的硬件组成。这对于理解服务器和后续的系统管理至关重要。

一台计算机主要由输入设备、主机设备、输出设备和外部存储设备组成。

以下是各类硬件设备的简要介绍：

*   **输入设备**：用于将用户的操作指令传递给计算机。例如：鼠标、键盘、触控屏。
*   **主机设备**：位于计算机机箱内部的核心部件。例如：
    *   **主板**：连接所有硬件的核心电路板。
    *   **CPU（中央处理器）**：负责数据计算和处理的核心部件。
    *   **内存**：用于临时存储运行中的程序和数据。
    *   **网卡、声卡、显卡**：分别负责网络通信、音频处理和图形显示。
*   **输出设备**：用于将计算机处理的结果反馈给用户。例如：显示器、耳机、打印机、投影仪。
*   **外部存储设备**：用于长期保存数据。例如：硬盘、光盘、U盘。

### CPU详解
CPU是计算机的“大脑”，其性能直接影响计算速度。

*   **厂商**：全球主要有两家CPU制造商：**Intel（英特尔）** 和 **AMD**。
*   **架构与位数**：常见的架构是 **x86**，目前主流CPU都是 **64位**。位数代表CPU一次能处理的数据量，64位CPU性能远强于早期的32位CPU。
*   **核心**：核心是CPU内的计算单元。核心数量越多，并行处理能力越强。现代CPU起步通常是四核或八核。
*   **缓存**：CPU缓存用于存储频繁访问的数据，其速度远快于内存和硬盘，能显著提升数据处理效率。可以这样理解三者的速度关系：**CPU（火箭）> 内存（飞机）> 硬盘（绿皮火车）**。

## Linux系统目录介绍 🌳

了解了硬件基础后，我们进入Linux系统的核心——目录结构。Linux采用倒置的树形结构，所有文件和目录都从根目录（`/`）开始。

![](img/043a51dfe06e1b1df82eb40cad2c9b51_26.png)

![](img/043a51dfe06e1b1df82eb40cad2c9b51_28.png)

根目录下有许多子目录，每个目录都有其特定的用途。以下是主要目录及其功能的介绍：

![](img/043a51dfe06e1b1df82eb40cad2c9b51_30.png)

![](img/043a51dfe06e1b1df82eb40cad2c9b51_32.png)

![](img/043a51dfe06e1b1df82eb40cad2c9b51_34.png)

*   **`/` （根目录）**：所有目录和文件的起点。
*   **`/bin` -> `/usr/bin`**：存放**用户和管理员必备的可执行二进制命令文件**。例如 `ls`、`cp`、`mv` 等命令的实际程序文件就存放在这里。它是一个指向 `/usr/bin` 的软链接。
*   **`/boot`**：存放**Linux系统内核及引导程序（如GRUB）所需的文件**。系统启动依赖此目录。
*   **`/dev`**：存放**硬件设备文件**。例如硬盘（`/dev/sda`）、终端等都在此目录下以文件形式存在。
*   **`/etc`**：存放**系统及服务的配置文件**，以及用户信息文件（如 `/etc/passwd`）。这是系统管理员需要频繁接触的目录。
*   **`/home`**：**普通用户的家目录**。每个用户在此目录下有一个以自己用户名命名的子目录。
*   **`/root`**：**系统超级管理员（root）的家目录**。
*   **`/lib`、`/lib64`**：存放**系统程序运行所需的共享库文件和内核模块**。多个程序可以共享这些库，以节省空间。
*   **`/opt`**：用于**安装额外的第三方可选应用程序软件包**。如果误删，可以手动重建。
*   **`/media`、`/mnt`**：**可移动介质（如光盘、U盘）和临时文件系统的挂载点**。它们也不是关键目录。
*   **`/proc`**：一个**虚拟文件系统**，其内的文件**对应内存中的信息**，如CPU信息、进程信息等，不占用磁盘空间。
*   **`/run`**：存放**系统启动后，程序或服务运行时产生的临时文件**，例如进程ID文件（`.pid`）。
*   **`/srv`**：存放**一些服务启动后需要访问的数据**。
*   **`/tmp`**：存放**临时文件**，所有用户都可访问。**重要数据切勿放在此目录**，因为其内容可能被定期清理。
*   **`/usr`**：存放**操作系统软件资源**，类似于第二个“根目录”。其子目录结构与根目录相似。
    *   **`/usr/local`**：**手动编译或安装的软件通常放置于此目录**，这是需要记住的常用目录。
*   **`/var`**：存放**经常变化的文件**，如日志、邮件等。
    *   **`/var/log`**：**系统日志文件目录**，是排查问题的重要位置。

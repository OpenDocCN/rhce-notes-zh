## 整数运算的常用方法

![](img/60ff11a968a4e4213483ffaa69906a45_8.png)

Shell中执行整数运算有多种方式，以下是几种常见的方法：

1.  **`expr` 命令**
2.  **`$(( ))` 语法**
3.  **`$[ ]` 语法**
4.  **`let` 命令**

![](img/60ff11a968a4e4213483ffaa69906a45_10.png)

其中，`let` 命令和 `$(( ))` 语法在实际脚本编写中使用最为频繁。

![](img/60ff11a968a4e4213483ffaa69906a45_12.png)

### 方法一：使用 `expr` 命令

![](img/60ff11a968a4e4213483ffaa69906a45_14.png)

`expr` 命令可以直接计算并显示表达式的结果。以下是基本运算示例：

![](img/60ff11a968a4e4213483ffaa69906a45_16.png)

```bash
# 直接计算 1 + 3
expr 1 + 3
# 输出：4

![](img/60ff11a968a4e4213483ffaa69906a45_18.png)

# 使用变量进行计算
number1=10
number2=20
expr $number1 + $number2
# 输出：30
```

需要注意的是，乘法符号 `*` 在Shell中有特殊含义（代表通配符），因此在使用 `expr` 进行乘法运算时必须使用转义符 `\`。

![](img/60ff11a968a4e4213483ffaa69906a45_20.png)

```bash
expr $number1 \* $number2
# 输出：200
```

如果想将运算结果赋值给一个新变量，语法相对繁琐：

```bash
sum=`expr $number1 + $number2`
# 或者
sum=$(expr $number1 + $number2)
echo $sum
# 输出：30
```

### 方法二：使用 `$(( ))` 语法

![](img/60ff11a968a4e4213483ffaa69906a45_22.png)

`$(( ))` 是进行整数运算非常方便的方式。它可以直接计算括号内的算术表达式。

![](img/60ff11a968a4e4213483ffaa69906a45_24.png)

```bash
number1=10
number2=20

![](img/60ff11a968a4e4213483ffaa69906a45_26.png)

# 加法
echo $((number1 + number2))
# 输出：30

![](img/60ff11a968a4e4213483ffaa69906a45_28.png)

# 减法
echo $((number1 - number2))
# 输出：-10

# 乘法（无需转义）
echo $((number1 * number2))
# 输出：200

![](img/60ff11a968a4e4213483ffaa69906a45_30.png)

# 除法（取整数商）
number1=7
number2=2
echo $((number1 / number2))
# 输出：3

# 取余（模运算）
echo $((number1 % number2))
# 输出：1

![](img/60ff11a968a4e4213483ffaa69906a45_32.png)

# 幂运算
echo $((2 ** 10))
# 输出：1024
```

![](img/60ff11a968a4e4213483ffaa69906a45_34.png)

这种方法的好处是，括号内可以直接使用变量名，无需添加 `$` 符号。运算结果也可以轻松赋值给变量。

![](img/60ff11a968a4e4213483ffaa69906a45_36.png)

![](img/60ff11a968a4e4213483ffaa69906a45_38.png)

```bash
result=$((2 ** 10))
echo $result
# 输出：1024
```

### 方法三：使用 `$[ ]` 语法

![](img/60ff11a968a4e4213483ffaa69906a45_40.png)

`$[ ]` 语法与 `$(( ))` 功能几乎完全相同，是另一种写法。

![](img/60ff11a968a4e4213483ffaa69906a45_42.png)

```bash
echo $[2 + 2]
# 输出：4
echo $[2 * 2]
# 输出：4
echo $[2 ** 3]
# 输出：8
```

同样，可以将结果赋值给变量。

```bash
number=$[2 ** 10]
echo $number
# 输出：1024
```

![](img/60ff11a968a4e4213483ffaa69906a45_44.png)

### 方法四：使用 `let` 命令

![](img/60ff11a968a4e4213483ffaa69906a45_46.png)

`let` 命令是进行算术运算的另一种强大工具，特别适合在需要改变变量自身值的场景中使用。

![](img/60ff11a968a4e4213483ffaa69906a45_48.png)

![](img/60ff11a968a4e4213483ffaa69906a45_49.png)

```bash
let sum=1+2
echo $sum
# 输出：3
```

`let` 命令更常见的用法是进行自增、自减等操作，这在循环控制中非常有用。

```bash
i=1
let i++  # 相当于 i=i+1
echo $i
# 输出：2

let i+=5 # 相当于 i=i+5
echo $i
# 输出：7
```

![](img/60ff11a968a4e4213483ffaa69906a45_51.png)

## 运算在脚本中的实际应用

![](img/60ff11a968a4e4213483ffaa69906a45_52.png)

理解了基本运算方法后，我们来看看它们在真实脚本场景中的应用。

![](img/60ff11a968a4e4213483ffaa69906a45_54.png)

### 应用示例一：计算内存使用率

![](img/60ff11a968a4e4213483ffaa69906a45_56.png)

一个常见的需求是计算当前内存的使用率百分比。这需要获取已用内存和总内存，然后进行计算。

```bash
#!/bin/bash
# 脚本名：mem_usage.sh

# 获取已使用的内存量（KB）
MEM_USED=`free | grep ^Mem | awk ‘{print $3}‘`
# 获取总内存量（KB）
MEM_TOTAL=`free | grep ^Mem | awk ‘{print $2}‘`

# 计算使用率百分比 (已使用 / 总计 * 100)
# 注意：先乘后除可以避免整数除法结果为0的问题
MEM_PERCENT=$((MEM_USED * 100 / MEM_TOTAL))

echo “当前内存使用率为：${MEM_PERCENT}%”
```

### 应用示例二：使用循环进行网络探测

在编写一个ping测试脚本时，我们通常需要控制ping的次数。这时，一个带有计数器的循环就离不开变量的运算。

```bash
#!/bin/bash
# 脚本名：ping_001.sh

target_ip=“10.18.40.42”
# 初始化计数器
i=1

# 使用while循环ping 5次
while [ $i -le 5 ]
do
    ping -c1 $target_ip &> /dev/null
    if [ $? -eq 0 ]; then
        echo “主机 ${target_ip} 第${i}次探测：存活 (Up)”
    else
        echo “主机 ${target_ip} 第${i}次探测：不存活 (Down)”
    fi
    # 关键步骤：使用let命令将计数器i的值加1
    let i++
done
```

在这个脚本中，`let i++` 确保了每次循环后计数器 `i` 的值都会增加，从而避免了无限循环，并让我们能追踪执行的次数。

## 小数运算的处理

![](img/60ff11a968a4e4213483ffaa69906a45_58.png)

![](img/60ff11a968a4e4213483ffaa69906a45_59.png)

上述方法主要针对整数运算。如果需要进行小数运算，可以借助其他工具。

### 使用 `bc` 计算器

`bc` 是Linux下的一个高精度计算器语言，支持小数运算。

```bash
# 交互式模式
bc
# 然后输入 1+2.5 等命令

# 非交互式模式，直接计算
echo “1+2.5” | bc
# 输出：3.5

# 设置精度（保留两位小数）
echo “scale=2; 6/4” | bc
# 输出：1.50

![](img/60ff11a968a4e4213483ffaa69906a45_61.png)

# 幂运算
echo “2^4” | bc
# 输出：16
```

![](img/60ff11a968a4e4213483ffaa69906a45_63.png)

### 使用 `awk` 命令

![](img/60ff11a968a4e4213483ffaa69906a45_65.png)

`awk` 本身也具有强大的算术运算能力。

![](img/60ff11a968a4e4213483ffaa69906a45_67.png)

```bash
echo | awk ‘{print 1/2}‘
# 输出：0.5
```

![](img/60ff11a968a4e4213483ffaa69906a45_69.png)

![](img/60ff11a968a4e4213483ffaa69906a45_70.png)

### 使用 `python` 等解释器

![](img/60ff11a968a4e4213483ffaa69906a45_72.png)

当然，也可以调用Python等脚本语言来进行复杂计算。

![](img/60ff11a968a4e4213483ffaa69906a45_74.png)

```bash
python3 -c “print(5.0 / 2)”
# 输出：2.5
```

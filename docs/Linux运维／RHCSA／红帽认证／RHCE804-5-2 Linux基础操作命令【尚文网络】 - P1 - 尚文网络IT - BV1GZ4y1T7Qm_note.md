## `touch` 命令：创建与修改时间戳 ⏰

![](img/1d44347c728980a8d4f0539b42062fde_3.png)

`touch` 命令用于生成一个空文件，或者修改文件的存取和修改时间记录值。

如果指定的文件不存在，`touch` 会创建一个空文件。如果文件已存在，`touch` 则会更新该文件的时间戳为当前系统时间。

**基本语法：**
```bash
touch 文件名
```

**示例：**
```bash
# 创建一个名为 testfile 的空文件
touch testfile

![](img/1d44347c728980a8d4f0539b42062fde_5.png)

# 使用 ls -l 查看文件详情，包括时间戳
ls -l testfile

# 使用 cat 命令确认文件内容为空
cat testfile
```

![](img/1d44347c728980a8d4f0539b42062fde_7.png)

如果再次对已存在的文件执行 `touch` 命令，文件的时间戳会被更新，而不会提示文件已存在。

## `cp` 命令：复制文件与目录 📋

![](img/1d44347c728980a8d4f0539b42062fde_13.png)

`cp` (copy) 命令用于将文件或目录从一个位置复制到另一个位置。使用时需要指定源文件路径和目标文件路径。

![](img/1d44347c728980a8d4f0539b42062fde_15.png)

**基本语法：**
```bash
cp [参数] 源文件路径 目标文件路径
```

**常用参数：**
*   `-i`：交互式复制，覆盖前提示。
*   `-f`：强制复制，直接覆盖已存在的目标文件。
*   `-r` 或 `-R`：递归复制，用于复制目录及其所有内容。
*   `-v`：显示详细的复制过程信息。
*   `-p`：保留源文件的属性（如权限、时间戳）。

![](img/1d44347c728980a8d4f0539b42062fde_17.png)

![](img/1d44347c728980a8d4f0539b42062fde_19.png)

**示例：**
```bash
# 将 /etc/passwd 文件复制到当前目录
cp /etc/passwd .

![](img/1d44347c728980a8d4f0539b42062fde_21.png)

# 使用 -v 参数显示复制详情
cp -v /etc/passwd ./password_copy

# 递归复制整个目录
cp -r /source/directory /destination/directory
```

## `mv` 命令：移动与重命名文件/目录 🚚

![](img/1d44347c728980a8d4f0539b42062fde_29.png)

![](img/1d44347c728980a8d4f0539b42062fde_31.png)

![](img/1d44347c728980a8d4f0539b42062fde_33.png)

`mv` (move) 命令有两个主要功能：移动文件/目录到新位置，以及对文件/目录进行重命名。

![](img/1d44347c728980a8d4f0539b42062fde_35.png)

**基本语法：**
```bash
mv 源文件/目录 目标文件/目录
```

**示例：**
```bash
# 将当前目录下的 password 文件移动到 /tmp 目录
mv password /tmp/

![](img/1d44347c728980a8d4f0539b42062fde_37.png)

# 重命名文件
mv old_filename new_filename

![](img/1d44347c728980a8d4f0539b42062fde_39.png)

# 移动并重命名
mv /path/to/file /new/path/newname
```

## `rm` 命令：删除文件与目录 🗑️

![](img/1d44347c728980a8d4f0539b42062fde_43.png)

![](img/1d44347c728980a8d4f0539b42062fde_45.png)

`rm` (remove) 命令用于删除文件或目录。与 `rmdir`（仅能删除空目录）不同，`rm` 功能更强大，但也更危险。

![](img/1d44347c728980a8d4f0539b42062fde_47.png)

**基本语法：**
```bash
rm [参数] 文件或目录名
```

![](img/1d44347c728980a8d4f0539b42062fde_49.png)

**常用参数：**
*   `-f`：强制删除，不进行任何提示。
*   `-i`：交互式删除，删除前逐一确认。
*   `-r` 或 `-R`：递归删除，用于删除目录及其所有内容。

![](img/1d44347c728980a8d4f0539b42062fde_51.png)

![](img/1d44347c728980a8d4f0539b42062fde_53.png)

**示例：**
```bash
# 删除单个文件（会提示确认）
rm myfile

# 强制删除文件（无提示）
rm -f myfile

![](img/1d44347c728980a8d4f0539b42062fde_55.png)

![](img/1d44347c728980a8d4f0539b42062fde_57.png)

# 递归强制删除整个目录及其内容（危险操作！）
rm -rf directory_name
```

**警告：** `rm -rf /` 或类似命令会强制删除根目录下的所有文件，导致系统崩溃，请务必谨慎使用。

## 文件内容查看命令 👀

![](img/1d44347c728980a8d4f0539b42062fde_59.png)

上一节我们介绍了如何管理文件，本节中我们来看看如何查看文件的内容。以下是几个常用的文件内容查看命令。

![](img/1d44347c728980a8d4f0539b42062fde_61.png)

### `cat` 命令
`cat` 命令用于连接文件并打印到标准输出设备，常用于查看较短的文件内容。

![](img/1d44347c728980a8d4f0539b42062fde_63.png)

![](img/1d44347c728980a8d4f0539b42062fde_65.png)

**基本语法：**
```bash
cat 文件名
```

**高级用法：**
*   **追加内容：** 使用 `>>` 将内容追加到文件末尾。
    ```bash
    cat file2 >> file1
    ```
*   **合并文件：** 使用 `>` 将多个文件内容合并到一个新文件。
    ```bash
    cat file1 file2 > newfile
    ```

![](img/1d44347c728980a8d4f0539b42062fde_67.png)

### `head` 与 `tail` 命令
*   `head`：显示文件开头部分的内容，默认显示前10行。
    ```bash
    head -n 15 filename  # 显示文件前15行
    ```
*   `tail`：显示文件末尾部分的内容，默认显示后10行。
    ```bash
    tail -n 20 filename  # 显示文件后20行
    ```
*   `tail -f`：实时追踪文件末尾的新增内容，常用于监控日志文件。
    ```bash
    tail -f /var/log/messages
    ```

![](img/1d44347c728980a8d4f0539b42062fde_69.png)

### `more` 与 `less` 命令
对于内容超过一屏的长文件，可以使用 `more` 或 `less` 命令分页查看。
*   `more`：按空格键向下翻页，按 `q` 键退出。只能向前翻页。
    ```bash
    more longfile.txt
    ```
*   `less`：功能比 `more` 更强大，可以使用上下箭头键向前或向后翻页，按 `q` 键退出。
    ```bash
    less longfile.txt
    ```

## 文件查找与筛选命令 🔍

![](img/1d44347c728980a8d4f0539b42062fde_73.png)

### `find` 命令
`find` 命令用于在指定目录下查找文件和目录，功能非常强大。本节我们只需了解其基本用途。
```bash
find /path/to/search -name "filename"
```

![](img/1d44347c728980a8d4f0539b42062fde_75.png)

![](img/1d44347c728980a8d4f0539b42062fde_76.png)

### `grep` 命令
`grep` 命令用于在文本中查找匹配指定模式（字符串）的行。
```bash
# 在文件中搜索包含 “root” 的行
grep "root" /etc/passwd

![](img/1d44347c728980a8d4f0539b42062fde_78.png)

# 结合管道符，从命令输出中筛选信息
ls -l | grep "password"
```

### `tree` 命令
`tree` 命令以树状图形式列出目录的结构，非常直观。
```bash
# 显示当前目录的树状结构
tree .

# 只显示目录，不显示文件
tree -d .

# 限制显示层级
tree -L 2 /path
```
如果系统未安装 `tree` 命令，可以使用包管理器安装，例如在基于RPM的系统上：`yum install -y tree`。

## 链接文件：`ln` 命令 🔗

![](img/1d44347c728980a8d4f0539b42062fde_80.png)

`ln` 命令用于在文件或目录之间创建链接。链接分为硬链接和软链接（符号链接）。

**硬链接：**
*   本质是指向文件索引节点（inode）的指针，系统不为其分配新的 inode。
*   **优点：** 节省空间，是传统的Linux文件整合方式。
*   **缺点：** 不能跨文件系统创建；只有超级用户能为目录创建硬链接。

**软链接（符号链接）：**
*   类似于Windows的快捷方式，是一个包含原文件路径信息的特殊文件。
*   **优点：** 可以跨文件系统创建，可以为目录创建。
*   **缺点：** 如果原文件被删除或移动，软链接将失效（成为“断链”）。

![](img/1d44347c728980a8d4f0539b42062fde_82.png)

**基本语法：**
```bash
# 创建硬链接
ln 源文件 目标链接名

![](img/1d44347c728980a8d4f0539b42062fde_84.png)

# 创建软链接（使用 -s 参数）
ln -s 源文件 目标链接名
```

![](img/1d44347c728980a8d4f0539b42062fde_85.png)

**示例：**
```bash
# 创建一个指向 /app/testfile 的软链接到 /tmp
ln -s /app/testfile /tmp/link_to_test

# 查看链接文件，箭头指向源文件
ls -l /tmp/link_to_test
```

![](img/1d44347c728980a8d4f0539b42062fde_87.png)

**重要提示：** 创建硬链接时，源文件和目标链接必须在同一个文件系统内，否则会报错 “Invalid cross-device link”。

## 命令回顾与总结 📝

![](img/1d44347c728980a8d4f0539b42062fde_95.png)

本节课中我们一起学习了Linux下文件与目录管理的核心命令：

1.  **创建与时间戳：** 使用 `touch` 创建空文件或更新文件时间。
2.  **复制操作：** 使用 `cp` 命令复制文件或目录，常用 `-r` 递归复制目录，`-p` 保留属性。
3.  **移动与重命名：** 使用 `mv` 命令移动文件/目录或对其进行重命名。
4.  **删除操作：** 使用 `rm` 命令删除文件或目录，`-r` 和 `-f` 参数需谨慎使用。
5.  **内容查看：**
    *   `cat` 查看文件。
    *   `head`/`tail` 查看文件头尾。
    *   `tail -f` 实时追踪日志。
    *   `more`/`less` 分页查看长文件。
6.  **查找与筛选：** 使用 `find` 查找文件，使用 `grep` 筛选文本，使用 `tree` 展示目录树。
7.  **链接文件：** 使用 `ln` 创建链接。理解硬链接（不可跨文件系统）与软链接（可跨文件系统，依赖源文件）的区别。

### **软件下载与准备**

首先，需要获取VMware Workstation 15的安装程序。可以通过提供的百度网盘链接下载。对于Windows 10操作系统，建议安装VMware 15版本。该版本兼容性更好，能避免一些潜在错误。

![](img/30fe831d20894dfeba7498d720b2eb17_7.png)

![](img/30fe831d20894dfeba7498d720b2eb17_8.png)

下载完成后，您将获得一个压缩包，解压后即可开始安装。

如果您使用的是macOS系统，则需要下载对应的DMG格式安装文件。

![](img/30fe831d20894dfeba7498d720b2eb17_13.png)

### **Windows系统安装步骤**

![](img/30fe831d20894dfeba7498d720b2eb17_21.png)

![](img/30fe831d20894dfeba7498d720b2eb17_22.png)

![](img/30fe831d20894dfeba7498d720b2eb17_24.png)

Windows系统下的软件安装过程通常较为简单，遵循“下一步”原则即可。以下是具体步骤：

![](img/30fe831d20894dfeba7498d720b2eb17_25.png)

![](img/30fe831d20894dfeba7498d720b2eb17_27.png)

![](img/30fe831d20894dfeba7498d720b2eb17_28.png)

1.  **启动安装程序**：双击解压后的安装文件，启动安装向导。
2.  **接受许可协议**：在出现的界面中，勾选“我接受许可协议中的条款”，然后点击“下一步”。
3.  **选择安装位置**：您可以选择将软件安装到默认的C盘，如果C盘空间不足，也可以点击“更改”按钮，选择安装到其他磁盘分区（如D盘）。
4.  **功能选项**：接下来的“增强型键盘驱动程序”选项保持默认不勾选即可，通常无需使用。
5.  **用户体验设置**：建议取消勾选“启动时检查产品更新”和“加入VMware客户体验提升计划”这两个选项，以避免不必要的更新和信息收集。
6.  **创建快捷方式**：保持“桌面”和“开始菜单程序文件夹”的默认勾选，以便后续快速启动软件。
7.  **开始安装**：确认设置无误后，点击“安装”按钮。安装过程大约需要5分钟，具体时间取决于您的电脑配置。
8.  **输入许可证密钥**：安装完成后，**请务必点击“许可证”按钮**，而不是直接点击“完成”。在弹出的窗口中，输入提供的许可证密钥，然后点击“输入”。完成激活后，即可正常使用VMware Workstation 15。

![](img/30fe831d20894dfeba7498d720b2eb17_29.png)

VMware 15版本引入了一些新功能，例如在创建虚拟机磁盘时，新增了**NVMe**磁盘类型选项，这能提供更好的磁盘性能。
